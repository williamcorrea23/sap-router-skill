## 九、常用 BAPI 速查

| BAPI 名称 | 功能 | 适用场景 |
|---|---|---|
| `BAPI_GOODSMVT_CREATE` | 创建货物移动凭证 | 程序触发MIGO操作（收货/发货/转仓）|
| `BAPI_GOODSMVT_CANCEL` | 取消货物移动凭证 | 撤销物料凭证 |
| `BAPI_INCOMINGINVOICE_CREATE` | 创建发票校验凭证 | MIRO自动化，避免用 `calc_tax_ind='X'` |
| `BAPI_INCOMINGINVOICE_CANCEL` | 取消发票凭证 | 撤销 MIRO 凭证 |
| `BAPI_PO_CREATE1` | 创建采购订单 | 采购订单自动创建 |
| `BAPI_PO_CHANGE` | 修改采购订单 | 修改 PO 行项目（数量/价格/交货日期）|
| `BAPI_PO_GET_DETAIL` | 读取采购订单明细 | 获取完整 PO 数据 |
| `BAPI_SALESORDER_CREATEFROMDAT2` | 创建销售订单 | SD 销售订单自动创建 |
| `BAPI_SALESORDER_CHANGE` | 修改销售订单 | 修改 SO 头/行数据 |
| `BAPI_SALESORDER_GETLIST` | 查询销售订单 | 根据条件检索订单 |
| `BAPI_DELIVERY_SAVEREPLICA` | 创建/修改交货单 | VL01N/VL02N 自动化 |
| `BAPI_OUTB_DELIVERY_CONFIRM_DEC` | 确认出货 | 触发交货单拣配确认 |
| `BAPI_ACC_DOCUMENT_POST` | 财务凭证过账 | FI 凭证自动创建 |
| `BAPI_COMPANYCODE_GETLIST` | 获取公司代码列表 | 读取 FI 公司代码数据 |
| `BAPI_MATERIAL_SAVEDATA` | 保存物料主数据 | 批量创建/修改物料 |
| `BAPI_MATERIAL_GET_DETAIL` | 读取物料主数据 | 获取物料详细信息 |
| `VMD_EI_API=>SYNCHRONIZE` | 物料主数据批量修改 | 工厂/存储地点数据扩展（类方法）|
| `ME_INFORECORD_MAINTAIN` | 创建/修改采购信息记录 | PIR 批量维护（注意必须同时传价格条件）|
| `BAPI_VENDOR_FIND` | 查找供应商 | 按条件搜索供应商主数据 |
| `BAPI_CUSTOMER_FIND` | 查找客户 | 按条件搜索客户主数据 |
