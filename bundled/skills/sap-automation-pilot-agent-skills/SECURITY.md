# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it responsibly.

**Please do NOT create a public GitHub issue for security vulnerabilities.**

Instead, please report security issues via the SAP Trust Center at [https://www.sap.com/about/trust-center/security/incident-management.html](https://www.sap.com/about/trust-center/security/incident-management.html).

When reporting, please include:

- A description of the vulnerability
- Steps to reproduce the issue
- Potential impact of the vulnerability
- Any suggested remediation (if applicable)

SAP will review your report and respond as quickly as possible. We appreciate your help in keeping this project and its users safe.
