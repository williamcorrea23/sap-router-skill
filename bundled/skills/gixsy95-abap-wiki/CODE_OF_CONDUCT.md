# Code of Conduct

This project adopts the [Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

In short: be respectful, assume good faith, focus criticism on code and ideas -
never on people. Harassment, discrimination, and personal attacks are not
tolerated anywhere in the project's spaces (issues, PRs, discussions).

Violations can be reported privately to
[gixsy95github@gmail.com](mailto:gixsy95github@gmail.com). Reports are handled
confidentially. The maintainer may edit, remove, or reject contributions and
interactions that violate this code, and ban repeat offenders.
