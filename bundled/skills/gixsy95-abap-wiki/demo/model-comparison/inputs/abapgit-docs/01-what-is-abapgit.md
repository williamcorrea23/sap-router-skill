# What is abapGit

abapGit is "a git client for ABAP developed in ABAP". It requires ABAP version
702 SP 8 or higher. It functions as a tool for importing and exporting code
between ABAP systems, enabling developers to perform mass operations that would
otherwise require manual execution.

## Purpose and features

abapGit allows developers with system access to manage ABAP artifacts through
version control. A key advantage is that repositories remain in plain text
format — unlike traditional transport files — allowing for code review before
importing to target systems.

## Security and community

While automated security checks may flag the tool due to its system-modifying
capabilities, abapGit maintains security through community oversight. All code
changes undergo pull request review, and 100+ repository watchers receive
notifications of every update.
