# README images

Screenshots referenced by the top-level `README.md`.

- `full-graph.png` - a real SAP S/4HANA knowledge base built with `abap_wiki`,
  shown as an Obsidian graph: thousands of custom ABAP objects clustered by
  package. **Anonymised**: the view carries no node labels and exposes no object
  names, table names, or business data - only the scale and shape of the graph.
- `example-graph.png` - the bundled synthetic example
  (`examples/token-saving/abap_wiki/`) as an Obsidian dependency graph: the
  `program-zexample_stock_alloc` node linked to the standard SAP tables
  (`MARA`/`MARC`/`MARD`) and classes (`CL_SALV_TABLE`/`CL_BCS`) it uses.

Both images are part of the public showcase. The example graph is fully
synthetic; the full graph is an anonymised, label-free visualisation.
