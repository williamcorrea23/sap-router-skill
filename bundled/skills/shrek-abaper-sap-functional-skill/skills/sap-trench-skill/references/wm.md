## 九、核心配置路径与概念

**打印控制配置**：
```
SPRO → 后勤执行 → 仓库管理 → 作业 → 定义打印控制
```

**仓库基础配置顺序（SPRO）**：
建仓库号（T300）→ 分配仓储地点（T320）→ 定义存储类型 → 定义存储区 → 定义仓位结构 → 定义需求类型 → 定义移动类型 → 设置自动转储单创建

**入库策略优先级**：固定仓位 → 待用存储 → 现有库存附加 → 空仓位 → 托盘 → 块存储
