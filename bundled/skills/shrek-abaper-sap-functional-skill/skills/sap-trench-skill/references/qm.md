# SAP QM（质量管理）知识库

QM 模块覆盖质量主数据、检验计划、检验批、检验结果记录、质量通知书全流程。

---

## 一、质检物料采购入库流程

有质检信息记录的物料，采购入库触发质检流程：

```
ME21N（创建采购订单）
    ↓
MIGO（101 收货）→ 系统自动创建检验批，库存状态：质检中（Quality Inspection）
    ↓
QA32（检验批选择）→ QE01（记录检验结果）
    ↓
QA11（记录使用决策）
    ↓
QA12（更改使用决策 / 触发后续动作）
    ├── 合格 → 转入非限制库存
    ├── 不合格 → 退货（105 移动类型）
    └── 有条件接受 → 非限制使用（特殊库存状态）
```
