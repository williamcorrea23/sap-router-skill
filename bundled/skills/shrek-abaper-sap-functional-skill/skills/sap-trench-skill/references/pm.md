## 八、常见问题

### 序列号在 MIGO 过账时报错（移动类型 309）

**现象**：MIGO 执行物料间调拨（移动类型 309）时，报错  
`Serial number X already exists for material Y`

**根因**：移动类型 309 带序列号转移，目标物料已存在相同序列号，系统禁止重复。

**解决方案**：
1. **归档冲突序列号**：通过 `IQ03` 找到冲突序列号 → `IQ04` 归档
2. **修改序列号归属**：`IQ02` → Edit → Special serial no. functions → 变更序列号或物料号

> 详细案例见 `troubleshooting.md` CASE-015
