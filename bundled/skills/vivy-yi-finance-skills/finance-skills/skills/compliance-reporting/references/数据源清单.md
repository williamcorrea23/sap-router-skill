# 数据源清单 — [ERP]/[REG]/[DIS]

## 数据源标签说明

| 标签 | 含义 | 典型来源 |
|---|---|---|
| [ERP] | 企业 ERP 系统 | SAP/用友/金蝶总账 |
| [REG] | 监管报送系统 | 国资委报送系统/统计局联网直报/证监会披露系统 |
| [DIS] | 披露管理系统/公告系统 | 巨潮资讯/上交所/联交所/公司官网 |

---

## [ERP] SAP / 用友 / 金蝶（总账）

**覆盖范围：**
- 月度/年度财务数据
- 主要财务指标
- 关联交易数据
- 担保/借款数据
- 税务申报数据

**查询路径：**

```
SAP 常用事务代码：
- S_ALR_00001236：财务报表
- FAGLB03：科目余额表
- S_ALR_00001238：关联方交易报表
- 担保台账：相关科目查询

用友/金蝶：
- 报表中心 → 主要财务指标
- 合同管理 → 担保借款台账
- 关联交易 → 关联方交易台账
```

---

## [REG] 监管报送系统

**覆盖范围：**
- 月度财务数据报送
- 季度/年度统计报送
- 国有资本经营预算
- 重点税源报表
- 行业特定监管报告

**报送系统：**

```
国资委系统：
- 国资监管信息平台
- 国有资本经营预算报送

统计局系统：
- 联网直报平台
- 企业创新调查

证监会系统（如上市公司）：
- 证监会信息披露电子化系统
- 创业板信息披露平台

税务局系统：
- 电子税务局
- 自然人税收管理系统（个税）

外汇管理局（如外资）：
- 资本项目信息系统
- 外汇收支申报
```

---

## [DIS] 披露管理系统 / 巨潮资讯 / 交易所

**覆盖范围：**
- 上市公司公告披露
- 临时公告（关联交易/担保/借款等）
- 定期报告（年报/半年报/季报）
- 监管问询与回复

**查询路径：**

```
巨潮资讯网：www.cninfo.com.cn
- 公告查询 → 公司公告列表
- 临时公告 → 按类型筛选
- 定期报告 → 年报/半年报/季报

上交所：www.sse.com.cn
- 信息披露 → 公告查询

联交所：www.hkex.com.hk（港股）
- 披露易 → 公告查询

公司官网：
- 投资者关系 → 公告列表
```

## 数据源组合使用

### 月度监管报送
```
[ERP] 月度科目余额 → 提取主要财务指标
→ 核对数据准确性
→ 内部审批（财务经理/CFO）
→ [REG] 填报 → 确认报送成功
→ 归档报送凭证
```

### 年度财务报告披露
```
[ERP] 年度财务数据 → 审计确认
→ 起草年报/半年报
→ 内部审批（董事会）
→ [DIS] 披露（巨潮/交易所）
→ 确认披露成功
```

### 关联交易临时披露
```
[ERP] 关联交易数据 → 判断披露阈值（> X 万 或 > 净资产 X%）
→ 起草关联交易公告
→ 内部审批（董事会/股东大会）
→ [DIS] 披露（临时公告）
→ 确认披露成功
```

### 监管问询回复
```
监管问询函 → 评估问询性质
→ [ERP] 准备数据支撑
→ 起草回复
→ 内部审批（CFO/CEO/董事会）
→ 报送监管机构
→ 归档回复记录
```
