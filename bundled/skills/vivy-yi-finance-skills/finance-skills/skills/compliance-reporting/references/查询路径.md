# 查询路径 — 合规报告

## 数据源优先级

| 优先级 | 数据源 | 说明 |
|---|---|---|
| 1 | ERP 总账（[ERP]）| 财务数据 | 最优先 |
| 2 | 监管报送系统（[REG]）| 报送入口 | 报送状态 |
| 3 | 披露管理系统（[DIS]）| 披露文件 | 披露记录 |

---

## 常用查询场景

### 场景 1：月度财务数据报送查询

**查询入口：**
```
ERP：
- 月度科目余额表
- 主要财务指标

监管报送系统：
- 报送状态查询
- 历史报送记录
```

**月度数据：**
```
□ 报送期间：[YYYY年MM月]
□ 主要财务指标：
□ 营业收入：[XXX万]
□ 利润总额：[XXX万]
□ 净利润：[XXX万]
□ 资产总额：[XXX万]
□ 负债总额：[XXX万]
□ 所有者权益：[XXX万]

□ 同比情况：
□ 收入同比：[±X%]
□ 利润同比：[±X%]

□ 报送状态：
□ 应报送日期：[YYYY-MM-DD]
□ 实际报送日期：[YYYY-MM-DD]
□ 报送状态：[已报送/待报送/延期]
```

## 数据源组合使用

### 月度监管报送
```
[ERP] 月度数据 → 核对数据 → 内部审批 → [REG] 报送 → 确认报送成功
```

### 年度报告披露
```
[ERP] 年度数据 → 审计确认 → 起草报告 → 内部审批 → [DIS] 披露 → 确认披露
```

### 关联交易披露
```
[ERP] 关联交易数据 → 判断披露阈值 → 起草披露文件 → 审批 → [DIS] 披露
```
