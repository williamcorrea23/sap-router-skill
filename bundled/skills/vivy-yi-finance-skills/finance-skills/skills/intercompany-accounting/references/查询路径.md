# 查询路径 — 内部往来核算

## 数据源优先级

| 优先级 | 数据源 | 说明 |
|---|---|---|
| 1 | ERP 总账（[ERP]）| 内部往来科目余额/明细 | 最优先 |
| 2 | 内部结算系统（[IC]）| 内部交易/对账 | 交易明细 |
| 3 | 资金系统（[CASH]）| 内部资金调拨 | 资金流 |

---

## 常用查询场景

### 场景 1：内部往来余额查询

**查询入口：**
```
ERP：
- 内部往来科目余额表
- 内部应收/应付账款

内部结算系统：
- 往来对账结果
```

**内部往来余额概览：**
```
□ 内部应收款合计：[XXX万]
□ 内部应付款合计：[XXX万]
□ 内部往来净额：[XXX万]

□ 按关联方分布：
□ [子公司A]：[应收 XXX万 / 应付 XXX万]
□ [子公司B]：[应收 XXX万 / 应付 XXX万]
□ [联营企业A]：[应收 XXX万 / 应付 XXX万]
```

## 数据源组合使用

### 月度内部对账
```
[ERP] 母公司往来 → [ERP] 子公司往来 → [IC] 对账调节 → 差异调平

### 关联交易汇总
```
[IC] 内部交易明细 → [ERP] 关联交易科目 → 生成关联交易台账 → 核对关联交易披露

### 关联担保监控
```
[ERP] 担保台账 → [CASH] 担保额度 → 担保余额监控 → 超限预警
```
