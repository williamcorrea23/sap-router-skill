# 数据源清单 — [ERP]/[IC]/[CASH]

## 数据源标签说明

| 标签 | 含义 | 典型来源 |
|---|---|---|
| [ERP] | 企业 ERP 系统（总账）| SAP/用友/金蝶总账 |
| [IC] | 内部结算系统 | SAP 内部交易/用友内部结算/金蝶内部往来 |
| [CASH] | 资金管理系统 | 司库系统/资金管理系统 |

---

## [ERP] SAP / 用友 / 金蝶（总账）

**覆盖范围：**
- 内部往来科目（应收/应付关联方）
- 内部销售收入/采购成本
- 内部借款（其他应收/应付关联方）
- 关联担保（或有负债）
- 内部费用分摊
- 财务费用（内部利息收入/支出）

**查询路径：**

```
SAP 常用事务代码：
- FAGLB03：科目余额表（内部往来科目）
- FBL5N：客户行项目（内部应收）
- FBL1N：供应商行项目（内部应付）
- S_ALR_00001238：关联方交易报表
- 持股公司报表：[持股结构查询]

用友/金蝶：
- 总账 → 内部往来 → 关联方往来明细
- 应收应付 → 关联方应收应付
- 或有负债 → 担保台账
```

---

## [IC] SAP 内部交易 / 用友内部结算 / 金蝶内部往来

**覆盖范围：**
- 内部交易订单/发票
- 内部对账单
- 内部定价规则
- 费用分摊计算
- 内部借款台账

**查询路径：**

```
SAP 内部交易（SAP CO）：
- 内部订单 → 内部交易明细
- 成本中心分摊 → 费用分摊结果
- 内部采购/销售 → 内部交易清单

用友内部结算：
- 内部结算 → 往来对账
- 费用分摊 → 分摊计算
- 内部借款 → 借款台账

金蝶内部往来：
- 往来管理 → 关联方对账
- 内部结算 → 交易明细
```

---

## [CASH] 司库系统 / 资金管理系统

**覆盖范围：**
- 内部资金调拨
- 内部借款（实际资金流动）
- 担保额度使用情况
- 资金池余额

**查询路径：**

```
司库系统：
- 账户管理 → 子公司账户余额
- 资金调拨 → 内部转账记录
- 融资管理 → 内部借款登记
- 担保管理 → 担保额度使用

银企直联：
- 内部转账记录
- 实际资金到账情况
```

## 数据源组合使用

### 月度内部对账闭环
```
[ERP] 母公司账面 → [ERP] 子公司账面 → [IC] 对账调节表
→ 识别差异 → 逐笔核查 → 调账确认 → 对账完成
```

### 关联交易汇总报告
```
[IC] 内部交易明细 → [ERP] 关联交易科目 → [ERP] 关联方往来余额
→ 生成关联交易台账 → 核对季报/年报披露
```

### 关联担保监控
```
[ERP] 担保合同台账 → [CASH] 担保额度使用 → 担保余额实时监控
→ 超限预警 → 报 CFO → 制定压缩计划
```

### 内部借款利息计算
```
[IC] 借款台账 → [IC] 利息计算（本金 × 利率 × 天数）
→ [ERP] 利息凭证生成 → [CASH] 实际利息收付
→ 核对利息收付凭证
```

## Connector 映射

|  | 数据需求 | Connector | 工具 | 关键字段 |
|------|---------|-----------|------|---------|
| 集团内往来余额 | SAP | `sap_gl_query` | `RACCT`（内部往来）, `KUNNR/LIFNR`（内部客户/供应商）, `DMBTR` |
| 集团内交易凭证 | SAP | `sap_gl_query` | `BELNR`, `BUKRS`（公司代码）, `DMBTR`, `BUDAT` |
| 关联交易清单 | SharePoint | `sp_finance_library_read` | `transaction_id`, `counterparty`, `amount`, `arm_length` |
| IC 余额调节 | SAP | `sap_ar_age` | `KUNNR`（IC客户）, `DMBTR` |
