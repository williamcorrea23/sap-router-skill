# 判断框架 — 内部往来核算异常处理

## 核心原则

**内部往来核算的目标是"往来清晰、定价公允、对账及时、披露合规"。**
遇到异常时，优先判断是"账务处理错误"还是"交易本身异常"，确保内部交易真实、公允、合规。

---

## 异常类型与处理方式

### 类型一：内部对账差异

**典型场景：**
- 母公司账面与子公司账面往来余额不一致
- 内部销售金额不一致（一方确认收入，另一方未确认采购）
- 内部费用分摊金额不一致

**判断流程：**

```
第一步：确认差异金额

□ 差异金额：[XXX万]
□ 差异方向：母公司多/少计
□ 涉及关联方：[子公司名称]
□ 涉及期间：[YYYY年MM月]

□ 差异分类：
□ 余额差异（期末往来余额不一致）
□ 发生额差异（本期交易金额不一致）

第二步：分析差异原因

□ 账务处理时间差异：
→ 一方已记账，另一方尚未记账
→ 银行转账在途（内部转账在途）
→ 发票/单据传递延迟

□ 账务处理错误：
→ 金额记错（多计/少计）
→ 方向记反（借方/贷方记反）
→ 关联公司记错（子公司A 记成子公司B）

□ 交易认定差异：
→ 一方确认为销售，另一方确认为代购
→ 费用分摊比例不一致
→ 利息计算基础不一致

□ 其他原因：
→ 汇率折算差异（境外子公司）
→ 会计政策差异（折旧年限/收入确认时点）

第三步：逐笔核对

□ 从 ERP 获取双方账面明细：
□ 母公司账面明细：[ERP 内部往来科目]
□ 子公司账面明细：[子公司 ERP]

□ 编制调节表：
□ 母公司余额：[XXX万]
□ 减：子公司已记账母公司未记账：[XXX万]
□ 加：母公司已记账子公司未记账：[XXX万]
□ 调整后余额：[XXX万]
□ 子公司账面余额：[XXX万]
□ 差异：[±XXX万]

□ 差异原因说明：[描述]

第四步：调账处理

□ 账务处理错误 → 调账：
□ 发现方调账：[调整分录]
□ 确认方确认：[确认分录]

□ 时间性差异 → 确认在途：
□ 确认为"内部往来-在途"
□ 下期及时调平

□ 交易认定差异 → 协调确认：
□ 双方财务沟通确认
□ 统一交易类型和金额
□ 补充必要附件
```

### 类型二：内部交易定价异常

**典型场景：**
- 内部销售价格明显偏离市场价
- 成本加成比例异常（高于或低于行业正常水平）
- 内部借款利率异常（高于或低于同期贷款利率）

**判断流程：**

```
第一步：确认定价信息

□ 内部交易内容：[描述]
□ 内部定价：[XXX元/XXX%]
□ 市场参考价：[XXX元/XXX%]
□ 偏离幅度：[±X%]

□ 定价依据：[成本加成/协议价/市场价]

第二步：分析定价异常程度

□ 偏离程度判断：
□ 偏离 < [X]%：可接受偏差
□ 偏离 [X]-[X]%：须说明原因
□ 偏离 > [X]%：存在转让定价风险，须评估

□ 风险评估：
□ 税务风险：[低/中/高]
□ 审计风险：[低/中/高]
□ 监管风险：[低/中/高]

第三步：评估转让定价风险

□ 是否符合独立交易原则：
□ 是 → 风险低，准备说明文件
□ 否 → 须调整定价或准备转让定价报告

□ 同期资料准备要求：
□ 须准备同期资料：[是/否]
□ 准备年度：[YYYY 年]
□ 准备截止日：[X 月 X 日]

□ 预约定价安排（APA）：
□ 是否申请 APA：[是/否]
□ 申请进度：[描述]

第四步：制定改善措施

□ 定价偏高（少缴税风险）：
□ 降低内部定价 → 须符合独立交易原则
□ 准备转让定价报告 → 证明定价合理性
□ 申请预约定价安排

□ 定价偏低（被税务调整风险）：
□ 调整内部定价 → 向市场看齐
□ 准备转让定价报告
□ 评估是否补税

□ 无合理依据：
□ 补充定价协议
□ 准备定价依据文件
□ 咨询税务专业意见
```

### 类型三：关联担保超限

**典型场景：**
- 累计关联担保余额 > 净资产 [X]%
- 单笔担保 > [X] 万
- 担保条件异常（免担保/超低担保费）

**判断流程：**

```
第一步：确认担保信息

□ 担保方：[公司名称]
□ 被担保方：[关联方名称]
□ 担保类型：[保证/抵押/质押]
□ 担保金额：[XXX万]
□ 担保期限：[YYYY-MM-DD 至 YYYY-MM-DD]
□ 担保费率：[X%]

□ 被担保方财务状况：
□ 资产负债率：[X]%
□ 流动比率：[X]
□ 盈利能力：[描述]

第二步：计算担保敞口

□ 公司净资产：[XXX万]
□ 累计担保余额：[XXX万]
□ 担保/净资产比例：[X%]

□ 担保限额检查：
□ 公司章程担保限额：[X] 万 或 净资产 [X]%
□ 本次担保：[XXX万]
□ 本次担保后累计：[XXX万]

□ 是否超出限额：[✅ 否 / 🔴 是]

□ 超限处理：
□ 须立即停止新增担保
□ 须制定压缩计划
□ 须披露并说明原因
```

### 类型四：内部资金占用

**典型场景：**
- 母公司向子公司提供借款 > 净资产 [X]%
- 无息或低息借款（税务风险）
- 借款长期未还

**判断流程：**

```
第一步：确认资金占用情况

□ 借出方：[公司名称]
□ 借入方：[子公司名称]
□ 借款金额：[XXX万]
□ 借款起始日：[YYYY-MM-DD]
□ 借款利率：[X]%/年 或 无息
□ 借款期限：[X] 个月

□ 是否签订借款协议：[是/否]
□ 是否经过审批：[是/否]

第二步：分析资金占用风险

□ 借款金额检查：
□ 公司净资产：[XXX万]
□ 借出金额：[XXX万]
□ 借出/净资产：[X%]

□ 利率检查：
□ 同期贷款利率：[X]%/年
□ 实际利率：[X]%/年 或 无息

□ 税务风险：
□ 无息或低息 → 税务可能要求按同期贷款利率调整
□ 调整金额：[XXX万] × 利率差 = [X] 万
□ 须代扣代缴个人所得税（如个人股东借款）

第三步：长期未还借款处理

□ 借款是否逾期：[是/否]
□ 原定还款日：[YYYY-MM-DD]
□ 已逾期：[X] 个月

□ 逾期处理：
□ 展期 → 重新签订借款协议
□ 催收还款 → 制定还款计划
□ 转作投资 → 转为股权投资

第四步：规范资金占用

□ 须补办手续：
□ 签订正式借款协议
□ 按规定利率计息
□ 完善内部审批流程

□ 须清理情形：
□ 股东借款 → 须归还或转作增资
□ 无息借款 → 须调整利率或补计利息
□ 逾期借款 → 须展期或催收
```

### 类型五：关联交易未及时披露

**典型场景：**
- 须披露的关联交易未在规定时间内公告
- 披露金额与实际不符
- 披露的定价政策与实际不符

**判断流程：**

```
第一步：确认未披露关联交易

□ 关联交易内容：[描述]
□ 交易金额：[XXX万]
□ 交易对方：[关联方名称]
□ 交易类型：[销售/采购/服务/担保]

□ 披露标准：
□ 须披露阈值：[X] 万
□ 实际交易额 > 阈值：[是/否]

□ 未披露原因：
□ 不了解披露要求
□ 审批流程延误
□ 认为是日常关联交易不需要披露

第二步：评估合规风险

□ 合规风险评估：
□ 违反信息披露规定：[是/否]
□ 可能面临的监管措施：[警示函/罚款/通报批评]
□ 审计风险：[高/中/低]

□ 对报表的影响：
□ 是否影响报表公允性：[是/否]
□ 是否须追溯调整：[是/否]

第三步：补救措施

□ 立即披露：
□ 补充临时公告
□ 说明未及时披露的原因
□ 披露交易详情和定价依据

□ 内部控制改进：
□ 关联交易识别流程改进
□ 关联交易台账管理
□ 定期关联交易审核

□ 沟通监管机构（如必要）：
□ 主动报告
□ 说明原因
□ 承诺改进
```
