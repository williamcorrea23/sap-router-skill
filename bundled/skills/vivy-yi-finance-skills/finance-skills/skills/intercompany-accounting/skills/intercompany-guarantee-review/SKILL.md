# /intercompany-guarantee-review — 关联担保审核


## Examples

→ 示例：用户说"帮我们审一下集团给子公司的担保合同，是否需要披露"，系统应调用本技能，评估集团担保的披露合规性。

→ 示例：用户说"子公司要用集团授信额度，需要签担保协议，帮我看看条款"，系统应调用本技能，审阅集团担保协议条款。

→ 示例：用户说"帮我们检查一下担保额度有没有超限"，系统应调用本技能，执行集团担保额度合规性检查。
## 第一步：获取担保数据

**从 [ERP]/[CASH] 获取担保台账：**
```
□ 担保提供方：[公司名称]
□ 净资产：[XXX万]
□ 担保限额：[净资产 X%] 或 [XXX万]

□ 担保余额合计：[XXX万]
□ 担保/净资产比例：[X%]

□ 获取字段：
  被担保方 | 担保类型 | 担保金额 | 担保期限 | 担保费率 | 状态
```

---

## 第二步：核查担保余额

**核查担保是否超限：**
```
□ 担保提供合计：[XXX万]
□ 担保/净资产比例：[X%]

□ 担保限额检查：
□ 公司章程限额：[X] 万 或 净资产 [X]%
□ 累计已提供担保：[XXX万]
□ 是否超限：[✅ 否 / 🔴 是]

□ 超限金额：[XXX万]
□ 超限处理：
□ 须立即停止新增担保
□ 须制定压缩计划
□ 须披露并说明原因
```

---

## 第三步：被担保方风险评估

**评估被担保方偿债能力：**
```
□ 主要被担保方：[X] 家

□ 风险评估：

□ [被担保方A]：[担保 XXX万]
□ 被担保方类型：[子公司/联营企业/第三方]
□ 持股比例：[X]%
□ 担保余额：[XXX万]

□ 被担保方财务状况：
□ 资产负债率：[X]%（[✅ 正常/⚠️ 偏高]）
□ 流动比率：[X]（[✅ 正常/⚠️ 偏低]）
□ 净利润：[XXX万]（[✅ 盈利/⚠️ 亏损]）

□ 代偿风险：[低/中/高]

□ [被担保方B]：[担保 XXX万]
□ 被担保方类型：[...]
□ 担保余额：[XXX万]
□ 财务状况：[...]
□ 代偿风险：[...]
```

---

## 第四步：到期预警

**核查担保到期情况：**
```
□ 个月内到期担保：[X] 笔，[XXX万]

□ 近期到期明细：
□ [被担保方A]：[担保 XXX万] — 到期 [YYYY-MM-DD] — 剩余 [X] 天
□ [被担保方B]：[担保 XXX万] — 到期 [YYYY-MM-DD] — 剩余 [X] 天

□ 是否存在逾期：[✅ 否 / ⚠️ 是]
□ 逾期担保：[X] 笔，[XXX万]

□ 到期处理建议：
□ 续保：[✅ 已申请/⚠️ 待申请/🔴 未申请]
□ 解除担保：[✅ 已解除/⚠️ 待解除]
□ 代偿风险：[✅ 已评估/⚠️ 须评估]
```

## 升级触发条件

- 担保余额超限
- 存在高代偿风险被担保方
- 担保即将到期未处理
- 须股东会审批的担保
