# /transfer-pricing-review — 转让定价审核


## Examples

→ 示例：用户说"关联交易定价要调整，需要准备什么文档给税局"，系统应调用本技能，准备转让定价文档要求评估。

→ 示例：用户说"帮我们审一下关联方服务费的定价是否合理"，系统应调用本技能，评估关联交易定价合理性。

→ 示例：用户说"集团内某公司要向我们收服务费，帮我算算成本加成是否在合理范围"，系统应调用本技能，执行成本加成合理性分析。
## 第一步：获取内部交易数据

**从 [ERP]/[IC] 获取关联交易明细：**
```
□ 审核期间：[YYYY年MM月/年度]
□ 关联交易合计：[XXX万]

□ 获取字段：
  关联方 | 交易类型 | 交易金额 | 定价方法 | 定价依据
```

---

## 第二步：分类核查定价

**按交易类型核查定价合理性：**

```
□ 内部销售（商品）：
□ 交易金额：[XXX万]
□ 定价方法：[成本加成/市场价/协议价]
□ 成本加成比例：[X]%
□ 市场正常加成幅度：[X]% — [X]%
□ 偏离度：[±X%]

□ 核查结论：[✅ 合理 / ⚠️ 偏高或偏低]

□ 内部服务：
□ 交易金额：[XXX万]
□ 定价方法：[成本加成/利润分割]
□ 成本加成比例：[X]%
□ 市场参考：[X]%（如有）

□ 核查结论：[✅ 合理 / ⚠️ 偏高或偏低]

□ 内部借款：
□ 借款金额：[XXX万]
□ 借款利率：[X]%
□ 同期贷款利率：[X]%
□ 偏离度：[±X%]

□ 核查结论：[✅ 合理 / ⚠️ 偏高或偏低]

□ 资产租赁：
□ 租赁金额：[XXX万]
□ 租赁资产：[描述]
□ 市场租金参考：[XXX万]

□ 核查结论：[✅ 合理 / ⚠️ 偏高或偏低]
```

---

## 第三步：分析偏离度

**评估转让定价风险程度：**

```
□ 偏离度评估：

□ 无偏离或轻微偏离（< [X]%）：
□ 风险等级：[低]
□ 处理方式：正常记录备查

□ 中度偏离（[X]-[X]%）：
□ 风险等级：[中]
□ 处理方式：准备定价说明文件

□ 重度偏离（> [X]%）：
□ 风险等级：[高]
□ 处理方式：须调整定价或准备转让定价报告

□ 主要偏离项：
□ [关联方A]：[交易类型] — 内部价 [XXX] — 市场价 [XXX] — 偏离 [X%]
□ [关联方B]：[交易类型] — 内部价 [XXX] — 市场价 [XXX] — 偏离 [X%]
```

---

## 第四步：评估转让定价风险

**综合评估税务和合规风险：**

```
□ 税务风险评估：
□ 是否符合独立交易原则：[✅ 是 / ⚠️ 否]
□ 潜在税务调整风险：[低/中/高]
□ 可能调整金额：[XXX万]

□ 转让定价报告准备：
□ 是否须准备同期资料：[是/否]
□ 准备年度：[YYYY 年]
□ 截止日：[YYYY-MM-DD]
□ 当前状态：[未准备/准备中/已完成]

□ 预约定价安排（APA）：
□ 是否申请 APA：[是/否]
□ 当前进度：[描述]
```
