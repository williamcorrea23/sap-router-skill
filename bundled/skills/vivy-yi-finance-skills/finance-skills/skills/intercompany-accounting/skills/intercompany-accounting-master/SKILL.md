# /intercompany-accounting-master — 内部往来核算管理主流程

## 完整关联交易管理流程

```
Step 1：内部往来对账（每月 [X] 日前完成）
  → 执行 intercompany-reconciliation
  → 确保往来余额一致

Step 2：关联交易审核（每月/季）
  → 执行 transfer-pricing-review
  → 确保定价公允

Step 3：关联担保审核（每月/季）
  → 执行 intercompany-guarantee-review
  → 确保担保合规

Step 4：关联交易汇总
  → 汇总全期关联交易
  → 核对关联交易台账
  → 确认披露完整性

Step 5：综合管理报告
  → 汇总 Step 1-4 发现的问题
  → 输出关联交易健康度报告
```
