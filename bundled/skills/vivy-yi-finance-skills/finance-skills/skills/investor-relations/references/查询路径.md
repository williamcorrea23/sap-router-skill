# 查询路径 — 投资者关系

## 查询入口

### 查询维度一：股东与股份

| 查询目标 | 系统入口 | 查询路径 | 刷新频率 |
|---|---|---|---|
| 股东名册 | [交易所系统] | 股东管理 → 股东名册 | 实时 |
| 股份变动 | [交易所系统] | 股份变动 → 增减持记录 | 实时 |
| 股份质押 | [交易所系统] | 质押管理 → 质押情况 | 实时 |

### 查询维度二：市场数据

| 查询目标 | 系统入口 | 查询路径 | 刷新频率 |
|---|---|---|---|
| 股价与成交量 | [行情系统] | 股票行情 → 实时/历史 | 实时 |
| 分析师覆盖 | [IR 软件] | 分析师管理 → 评级/目标价 | 实时 |
| 机构持仓 | [外部数据] | 机构持仓 → 基金季报 | 季度 |

## 常用查询组合

**场景：机构投资者问询**
```
1. [交易所系统] → 股东名册 → 确认股东身份
2. [行情系统] → 股价查询 → 近期股价表现
3. [IR 软件] → 分析师管理 → 查询最新评级
4. [BI 系统] → 财务数据 → 准备业绩数据支撑
```

**场景：业绩路演准备**
```
1. [BI 系统] → 财务数据 → 提取核心业绩数据
2. [行情系统] → 股价查询 → 提取历史股价走势
3. [IR 软件] → 分析师管理 → 分析评级分布与目标价
4. [外部数据] → 竞品数据 → 准备行业对标材料
```
