# 查询路径 — 投融资管理

## 数据源优先级

| 优先级 | 数据源 | 说明 |
|---|---|---|
| 1 | 投资管理系统（[INVEST]）| 投资项目/回报/进展 | 最优先 |
| 2 | ERP 总账（[ERP]）| 投资账务/处置损益 | 账务核对 |
| 3 | 资金系统（[CASH]）| 融资余额/授信/还款 | 融资管理 |
| 4 | 行情数据（[MARKET]）| 股票/债券/商品价格 | 市值评估 |

---

## 常用查询场景

### 场景 1：投资项目盘点

**查询入口：**
```
投资管理系统：
- 投资项目清单
- 投资成本/账面价值
- 投资回报/分红
- 被投公司财务数据

ERP：
- 长期股权投资科目
- 其他非流动金融资产
- 投资收益科目
```

**投资项目概览：**
```
□ 被投公司/项目：[X] 个
□ 投资成本合计：[XXX万]
□ 账面价值合计：[XXX万]
□ 浮盈/浮亏：[±XXX万]

□ 上市公司：[X] 个，账面 [XXX万]
□ 非上市公司：[X] 个，账面 [XXX万]
□ 基金/理财：[X] 个，账面 [XXX万]
```

## 数据源组合使用

### 投资项目月度复盘
```
[INVEST] 投资项目进展 → [MARKET] 市值/行情 → [ERP] 投资收益账务 → 生成投资月报
```

### 融资管理月度复盘
```
[CASH] 融资余额/还款 → [CASH] 授信额度 → [ERP] 财务费用 → 生成融资月报
```

### 流动性预警
```
[CASH] 现金余额 → [CASH] 未来 [X] 月还款 → [CASH] 授信可用 → [ERP] 每月支出 → 生成流动性预警
```
