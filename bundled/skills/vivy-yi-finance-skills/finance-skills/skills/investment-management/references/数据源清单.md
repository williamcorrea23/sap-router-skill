## [INVEST] 投资管理系统

**覆盖范围：**
- 投资项目清单（投资成本/持股比例/账面价值）
- 被投公司基本信息（行业/地区/成立时间）
- 被投公司财务数据（收入/利润/资产）
- 投资回报（分红/利息/处置收益）
- 投后管理记录（董事会/经营跟踪）

**查询路径：**

```
投资管理系统：
- 项目组合 → 投资项目清单
- 项目详情 → 投资成本/回报/进展
- 被投公司 → 财务数据/经营情况
- 投资回报 → 分红/处置记录
```

---

## [ERP] SAP / 用友 / 金蝶（总账）

**覆盖范围：**
- 长期股权投资科目
- 其他非流动金融资产
- 交易性金融资产
- 应付债券
- 短期借款/长期借款
- 投资收益/财务费用
- 预计负债（或有负债）

**查询路径：**

```
SAP 常用事务代码：
- FAGLB03：科目余额表
- FB03：凭证明细
- S_ALR_00001237：投资台账
- 持股平台账务：[股权管理办法]

用友/金蝶：
- 长期股权投资 → 投资明细账
- 金融资产 → 公允价值变动
- 借款 → 融资台账
```

---

## [CASH] 司库系统 / 资金管理系统

**覆盖范围：**
- 银行账户余额（实时）
- 融资余额明细（银行借款/债券/融资租赁）
- 授信额度（总额/已用/可用）
- 还款计划（到期日/金额）
- 担保/抵押登记
- 资金预测（未来 [X] 个月）

**查询路径：**

```
司库系统：
- 账户管理 → 银行账户余额
- 融资管理 → 融资余额/授信/还款
- 担保管理 → 担保/抵押登记
- 资金预测 → 未来现金流预测

银企直联：
- 实时余额查询
- 电子回单
- 付款指令
```

---

## [MARKET] Wind / 同花顺 / 彭博

**覆盖范围：**
- 上市公司股价（实时/历史）
- 债券价格/收益率
- 行业指数
- 宏观经济数据
- 外汇/大宗商品价格

**查询路径：**

```
Wind/同花顺：
- 股票行情 → 持仓股票价格
- 债券行情 → 持仓债券估值
- 行业数据 → 行业平均估值

交易所：
- 定期报告披露查询
- 监管公告查询

彭博（机构用户）：
- 综合行情分析
- 估值模型数据
```

## 数据源组合使用

### 投资项目月度复盘
```
[INVEST] 投资项目 → [MARKET] 上市公司市值 → [ERP] 投资收益账务
```

### 融资管理月度复盘
```
[CASH] 融资余额 → [CASH] 授信额度 → [ERP] 财务费用 → [MARKET] 市场利率
```

### 流动性预警
```
[CASH] 现金余额 → [CASH] 未来 [X] 月到期 → [CASH] 可用授信 → 流动性分析
```

### 投资组合风险评估
```
[INVEST] 投资组合 → [MARKET] 行业/市值 → 风险敞口计算 → 预警
```
