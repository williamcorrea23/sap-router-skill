# 判断框架 — 投融资管理异常处理

## 核心原则

**投融资管理的目标是"把每一分钱投向回报最高的地方，同时确保公司不会因为融资而陷入危机"。**
遇到异常时，优先判断是"市场波动"（短期）还是"基本面恶化"（长期），并确保风险敞口在可控范围内。

---

## 异常类型与处理方式

### 类型一：投资项目亏损（已确认损失）

**典型场景：**
- 被投公司经营不善，确认投资损失
- 项目失败，投资无法回收
- 清算/破产导致本金损失

**判断流程：**

```
第一步：确认损失金额

□ 投资成本：[X] 万
□ 已确认损失：[X] 万
□ 剩余账面价值：[X] 万

□ 损失阶段：
  - 亏损（< 50% 损失）：仍有残余价值
  - 重大亏损（50-90% 损失）：残余价值有限
  - 全损（> 90% 损失）：通常全额确认损失

□ 损失确认依据：
  - 被投公司审计报告
  - 第三方评估报告
  - 市场交易价格（如上市公司）
  - 清算分配方案

→ 须审计/评估报告支撑，不能仅凭主观判断

第二步：分析损失原因

□ 被投公司管理层问题：
  → 战略失误/执行不力
  → 财务造假/治理问题
  → 核心人员流失

□ 行业/市场因素：
  → 行业整体下行
  → 政策不利变化
  → 宏观经济冲击

□ 我方判断失误：
  → 尽职调查不充分
  → 估值模型缺陷
  → 风险识别不足

□ 不可抗力：
  → 自然灾害/战争
  → 监管政策突变

第三步：评估剩余敞口

□ 剩余账面价值：[X] 万
□ 是否须进一步计提减值？
□ 是否须调整估值假设？

□ 后续处置方案：
  → 继续持有等待反转（须有明确理由）
  → 转让退出（积极寻找买家）
  → 清算退出（最坏情况）

第四步：制定改善措施

□ 投资决策改进：
  → 尽职调查清单完善
  → 估值模型压力测试
  → 风险条款增加（对赌/回购）

□ 投后管理改进：
  → 董事会/观察员席位争取
  → 季度经营跟踪机制
  → 预警指标设定
```

### 类型二：投资项目浮亏（未确认损失）

**典型场景：**
- 上市公司股票/基金价格下跌（浮亏）
- 非上市投资估值下调（浮亏）
- 理财产品净值下跌（浮亏）

**判断流程：**

```
第一步：确认浮亏金额

□ 投资成本：[X] 万
□ 当前市值：[X] 万
□ 浮亏金额：[X] 万
□ 浮亏比例：[X]%

□ 是否触及止损线？
□ 是否超出风险偏好？

第二步：判断是"市场波动"还是"基本面恶化"

市场波动（短期，可接受）：
□ 大盘整体下跌，但被投公司基本面正常
□ 行业周期性调整，不影响长期价值
□ 流动性紧张导致的暂时性下跌
→ 继续持有，定期监控

基本面恶化（长期风险，须关注）：
□ 被投公司经营指标恶化（收入/利润/用户）
□ 行业竞争格局恶化（市场份额下降）
□ 政策不利变化（行业整体受压）
□ 管理层变动/核心人员流失
→ 须深入评估是否须计提减值

第三步：评估风险敞口

□ 浮亏金额 vs 净资产：[X%]
□ 单一投资浮亏上限：[X] 万

□ 是否须止损？
□ 止损后的损失是否可承受？
□ 是否有机会通过补仓摊薄成本？

第四步：制定应对策略

□ 如判断为市场波动：
  → 维持现状，定期（每月）复盘
  → 设置预警指标（跌破 [X] 元须重新评估）
  → 必要时分散持仓

□ 如判断为基本面恶化：
  → 深入评估（联系被投公司/行业专家）
  → 考虑减持/退出
  → 调整估值模型参数
```

### 类型三：融资成本过高

**典型场景：**
- 贷款利率 > 目标 [X]%
- 担保费率 > 市场水平
- 融资顾问费 > 行业标准
- 综合融资成本 > 利润增长率

**判断流程：**

```
第一步：确认融资成本构成

□ 贷款本金：[X] 万
□ 年利率：[X]%
□ 年利息：[X] 万
□ 担保费：[X] 万
□ 顾问费：[X] 万
□ 其他费用：[X] 万

□ 综合融资成本（IRR）：[X]%

□ 是否超过公司设定的上限 [X]%？
□ 是否超过投资项目的 IRR？

第二步：分析成本过高的原因

□ 自身资质问题：
  → 信用评级下降
  → 资产负债率过高
  → 现金流紧张
  → 抵押物不足

□ 市场环境问题：
  → 银行整体收紧信贷
  → 行业风险高（银行不愿贷）
  → 货币政策收紧

□ 融资结构问题：
  → 短期贷款占比过高（滚动再融资风险）
  → 担保/抵押方式受限
  → 缺少长期低成本资金

第三步：制定改善方案

□ 降低融资成本：
  → 改善财务指标（降低负债率）
  → 增加抵押/担保物
  → 争取信用贷款（无须担保）
  → 对比 [X] 家银行，选择最优报价

□ 调整融资结构：
  → 增加长期贷款比例
  → 发行公司债券（利率更低）
  → 引入低成本战略投资者

□ 改善现金流：
  → 加速回款，降低融资需求
  → 处置低效资产，回笼资金
  → 协商延长付款账期

第四步：融资成本管控机制

□ 新增融资须经 CFO 审批（> [X] 万）
□ 综合融资成本须 < [X]% 或 < 投资 IRR
□ 定期（每季度）复盘融资成本
□ 优先使用低成本融资渠道
```

### 类型四：担保/抵押过度

**典型场景：**
- 累计担保额 > 净资产 [X]%
- 单一客户担保额 > [X] 万
- 抵押物价值不足覆盖贷款
- 反担保措施缺失

**判断流程：**

```
第一步：确认担保敞口

□ 对外担保总额：[X] 万
□ 净资产：[X] 万
□ 担保/净资产比例：[X%]

□ 已确认代偿风险：[X] 万
□ 历史代偿记录：[X] 次/[X] 万

□ 主要被担保方：
  - [客户A]：[X] 万
  - [客户B]：[X] 万

第二步：评估被担保方资质

□ 被担保方财务状况：
  → 资产负债率：[X]%（< [X]% 安全）
  → 流动比率：[X]（> [X] 安全）
  → 盈利能力：[描述]

□ 被担保方行业风险：
  → 行业周期：[上升/稳定/下降]
  → 竞争格局：[描述]
  → 政策风险：[描述]

□ 被担保方历史信用：
  → 历史代偿次数：[X]
  → 历史逾期记录：[X]
  → 实际控制人信用：[描述]

第三步：判断担保风险等级

担保风险等级：[低/中/高/极高]

□ 低风险：被担保方资质优良，反担保充足
□ 中风险：被担保方资质尚可，需关注
□ 高风险：被担保方资质恶化，须预防性措施
□ 极高风险：被担保方可能违约，须准备代偿

第四步：制定管控措施

□ 存量担保管控：
  → 建立被担保方季度评估机制
  → 设置预警指标（财务指标恶化触发）
  → 要求被担保方提供反担保
  → 逐步压缩担保额度

□ 新增担保管控：
  → 新增担保须 CFO 审批
  → 须有反担保措施
  → 单一客户担保上限：[X] 万
  → 须定期跟踪被担保方经营

□ 代偿准备：
  → 按风险等级计提预计负债
  → 保持充足现金储备应对代偿
  → 建立代偿应急方案
```

### 类型五：流动性危机风险

**典型场景：**
- 流动比率 < [X]，短期偿债能力不足
- 现金及等价物 < [X] 个月刚性支出
- 大额债务集中到期
- 银行授信额度即将用尽

**判断流程：**

```
第一步：评估流动性状况

□ 现金及等价物：[X] 万
□ 每月刚性支出：[X] 万
□ 可用月数：[X] 个月

□ 流动资产：[X] 万
□ 流动负债：[X] 万
□ 流动比率：[X]

□ 银行授信总额：[X] 万
□ 已使用授信：[X] 万
□ 可用授信：[X] 万

□ 下 [X] 个月到期债务：[X] 万

第二步：识别流动性紧张原因

□ 收入端问题：
  → 收入下降，应收账款回款慢
  → 客户集中度高，大客户流失
  → 行业周期性，旺季过后

□ 支出端问题：
  → 债务集中到期（再融资压力）
  → 资本支出过大（扩张期）
  → 运营成本过高（亏损期）

□ 融资端问题：
  → 银行收紧信贷
  → 再融资困难
  → 担保物不足

第三步：判断危机等级

流动性危机等级：[正常/关注/预警/危机]

□ 正常：现金 > [X] 个月支出，流动比率 > [X]
□ 关注：现金 [X]-[X] 个月支出，流动比率 [X]-[X]
□ 预警：现金 < [X] 个月支出，流动比率 < [X]
□ 危机：现金不足 [X] 个月支出，即将技术性违约

第四步：制定应对方案

□ 如为关注状态：
  → 加强应收账款催收
  → 提前与银行沟通续授信
  → 减少非必要支出
  → 准备应急融资方案

□ 如为预警状态：
  → 立即启动应急机制
  → 停止所有非必要支出
  → 争取银行不抽贷、压降
  → 评估资产处置可能性
  → 引入战略投资者

□ 如为危机状态：
  → 申请债务展期/重组
  → 处置非核心资产
  → 引入外部救援资金
  → 必要时申请破产保护
```

## 置信度评估

| 置信度 | 条件 | 操作 |
|---|---|---|---|
| 高 | 投资数据完整 + 市值更新 + 估值模型更新 | ✅ 正常分析 |
| 中 | 投资数据完整，市值/估值待更新 | ⚠️ 标注不确定性 |
| 低 | 投资项目数据不完整或存在争议 | 🔴 暂停分析，核查 |
