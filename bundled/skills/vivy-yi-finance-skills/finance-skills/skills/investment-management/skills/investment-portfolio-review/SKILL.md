# /investment-portfolio-review — 投资组合复盘


## Examples

→ 示例：用户说"帮我们审一下目前的投资组合配置，有没有过度集中"，系统应调用本技能，执行投资组合集中度分析。

→ 示例：用户说"有一笔理财快到期了，帮我分析一下要不要续投"，系统应调用本技能，评估理财续投决策。

→ 示例：用户说"帮我们看一下投资组合的整体收益率，和基准比怎么样"，系统应调用本技能，计算投资组合收益与基准对比。
## 第一步：获取投资数据

**从 [INVEST]/[MARKET]/[ERP] 获取投资数据：**
```
□ 投资项目清单：全部 [X] 个
□ 投资成本合计：[XXX万]
□ 账面价值合计：[XXX万]

□ 获取字段：
  被投公司 | 投资成本 | 账面价值 | 持股比例 | 市值/估值
```

```
  - [公司B]：[XXX万]（持股 [X]%）— 市值 [XXX万] — 浮盈/浮亏 [±XXX万]
```

## 第三步：浮盈/浮亏分析

**浮盈/浮亏汇总：**

```
□ 投资成本合计：[XXX万]
□ 当前市值/估值合计：[XXX万]
□ 浮盈/浮亏：[±XXX万]
□ 浮盈/浮亏率：[±X%]

□ 浮盈项目：[X] 个（[XXX万]）
□ 浮亏项目：[X] 个（[XXX万]）
□ 平价项目：[X] 个
```

**重大浮盈/浮亏项目：**

```
□ 浮盈最大：[公司A] — [±XXX万]（[±X%]）
□ 浮亏最大：[公司B] — [±XXX万]（[±X%]）

□ 须关注浮亏项目（> [X] 万）：
  - [公司B]：[浮亏 XXX万] — 原因 [描述]
  - [公司C]：[浮亏 XXX万] — 原因 [描述]
```

## 第四步：风险敞口评估

**行业/地域分散度：**

```
□ 单一行业最大敞口：[行业A] — [X]%（< [X]% 限制）
□ 单一地域最大敞口：[地区A] — [X]%（< [X]% 限制）
□ 海外投资敞口：[XXX万]（< 净资产 [X]%）
```

**流动性风险：**

```

```
