## 常用查询组合

**场景：LCR 预警处理**
```
1. [司库系统 TRSY] → 流动性 → LCR 仪表板 → 确认当前 LCR
2. [司库系统 TRSY] → 现金头寸 → 分析 HQLA 构成
3. [司库系统 TRSY] → 资金计划 → 分析 30 天净流出
4. [司库系统 TRSY] → 投资管理 → 评估可变现投资
```

**场景：现金池调拨**
```
1. [司库系统 TRSY] → 现金管理 → 查询各账户余额
2. [司库系统 TRSY] → 现金池 → 查询池内额度
3. [银行系统 BANK] → 账户管理 → 确认到账情况
4. [司库系统 TRSY] → 调拨记录 → 查询历史调拨
```
