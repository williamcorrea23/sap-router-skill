# 判断框架 — 应付账款异常处理

## 核心原则

**应付账款管理的目标是"维护供应商关系的同时确保资金安全"。**
遇到异常时，优先判断是"合理商业原因"还是"潜在风险"，对高风险事项保持警惕。

---

## 异常类型与处理方式

### 类型一：发票与订单金额不匹配

**典型场景：**
- 发票金额 > 订单金额（供应商多收了）
- 发票金额 < 订单金额（供应商少收了）
- 发票数量 vs 订单数量不一致
- 税率错误（合同约定 13%，发票开 9%）

**判断流程：**

```
第一步：确认是否为合理差异

合理差异（可接受）：
  - 实际收货数量 vs 订单数量（按实际数量结算）
  - 汇率波动（海外采购，汇率变化导致小额差异）
  - 供应商提前备货产生的价格差异（须有沟通记录）

不合理差异（须核查）：
  - 供应商单方面加价
  - 税率错开（13% → 9%，供应商少交税）
  - 数量/金额差异 > 5% 且无合理解释

第二步：处理方式

差异 ≤ [X] 元（或 ±[X]%）：
  → ⚠️ AP 主管审批后可付款
  → 记录差异原因

差异 > [X] 元：
  → 🔴 暂停付款
  → 联系供应商确认，必要时重新开票
  → 收到正确发票前不得付款
```

---

### 类型二：付款条款变更要求

**典型场景：**
- 供应商要求将月结 30 天改为月结 60 天
- 供应商要求提前付款（供应链金融/折扣）
- 供应商要求改变收款账户

**判断标准：**

```
提前付款（供应商要求提前回款）：
  - 折扣提前付款（2/10 net 30）：评估是否合算
    → 公式：年化成本 = 折扣率 / (1-折扣率) × 360 / (到期日-折扣天数)
    → 如年化成本 < 资金成本，接受；否则拒绝
  - 无折扣提前付款：拒绝，除非有战略合作理由

付款账户变更（供应商要求变更收款账户）：
  → 🔴 高风险信号
  → 步骤：
    1. 暂停付款，联系供应商原对接人确认
    2. 要求供应商提供书面变更申请（加盖公章）
    3. 与历史付款记录核对（账户名/账号/银行）
    4. 如账户信息变化大，升级至财务经理 + 法务

付款条款恶化（延长账期）：
  → ⚠️ 须评估对供应商关系的影响
  → 如影响核心物料供应，建议接受
  → 如无实质影响，可谈判
```

### 类型三：逾期未付款

**典型场景：**
- 应付已到期但未安排付款
- 供应商催款
- 连续多个月逾期

**判断标准：**

| 逾期天数 | 风险等级 | 处理方式 |
|---|---|---|
| 0-7 天 | ✅ 正常 | 下周付款日安排 |
| 7-30 天 | ⚠️ 关注 | 须在付款优先级中排前 |
| 30-60 天 | 🔴 警告 | 须升级，联系供应商说明情况 |
| > 60 天 | 🔴 高风险 | 须升级至财务总监，处理法律风险 |

**逾期原因排查：**
```
□ 是否因为资金不足无法支付？
□ 是否因为三单不匹配无法确认应付？
□ 是否因为供应商发票未到？
□ 是否因为内部审批延迟？
□ 是否因为供应商争议（质量/数量问题）？
```

---

### 类型四：预付款风险

**典型场景：**
- 新供应商要求高比例预付款
- 预付款后供应商交付延迟
- 预付款到期未收到货/服务

**判断标准：**

```
预付款审批规则：
  预付 ≤ 30%：
    → AP 主管审批后可执行
  预付 30-50%：
    → 财务经理审批
  预付 > 50%：
    → 财务总监 + CEO 审批

预付款风险检查：
□ 供应商是否为 A 级（战略供应商）？
□ 是否有合同 + 里程碑付款机制？
□ 预付款到期是否有交付保障？
□ 供应商是否有违约记录？

预付款到期处理：
  到期未交付：
    → 🔴 立即升级至财务经理
    → 启动供应商交涉
    → 必要时启动法律程序追回预付款
```

---

### 类型五：供应商争议

**典型场景：**
- 供应商对付款金额有异议
- 供应商要求跳过争议处理直接付款
- 质量扣款/罚款争议

**处理流程：**

```
第一步：确认争议性质

金额争议：
  → 要求供应商提供明细
  → 与合同/订单/收货单核对
  → 确定争议金额

质量扣款：
  → 要求质量部门出具检验报告
  → 如质量不达标，扣款合理
  → 如质量达标，须正常付款

第二步：谈判策略

优先和解（付款金额的一部分作为和解费）：
  → 和解金额 ≤ 争议金额的 20%
  → 和解须有书面协议

坚持原立场（证据充分时）：
  → 向供应商出示合同/收货单据
  → 如供应商坚持，通过法律途径解决

第三步：付款决策

无争议部分：
  → 按时付款

争议部分：
  → 暂扣争议金额，待争议解决后支付
  → 或按和解协议支付
```

---

## 置信度评估

| 置信度 | 条件 | 操作 |
|---|---|---|
| 高 | 三单完全匹配 + 无争议 | ✅ 可安排付款 |
| 中 | 差异可解释或有部分争议 | ⚠️ 解决后付款 |
| 低 | 重大差异或高风险信号 | 🔴 暂停付款，升级 |

---

## 升级触发条件

- 单笔应付 > [填空：如 50 万元]
- 预付款比例 > 30%
- 逾期 > 30 天
- 供应商要求变更收款账户
- 供应商争议金额 > [填空：如 10 万元]
- 发现疑似欺诈（假发票/假供应商）
- 同一供应商连续 3 个月付款异常
