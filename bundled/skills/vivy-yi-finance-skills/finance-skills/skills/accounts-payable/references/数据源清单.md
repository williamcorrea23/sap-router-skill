# 数据源清单 — [ERP]/[PO]/[RCV]/[INV]/[CASH]

## 数据源标签说明

| 标签 | 含义 | 典型来源 |
|---|---|---|
| [ERP] | 企业 ERP 系统（AP 模块）| SAP FI/AP / 用友 AP / 金蝶 AP |
| [PO] | 采购订单/合同系统 | SAP MM / 用友采购 / 金蝶采购 |
| [RCV] | 收货/验收系统 | SAP MM / 用友库存 / 金蝶仓存 |
| [INV] | 供应商发票系统 | SAP MIRO / 用友应付 / 影像扫描系统 |
| [CASH] | 资金/现金管理系统 | 司库系统/资金系统/网上银行 |

---

## [ERP] SAP FI/AP 模块（应付账款）

**覆盖范围：**
- 供应商主数据（供应商名称/账户/付款条件）
- 应付账款明细（各供应商未清余额/到期日）
- 已过账发票（应付凭证）
- 已过账付款（付款凭证）
- 供应商行项目（所有应收应付明细）
- 预付款凭证
- 争议项标记

**查询路径：**

```
SAP 常用事务代码：
- FK01N：显示供应商（应付视图）
- FBL1N：供应商行项目（未清项分析）
- FB03：显示凭证（查看付款记录）
- MIRO：发票校验
- MIRA：发票校验记录查询
- FCH5：显示付款计划

常用报表：
- 供应商余额表（按供应商）
- 到期分析表（按到期日）
- 账龄分析表（逾期分析）
- 付款预测表（未来到期金额）
```

---

## [PO] SAP MM 采购模块（采购订单）

**覆盖范围：**
- 采购订单主数据（供应商/物料/数量/单价/付款条件）
- 框架合同（长期采购协议）
- 合同价格（年度价/协议价）
- 订单状态（已创建/已交货/已关闭）

**查询路径：**

```
SAP MM 常用事务代码：
- ME21N：创建采购订单
- ME23N：显示采购订单
- ME2L：按供应商查看采购订单
- ME2M：按物料查看采购订单

用友/金蝶：
- 采购管理 → 采购订单查询
- 合同管理 → 框架合同查询
```

---

## [RCV] SAP MM 收货模块（采购入库）

**覆盖范围：**
- 采购收货记录（数量/日期/仓库）
- 服务验收单（里程碑验收）
- 质量检验结果
- 退货记录

**查询路径：**

```
SAP MM 常用事务代码：
- MIGO：采购收货
- MB03：显示物料凭证
- ME2N：按采购订单查看收货

用友/金蝶：
- 库存管理 → 采购入库单
- 质量管理 → 来料检验单
```

---

## [INV] 供应商发票系统

**覆盖范围：**
- 供应商发票登记（发票代码/号码/金额/日期）
- 发票影像（扫描件/电子发票）
- 发票验真（与税局系统对接）
- 异常发票记录（作废/红冲/失控）

**查询路径：**

```
SAP MIRO：
- 发票校验：输入发票信息，与 PO/GRN 匹配

发票影像系统：
- 扫描系统 → 查看发票影像
- OCR识别 → 自动提取发票信息

国家税务总局发票查验平台：
- https://inv-veri.chinatax.gov.cn
- 增值税专用发票/普通发票验真
```

---

## [CASH] 资金/现金管理系统

**覆盖范围：**
- 银行账户余额（各银行账户可用余额）
- 现金流预测（未来 N 天到期应付/应收）
- 付款排期（已排期待付款项）
- 承兑汇票状态

**查询路径：**

```
司库系统（如有）：
- 可用资金查询
- 付款排期
- 现金流预测

网上银行：
- 企业网银 → 账户余额查询
- 付款指令状态查询

ERP 资金模块：
- SAP FCH5：显示付款计划
- SAP FBL5N：客户行项目（配合 AR 使用）
```

## 数据源组合使用

### 付款排期
```
[CASH] 可用资金 → [ERP] 到期应付清单 → [PO] 订单优先级 → 排期付款
```

### 三单匹配
```
[PO] 采购订单 → [RCV] 收货确认 → [INV] 供应商发票 → 匹配校验
```

### 供应商争议
```
[ERP] 争议项 → [INV] 发票影像 → [RCV] 收货记录 → [PO] 合同条款 → 争议处理
```
