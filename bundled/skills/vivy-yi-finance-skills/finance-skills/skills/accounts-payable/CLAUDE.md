## 供应商管理配置

### 供应商分级

| 等级 | 评级标准 | 付款优先级 | 付款条款 |
|---|---|---|---|
| A 级 | 战略供应商/长期合作 | 优先付款 | 月结 30 天 / EOM |
| B 级 | 重要供应商/良好记录 | 正常付款 | 月结 45 天 |
| C 级 | 一般供应商 | 正常付款 | 月结 60 天 / 现结 |
| D 级 | 新供应商/问题供应商 | 谨慎付款 | 预付款或现结 |

### 供应商付款条款

```
标准付款条款：
  - 月结 30 天：每月末确认应付，次月 [X] 日支付
  - 月结 45 天：每月末确认应付，次月 [X+15] 日支付
  - 月结 60 天：每月末确认应付，隔月 [X] 日支付
  - 现结：收货/服务完成即支付
  - 预付款：订单确认后先付 [X]%，收货后付尾款

特殊付款条款：
  - 供应链金融：可选择提前付款（供应商承担贴息）
  - 承兑汇票：6 个月银行承兑汇票，到期支付
  - 信用证：见单即付 / 远期信用证
```

## 付款审批层级

### 付款审批矩阵

| 单笔金额 | 审批层级 |
|---|---|
| ≤ 5 万 | AP 专员 → AP 主管 |
| 5-20 万 | AP 专员 → AP 主管 → 财务经理 |
| 20-50 万 | AP 专员 → AP 主管 → 财务经理 → 财务总监 |
| > 50 万 | AP 专员 → AP 主管 → 财务经理 → 财务总监 → CEO |

### 特殊付款类型

```
紧急付款（超出正常付款周期的付款）：
  - 须额外说明原因
  - 须部门负责人 + 财务经理双重审批
  - 手续费（如有）由申请部门承担

预付款申请：
  - 须提供合同/订单
  - 预付比例 ≤ 30%（标准情况）
  - 超 30% 须财务总监审批
  - 须在收货/服务完成后 [X] 天内取得发票并核销
```

## 审计与合规

### 三单匹配规则

```
三单匹配（Three-way Match）：
  采购订单（PO）= 收货单（GRN）= 供应商发票（Invoice）

完全匹配：允许付款
  → 三单金额完全一致

可容忍差异：
  → 发票 vs PO：± [X]% 或 ± [X] 元（取较大者）
  → 收货 vs 发票：数量差异 ≤ [X]%，金额差异 ≤ [X] 元

差异处理：
  → 金额差异 ≤ [X] 元：AP 主管审批后可付款
  → 金额差异 > [X] 元：须联系供应商确认后重新开票
  → 数量差异：按实际收货数量付款
```

### 反腐合规

```
付款合规红线：
  - 不得向供应商个人账户支付（须对公账户）
  - 不得分拆付款（规避审批限额）
  - 不得虚构交易付款
  - 大额预付须有合同 + 里程碑支撑
  - 付款摘要须与合同/订单一致
```

## 地区/行业差异

```
供应商注册地差异：
  - 本地供应商：标准付款条款
  - 异地供应商：增加银行转账时间（+ 1-2 天）
  - 海外供应商：TT/LC 付款，须提前 [X] 天发起

行业差异：
  - 制造业：大额设备/原材料采购，预付款比例可提高
  - 服务业：人力成本月结，无预付款需求
  - 零售业：快周转，现结为主
```
