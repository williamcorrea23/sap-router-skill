## 预算控制参数

### 超预算处理规则

| 超预算幅度 | 处理方式 |
|---|---|
| ≤ 5% | ⚠️ 软控制 — 须部门负责人审批后可以报销 |
| 5-10% | 🔴 须财务经理审批 |
| 10-20% | 🔴 须财务总监审批 |
| > 20% | 🔴 须 CEO 审批（重大预算调整）|

### 预算冻结规则

```
触发条件：
  - 部门预算执行率已超过 90%
  - 公司整体预算执行率异常

冻结范围：
  - 当月新增预算外采购/服务申请
  - 超预算 10% 以上的费用报销

冻结解除：
  - 下月预算释放（自然释放）
  - 中期预算调整获批后解除
```

## 预算编制方法

### 方法一：增量预算（Incremental Budgeting）

```
适用场景：业务稳定的成熟企业
方法：
  1. 以上年实际为基础
  2. 按固定比例（如 3-5%）增加或减少
  3. 调整已知的新增/削减项
优点：编制简单，易于对比
缺点：可能固化无效支出
```

### 方法二：零基预算（Zero-Based Budgeting）

```
适用场景：成本压力大，需优化资源配置的企业
方法：
  1. 每个预算项目都从"零"开始论证
  2. 不自动延续上期预算
  3. 按优先级排序，资源用完即止
优点：优化成本结构，识别浪费
缺点：编制工作量大，每年只能对部分科目做
```

### 方法三：作业预算（Activity-Based Budgeting）

```
适用场景：制造成本占比高，流程清晰的企业
方法：
  1. 分析各作业活动的资源消耗
  2. 按作业量分配预算
  3. 关注产能利用率
优点：精确分配资源
缺点：基础数据要求高
```

## 预算分析维度

### 差异分解

```
总差异 = 实际 - 预算

分解方式（按业务类型选择）：

制造类：
  总差异 = 销量差异 + 单位成本差异
  销量差异 = (实际销量 - 预算销量) × 预算单价
  单位成本差异 = 实际销量 × (实际单价 - 预算单价)

费用类：
  总差异 = 开支差异 + 效率差异
  开支差异 = 实际开支 - 预算开支
  效率差异 = 预算单价 × (实际用量 - 预算用量)

收入类：
  总差异 = 销量差异 + 价格差异
  销量差异 = (实际销量 - 预算销量) × 预算单价
  价格差异 = 实际销量 × (实际单价 - 预算单价)
```

### 预算执行率监控

```
计算公式：
  执行率 = 实际发生额 / 预算金额 × 100%

监控阈值：
  预算执行率 > 100%：🔴 超预算
  预算执行率 90-100%：⚠️ 接近预算上限
  预算执行率 70-90%：✅ 正常范围
  预算执行率 < 70%：⚠️ 进度偏慢，可能影响年度目标
```

### 预算准确性评估

| 置信度 | 条件 | 输出标注 |
|---|---|---|
| 高 | 实际/预算差异在 ±10% 以内 | ✅ 预算编制准确 |
