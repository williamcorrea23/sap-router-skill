# 数据源清单 — [BK]/[ERP]/[FK]/[OA]/[HR]

## 数据源标签说明

| 标签 | 含义 | 典型来源 |
|---|---|---|
| [BK] | 预算管理系统 | SAP BPC/用友全面预算/金蝶预算管理 |
| [ERP] | 企业 ERP 系统 | SAP/用友/金蝶总账 |
| [FK] | 费控平台 | 分贝通/胜意/Coupa |
| [OA] | OA 审批系统 | 钉钉/飞书/企微 |
| [HR] | HR 系统 | 薪酬系统/社保申报系统 |

---

## [BK] SAP BPC（预算管理系统）

**覆盖范围：**
- 年度预算编制数据（按成本中心/科目/期间）
- 预算调整记录
- 预算执行数据（实际 vs 预算）
- 预算审批流程
- 多版本预算（预算 v1/v2/实际）

**查询路径：**

```
BPC Web：
- 管理界面 → 预算执行报表
- 支持按公司代码/成本中心/科目/期间筛选
- 可导出 Excel/PDF

BPC Excel Plugin：
- 连接到 BPC 系统
- 可在 Excel 中查询和更新预算数据

常用报表：
- 预算执行情况（按部门/按科目）
- 预算预警清单
- 预算调整历史
```

**不覆盖：**
- 实际发生额（须从 [ERP] 获取）
- 承诺额（须从 [FK] 获取）

---

## [BK] 用友全面预算/金蝶预算管理

**覆盖范围：**
- 预算编制/调整/执行数据
- 预算审批流程
- 多维度预算分析

**查询路径：**

```
用友全面预算：
- 预算管理 → 预算执行 → 执行情况查询
- 预算管理 → 预算分析 → 差异分析
- 支持多维度（部门/项目/期间）

金蝶预算管理：
- 预算管理 → 执行分析 → 预算执行表
- 预算管理 → 预算分析 → 预算执行分析
```

---

## [ERP] SAP FI/CO 总账

**覆盖范围：**
- 费用实际发生额（按科目/成本中心）
- 成本中心实际成本
- 项目实际支出
- 人力成本数据（从 HR 系统过账）

**查询路径：**

```
SAP FI：
- FB03：显示凭证
- FBV3：试算表（按科目范围）
- FK08N：成本中心余额

SAP CO：
- KOb1：成本中心实际
- CJ30：项目预算执行

常用报表：
- 费用分析（按部门/按科目）
- 成本中心报表
- 项目成本报表
```

---

## [FK] 分贝通/胜意费控

**覆盖范围：**
- 预算占用情况（已审批/已报销/剩余）
- 承诺额（已审批但未报销）
- 预算预警记录
- 超预算审批记录

**查询路径：**

```
分贝通：
- 企业后台 → 预算管理 → 预算执行
- 企业后台 → 预警管理 → 预算预警
- 承诺额 → 待报销承诺额

胜意费控：
- 预算中心 → 预算执行查询
- 预算中心 → 承诺额分析
```

## [OA] 钉钉/飞书/企业微信

**覆盖范围：**
- 预算外申请审批记录
- 采购申请（金额/供应商）
- 合同审批记录
- 预算调整申请（如须 OA 审批）

**查询路径：**

```
钉钉：
- 管理后台 → 审批记录 → 预算相关审批

飞书：
- 管理后台 → 审批 → 预算相关审批

企微：
- 管理后台 → 审批 → 预算相关审批
```

---

## [HR] 薪酬系统/社保系统

**覆盖范围：**
- 人力成本预算执行（工资/奖金/社保）
- 各部门人力成本实际
- 年度加薪/招聘计划执行

**查询路径：**

```
HR 薪酬系统：
- 薪酬报表 → 各成本中心人力成本
- 人员变动报表 → 入离职情况

社保系统：
- 社保申报 → 月度缴费明细
- 可导出各成本中心社保缴纳情况

数据接口（ERP）：
- 薪酬数据通过接口过账到 SAP/用友
- 可在 ERP 中查询人力成本凭证
```

## 数据源组合使用

### 预算执行分析（常规）
```
[BK] 预算编制数据 → [ERP] 实际发生额 → [FK] 承诺额 → 合并生成执行率报告
```

### 超预算预警
```
[BK] 预算执行数据 + [ERP] 实时发生额 + [FK] 承诺额 → 计算剩余预算 → 超预警阈值时触发预警
```

### 预算调整分析
```
[BK] 预算调整历史 → [OA] 调整申请审批记录 → [ERP] 调整后实际执行情况
```

### 资本支出追踪
```
[OA] 采购申请/合同审批 → [ERP] 资产模块 → [BK] 资本支出预算 → 预算 vs 实际对比
```

## Connector 映射

| 数据需求 | Connector | 工具 | 关键字段 |
|---------|-----------|------|---------|
| 预算编制/执行 | Anaplan | `anaplan_model_run` | `BudgetSet`, `Department`, `Amount`, `Actual` |
| 实际费用 | SAP | `sap_gl_query` | `KOSTL`, `RACCT`, `DMBTR`, `Period` |
| 承诺额/占用 | SAP Concur | `concur_expense_list` | `report_status`, `amount`, `cost_center` |
| 人力成本 | Workday HCM | `workday_payroll_query` | `cost_center`, `gross`, `tax`, `social` |
