# 判断框架 — 预算管理异常处理

## 核心原则

**预算管理的目标是"让资源用在最重要的事情上"，不是"花了多少钱"。**
遇到超预算情形时，优先判断是"应该花的战略性超支"还是"应该避免的失控超支"。

---

## 异常类型与处理方式

### 类型一：单项费用超预算

**典型场景：**
- 咨询费超预算 30%（外部顾问临时增加）
- 差旅费超预算 20%（重要客户拜访增加）
- 广告宣传费超预算 50%（竞品压力加大）

**判断流程：**

```
第一步：判断超支性质

战略性超支（可接受）：
  - 为抓住重要业务机会而临时增加
  - 有明确的业务回报预期
  - 已获得上级口头同意

失控超支（不可接受）：
  - 无业务理由的随意超支
  - 员工为消化预算而突击花钱
  - 重复采购/不必要的花费

第二步：判断超支幅度

≤ 5%：
  → ⚠️ 须部门负责人审批，可报销
  → 附上超支说明

5-10%：
  → 🔴 须财务经理审批
  → 须提供业务理由 + 补救措施

10-20%：
  → 🔴 须财务总监审批
  → 须提交预算调整申请

> 20%：
  → 🔴 须 CEO 审批
  → 如无充分理由，不予报销
```

---

### 类型二：部门整体预算超支

**典型场景：**
- 研发部门整体超支 15%（人力成本上涨）
- 销售部门整体超支 10%（渠道费用增加）
- 市场部门整体超支 25%（新品上市推广）

**判断流程：**

```
判断维度：
1. 超支金额是否在可接受范围内（< [填空：如 10 万元]）
2. 超支是否有业务正当理由
3. 是否影响其他部门的预算执行
4. 下月是否有明确的降本措施

处理方式：

超支 < 5% 且有正当理由：
  → ⚠️ 记录在案，下期预算调整时考虑
  → 须部门负责人书面说明

超支 5-15%：
  → 🔴 冻结该部门下月非必要支出
  → 须提交整改计划

超支 > 15%：
  → 🔴 财务经理介入审查
  → 须提交详细原因分析及补救计划
  → 可能影响该部门年度绩效评分
```

### 类型三：预算执行率异常

**典型场景：**
- 执行率 < 50%（预算花不出去）
- 执行率 > 100%（超预算）
- 执行率波动剧烈（某月极低，某月极高）

**判断标准：**

| 执行率 | 原因分析 | 处理 |
|---|---|---|
| < 50% | 预算编制过大，或业务进度延迟 | ⚠️ 复盘预算假设 |
| 50-70% | 进度偏慢，可能影响年度目标 | ⚠️ 关注 Q3/Q4 执行 |
| 70-90% | 正常范围 | ✅ 无需干预 |
| 90-100% | 接近预算上限 | ⚠️ 监控下月超支风险 |
| > 100% | 超预算 | 🔴 按超预算流程处理 |

**执行率波动剧烈排查：**
```
季度内月度执行率标准差 > 15%：
  → 疑似"季度末集中花钱"
  → 检查是否有平滑花钱的动机
  → 建议优化预算分配节奏
```

---

### 类型四：预算调整申请异常

**典型场景：**
- Q1 结束即申请预算调整（说明预算编制有问题）
- 调整幅度 > 30%（调整过大）
- 频繁调整（> 2次/年）

**判断标准：**

| 异常类型 | 判断 | 处理 |
|---|---|---|
| Q1 即申请调整 | 预算编制质量差 | 🔴 不批准，要求复盘 |
| 调整幅度 > 30% | 预算基础有重大变化 | 🔴 评估后批准或否决 |
| 频繁调整 > 2次/年 | 预算管控失效 | 🔴 财务经理审查预算流程 |

---

### 类型五：资本支出预算超支

**典型场景：**
- 设备采购超预算（市场价格波动）
- 工程建设超预算（工期延长/设计变更）
- IT 项目超预算（需求变更/技术难度低估）

**判断标准：**

| 超支幅度 | 处理 |
|---|---|
| ≤ 5% | ⚠️ 须部门负责人 + 采购确认，说明原因 |
| 5-15% | 🔴 须财务总监审批 |
| 15-30% | 🔴 须 CEO 审批 + 重新评估 ROI |
| > 30% | ❌ 暂停项目，财务委员会重新审批 |

**资本支出特殊规则：**
```
资本支出超支必须追加预算，不允许从运营费用弥补
工程类资本支出须有独立的投资回报分析（IRR/NPV）
```

## 置信度评估

### 预算分析置信度

| 置信度 | 条件 | 操作 |
|---|---|---|
| 高 | 数据完整 + 差异可追溯 + 无异常值 | ✅ 报告可信 |
| 中 | 有少量数据缺失或异常值 | ⚠️ 标注不确定性 |
| 低 | 数据缺失 > 20% 或存在系统性异常 | 🔴 报告不具参考价值 |

---

## 升级触发条件

满足以下任一条件，升级至财务经理：

- 单项费用超预算 > 20%
- 部门整体超支 > 15%
- 资本支出超支 > 5%（且无合理解释）
- 收入预算低于预期 > 20%
- 预算调整申请理由不充分
- 连续 3 个月执行率异常（< 60% 或 > 110%）
- 疑似预算舞弊（执行率刚好 100% 附近，或突击花钱）
