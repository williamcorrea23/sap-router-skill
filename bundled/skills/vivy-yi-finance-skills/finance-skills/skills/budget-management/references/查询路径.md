# 查询路径 — 预算管理

## 数据源优先级

| 优先级 | 数据源 | 说明 |
|---|---|---|
| 1 | 预算管理系统（SAP BPC/用友预算/金蝶预算）| 预算编制/调整/执行数据 | 最优先 |
| 2 | ERP 总账（SAP FI/用友/金蝶）| 实际发生额 | 次优先 |
| 3 | 费控平台（分贝通/胜意）| 承诺额/已占用预算 | 承诺额查询 |
| 4 | OA 审批系统（钉钉/飞书/企微）| 预算外申请/采购申请 | 申请审批查询 |
| 5 | HR 系统 | 人力成本预算执行（工资/社保）| 人力成本核对 |

---

## 常用查询场景

### 场景 1：预算执行率查询

**查询入口：**
```
SAP BPC：
- BPC 管理界面 → 预算执行报表
- 可按公司代码/成本中心/科目筛选

用友全面预算：
- 预算管理 → 预算执行 → 执行情况查询
- 支持多维度（部门/项目/期间）

金蝶预算管理：
- 预算管理 → 执行分析 → 预算执行表
- 支持 Excel 导出
```

**执行率计算：**
```
公式：执行率 = 实际发生额 / 预算金额 × 100%

查询维度：
- 按部门：各成本中心的执行率
- 按科目：各费用科目的执行率
- 按期间：月度/季度/年度执行率
- 按项目：各项目的预算执行情况
```

---

### 场景 2：超预算预警查询

**查询入口：**
```
费控平台（如分贝通/胜意）：
- 预算预警 → 设置阈值 → 查询超预算预警
- 可查看各部门的预算预警记录

ERP + OA：
- ERP 导出预算执行表
- OA 导出待审批报销单
- 两者比对，识别超预算风险
```

**预警触发规则：**
```
预警规则（可配置）：
  - 执行率 > 80%：黄色预警
  - 执行率 > 90%：橙色预警
  - 执行率 > 100%：红色预警

预警通知：
  - 系统自动通知部门负责人
  - 超 100% 时通知财务经理
```

### 场景 3：预算调整查询

**查询入口：**
```
预算管理系统：
- SAP BPC：管理 → 预算调整记录
- 用友：预算调整 → 调整历史

OA 审批（如调整须审批）：
- 预算调整申请 → 审批记录
```

**调整类型判断：**
```
调整类型：
  - 正式调整（预算变更，须审批）
  - 期中动用（预算内科目间调剂）
  - 紧急调整（突发业务，须加急审批）

调剂规则（科目间调剂）：
  - 同一部门内调剂：部门负责人审批
  - 跨部门调剂：财务总监审批
  - 跨期间调剂（如 Q1 用 Q2 预算）：CEO 审批
```

### 场景 4：承诺额查询（已审批未报销）

**查询入口：**
```
费控平台：
- 分贝通 → 承诺额管理 → 待报销承诺额查询
- 胜意 → 预算中心 → 承诺额分析

ERP（采购订单/合同）：
- SAP ME21N：查询未清采购订单
- 用友：应付款管理 → 待结算采购订单
```

**承诺额计算：**
```
承诺额 = 已审批的采购订单/合同金额 - 已报销金额

承诺额类型：
  - 采购订单承诺额（已下订单但未收货/未报销）
  - 合同承诺额（合同约定付款但未到账）
  - 差旅预借承诺额（员工预借差旅费未报销）

承诺额对预算的影响：
  预算可用余额 = 预算金额 - 实际发生额 - 承诺额
```

### 场景 5：部门预算与实际对比

**查询入口：**
```
ERP 总账：
- SAP：FK08N 按成本中心查询科目余额
- 用友：总账 → 部门辅助账

BPC（预算系统）：
- 按成本中心导出预算 vs 实际对比表
```

**对比分析维度：**
```
对比维度：
  - 预算 vs 实际（绝对金额）
  - 执行率（相对）
  - 差异金额 = 实际 - 预算
  - 差异率 = (实际 - 预算) / 预算 × 100%

重点关注：
  - 超预算 > 10% 的部门/科目
  - 执行率 < 60% 或 > 110% 的异常项
  - 同比差异（与去年同月对比）
```

---

### 场景 6：资本支出预算执行

**查询入口：**
```
ERP 固定资产模块：
- SAP AW01N：资产主记录
- SAP ANEP：资本支出计划 vs 实际

合同管理系统（如有）：
- 查询已签合同/已付工程款
```

**资本支出分析：**
```
分析维度：
  - 预算 vs 实际（已付款 + 已审批未付 + 承诺额）
  - 项目进度 vs 预算进度
  - 单项工程/设备的预算偏差

特殊关注：
  - 在建工程长期未转固（> 1 年）
  - 资本支出预算与实际差异 > 15%
  - 追加预算申请（资本支出超支处理）
```

## 查询结果处理

### 数据完整性判断

```
数据完整（所有系统可用）：
→ 高置信度，可直接生成预算执行报告

数据部分缺失（部分系统不可用）：
→ 中置信度，用 Excel 手工补充缺失数据
→ 报告中标注数据来源

数据不可用（所有系统不可用）：
→ 低置信度，暂停预算分析
→ 使用上次报告数据 + 手工台账估算
→ 标注数据截止时间和估算方法
```
