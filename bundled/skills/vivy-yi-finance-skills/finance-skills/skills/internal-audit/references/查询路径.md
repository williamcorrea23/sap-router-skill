# 查询路径 — 内部审计（无 Connector 时）

## 手动流程说明

当 ERP/专业审计系统不可用时，按以下步骤手动完成内部审计任务。

---

## 步骤一：审计计划制定

```
□ 从 CLAUDE.md 获取本年度审计范围和重点
□ 收集被审计单位基本信息（组织架构/业务流程）
□ 执行风险评估，确定审计重点
□ 编制审计方案（含时间表/人员安排）
□ 输出：年度审计计划 / 单项审计方案
```

---

## 步骤二：现场审计执行

```
□ 访谈：
□ 了解业务流程和控制环境
□ 访谈对象：财务经理/业务负责人/关键岗位
□ 记录访谈纪要

□ 文件审查：
□ 抽样检查凭证/合同/审批记录
□ 核对账实是否一致
□ 检查审批流程是否合规

□ 数据分析：
□ 导出交易数据进行分析
□ 筛选异常交易
□ 输出：测试底稿和发现清单
```

---

## 步骤三：审计发现沟通

```
□ 编制审计发现初稿
□ 与被审计单位管理层沟通（须在报告前）
□ 收集整改承诺和计划
□ 确认事实认定的准确性
□ 输出：审计发现沟通纪要
```

---

## 步骤四：审计报告撰写

```
□ 按审计报告格式撰写：
□ 背景/范围/方法
□ 审计发现（按严重程度排列）
□ 管理层整改意见
□ 附件（测试底稿/数据表）

□ 报告分发：
□ 被审计单位
□ 审计委员会
□ CFO（如涉及重大发现）
□ 输出：正式审计报告
```

---

## 步骤五：整改跟踪

```
□ 建立整改跟踪台账
□ 定期（每月/每季）跟进整改进度
□ 验证整改效果（文件审查/实地复查）
□ 报告审计委员会整改完成情况
□ 输出：整改验证报告
```
