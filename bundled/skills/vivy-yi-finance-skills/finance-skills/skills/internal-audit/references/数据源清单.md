# 数据源清单 — 内部审计

## 审计数据需求与来源

### 财务与运营数据

| 数据项 | 用途 | 首选数据源 | 备选数据源 |
|--------|------|-----------|-----------|
| 会计凭证/账簿 | 交易真实性验证 | ERP 总账 | 财务共享中心 |
| 银行对账单 | 资金流水核查 | 资金系统/网上银行 | 银行回单 |
| 采购订单/合同 | 采购完整性 | SRM 系统 | 合同管理系统 |
| 销售订单/出库单 | 收入完整性 | CRM/ERP 销售模块 | 销售报表 |
| 库存记录 | 存货真实性 | WMS/ERP | 仓库台账 |
| 合同台账 | 关联方/担保 | 法务合同系统 | Excel 台账 |

| 合规政策文档 | 合规性对照 | 法务/合规部 | 制度汇编 |
