# 数据源清单 — [TAX]/[INV]/[ERP]/[HR]

## 数据源标签说明

| 标签 | 含义 | 典型来源 |
|---|---|---|
| [TAX] | 电子税务局 | 国家税务总局/各省电子税务局 |
| [TAX-VAT] | 增值税发票管理系统 | 增值税发票综合服务平台（勾选平台）|
| [TAX-GOLD] | 金税系统 | 增值税风险监控系统 |
| [INV] | 开票系统 | 百望/航天信息/金税盘/税盘 |
| [ERP] | 企业 ERP 系统 | SAP/用友/金蝶总账 |
| [HR] | HR 薪酬系统 | 薪酬管理系统/社保公积金系统 |

---

## [TAX] 电子税务局

**覆盖范围：**
- 增值税申报（一般纳税人/小规模纳税人）
- 企业所得税申报（预缴/年度汇算）
- 个人所得税代扣代缴
- 附加税（城建税/教育费附加/地方教育附加）申报
- 印花税申报
- 社保费申报
- 完税证明开具
- 税收优惠备案

**查询路径：**

```
电子税务局首页：
- 我要办税 → 税费申报及缴纳
- 申报查询 → 查看已申报记录
- 缴款查询 → 查看已缴纳税款记录
- 证明开具 → 完税证明/纳税证明

常用报表：
- 增值税及附加税费申报表
- 企业所得税月（季）度申报表
- 关联申报表（如有跨境关联交易）
```

**不覆盖：**
- 发票明细数据（须从 [INV] 获取）
- 进项发票认证状态（须从 [TAX-VAT] 获取）

---

## [TAX-VAT] 增值税发票管理系统（勾选平台）

**覆盖范围：**
- 进项发票勾选认证
- 进项税额统计
- 发票状态查询（正常/作废/红冲/失控/异常）
- 异常凭证记录
- 滞留票信息

**查询路径：**

```
增值税发票综合服务平台：
- 抵扣勾选 → 发票抵扣勾选/统计
- 抵扣记录 → 查看历史抵扣记录
- 异常凭证 → 查看被认定为异常凭证的发票
- 信息发送 → 查看税务机关发送的各类通知

功能说明：
- 抵扣期限：自开票之日起 360 天内须勾选认证
- 逾期抵扣：须向税务机关申请逾期抵扣审批
```

---

## [TAX-GOLD] 金税系统（风险监控）

**覆盖范围：**
- 增值税风险预警（税负率/进销项/异常发票）
- 企业所得税风险监控（收入成本配比/期间费用异常）
- 发票风险（失控票/异常凭证/虚开发票）
- 纳税信用评级
- 协查函接收

**查询路径：**

```
金税系统（纳税人端）：
- 风险监控 → 风险预警处理
- 发票管理 → 异常发票查询
- 协查管理 → 收到协查函处理

常见风险预警：
- 增值税税负率偏低
- 进项税额长期大于销项税额
- 农产品收购发票金额异常
- 旅客运输发票抵扣异常
- 关联销售额与申报收入差异大
```

## [INV] 开票系统（百望/航天信息）

**覆盖范围：**
- 销项发票开具（增值税专用发票/普通发票/电子发票）
- 销项发票作废/红冲
- 销项发票统计（按税率/期间/客户）
- 发票作废率监控
- 库存发票数量

**查询路径：**

```
百望云：
- 发票管理 → 销项发票查询
- 统计报表 → 按月/按季度汇总
- 支持按税率/客户/商品名称筛选

航天信息（诺言）：
- 发票管理 → 发票查询
- 汇总统计 → 报表明细

常用报表：
- 销项发票汇总表（月度）
- 销项发票明细表（逐张）
- 作废发票统计
```

## [ERP] SAP FI / 用友 / 金蝶总账

**覆盖范围：**
- 应交税费明细账（各税种余额）
- 税金计提分录（每月计提凭证）
- 税金缴纳分录（缴税凭证）
- 所得税费用明细
- 递延所得税资产/负债

**查询路径：**

```
SAP FI：
- FB03：显示凭证
- FS10N：科目余额（应交税费各明细科目）
- FBL5N：供应商发票（待开发票）

用友：
- 总账 → 科目余额表 → 筛选应交税费
- 总账 → 明细账 → 查看各税种变动

金蝶：
- 科目余额表 → 筛选应交税费
- 明细分类账 → 查看各税种发生额

常用科目：
- 应交税费 — 应交增值税（进项税额/销项税额/已交税金/转出未交增值税）
- 应交税费 — 应交企业所得税
- 应交税费 — 应交个人所得税
- 应交税费 — 应交城市维护建设税
- 所得税费用 — 当期所得税
```

## [HR] 薪酬系统

**覆盖范围：**
- 员工工资薪金明细
- 社保公积金缴纳明细（个人部分/单位部分）
- 专项附加扣除信息采集
- 实习生/非全职员工薪金
- 股息红利/稿酬/特许权使用费等分类所得

**查询路径：**

```
薪酬系统：
- 工资明细 → 当月税前工资/社保公积金/个税
- 社保公积金 → 月度缴纳明细
- 个税申报 → 推送至自然人电子税务局

数据接口：
- 薪酬系统 → [ERP] 过账（工资凭证）
- 薪酬系统 → [TAX] 推送个税申报数据
```

## 数据源组合使用

### 增值税申报准备
```
[INV] 销项发票数据 + [TAX-VAT] 进项发票勾选数据 + [ERP] 应交增值税余额
→ 核对申报表销项/进项/应纳税额
```

### 企业所得税预缴申报
```
[ERP] 利润表（累计）+ [TAX] 已预缴数据 + [ERP] 应交企业所得税余额
→ 核对申报表实际利润/已预缴/应补（退）税
```

### 个税代扣代缴
```
[HR] 工资明细 + [HR] 社保公积金 + [HR] 专项附加扣除
→ 生成个税申报数据，推送至 [TAX]
```

### 税务风险自查
```
[TAX-GOLD] 风险预警清单 + [TAX-VAT] 异常凭证记录 + [INV] 销项发票统计
→ 逐项核查风险，合理说明或整改
```

## Connector 映射

| 数据需求 | Connector | 工具 | 关键字段 |
|---------|-----------|------|---------|
| 税务申报 | TaxDome | `taxdome_client_portal` | `client_id`, `filing_type`, `status` |
| 增值税发票 | Vertex O系列 | `vertex_quote_request` | `tax_code`, `jurisdiction`, `amount` |
| ERP 总账 | SAP | `sap_gl_query` | `RACCT`（应交税费明细）, `KOSTL`, `BELNR` |
| 薪酬数据 | Workday HCM | `workday_payroll_query` | `employee_id`, `gross`, `tax_withheld` |
| 风险预警 | 金税系统 | 手动登录 | 风险等级/税负率/进销项配比 |
