# 查询路径 — 月末结账

## 数据源优先级

| 优先级 | 数据源 | 说明 |
|---|---|---|
| 1 | ERP 总账模块（SAP FI/用友总账）| 凭证/余额表/报表 | 最优先 |
| 2 | ERP 子模块（AP/AR/FA/CO）| 往来核销/资产折旧/成本 | 次优先 |
| 3 | 费控平台（分贝通/胜意）| 未报销费用预提 | 费用预提用 |
| 4 | 银行网银/银企直连 | 银行对账单 | 银企对账用 |
| 5 | 影像管理系统 | 凭证附件影像 | 抽查核实用 |

## 常用查询场景

### 场景 1：试算表不平衡排查

**查询入口：**
```
SAP：
- FBV3：试算表（按科目区间）
- F.28：综合演练算（总账科目范围）
- FB03：逐笔查询凭证

用友 U8：
- 总账 → 凭证 → 试算平衡表
- 总账 → 账表 → 科目余额表

金蝶 K3：
- 总账 → 试算表
- 总账 → 凭证查询
```

**排查步骤：**
```
第一步：导出当月所有凭证汇总（按科目借方+贷方合计）
第二步：计算每个科目的期末余额 = 期初余额 + 借方合计 - 贷方合计
第三步：将计算余额与系统余额表对比
第四步：差异最大的科目即为问题科目
第五步：导出该科目所有凭证，逐一核对借贷方向
```

### 场景 2：银行对账

**查询入口：**
```
银企直连（实时）：
- SAP：F.28 或 FK02N 查询银行账户余额
- 用友：应收款管理 → 银行对账

网银导出（离线）：
- 各银行企业网银导出当月银行对账单（CSV/Excel）
- 对比银行系统余额与企业账面余额
```

**银企对账步骤：**
```
第一步：从 ERP 导出银行账户当月所有付款凭证
第二步：从网银导出当月银行对账单
第三步：在 Excel 中匹配：
  - 企业已付且银行已付 → 已达账（标记 ✅）
  - 企业已付但银行未付 → 银行未达（标记 ⚠️）
  - 银行已付但企业未付 → 企业未达（标记 ⚠️）
  - 双方均无 → 疑似错误（标记 ❌）
第四步：汇总银行调节表
第五步：差异为 0 → 对账完成
  差异 > 0 → 排查未达账项
```

### 场景 3：往来核销查询（AP/AR）

**查询入口：**
```
SAP AP：
- FK01N：供应商行项目（显示未清项）
- F-44：手动清账
- F-53：付款清账

SAP AR：
- FD10N：客户行项目
- F-28：收款清账

用友 AP：
- 应付款管理 → 供应商清账
- 应付款管理 → 对账单

用友 AR：
- 应收款管理 → 客户清账
- 应收款管理 → 账龄分析
```

**核销判断标准：**
```
正常清账：
- 供应商应付金额 = 付款金额（精确匹配）
- 客户应收金额 = 收款金额（精确匹配）

部分清账：
- 应付/应收 > 付款/收款 → 仍有余额 → 保留在未清项
- 应付/应收 < 付款/收款 → 多付/多收 → 退回或转入其他应付款

长期未清（> 90 天）：
→ 触发催办流程
→ 标注账龄
→ 财务经理评估是否提坏账准备
```

### 场景 4：固定资产折旧检查

**查询入口：**
```
SAP FA：
- AW01N：资产主记录
- AFABN：折旧运行
- ABLN：资产价值的变化

用友：
- 固定资产 → 计提折旧 → 折旧清单
- 固定资产 → 变动清单

金蝶：
- 固定资产 → 折旧清单
- 固定资产 → 变动记录
```

**检查点：**
```
检查1：本月新增资产是否已录入系统并计提折旧
  → 新增资产当月不折旧（系统默认）
  → 检查是否正确

检查2：本月报废/处置资产是否已做处置凭证
  → 报废资产不再计提折旧
  → 检查是否正确

检查3：折旧金额与上月对比是否有异常波动（±10% 以上）
  → 如有，排查资产原值/折旧年限是否有变动
```

### 场景 5：费用预提检查

**查询入口：**
```
费控平台（如有）：
- 分贝通 → 费用预提 → 查看已预提项目
- 胜意费控 → 预提管理

手工台账（无费控系统时）：
- 从 OA 导出当月所有未完成审批的报销单
- 识别属于本期费用但报销未完成的
- 计算预提金额
```

**预提标准项目：**
```
□ 当月已发生工资（次月发放，本月须预提）
□ 社会保险（当月计提）
□ 住房公积金（当月计提）
□ 奖金/绩效（已计提未发放）
□ 房租/物业（当月应付未付）
□ 水电费（当月耗用，收到账单后付款）
□ 咨询费/律师费（已服务，未开票）
□ 差旅费（员工已出差，报销未提交）
```

### 场景 6：报表钩稽验证

**查询入口：**
```
ERP 报表模块：
- SAP：S_ALR_87012341（资产负债表）
- SAP：S_ALR_87012342（利润表）
- SAP：S_P99_41000021（现金流量表）

用友：
- UFO1：自定义报表
- UFO2：财务报表（资产负债表/利润表）

金蝶：
- 报表平台：资产负债表/利润表/现金流量表
```

**标准钩稽检查：**
```
检查1：资产负债表平衡
  资产总计 = 负债总计 + 所有者权益总计

检查2：未分配利润勾稽
  期末未分配利润 = 期初未分配利润 + 本期净利润 - 本期分红

检查3：现金勾稽
  资产负债表货币资金 = 现金流量表现金及等价物期末余额

检查4：所有者权益变动勾稽
  期末所有者权益 = 期初 + 本期增碱变动合计
```

## 查询结果处理

### 情形一：数据完整且一致
→ 高置信度，结账可继续

### 情形二：数据有缺失或格式问题
→ 中置信度，标注数据来源，手工补充

### 情形三：数据矛盾
→ 低置信度，停止结账，排查数据源

### 情形四：系统无法导出
→ 降级为手工台账，延长结账时间
