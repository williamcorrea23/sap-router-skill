# 数据源清单 — [ERP]/[OA]/[FK]/[BANK]/[TAX]/[IMG]

## 数据源标签说明

| 标签 | 含义 | 典型来源 |
|---|---|---|
| [ERP] | 企业 ERP 系统 | SAP/用友/金蝶 |
| [OA] | OA 审批系统 | 钉钉/飞书/企微 |
| [FK] | 费控平台 | 分贝通/胜意/Coupa |
| [BANK] | 银行系统 | 网银/银企直连 |
| [TAX] | 税务系统 | 税局客户端/电子税务局 |
| [IMG] | 影像管理系统 | 凭证影像/发票影像 |

## [ERP] SAP FI/CO 模块

**覆盖范围：**
- 总账凭证（所有会计分录）
- 科目余额表/试算表
- 资产负债表/利润表/现金流量表
- 供应商主数据及应付账款行项目
- 客户主数据及应收账款行项目
- 固定资产主数据及折旧记录
- 成本中心/项目实际与计划对比

**查询路径：**

```
SAP FI：
- FB03：显示凭证
- FBV3：试算表（按科目范围）
- F.28：综合演练算
- FK01N：供应商行项目（未清项）
- FD10N：客户行项目（未清项）
- FBL1N：供应商行项目明细
- FBL5N：客户行项目明细

SAP CO：
- KOl1：成本中心计划
- KOb1：成本中心实际
- KB11N：手动预算过账
- CJ30：项目预算执行

SAP FA：
- AW01N：资产主记录
- AFABN：折旧运行
- ABLN：资产价值变化
```

**不覆盖：**
- OA 审批流状态（须从 [OA] 获取）
- 发票影像（须从 [IMG] 获取）
- 银行交易明细（须从 [BANK] 获取）

## [ERP] 用友 U8 / U9

**覆盖范围：**
- 总账模块：凭证/余额表/报表
- 应收款管理：客户应收/收款/核销
- 应付款管理：供应商应付/付款/核销
- 固定资产：折旧计提/变动
- 成本管理：费用分摊/成本计算

**查询路径：**

```
用友 U8：
- 总账 → 凭证 → 试算平衡表
- 总账 → 账表 → 科目余额表
- 总账 → UFO1 自定义报表
- 应收款管理 → 客户对账单
- 应付款管理 → 供应商对账单
- 固定资产 → 折旧清单
- 固定资产 → 变动清单
```

## [ERP] 金蝶 K3 / 云星空

**覆盖范围：**
- 总账：凭证/余额表/财务报表
- 应收款管理：客户往来/核销/账龄
- 应付款管理：供应商往来/核销/账龄
- 固定资产：折旧/变动
- 预算管理：预算执行对比

**查询路径：**

```
金蝶 K3：
- 总账 → 试算表
- 总账 → 凭证查询
- 应收款管理 → 往来对账单
- 应付款管理 → 往来对账单
- 固定资产 → 折旧清单
- 报表平台 → 资产负债表/利润表
```

## [OA] 钉钉/飞书/企业微信

**覆盖范围：**
- 差旅申请记录（用于核对差旅预提）
- 费用报销审批状态（当月报销是否已关闭）
- 合同审批记录（用于核对采购入账）

**查询路径：**

```
钉钉：
- 管理后台 → 审批记录 → 筛选日期范围/状态

飞书：
- 管理后台 → 审批 → 同上

企微：
- 管理后台 → 审批 → 同上
```

---

## [FK] 分贝通/胜意费控

**覆盖范围：**
- 当月已申请但未完成报销的费用（预提数据）
- 各部门的预算占用情况
- 差旅标准校验记录

**查询路径：**

```
分贝通：
- 企业后台 → 费用预提 → 当月预提明细
- 企业后台 → 预算管理 → 按部门/项目

胜意费控：
- 预算中心 → 预算查询 → 预提数据导出
```

---

## [BANK] 银行网银/银企直连

**覆盖范围：**
- 各银行账户的当期交易明细
- 银行对账单（用于银企对账）
- 银行承兑汇票记录

**查询路径：**

```
银企直连（实时）：
- SAP FK08N：显示银行账户
- 用友：银企直连 → 银行对账

网银导出（离线）：
- 各银行企业网银后台导出 CSV/Excel
- 对比企业账面与银行对账单
```

---

## [TAX] 税局系统/电子税务局

**覆盖范围：**
- 增值税申报数据（销售额/进项税额/应纳税额）
- 企业所得税预缴数据
- 个人所得税代扣代缴数据
- 社保/公积金缴纳数据

**查询路径：**

```
国家税务总局电子税务局：
- https://etax.chinatax.gov.cn
- 增值税申报表（按月）
- 企业所得税预缴申报表（按季）

本地税局客户端（如有）：
- 增值税申报查询
- 企业所得税预缴查询
```

## 数据源组合使用

### 常规月度结账
```
[ERP 总账] 试算表 → [ERP AP/AR] 往来核销 → [ERP FA] 折旧 → [FK] 预提 → [BANK] 银企对账 → [TAX] 申报核对 → [IMG] 凭证影像抽查
```

### 报表钩稽检查
```
[ERP] 导出三表 → 执行钩稽公式 → 如不一致：[ERP] 逐凭证排查 → 定位差异科目 → [IMG] 抽查该科目凭证附件
```

### 银行对账
```
[BANK] 导出对账单 → [ERP] 导出银行凭证 → Excel 匹配 → 生成银行调节表 → 差异 = 0 ✅ / 差异 > 0 排查
```
