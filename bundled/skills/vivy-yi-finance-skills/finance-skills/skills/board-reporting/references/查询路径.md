# 查询路径 — 董事会汇报

## 查询入口

### 查询维度一：Board Deck 数据

| 查询目标 | 系统入口 | 查询路径 | 刷新频率 |
|---|---|---|---|
| 财务业绩数据 | [BI 系统] | Board 视图 → 核心指标 | 每日 |
| 现金流数据 | [ERP/司库] | 资金视图 → 现金流仪表板 | 每日 |
| 预算vs实际 | [预算系统] | 预算管理 → 预算执行跟踪 | 每日 |

### 查询维度二：辅助信息

| 查询目标 | 系统入口 | 查询路径 | 刷新频率 |
|---|---|---|---|
| 历史 Board Deck | [文档库 DOC] | 董事会材料 → 历史 Deck | 实时 |
| 审计报告 | [审计系统 AUD] | 审计工作台 → 审计报告 | 实时 |
| 监管合规状态 | [合规系统 CMP] | 合规仪表板 → 整体状态 | 实时 |

## 常用查询组合

**场景：准备 Board Deck**
```
1. [BI 系统] → Board 视图 → 提取核心财务数据
2. [ERP] → 查询业务明细 → 补充业务背景
3. [合规系统 CMP] → 查询合规状态 → 补充合规信息
4. [BI 系统] → 历史趋势 → 提取历史数据对比
5. [文档库 DOC] → 历史 Deck → 参考历史格式
```

**场景：审计委员会问询回复**
```
1. [审计系统 AUD] → 查询审计发现 → 准备回复材料
2. [BI 系统] → 查询相关数据 → 准备数据支撑
3. [合规系统 CMP] → 查询整改进展 → 准备整改进度
```
