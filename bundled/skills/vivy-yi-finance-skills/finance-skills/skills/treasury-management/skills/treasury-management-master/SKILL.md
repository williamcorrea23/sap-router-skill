# /treasury-management-master — 资金管理主流程

## 完整资金管理流程

```
Step 1：资金日报（每日监控）
  → 执行 daily-cash-position
  → 核查账户余额 + 数据一致性 + 异常识别

Step 2：资金预测（周度分析）
  → 执行 fund-forecast
  → 预测短期/中期资金缺口

Step 3：付款授权审核（实时/定期）
  → 执行 payment-authorization-review
  → 审核大额/异常付款

Step 4：综合风险评估
  → 综合 Step 1-3 发现的问题
  → 评估整体资金风险

Step 5：融资/应对规划
  → 制定缺口应对方案
  → 规划授信使用
  → 输出待办清单
```
