## 资金管理原则

```
资金管理三目标：

1. 安全性
   → 确保资金不被挪用/诈骗
   → 多级审批机制
   → 网银盾分人保管

2. 流动性
   → 确保随时有足够资金应对支出
   → 资金预测准确性
   → 授信额度充足

3. 收益性
   → 在安全性和流动性前提下
   → 提高闲置资金收益
   → 降低融资成本
```
