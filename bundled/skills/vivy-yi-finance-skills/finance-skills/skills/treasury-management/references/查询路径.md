# 查询路径 — 资金管理

## 数据源优先级

| 优先级 | 数据源 | 说明 |
|---|---|---|
| 1 | 资金系统（[CASH]）| 账户余额/调拨/付款 | 最优先 |
| 2 | 网上银行（[BANK]）| 实时余额/流水 | 实时核对 |
| 3 | ERP 总账（[ERP]）| 账务核对 | 账务一致性 |
| 4 | OA 审批系统（[OA]）| 付款审批 | 合规控制 |

## 数据源组合使用

### 每日资金核对
```
[BANK] 实时余额 → [CASH] 账户余额 → 对比差异 → 识别未达账项
```

### 资金预测（周度）
```
[ERP] 应收应付到期 → [OA] 待付款申请 → [BK] 预算支出 → [CASH] 预测模型 → 生成预测
```

### 流动性预警
```
[CASH] 实时余额 → [BANK] 明日到期 → [CASH] 预测 → 缺口预警 → 升级
```
