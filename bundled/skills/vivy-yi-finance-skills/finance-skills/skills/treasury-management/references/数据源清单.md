# 数据源清单 — [CASH]/[BANK]/[ERP]/[OA]

## 数据源标签说明

| 标签 | 含义 | 典型来源 |
|---|---|---|
| [CASH] | 资金系统/司库 | 司库系统/资金管理系统/用友资金/金蝶资金 |
| [BANK] | 网上银行 | 企业网银/银企直联 |
| [ERP] | 企业 ERP 系统（总账）| SAP/用友/金蝶总账 |
| [OA] | OA 审批系统 | 泛微/钉钉/飞书/用友 OA |

---

## [CASH] 司库系统 / 资金管理系统

**覆盖范围：**
- 银行账户余额（多银行汇总）
- 内部调拨指令
- 付款指令
- 资金预测模型
- 授信额度管理
- 外汇敞口

**查询路径：**

```
司库系统：
- 账户管理 → 账户余额汇总
- 付款指令 → 付款状态跟踪
- 资金预测 → 未来资金预测
- 授信管理 → 银行授信额度
- 外汇管理 → 外汇敞口/合约

用友/金蝶资金：
- 资金日报
- 账户余额
- 调拨记录
- 预测模型
```

---

## [BANK] 网上银行 / 银企直联

**覆盖范围：**
- 单个银行账户实时余额
- 银行流水（交易明细）
- 付款指令状态（已提交/银行处理中/已完成）
- 授信额度（银行端）
- 电子回单

**查询路径：**

```
企业网银：
- 账户管理 → 账户余额
- 交易明细 → 流水查询
- 付款管理 → 指令状态

银企直联：
- 实时余额查询
- 自动付款指令
- 电子回单自动归档
```

---

## [ERP] SAP / 用友 / 金蝶（总账）

**覆盖范围：**
- 现金/银行存款科目余额
- 应收账款明细（客户维度）
- 应付账款明细（供应商维度）
- 外币汇兑损益
- 利息收入/支出

**查询路径：**

```
SAP 常用事务代码：
- FAGLB03：总账科目余额
- FBL5N：客户行项目（应收账款）
- FBL1N：供应商行项目（应付账款）
- FB03：显示凭证
- F-53：付款凭证

用友/金蝶：
- 总账 → 现金/银行存款
- 应收款管理 → 客户应收明细
- 应付款管理 → 供应商应付明细
```

---

## [OA] 泛微 / 钉钉 / 飞书 / 用友 OA

**覆盖范围：**
- 付款审批流程
- 内部调拨审批
- 合同审批（付款条款）
- 费用报销审批

**查询路径：**

```
OA 系统：
- 审批流程 → 待我审批
- 付款申请 → 申请记录
- 合同管理 → 合同付款条款
- 历史审批 → 已完成审批

钉钉/飞书：
- 审批 → 付款审批
- 审批 → 差旅报销
```

## 数据源组合使用

### 每日资金核对
```
[BANK] 实时余额 → [CASH] 账户余额 → 对比 → 识别未达账项 → 调账
```

### 资金预测（周度）
```
[OA] 待付款申请 → [ERP] 应收应付到期 → [BK] 预算支出 → [CASH] 预测 → 输出预测报告
```

### 大额付款审核
```
[OA] 付款申请 → [ERP] 合同/订单 → [CASH] 账户余额 → [BANK] 授信 → 综合判断
```

### 流动性预警
```
[CASH] 实时余额 → [OA] 即将到期付款 → [ERP] 大额应收 → [BANK] 授信可用 → 预警判断
```

## Connector 映射

| 数据需求 | Connector | 工具 | 关键字段 |
|---------|-----------|------|---------|
| 银行余额/流水 | Kyriba | `kyriba_cash_positions` | `account_id`, `balance`, `currency` |
| 付款指令 | Kyriba | `kyriba_payment_submit` | `payment_id`, `amount`, `beneficiary` |
| 现金流预测 | Kyriba | `kyriba_forecast` | `date`, `inflows`, `outflows`, `balance` |
| ERP 总账 | SAP | `sap_gl_query` | `RACCT`（现金/银行存款）, `KOSTL` |
| OA 审批 | 飞书/钉钉 | MCP HTTP | 审批流 ID/状态/金额 |
| 外汇敞口 | Bloomberg | `bloomberg_fx_rate` | `currency_pair`, `spot_rate`, `fwd_points` |
