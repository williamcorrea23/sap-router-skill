# /master-document-preparation — 同期资料文档撰写

## 主体文档（Master File）框架

```
一、集团组织架构
  □ 股权结构图（含各关联方持股比例）
  □ 集团业务概述
  □ 主要业务线分布

二、价值链分析
  □ 各成员企业在价值链中的位置
  □ 关键价值驱动因素
  □ 集团内部协作模式

三、关联交易概况
  □ 关联交易类型汇总
  □ 各类型交易金额占比
  □ 主要关联方交易明细

四、功能、风险与资产分布
  □ 各成员企业承担的功能（研发/制造/销售/服务）
  □ 各成员企业承担的风险（市场风险/信用风险/运营风险）
  □ 各成员企业使用的有形/无形资产

五、可比性分析摘要
  □ 筛选标准
  □ 可比公司数量
  □ 利润水平指标区间与实际值对比

六、转让定价政策
  □ 各类型交易的 TP 方法选择
  □ 定价原则
  □ 内部协议框架
```

---

## 本地文档（Local File）框架

```
一、企业基本信息
  □ 企业名称、注册地、主营业务
  □ 组织架构（本地）
  □ 关联关系（母公司/子公司持股情况）

二、关联交易详情
  □ 关联交易类型
  □ 交易金额（按关联方/类型细分）
  □ 关联交易合同摘要

三、可比性分析
  □ 筛选标准与过程
  □ 可比公司信息
  □ 财务指标计算与调整
  □ 利润水平对比

四、转让定价计算
  □ 各关联交易的具体定价计算
  □ 方法选择依据
  □ 调整（如有）

五、功能风险分析与收益分配
  □ 企业承担的功能
  □ 企业承担的风险
  □ 收益分配与功能风险匹配性评估
```

## 常见文档缺陷预警

```
⚠️ 常见问题：
□ 可比公司 < 3 家 → 须补充或说明理由
□ 方法选择无依据 → 须补充方法论说明
□ 财务数据与审计报告不一致 → 须核对调整
□ 功能风险分布与实际情况不符 → 须重新评估
□ 关联交易金额与申报表不一致 → 须追溯调整

🔴 高风险问题：
□ 主体文档与本地文档数据矛盾
□ 可比性分析使用了非独立第三方数据
□ 未披露重要的关联方交易
□ 利润水平持续偏离警戒线且无合理解释
```
