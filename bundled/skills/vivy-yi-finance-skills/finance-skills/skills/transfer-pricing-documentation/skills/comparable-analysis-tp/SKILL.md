# /comparable-analysis-tp — 可比性分析

## 第一步：确认分析范围

```
□ 关联交易类型：
□ 有形资产销售 → 分析毛利率
□ 服务提供 → 分析成本加成率
□ 无形资产许可 → 分析利润分割率
□ 金融交易 → 分析利率区间

□ 分析期间：[最近 X 年]
□ 可比公司数量要求：≥ [X] 家（越多越好，建议 ≥ 5 家）
```

---

## 第二步：可比公司筛选

```
□ 筛选标准（从 CLAUDE.md 读取）：
□ 行业：[X]
□ 地理位置：[X]
□ 财务数据年份：[最近 X 年]

□ 筛选步骤：
□ 初筛：按行业分类找到潜在可比公司
□ 再筛：剔除财务数据异常公司（亏损/极端比率）
□ 复核：确认剩余公司的功能风险与被评估企业可比

□ 筛选结果：
□ 初筛数量：[X] 家
□ 最终可比公司：[X] 家
□ 剔除原因：[...]
```

---

## 第三步：财务指标计算

```
□ 毛利率（适用于有形资产销售）：
□ 可比公司区间：[X]% - [X]%
□ 企业实际值：[X]%
□ 偏离幅度：[±X]%
□ 风险评估：[✅ 在区间内 / ⚠️ 偏离 / 🔴 严重偏离]

□ 营业利润率（适用于服务提供）：
□ 可比公司区间：[X]% - [X]%
□ 企业实际值：[X]%
□ 偏离幅度：[±X]%
□ 风险评估：[✅ 在区间内 / ⚠️ 偏离 / 🔴 严重偏离]

□ 完全成本加成率（适用于服务提供）：
□ 可比公司区间：[X]% - [X]%
□ 企业实际值：[X]%
□ 偏离幅度：[±X]%

□ 资产报酬率（适用于无形资产/金融交易）：
□ 可比公司区间：[X]% - [X]%
□ 企业实际值：[X]%
□ 偏离幅度：[±X]%
```

---

## 第四步：差异调整

```
□ 是否须做差异调整：[是/否]
□ 差异类型：
□ 地理市场差异 → 已调整 / 不须调整
□ 产品组合差异 → 已调整 / 不须调整
□ 规模差异 → 已调整 / 不须调整
□ 生命周期差异 → 已调整 / 不须调整

□ 调整方法：[描述]
□ 调整后指标区间：[X]% - [X]%
□ 企业实际值：[X]% — 是否在调整后区间内：[是/否]
```

---

## 第五步：风险判断

```
□ 利润水平指标风险：
□ ✅ 低风险：企业值在可比公司 [X]% - [X]% 区间内
□ ⚠️ 中风险：企业值偏离区间 [X]% 以内，须说明
□ 🔴 高风险：企业值偏离区间超过 [X]%，须调整定价或补充分析

□ 方法选择合理性：
□ ✅ 合理：方法与交易类型匹配，有充分依据
□ ⚠️ 存疑：方法基本合理，但可比公司不足
□ 🔴 不当：方法选择无依据或与交易本质不符

□ 综合结论：
□ [一句话描述整体风险水平]
□ [具体建议：调整定价/补充分析/申请 APA/维持现状]
```

## 升级触发条件

- 可比公司数量 < 3 家（须说明理由并获 TP 负责人批准）
- 利润水平严重偏离（超出警戒线 [X]% 以上）
- 无可比公司须用成本加成法（须额外说明）
