# /beps-compliance — BEPS 行动 13 合规检查

## BEPS 行动 13 三层结构

```
Layer 1 — 主体文档（Master File）
  目标：跨国集团整体转让定价政策
  受众：集团各税务机关
  内容：组织架构/价值链/集团整体政策/无形资产/财务/可比性

Layer 2 — 本地文档（Local File）
  目标：本企业关联交易详情
  受众：本地税务机关
  内容：企业信息/关联交易/本地可比性分析/定价计算

Layer 3 — 国别报告（CbCR）
  目标：各管辖区汇总数据
  受众：集团最终母公司所在国税务机关（交换给相关管辖区）
  内容：各成员实体收入/利润/税款/员工数/资产/纳税
```

## 升级触发条件

- 须准备的文档层级超出原计划
- 主体文档与本地文档数据不一致
- CbCR 数据无法与集团合并报表核对
- 税务机关就 BEPS 合规提出质疑
