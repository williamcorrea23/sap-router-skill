### 1.3 同期资料三件套（2016 年 42 号公告）

中国自 2016 年起引入"三件套"制度：

| 文档 | 适用条件 | 核心内容 | 准备时点 |
|------|---------|---------|---------|
| **主体文档（Master File）** | 集团合并收入 > 50 亿元 | 集团组织、价值链、TP 政策 | 6 月 20 日前 |
| **本地文档（Local File）** | 单边关联交易 > 2 亿元 | 企业功能/风险/资产、关联交易 | 汇算清缴时 |
| **国别报告（CbCR）** | 集团合并收入 > 55 亿元 | 国别收入、税前利润、所得税、员工数 | 12 个月内 |

**中国本地文档阈值**（2024 年调整）：

| 触发条件 | 阈值 |
|---------|------|
| 有形资产所有权转让 | > 2 亿元 |
| 金融资产转让 | > 1 亿元 |
| 无形资产所有权转让 | > 1 亿元 |
| 无形资产使用权许可 | > 5000 万元 |
| 劳务/服务交易 | > 5000 万元 |
| 总体关联交易金额 | > 4 亿元 |

### 1.4 BEPS 2.0 支柱二（GloBE 规则）

2023 年起，OECD/G20 推出全球最低税（GMT）规则：

```
GloBE Rules (Global Anti-Base Erosion):
├── IIR（Income Inclusion Rule）— 收入纳入规则
│   → 母公司就低税子公司补税至 15%
├── UTPR（Undertaxed Profits Rule）— 低税利润规则
│   → 其他管辖区就剩余低税利润补税
├── QDMTT（Qualified Domestic Minimum Top-up Tax）— 合格本地最低税
│   → 收入来源国先于 IIR 补税
└── Subject To Tax Rule（STTR）— 征税不足规则
    → 允许来源国对部分特定跨境付款预提
```

**中国 GloBE 实施状态**（截至 2026-06）：
- 已立法：2024 年 12 月发布《全球最低税》征求意见稿
- 预计生效：2027 财年（收入年度 ≥ 7.5 亿欧元的集团）

## 二、转让定价方法详解

### 2.1 五种 OECD 公认方法

| 方法 | 适用情形 | 计算公式 | 数据需求 |
|------|---------|---------|---------|
| **CUP** 可比非受控价格法 | 有可比非受控交易 | `可比交易价格` | 高（需要可比交易） |
| **RPM** 再销售价格法 | 营销分销商 | `(再销售价格 - 合理毛利) ÷ 再销售价格 = 毛利率` | 中 |
| **CPM** 成本加成法 | 制造/服务 | `(成本 × (1 + 成本加成率)) = 售价` | 中 |
| **PSM** 利润分割法 | 高价值无形资产/多方交易 | `按功能/风险/贡献分割合并利润` | 高 |
| **TNMM** 交易净利润法 | 多数常规交易 | `营业利润 ÷ 适当基数 = 利润率` | 中低（最常用） |

### 2.2 方法选择决策树

```
有形资产交易：
├── 有可比非受控交易？
│   ├── 是 → CUP（首选）
│   └── 否 → 角色分析
│       ├── 营销分销商 → RPM
│       ├── 制造商/服务商 → CPM
│       └── 其他 → TNMM
└── 高价值无形资产/多方交易？
    └── PSM

服务交易：
├── 服务性质简单可比 → CUP
├── 服务是"内部"还是"外部"？
│   ├── 内部服务（股东活动/重复服务）→ 收费豁免
│   └── 真实服务 → CPM（成本加成）
└── 服务有独立交易可比 → CUP

无形资产：
├── 有可比许可 → CUP
├── 无形资产独特 → PSM
├── 评估 DEMPE 功能 → 剩余利润分割法

金融交易：
├── 内部贷款 → 利率应符合独立交易原则
│   ├── 评级法（基于债务人信用评级）
│   ├── 信用利差法
│   └── 商业票据法
└── 担保 → 应区分股东担保 vs 商业担保
```

### 2.3 TNMM 利润率基准（行业参考）

按行业类型给出 TNMM 四分位区间（参考 ORBA 2024 报告）：

| 行业 | 全成本加成（成本加成法） | 营业利润率（TNMM） |
|------|-----------------------|------------------|
| 制造业 | 5% - 15% | 3% - 10% |
| 批发分销 | 3% - 8% | 1% - 5% |
| 零售 | 5% - 12% | 2% - 6% |
| 软件开发 | 15% - 30% | 10% - 25% |
| 咨询服务 | 10% - 25% | 8% - 20% |
| 物流仓储 | 5% - 10% | 3% - 7% |
| 金融服务 | 不适用 | 视产品/服务而定 |

**注意**：上表为参考区间。同期资料应使用**自有可比性分析**得出的区间，不能直接照搬。

## 三、可比性分析详细步骤

可比性分析是同期资料的**核心**，也是税务机关质询的重点。

### 3.1 五大可比性因素

OECD 要求从以下五个维度评估可比性：

```
1. 财产或服务的特征（Property or Services Characteristics）
2. 功能分析（Functional Analysis）
3. 合同条款（Contractual Terms）
4. 经济情况（Economic Circumstances）
5. 经营策略（Business Strategies）
```

### 3.2 可比公司筛选流程

**步骤 1：确定功能画像**
```
根据受测方实际承担的功能（DEMPE），确定可比公司应具备的相似功能：
- 研发/营销/销售/生产/物流/管理
- 风险承担类型（市场风险/存货风险/汇率风险/信用风险）
- 资产类型（有形资产/无形资产）
```

**步骤 2：行业代码匹配**
```
- 优先使用 GICS / NAICS / SIC 行业代码
- 颗粒度：4 位 GICS 或 6 位 NAICS
- 多个行业代码？选主导行业（收入占比 > 60%）
```

**步骤 3：地理市场筛选**
```
- 地区筛选原则：可比公司应在相似市场运营
- 中国：境内可比
- 东南亚：可比度需折扣
- 美国/欧洲：通常不与中国可比
```

**步骤 4：财务指标筛选**
```
筛选指标：
- 收入规模：可比公司收入应在受测方 0.5x - 2x 范围内
- 盈利能力：ROA / ROE / 营业利润率
- 资本结构：资产负债率
- 营运资金：流动比率

剔除标准：
- 亏损连续 2 年以上（除非行业普遍亏损）
- 重大的非经常性损益
- 重组期/破产期/上市前
- 财务数据不全
```

**步骤 5：可比性差异调整**
```
剩余可比性差异（如规模、地理、资本结构）需要调整：
- 价格调整（基于规模经济系数）
- 工作资本调整
- 资产调整

调整后仍存在不可调整差异 → 扩展样本或重新筛选
```

### 3.3 可比数据库使用

**主流数据库对比**：

| 数据库 | 覆盖范围 | 数据质量 | 中国数据 | 价格 |
|--------|---------|---------|---------|------|
| **Capital IQ** | 全球 | 极高 | 较好 | 昂贵 |
| **Bloomberg** | 全球 | 高 | 一般 | 昂贵 |
| **RoyaltyStat** | 全球无形资产 | 高 | 一般 | 中等 |
| **Romson** | 中国 + 亚洲 | 高 | 极好 | 中等 |
| **BVD Orbis** | 全球中小企业 | 中 | 较好 | 中等 |
| **中国税务年鉴** | 中国行业平均 | 高 | 极好 | 公开 |
| **国泰安 CSMAR** | 中国上市公司 | 高 | 极好 | 中等 |

**中国本地数据库推荐**：
- 上交所/深交所披露年报（免费，但需手工整理）
- Wind / 同花顺 / 巨潮资讯
- 上市公司公告中的"重要关联交易"

### 3.4 四分位区间法

**关键概念**：
```
四分位区间（Interquartile Range, IQR）：
Q1 (25th percentile) - Q3 (75th percentile) = 50% 中间区间
中位数 (Median) = 50th percentile
```

**应用规则**：
```
中国实务：
- 营业利润率落在 Q1-Q3 之间 → 合规
- 低于 Q1 → 风险（可能被调整）
- 高于 Q3 → 风险（可能引发关注）
- 偏离 Q1/Q3 超过 5 个百分点 → 重大风险

OECD 实务：
- 优先使用中位数（中点）
- 区间为 Q1-Q3
- 偏离需要解释
```

### 3.5 完整可比性分析案例

**场景**：
- 受测方：江苏某机械制造公司，向香港母公司销售产品
- 关联交易金额：3.5 亿元/年
- 受测方承担功能：来料加工、组装、二级研发
- 行业：通用机械制造

**筛选过程**：
```
1. 行业代码：GICS 20101015（机械制造）
2. 地理：中国境内
3. 报告期：最近 3 年
4. 收入规模：3 亿 - 7 亿元
5. 财务筛选：
   - 营业利润率：-5% - 15%
   - 资产负债率：30% - 70%
6. 初始候选：42 家公司
7. 剔除：
   - 亏损公司 3 年连续：8 家
   - 重组期：2 家
   - 主营不一致：5 家
   - 数据不全：3 家
8. 最终可比：24 家
9. 财务调整：规模、资本结构、营运资金

四分位区间（调整后）：
- Q1: 4.5%
- 中位数: 7.2%
- Q3: 10.1%
```

**结论**：
- 受测方营业利润率：8.3%
- 落在 Q1-Q3 区间内 → **合规**
- 建议：基于 TNMM，加成水平在中位数附近

## 四、主体文档（Master File）准备

### 4.1 文档结构（OECD 模板）

```
第一章：组织架构
1.1 集团组织图
1.2 关联方列表（含管辖区、股权关系）
1.3 重大变化（年度内股权/架构变更）

第二章：业务描述
2.1 集团业务范围
2.2 收入构成（按产品/地区）
2.3 主要客户/供应商
2.4 价值链分析（核心流程：研发-生产-营销-分销-售后）
2.5 关键无形资产

第三章：转让定价政策
3.1 集团总体 TP 政策
3.2 按交易类型的方法选择
3.3 定价/加成率/利率政策

第四章：经济与法律环境
4.1 主要管辖区的税收环境
5.2 政府激励/补贴

第五章：财务活动
5.1 集团内金融服务
5.2 内部资金流转

第六章：无形资产和合同研发
6.1 DEMPE 功能
6.2 成本分摊协议
6.3 主要合同安排

第七章：集团内服务
7.1 服务类型和定价

第八章：转让定价方法说明
8.1 方法选择理由
8.2 可比性分析概要

附录：详细财务数据
```

### 4.2 准备要点

**数据完整性**：
```
必填：
- 集团合并财务报表（最近 3 年）
- 各管辖区子公司财务报表
- 关联交易明细（按关联方/交易类型）
- 价值链分析证据（组织图/合同/业务说明）

建议：
- 同业可比公司财务数据
- 行业研究报告
- 重大重组/交易证据
- 集团内部政策文档
```

**常见错误**：
```
- 集团组织图过于简略（只到母公司层）
- 关联交易披露不完整（漏报小额交易）
- 价值链分析过于抽象（缺少具体证据）
- 财务数据汇总后没有分国别细分
- 没有解释 TP 政策与商业实质的一致性
```

## 六、国别报告（CbCR）详解

```
Table 1: 概览

- 税前利润
- 已交所得税
- 应计所得税
- 股本
- 累计利润
- 员工数

Table 2: 按管辖区列示收入
- 关联方收入
- 第三方收入
- 总收入

Table 3: 按管辖区列示

- 注册资本
- 累计利润
- 员工数
- 有形资产（非现金，非无形资产）

- 业务活动描述

```

### 6.2 报告时点

| 报告 | 时点 |
|------|------|
| 集团最终控股企业 | 财年结束后 12 个月内 |

```
已生效：
- 香港、澳门（2018-）
- 英国、德国、法国、荷兰、瑞士、新加坡
  日本、韩国、澳大利亚、新西兰
```

## 七、APA（预约定价安排）详解

### 7.1 APA 三种类型

| 类型 | 含义 | 申请方 | 期限 |
|------|------|--------|------|
| **单边 APA** | 一国税务机关与纳税人 | 仅纳税人本国 | 3-5 年 |
| **双边 APA** | 两国税务机关与纳税人 | 两国纳税人共同 | 3-5 年 |
| **多边 APA** | 多国税务机关与纳税人 | 多国纳税人 | 3-5 年 |

**中国实务**：鼓励双边/多边，避免单边。

### 7.2 APA 申请流程（中国 2017 年 9 号公告）

```
1. 预备会谈（Pre-filing Meeting）
   - 提交 APA 申请意向书
   - 税务机关评估可行性
   - 形成会谈纪要

2. 正式申请
   - 提交《预约定价安排申请书》
   - 准备 5 类材料：
     a. 申请书
     b. 集团组织架构
     c. 价值链分析
     d. 关联交易详情
     e. 可比性分析

3. 评估与协商
   - 税务机关评估（6-9 个月）
   - 多次协商谈判
   - 草案确认

4. 签署执行
   - 签订 APA
   - 年度执行情况报告
   - 续签
```

### 7.3 APA 适用场景

**推荐场景**：
- 集团内重大跨境交易（金额 > 1 亿元）
- 涉及高价值无形资产
- 未来 5 年交易结构稳定
- 双方税务机关合作意愿强

**不推荐场景**：
- 交易类型不清晰
- 财务数据历史不足
- 集团结构频繁变化
- 涉及多国复杂利益

### 7.4 APA 优势与风险

**优势**：
```
- 确定性（避免事后调整）
- 减少税务争议
- 提升合规形象
- 锁定关键方法论
```

**风险**：
```
- 申请成本高（咨询费 + 内部资源）
- 谈判期长（6-24 个月）
- 一旦签署，方法论锁定（灵活性下降）
- 续签需重新评估
- 需年度执行报告
```

## 十一、转让定价风险评估矩阵

| 风险等级 | 触发条件 | 处理方式 |
|---------|---------|---------|
| 🔴 高 | - 利润水平偏离 Q1/Q3 超过 5 个百分点<br>- 单边 APA 受限<br>- 重大交易未文档化<br>- 监管处罚历史 | 立即申请 APA/重整 TP 政策 |
| ⚠️ 中 | - 利润水平接近 Q1/Q3 边界<br>- 关联交易占总收入 > 30%<br>- 涉及高税负管辖区 | 加强文档 + 准备 APA |
| ✅ 低 | - 利润水平稳定在 Q1-Q3 中间<br>- 关联交易结构清晰<br>- 文档完整 | 维持现状 |
