# 判断框架 — 转让定价文档合规管理

## 核心原则

转让定价文档的核心目标是**证明关联交易符合独立交易原则**。OECD BEPS 行动计划 13 和中国税法要求企业准备充分、可验证的文档支持。遇到异常时，优先判断"是否符合独立交易原则"。

---

## 文档完整性判断

### 第一步：确认须准备的文档类型

```
□ 是否超过同期资料准备门槛：[是/否]
  → 门槛：关联交易 > [X] 万元 或 关联交易 > 净资产 [X]%

□ 是否须准备主体文档：
  → 条件：海外关联方 + 年度关联交易 > [X] 万元

□ 是否须准备国别报告：
  → 门槛：集团合并收入 > [X] 亿元（人民币）

□ 是否须提交关联申报表：
  → 须在每年 [X] 月 [X] 日前提交
```

### 第二步：检查文档完整性

```
□ 主体文档完整性：
□ 组织架构描述
□ 价值链分析
□ 关联交易概况
□ 功能/风险/资产分布
□ 可比性分析
□ 转让定价方法说明

□ 本地文档完整性：
□ 企业基本信息
□ 关联交易详情
□ 可比性分析
□ 转让定价计算过程

□ 国别报告完整性：
□ 跨国集团各成员实体收入
□ 员工数、资产、利润分配
□ 各管辖区纳税情况
```

## 转让定价风险判断

### 利润水平指标预警

```
利润率类型      警戒线下限      警戒线上限      超限风险
─────────────────────────────────────────────────────
毛利率          [X]%           [X]%           🔴 偏离 → 税务调整风险
营业利润率      [X]%           [X]%           🔴 偏离 → 转让定价风险
完全成本加成率  [X]%           [X]%           🔴 偏离 → 定价不合理
资产报酬率      [X]%           [X]%           🔴 偏离 → 功能风险不匹配
```

### 风险分级

```
✅ 低风险：
□ 利润水平在警戒线内
□ 有充分可比公司支撑
□ 文档完整且及时更新
□ 无税务机关查询记录

⚠️ 中风险：
□ 利润水平接近警戒线
□ 可比公司数量 < 3 家
□ 存在特殊无形资产或特殊交易
□ 历史上有轻微调整记录

🔴 高风险：
□ 利润水平超出警戒线
□ 无可比公司（须用成本加成法）
□ 涉及跨境无形资产交易
□ 已有税务机关反转让定价调查
□ 关联交易金额重大变动
```

## 文档质量问题判断

### 常见文档缺陷

```
□ 可比性分析不足：
□ 可比公司 < 3 家
□ 行业分类不准确
□ 财务指标调整不充分
□ 差异调整不完整

□ 功能分析不准确：
□ 功能/风险/资产分布与实际不符
□ 价值贡献分析不完整
□ 收益分配与功能不匹配

□ 方法选择不当：
□ 有形资产转让用了利润分割法
□ 服务提供用了转售价格法
□ 未说明选择理由

□ 数据陈旧：
□ 可比数据超过 [X] 年
□ 未使用最新财务数据
□ 关键假设未更新
```

## BEPS 行动 13 合规检查

```
行动 13 三层结构：
□ 主体文档（Master File）— 跨国集团整体信息
□ 本地文档（Local File）— 企业自身关联交易详情
□ 国别报告（CbCR）— 各管辖区汇总数据

国别报告（CbCR）补充测试：
□ 是否在集团合并收入 > [X] 亿元时准备
□ 是否包含所有跨境关联交易
□ 数据是否与财务账目一致
□ 是否在截止日前提交
```

---

## 升级触发条件

- 利润水平指标超出警戒线 → 🔴 高风险
- 税务机关发出转让定价查询函
- 须启动 APA 申请或续期
- 年度关联交易金额重大变动（> ±[X]%）
- 境外关联方注册地变更为高风险管辖区
