# 查询路径 — 转让定价文档（无 Connector 时）

## 手动流程说明

当 ERP/专业系统不可用时，按以下步骤手动收集转让定价文档所需数据。

---

## 步骤一：关联方识别与清单确认

```
□ 从工商登记获取全部关联方清单
□ 确认境外关联方注册地及当地税率
□ 核实关联交易类型（销售/采购/服务/借款/担保/费用分摊）
□ 整理关联方与本公司的持股关系图
□ 输出：关联方清单（含持股比例、注册地、交易类型）
```

---

## 步骤二：关联交易数据收集

```
□ 从总账导出全年关联交易明细
□ 按关联方分类汇总：
□ 销售商品：[X] 万元
□ 采购商品：[X] 万元
□ 提供服务：[X] 万元
□ 借款利息：[X] 万元
□ 费用分摊：[X] 万元
□ 担保余额：[X] 万元

□ 确认关联交易超过准备门槛：[是/否]
□ 如超过，准备同期资料
```

---

## 步骤三：转让定价方法确认

```
□ 按关联交易类型确认 TP 方法：
□ 有形资产转让 → [方法]
□ 服务提供 → [方法]
□ 无形资产许可 → [方法]
□ 金融交易 → [方法]

□ 确认方法选择的合理性依据
□ 输出：各交易类型的 TP 方法说明
```

---

## 步骤四：可比性分析

```
□ 确定可比公司筛选标准
□ 行业：[X]
□ 地理市场：[X]
□ 财务指标区间：[X]% - [X]%

□ 从公开数据源（年报/Bloomberg）收集可比公司数据
□ 计算并调整财务指标
□ 确认利润水平在警戒线内
□ 输出：可比性分析表（含可比公司列表、指标计算）
```

---

## 步骤五：文档撰写与核对

```
□ 按主体文档框架撰写：
□ 组织架构 → 价值链 → 交易概况 → 功能分析 → 可比性 → 方法说明

□ 按本地文档框架撰写：
□ 企业信息 → 关联交易详情 → 可比性分析 → 定价计算

□ 核对完整性清单
□ 确认所有数据有来源支撑
□ 输出：完整的同期资料文档包
```

---

## 查询路径总结

```
关联方清单 → 关联交易数据 → TP 方法确认 → 可比性分析 → 文档撰写
     ↓              ↓              ↓              ↓             ↓
工商/企查查    ERP总账/手工    方法论框架      Bloomberg/     Word/PDF
                              /CLAUDE.md     年报
```
