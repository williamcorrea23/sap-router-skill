# 判断框架 — 成本控制异常处理

## 核心原则

**成本控制的目标是"不花冤枉钱，也不省该花的钱"。**
遇到成本异常时，优先判断是"市场因素"（外部）、"效率问题"（内部管理）还是"预算失准"（计划问题），再决定处理方式。

---

## 异常类型与处理方式

### 类型一：直接材料超支

**典型场景：**
- 材料成本率 > 目标 [X%]
- 材料单耗 > 标准 [X%]
- 材料采购价 > 预算价 [X%]

**判断流程：**

```
第一步：确认数据准确性

□ 数据来源是否正确（ERP 取数口径）？
□ 是否有产品结构变更（影响配方）？
□ 是否有会计政策变更（成本归属）？

→ 如数据有问题：修正数据后再分析

第二步：区分"量差"还是"价差"

材料成本差异 = 量差 + 价差

□ 量差 =（实际用量 - 标准用量）× 标准单价
  → 量大：生产损耗增加/工艺问题/良率下降
  → 量小：节约用料（好）但需确认质量

□ 价差 =（实际单价 - 标准单价）× 实际用量
  → 价高：市场涨价/供应商议价弱/采购时机差
  → 价低：折扣/谈判好/市场降价（好）

第三步：归因到具体原因

材料量差归因：
  □ 工艺变更导致损耗增加
  □ 工人操作不当（培训/监督问题）
  □ 设备老化导致良率下降
  □ 原材料质量不稳定
  □ 产品结构变化（小批量/大批量）

材料价差归因：
  □ 市场价格整体上涨（行业因素）
  □ 采购时机选择失误
  □ 供应商合同到期未续签
  □ 采购批量小未享批量折扣
  □ 替代材料引入（质量 vs 成本权衡）

第四步：制定改善措施

材料量差改善：
  → 工艺优化，减少损耗
  → 加强操作培训
  → 设备维护/更新计划
  → 来料质量检验加强
  → 生产批量经济化

材料价差改善：
  → 集中采购，争取批量折扣
  → 长期合同锁定价格
  → 供应商整合，淘汰高价供应商
  → 替代材料评估（需验证质量）
  → 套期保值（原材料价格波动大时）
```

### 类型二：直接人工超支

**典型场景：**
- 直接人工成本率 > 目标 [X%]
- 单位产品工时 > 标准 [X%]
- 人均产出 < 目标 [X%]

**判断流程：**

```
第一步：确认数据准确性

□ 工时统计来源是否正确（MES/考勤系统）？
□ 产出计量是否准确（良品数/总产量）？
□ 人工成本是否含社保/公积金（全口径）？

→ 如数据有问题：修正数据后再分析

第二步：区分"效率差"还是"价格高"

人工成本差异 = 效率差异 + 工资率差异

□ 效率差异 =（实际工时 - 标准工时）× 标准工资率
  → 效率低：人员技能不足/工艺复杂/设备瓶颈
  → 效率高：学习曲线改善（好）

□ 工资率差异 =（实际工资率 - 标准工资率）× 实际工时
  → 工资率高：招聘薪资上涨/高技能人员占比高
  → 工资率低：人员替换/低工资地区调配

第三步：归因到具体原因

效率差异归因：
  □ 新员工占比高（学习曲线）
  □ 产品工艺变更（复杂度上升）
  □ 设备故障导致等待
  □ 生产计划不均衡（加班/待料）
  □ 培训计划执行情况

工资率差异归因：
  □ 市场薪资上涨（行业整体）
  □ 年度调薪执行
  □ 高技能岗位招聘困难
  □ 人员结构变化（高工资占比上升）

第四步：制定改善措施

效率改善：
  → 技能培训提升
  → 产线平衡优化
  → 设备预防性维护
  → 生产计划精细化
  → 引入计件工资（激励效率）

成本改善：
  → 招聘策略优化（技能匹配 vs 成本）
  → 年度调薪预算控制
  → 灵活用工（季节性峰值）
  → 地区薪资差异化（成本中心布局）
```

### 类型三：制造费用超支

**典型场景：**
- 制造费用率 > 目标 [X%]
- 产能利用率 < [X]%（固定费用摊薄）
- 某项费用科目超支 > [X]%

**判断流程：**

```
第一步：确认数据准确性

□ 费用分摊方法是否合理？
□ 产出计量（产量/机器小时）是否准确？
□ 是否有跨期费用错误分摊？

→ 如数据有问题：修正数据后再分析

第二步：判断是"固定费用"还是"可变费用"问题

固定制造费用（折旧/房租/管理人员工资）：
  □ 产能利用率是否下降？
  □ 固定费用是否随规模增加而增加？
  → 产能利用率 < [X]%：固定费用摊薄，单位成本上升

可变制造费用（水电/物料/维修）：
  □ 是否随产量变动而合理变动？
  □ 单位费用是否超标？
  → 单位费用上升：能源涨价/维修频繁/物料消耗增加

第三步：归因到具体科目

□ 哪个费用科目超支最多？
  - 折旧：新增资产/延长折旧年限
  - 水电：能源涨价/设备老化
  - 维修：设备老化/保养不足
  - 物料：低值易耗品消耗增加
  - 间接人工：辅助人员超编

□ 是否有一次性大额维修/改造支出？
□ 产能利用率下降原因（订单不足/设备故障）？

第四步：制定改善措施

固定费用改善：
  → 产能利用率提升（接单优化/淡季促销）
  → 设备共享（跨产线利用）
  → 固定费用削减（外包/关停低效产线）
  → 租赁 vs 自有评估（轻资产化）

可变费用改善：
  → 能源管理（错峰用电/设备更新）
  → 预防性维护（减少紧急维修）
  → 物料消耗定额管理
  → 包装/周转材料循环利用
```

### 类型四：销售费用超支

**典型场景：**
- 销售费用率 > 目标 [X]%
- 某项费用（佣金/差旅/市场费）超支 > [X]%
- 人均销售费用持续上升

**判断流程：**

```
第一步：区分费用类型

可控费用（可直接控制）：
  □ 差旅费：销售拜访客户发生
  □ 招待费：客户关系维护
  □ 市场推广费：广告/展会/活动

半可控费用（与业绩挂钩）：
  □ 销售佣金：业绩完成率越高费用越高
  □ 销售提成：与回款挂钩

固定费用（相对稳定）：
  □ 销售人员工资/社保
  □ 销售部门管理费用

第二步：判断超支是否合理

合理性判断（外部因素）：
  □ 进入新市场（前期投入大）
  □ 大客户攻关期（关系维护密集）
  □ 行业旺季（费用自然增加）
  □ 竞争对手动作（应对性投入）
  → 须有明确的投入产出预期

不合理超支（内部问题）：
  □ 销售政策执行不严（报销超标）
  □ 人员扩张失控（收入增长慢于人员增长）
  □ 客户ROI评估缺失（投入大产出小）
  □ 供应商/客户关系维护浪费
  → 须立即整改

第三步：制定改善措施

可控费用改善：
  → 差旅标准强化（超标审批）
  → 招待费事前审批
  → 市场活动ROI评估（每活动必须评估效果）
  → 数字化销售工具（减少出差）

半可控费用改善：
  → 佣金政策调整（阶梯制/封顶制）
  → 提成与回款挂钩（而非发货）
  → 费用率与业绩挂钩浮动

固定费用改善：
  → 销售组织扁平化
  → 低效人员淘汰
  → 区域整合（减少重叠覆盖）
```

### 类型五：管理费用超支

**典型场景：**
- 管理费用率 > 目标 [X]%
- 管理人员人均成本 > 目标 [X] 万
- 某项费用科目超支 > [X]%

**判断流程：**

```
第一步：区分费用类型

人员费用（最大占比，约 [X]%）：
  □ 管理人员工资/奖金/社保
  □ 招聘/培训费用

固定费用：
  □ 折旧摊销
  □ 房租/物业
  □ IT 系统

可控费用：
  □ 办公费用
  □ 差旅费用
  □ 咨询顾问费

第二步：判断超支原因

人员费用超支：
  □ 人员是否超编？（管理幅度是否合理）
  □ 薪酬是否过高？（市场对比）
  □ 招聘是否失控？（人员补充计划）

固定费用超支：
  □ IT 系统升级（一次性投入）
  □ 办公场地扩张
  □ 折旧到期换新

可控费用超支：
  □ 咨询项目超预算
  □ 差旅/办公标准提高
  □ 外部活动增加

第三步：制定改善措施

人员费用改善：
  → 组织架构扁平化（减少管理层级）
  → 岗位合并/职责整合
  → 人才盘活（内部调配 vs 外部招聘）
  → 管理岗竞聘（能上能下）

固定费用改善：
  → IT 系统共享（减少重复建设）
  → 办公场地整合（远程+共享办公）
  → 设备更新评估（自购 vs 租赁）

可控费用改善：
  → 咨询项目预算审批制
  → 差旅标准分级（按级别）
  → 办公用品集采（降低采购成本）
```

### 类型六：研发费用异常

**典型场景：**
- 研发费用率 > 目标 [X]%
- 研发项目进度滞后导致费用积压
- 研发成果转化率低

**判断流程：**

```
第一步：确认研发费用口径

□ 研发费用是否正确归集（区分研发 vs 非研发）？
□ 资本化 vs 费用化处理是否合规？
□ 是否有与政府补贴项目混用？

第二步：判断超支类型

投入超支：
  □ 人员扩张（研发人员 > [X] 人）
  □ 设备/材料采购超预算
  □ 外部合作/委托研发超支

效率问题：
  □ 项目进度滞后（费用花出去但成果未出）
  □ 试验失败率高（重复试验消耗）
  □ 人员利用率低（有效工时 < [X]%）

第三步：评估研发效率

□ 研发投入产出比：研发费用 / 新增收入
□ 项目成功率：成功项目 / 总项目
□ 成果转化率：商业化项目 / 成功项目

□ 研发效率下降 → 须评估项目组合
□ 研发效率稳定但投入大 → 评估战略优先级

第四步：制定改善措施

投入控制：
  → 研发项目预算审批（分级授权）
  → 里程碑付款（按成果支付）
  → 外部资源利用（高校/外包）减少自建

效率提升：
  → 研发流程标准化（减少试验失败）
  → 项目管理工具（进度可视化）
  → 人员技能提升（减少试错）
  → 知识管理（避免重复研发）

组合优化：
  → 定期项目评审（资源重新分配）
  → 终止低价值项目（释放资源）
  → 聚焦核心产品（减少分散投入）
```

## 置信度评估

| 置信度 | 条件 | 操作 |
|---|---|---|---|
| 高 | 数据完整 + 标准成本设定 + 月末结账 | ✅ 正常分析 |
| 中 | 数据完整，标准成本待更新 | ⚠️ 标注口径差异 |
| 低 | 数据不完整或未结账 | 🔴 暂停分析，核查 |

- 成本率超预算 > [X]%
