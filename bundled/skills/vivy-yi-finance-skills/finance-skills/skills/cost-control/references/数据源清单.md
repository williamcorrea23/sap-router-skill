# 数据源清单 — [ERP]/[COST]/[MES]/[BK]

## 数据源标签说明

| 标签 | 含义 | 典型来源 |
|---|---|---|
| [ERP] | 企业 ERP 系统（总账）| SAP/用友/金蝶总账 |
| [COST] | 成本核算系统 | SAP CO/用友成本/金蝶成本 |
| [MES] | 生产执行系统 | 生产管理系统/工时系统 |
| [BK] | 预算管理系统 | SAP BPC/用友预算/金蝶预算 |

---

## [ERP] SAP / 用友 / 金蝶（总账）

**覆盖范围：**
- 成本科目余额（直接材料/直接人工/制造费用）
- 费用科目余额（销售/管理/研发）
- 辅助核算（按成本中心/项目/产品）
- 成本凭证明细

**查询路径：**

```
SAP 常用事务代码：
- KOB1N：成本element报表
- KKS1：生产成本核算
- KB11N：成本中心计划/实际对比
- S_ALR_00001236：成本中心费用分析
- COOIS：生产订单信息系统

用友/金蝶：
- 成本管理 → 成本明细账
- 辅助核算 → 按成本中心
- 报表 → 成本分析报表
```

---

## [COST] SAP CO / 用友成本 / 金蝶成本

**覆盖范围：**
- 标准成本设定（材料/工时/费用）
- 实际成本 vs 标准成本差异
- 成本差异分析（量差/价差/效率/费率）
- 成本分摊规则

**查询路径：**

```
SAP CO：
- 标准成本估算：CK11N/CK24
- 成本差异分析：KKBC_PCK/KKBC_KKO
- 在制品计算：K，照
- 成本分摊：KSII

用友/金蝶：
- 标准成本管理 → 标准成本设定
- 成本差异 → 差异分析报表
```

---

## [MES] 生产执行系统 / 工时系统

**覆盖范围：**
- 产量数据（总产量/良品/废品）
- 工时数据（实际工时/标准工时/加班）
- 设备运行数据（运行时间/停机时间）
- 物料消耗数据（领料/退料/损耗）
- 良率数据

**查询路径：**

```
MES 系统：
- 生产报工 → 产出数量
- 工时登记 → 实际工时
- 设备管理 → OEE/运行时间
- 质量管理 → 良率/废品

工时系统：
- 员工工时 → 直接工时/间接工时
- 项目工时 → 研发项目工时
```

---

## [BK] SAP BPC / 用友预算 / 金蝶预算

**覆盖范围：**
- 年度成本预算（按成本中心/科目）
- 预算版本（初始/调整/预测）
- 预算执行进度
- 滚动预测

**查询路径：**

```
SAP BPC：
- 预算编制 → 成本预算
- 预算控制 → 执行进度
- 预算分析 → 差异分析

用友/金蝶：
- 预算管理 → 成本预算编制
- 预算执行 → 执行分析
```

---

## 数据源组合使用

### 标准成本差异分析
```
[COST] 标准成本 → [ERP] 实际成本 → [MES] 产量/工时 → 计算差异（量差/价差）
```

### 成本中心绩效分析
```
[ERP] 成本中心费用 → [COST] 成本分摊 → [MES] 产出数据 → 计算单位成本
```

### 成本预算执行分析
```
[BK] 成本预算 → [ERP] 实际成本 → [COST] 差异分析 → 预算执行报告
```

## Connector 映射
