# 查询路径 — 成本控制

## 数据源优先级

| 优先级 | 数据源 | 说明 |
|---|---|---|
| 1 | ERP 总账（[ERP]）| 成本明细/科目余额 | 最优先 |
| 2 | 成本核算系统（[COST]）| 标准成本/实际成本/差异 | 成本分析 |
| 3 | 生产管理系统（[MES]）| 产量/工时/良率 | 生产效率 |
| 4 | 预算管理系统（[BK]）| 成本预算/执行 | 对比基准 |

---

## 常用查询场景

### 场景 1：直接材料成本分析

**查询入口：**
```
ERP：
- 科目余额表：营业成本 - 直接材料
- 明细账：材料采购/领料/入库

成本核算系统：
- 标准成本 vs 实际成本差异
- 材料成本差异分析（量差/价差）

MES：
- 材料领料记录
- 产出良品数量
- 物料损耗率
```

**材料成本分析维度：**
```
□ 材料成本总额：[XXX万]
□ 材料成本率：[X%]（目标 [X%]）

□ 材料量差：[±XXX万]
□ 材料价差：[±XXX万]

□ 主要超支材料：
  - [材料A]：[XXX万]（超支 [XXX万]）
  - [材料B]：[XXX万]（超支 [XXX万]）

□ 主要超支产品：
  - [产品A]：[XXX万]（超支 [XXX万]）
```

## 数据源组合使用

### 月度成本分析
```
[ERP] 成本明细 → [COST] 标准成本差异 → [MES] 产出数据 → 生成成本分析报告
```

### 成本中心分析
```
[ERP] 成本中心报表 → [COST] 分摊差异 → [MES] 产线效率 → 成本中心绩效评估
```

### 成本改善分析
```
[ERP] 历史成本趋势 → [BK] 预算目标 → [MES] 效率数据 → 改善机会识别
```
