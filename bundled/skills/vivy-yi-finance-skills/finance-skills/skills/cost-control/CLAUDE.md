## 成本控制方法

```
成本控制三板斧：

1. 事前：预算控制
   → 成本预算编制
   → 标准成本设定
   → 审批权限控制

2. 事中：差异监控
   → 实际 vs 预算监控
   → 标准成本差异分析
   → 超标预警

3. 事后：分析改善
   → 差异归因
   → 责任落实
   → 改善措施
```
