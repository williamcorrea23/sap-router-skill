## 付款优先级配置

### 付款优先级排序

```
P0（立即支付）：
  - 员工工资/社保（每月固定）
  - 税款缴纳（申报截止日）
  - 到期银行贷款本息
  - 已到期法院判决/仲裁裁决

P1（优先支付）：
  - 供应商大额应付（已到期且催款紧急）
  - 水电费/租金等固定支出
  - 重要客户大额采购款（维持客户关系）

P2（正常支付）：
  - 常规供应商付款（按账期）
  - 日常运营费用
  - 一般报销款

P3（可延后）：
  - 非紧急供应商付款
  - 一般运营费用
  - 员工差旅费（无紧急出差）

P4（战略延后）：
  - 可延后的供应商付款（与供应商协商展期）
  - 非必要支出
```


## 地区/行业差异

```
银行服务差异：
  - 四大行（工农中建）：网点多，服务稳定，但审批较慢
  - 股份制银行（招商/兴业）：服务灵活，审批较快
  - 城商行/农商行：本地服务好，但跨区域服务有限

行业资金特点：
  - 制造业：资金周转慢，存货占用大，须关注资金缺口
  - 零售/电商：资金周转快，应收少，但备货资金需求大
  - 项目型业务：资金阶段性需求大，须提前安排融资
  - 政府/教育：账期长，但资金来源稳定
```
