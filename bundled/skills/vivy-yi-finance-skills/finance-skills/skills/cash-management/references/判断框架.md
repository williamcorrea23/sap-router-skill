# 判断框架 — 现金管理异常处理

## 核心原则

**现金管理的目标是"确保公司随时有足够的钱应对所有支出，同时不让钱闲置"。**
遇到异常时，优先判断是"短期流动性问题"还是"长期资金结构问题"，对资金缺口风险保持警惕。

---

## 异常类型与处理方式

### 类型一：资金缺口（即将到期无对应资金）

**典型场景：**
- 某日须支付 [XXX万] 但账户余额不足
- 大额应收款逾期未收回同时有大额应付到期
- 银行贷款到期但无足够资金还款

**判断流程：**

```
第一步：判断缺口规模

小额缺口（< 50 万）：
  → 优先调拨其他账户资金
  → 或加速催收应收款
  → 或延后非紧急支出

大额缺口（50-200 万）：
  → 启动银行融资（短期流贷/银行承兑）
  → 加速催收大客户应收款
  → 协商供应商延后付款

超大额缺口（> 200 万）：
  → 🔴 立即升级至财务总监
  → 评估资产变现可行性
  → 评估引入战略投资者可行性

第二步：判断缺口原因

临时性缺口（收入季节性/付款集中）：
  → 短期融资解决
  → 提前规划后续资金安排

结构性缺口（收入持续下滑/负债过高）：
  → 🔴 须系统性解决
  → 调整业务结构
  → 引入长期融资
  → 须升级至 CEO

第三步：紧急融资优先级

紧急程度排序（从高到低）：
  1. 员工工资 → 绝对优先，不可延后
  2. 税款 → 不可延后，有法律风险
  3. 到期贷款 → 不可延后，影响信用
  4. 大供应商 → 协商延后或部分支付
  5. 一般支出 → 延后
```

### 类型二：资金来源异常（大额到账/大额转出）

**典型场景：**
- 客户突然大额回款（金额异常大）
- 账户突然转出大额资金（无对应审批）
- 收到不明来源资金

**判断标准：**

```
🔴 高风险信号（立即冻结）：

账户异常转出：
  - 转出金额 > [XXX万] 且无 OA 审批
  - 转至非公司账户
  - 转至非已知供应商账户
  - 收款账户与合同不符
  → 🔴 立即冻结账户，联系银行，确认后报警

不明来源资金：
  - 无法确认付款方
  - 金额与任何应收款不匹配
  - 付款方要求退回至其他账户
  → 🔴 不可退回原账户
  → 🔴 须确认资金合法性后再处理
  → 如涉及诈骗/洗钱，须立即报警

⚠️ 中风险信号（须核实）：

大额预收：
  - 客户预付大额定金（金额 > [XXX万]）
  - 预收理由不充分
  → ⚠️ 核实合同和客户背景后再确认

资金摆布：
  - 账户间大额调拨（金额 > [XXX万]）
  - 无提前资金计划
  → ⚠️ 核实资金计划后再执行
```

### 类型三：银行账户问题

**典型场景：**
- 银行账户被冻结/限制
- 银行拒绝支付
- 账户余额与 ERP 不一致

**判断标准：**

```
🔴 银行账户冻结：

原因：
  - 法律诉讼冻结（最常见）
  - 税务稽查冻结
  - 银行风控冻结

处理：
  → 🔴 立即联系银行确认冻结原因
  → 联系法务评估解决方案
  → 如为税务问题，配合税务稽查
  → 如为法律诉讼，评估应诉策略
  → 🔴 立即启动备用账户

⚠️ 账户限额/限制：

原因：
  - 银行风控（交易异常）
  - 账户信息过期
  - 证件过期

处理：
  → ⚠️ 联系银行解除限制
  → 更新账户信息/证件
  → 准备证明材料
```

### 类型四：资金集中度风险

**典型场景：**
- 单一客户收款占当月收入 > [X]%
- 单一供应商付款占当月支出 > [X]%
- 资金来源过于集中

**判断标准：**

```
单一客户依赖风险：

单一客户收款占比 > 50%：
  → ⚠️ 高风险，须立即关注
  → 该客户经营一旦出问题，我方资金链断裂

单一客户收款占比 30-50%：
  → ⚠️ 中风险，须提前安排备用资金

单一客户收款占比 < 30%：
  → ✅ 可接受

处理措施：

高风险（单一客户 > 50%）：
  → 🔴 立即开发新客户
  → 减少对该客户的依赖
  → 提前收回应收款，降低风险敞口

中风险（单一客户 30-50%）：
  → ⚠️ 加强该客户信用管理
  → 加快该客户应收款催收
  → 储备备用资金
```

### 类型五：资金挪用风险

**典型场景：**
- 员工申请将公司资金转至个人账户
- 供应商收款账户变更（高风险）
- 付款申请与合同不符

**判断标准：**

```
🔴 高风险信号（立即拒绝）：

资金转至个人账户：
  - 任何将公司资金转至个人账户的申请
  - "过账"/"帮忙转账"等任何理由
  → 🔴 立即拒绝，升级至财务经理
  → 如涉及职务侵占，须报警

供应商账户变更：
  - 供应商要求变更收款账户
  - 新账户与合同不符
  - 要求付至第三方账户
  → 🔴 立即冻结该笔付款
  → 🔴 联系供应商原对接人核实
  → 核实前不得付款

付款申请与合同不符：
  - 金额/账户/收款方与合同不一致
  - 无合理说明
  → 🔴 立即拒绝，核实后再处理
```

## 升级触发条件

- 资金缺口 > [填空：如 100 万] 且无资金来源
- 账户余额低于安全线
- 须新增银行融资 > [填空：如 100 万]
- 银行账户被冻结
- 发现资金挪用嫌疑
- 不明来源资金入账
- 大额异常转出（> [填空：如 50 万]）
