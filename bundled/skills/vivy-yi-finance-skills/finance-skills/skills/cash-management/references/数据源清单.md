# 数据源清单 — [CASH]/[BANK]/[ERP]/[OA]/[BK]

## 数据源标签说明

| 标签 | 含义 | 典型来源 |
|---|---|---|
| [CASH] | 资金系统/司库 | 司库系统/资金管理系统/用友资金/金蝶资金 |
| [BANK] | 网上银行 | 企业网银/银企直联 |
| [ERP] | 企业 ERP 系统 | SAP/用友/金蝶总账 |
| [OA] | OA 审批系统 | 泛微/钉钉/飞书/致远 |
| [BK] | 预算管理系统 | SAP BPC/用友预算/金蝶预算 |

---

## [CASH] 资金系统/司库

**覆盖范围：**
- 所有银行账户余额（实时）
- 资金调拨记录
- 内部账户间调拨
- 资金计划执行情况
- 银行授信额度管理

**查询路径：**

```
司库系统：
- 账户余额总览 → 一屏看所有账户
- 资金调拨申请 → 发起调拨
- 资金计划 → 查看资金计划 vs 实际

常用报表：
- 资金日报
- 资金余额汇总表
- 资金调拨明细
- 授信额度使用表
```

---

## [BANK] 网上银行

**覆盖范围：**
- 银行账户实时余额
- 银行流水（收支明细）
- 到账通知
- 付款指令状态
- 银行回单

**查询路径：**

```
企业网银：
- 账户管理 → 余额查询
- 历史明细 → 按日期筛选流水
- 转账汇款 → 发起付款
- 电子回单 → 下载回单

银企直联（如有）：
- ERP 直联银行 → 自动获取流水
- 付款指令自动发送
```

---

## [ERP] 企业 ERP 系统

**覆盖范围：**
- 现金/银行存款科目余额
- 银行收款凭证
- 银行付款凭证
- 到期应付账款明细
- 银行贷款明细

**查询路径：**

```
SAP 常用事务代码：
- FAGLB03：总账科目余额报表
- FBL5N：供应商行项目（AP）
- FBL1N：客户行项目（AR）
- FB03：显示凭证
- FBL3N：总账行项目

用友/金蝶：
- 现金银行 → 现金日记账
- 应收应付 → 到期清单
- 资金管理 → 借款明细
```

---

## [OA] OA 审批系统

**覆盖范围：**
- 付款申请审批流程
- 资金调拨申请
- 费用报销审批
- 供应商付款信息维护

**查询路径：**

```
OA 系统：
- 审批管理 → 待我审批
- 付款申请 → 已审批待支付
- 历史审批 → 审批记录查询

常用报表：
- 待支付清单（按到期日排序）
- 审批中付款申请
- 已完成付款记录
```

---

## [BK] 预算管理系统

**覆盖范围：**
- 资金计划（周/月/季/年）
- 预算执行情况
- 资金预测模型
- 偏差分析

**查询路径：**

```
预算管理系统：
- 资金计划 → 未来收支计划
- 预算执行 → 实际 vs 计划
- 预测模型 → 资金缺口预测

常用报表：
- 资金计划表（周度）
- 资金计划表（月度）
- 资金缺口预警
```

---

## 数据源组合使用

### 每日资金管理
```
[CASH] 实时余额 → [BANK] 流水核对 → [ERP] 账面余额 → [OA] 待支付清单 → 资金日报
```

### 付款安排
```
[OA] 审批通过清单 → [CASH] 账户余额 → [BANK] 实时余额 → [BK] 资金计划 → 付款计划
```

### 资金缺口预警
```
[CASH] 未来30天支出 → [OA] 待支付申请 → [ERP] 到期应付 → [BK] 计划收支 → 缺口预测
```

## Connector 映射

|  | 数据需求 | Connector | 工具 | 关键字段 |
|------|---------|-----------|------|---------|
| 银行账户余额/流水 | Kyriba | `kyriba_bank_statements` | `account_id`, `balance`, `date`, `transaction` |
| 现金流预测 | Kyriba | `kyriba_forecast` | `date`, `inflows`, `outflows`, `balance` |
| 付款指令状态 | Kyriba | `kyriba_payment_status` | `payment_id`, `status`, `amount` |
| ERP 现金余额 | SAP | `sap_gl_query` | `RACCT`（现金/银行存款）, `DMBTR` |
