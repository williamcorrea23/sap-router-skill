# /cash-management-master — 现金管理主流程

## 完整资金管理体系

```
日监控（每日）：
  → 执行 cash-position-monitoring
  → 监控账户余额，核查三方一致性
  → 识别当日资金风险

周计划（每周）：
  → 执行 payment-scheduling-cm
  → 编排本周付款计划
  → 识别周度资金缺口

月预测（每月）：
  → 执行 cash-flow-forecasting
  → 滚动预测未来 3 个月
  → 输出融资计划建议
```
