# 查询路径 — 管理报告

## 查询入口

### 查询维度一：管理报告数据

| 查询目标 | 系统入口 | 查询路径 | 刷新频率 |
|---|---|---|---|
| 月度财务数据 | [BI 系统] | 管理报告 → 月度财务 | 每日 |
| 业务运营数据 | [CRM/OPS] | 运营仪表板 → 业务指标 | 实时 |
| 预算执行数据 | [预算系统] | 预算管理 → 执行跟踪 | 每日 |
| 现金流预测 | [司库系统 TRSY] | 资金视图 → 现金流预测 | 每日 |

### 查询维度二：分析与洞察

| 查询目标 | 系统入口 | 查询路径 | 刷新频率 |
|---|---|---|---|
| 同比/环比分析 | [BI 系统] | 分析中心 → 趋势分析 | 实时 |
| 异常预警 | [BI 系统] | 预警中心 → 活跃预警 | 实时 |
| 部门绩效 | [HR 系统] | 部门绩效 → 业绩看板 | 每周 |

## 常用查询组合

**场景：月度管理报告编制**
```
1. [BI 系统] → 管理报告 → 月度数据
2. [ERP] → 查询业务明细 → 补充业务分析
3. [CRM] → 查询销售数据 → 补充收入分析
4. [BI 系统] → 趋势分析 → 提取同比环比
5. [BI 系统] → 预警中心 → 标注须关注事项
```

**场景：指标预警跟进**
```
1. [BI 系统] → 预警中心 → 查看预警详情
2. [ERP/CRM] → 查询相关业务数据 → 分析预警原因
3. [BI 系统] → 历史数据 → 查询类似预警历史处理方式
```
