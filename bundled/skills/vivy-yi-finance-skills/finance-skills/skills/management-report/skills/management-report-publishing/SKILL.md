# /management-report-publishing — 管理报告发布


## Examples

→ 示例：用户说"月度管理报告已经完成了数据核对，帮我发布给管理层"，系统应调用本技能，执行发布前最终检查并输出报告包。

→ 示例：用户说"报告要在截止日期前发出去，但数据还没齐，先发一版可以吗"，系统应调用本技能，评估风险并给出分阶段发布建议。

→ 示例：用户说"这次的管理报告需要翻译成英文，发给境外董事"，系统应调用本技能，生成英文版管理报告。

## 升级触发条件

- CFO 审核提出重大修改意见须立即处理
- 报告包含保密信息但未完成脱敏
- 报告无法在分发截止日前完成审阅
- 分发对象（CEO 等高管）未确认收到
- 报告发布后发现重大数据错误须重新发布
- 报告内容与公司披露政策存在冲突
