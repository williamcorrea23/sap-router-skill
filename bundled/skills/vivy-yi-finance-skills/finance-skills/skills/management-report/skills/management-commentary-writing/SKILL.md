# /management-commentary-writing — 管理报告文字撰写


## Examples

→ 示例：用户说"帮我写一段管理层对毛利率下降的解释，要客观但不显得甩锅"，系统应调用本技能，生成管理层 Commentary 草稿。

→ 示例：用户说"管理报告的 KPI 部分需要一个执行摘要，说明未达标原因和整改措施"，系统应调用本技能，生成 KPI 执行摘要。

→ 示例：用户说"这次 Board Deck 里业务和财务对收入数字口径不一致，帮我写一段协调说明"，系统应调用本技能，生成跨部门数据协调 Commentary。
## 第一步：信息梳理

**核心数据确认：**
```
□ 财务核心数据：
  → 收入：[X] 万（vs 预算 [+/-X]%，vs 去年 [+/-X]%）
  → 毛利：[X] 万（毛利率 [X]%）
  → EBITDA：[X] 万（EBITDA 率 [X]%）
  → 净利润：[X] 万

□ 运营核心数据：
  → [指标1]：[X]（vs 目标 [+/-X]%）
  → [指标2]：[X]（vs 目标 [+/-X]%）
```

**须涵盖的核心信息：**
```
□ 必选内容：
  → 业绩概述：[X] 段
  → 业务亮点：[X] 点
  → 风险说明：[X] 点
  → 下期展望：[X] 段

□ 可选内容（如有特殊事项）：
  → 战略进展：[描述]
  → 重大事件：[描述]
```

---

## 第二步：业绩概述撰写

**业绩概述框架：**
```
□ 第一段（总起）：
  → 本期业绩一句话总结：[描述]
  → 整体评价：[超预期/符合/低于预期]

□ 第二段（收入）：
  → 收入表现：[描述]
  → 增长驱动：[描述 1-2 点]

□ 第三段（利润）：
  → 利润表现：[描述]
  → 关键变化：[描述 1-2 点]
```

**撰写示例（参考风格）：**
```
"本期收入同比增长[X]%，主要得益于[A业务]强劲增长及[B产品]市场份额提升。
毛利率[同比提升/下降][X]个百分点，主要受[原材料价格/产品组合变化]影响。
整体业绩[超预期/符合/低于预期]，主要偏差在于[...]。"
```

---

## 第三步：业务亮点提炼

**亮点提炼（通常 2-4 个）：**
```
□ 亮点 1：[标题] — [数据佐证]
  → 文字表述："[描述，如'某业务线收入同比增长[X]%，连续[X]个季度保持[20%+增速]，得益于[...]']"

□ 亮点 2：[...]
```

**亮点选择标准：**
```
□ 优先选择：
  → 超预期的亮点（vs 预算/目标）
  → 趋势向好的指标（连续改善）
  → 战略举措的进展

□ 谨慎提及：
  → 低于预期但管理层有合理解释
  → 一次性因素导致的正向贡献
```

---

## 第四步：风险说明撰写

**风险说明原则：**
```
□ 平衡性：
  → 既要诚实披露风险，也不制造不必要的焦虑
  → 避免"报喜不报忧"，也避免过度悲观

□ 具体性：
  → 避免泛泛而谈，要有具体数据和影响
  → 说明已采取或计划采取的应对措施
```

**风险说明框架：**
```
□ 已知风险（可量化）：
  → 风险 1：[描述] — 影响 [量化描述] — 应对 [措施]

□ 潜在风险（监控中）：
  → 风险描述：[描述]
  → 监控指标：[指标]
  → 应对准备：[措施]
```

**撰写示例：**
```
"本期[某指标]表现承压，主要受[外部因素A]和[内部因素B]影响。
我们将通过[具体措施]应对，预计将在[时间]改善。
同时，我们密切关注[潜在风险]，已制定[应对预案]。"
```

---

## 第五步：下期展望撰写

**展望撰写原则：**
```
□ 适度乐观但不过度承诺：
  → 提供方向性指引，而非精确预测
  → 说明关键假设和前提条件

□ 可量化 + 定性结合：
  → 量化指引：收入 [X-Y] 万，[+/-X]%
  → 定性指引：重点方向 [描述]
```

**撰写示例：**
```
"展望[下期]，[核心业务]预计保持[稳定/增长]趋势。
基于当前可见的订单和项目，我们预计[下期指标]将[描述]。
上述展望基于以下假设：[假设1]、[假设2]。
我们将持续关注[不确定因素]的发展。"
```

```
【执行摘要】（约 [X] 字）

[文字内容]

【业务亮点】（约 [X] 字）

  [描述，含数据]

• 亮点 2：[标题]
  [描述，含数据]

【风险说明】（约 [X] 字）

• 已知风险：[描述] — 影响 [量化] — 应对 [措施]
• 潜在风险：[描述] — 监控 [指标] — 准备 [预案]

【下期展望】（约 [X] 字）

═══════════════════════════════════════

审阅建议：[CFO 审核重点关注：风险说明措辞 / 下期展望措辞]
═══════════════════════════════════════
```

## 升级触发条件

- CFO 对风险说明措辞有重大修改意见
- 下期展望涉及对外披露（影响投资者预期）
- 文字稿与财务数据存在不一致
- 业绩概述的定性判断与数据表现明显矛盾
- 涉及敏感信息（如未公开的战略举措）
