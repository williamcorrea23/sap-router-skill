# 查询路径 — 商业洞察

## 查询入口

### 查询维度一：业务数据

| 查询目标 | 系统入口 | 查询路径 | 刷新频率 |
|---|---|---|---|
| 销售数据 | [CRM/OPS] | 销售分析 → 业绩看板 | 实时 |
| 客户数据 | [CRM] | 客户分析 → 客户分层 | 实时 |
| 产品数据 | [ERP/OPS] | 产品分析 → 贡献度 | 每日 |
| 运营数据 | [OPS] | 运营仪表板 → 效率指标 | 实时 |

### 查询维度二：财务与市场

| 查询目标 | 系统入口 | 查询路径 | 刷新频率 |

| 财务业绩 | [BI 系统] | 财务分析 → 核心指标 | 每日 |

## 常用查询组合

**场景：收入增长专项分析**
```
1. [CRM] → 销售分析 → 增长来源拆解
2. [ERP] → 收入确认 → 确认收入数据口径
3. [BI 系统] → 趋势分析 → 增长趋势
4. [外部数据] → 行业数据 → 行业增速对比
5. [CRM] → 客户分析 → 客户流失/新增分析
```

**场景：成本优化机会识别**
```
1. [ERP] → 成本中心 → 成本明细
2. [BI 系统] → 成本分析 → 同比/环比
3. [外部数据] → 行业数据 → 行业成本基准
4. [ERP] → 采购数据 → 主要供应商成本变化
```
