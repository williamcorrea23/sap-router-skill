# 查询路径 — 资本配置

## 查询入口

### 查询维度一：投资评估

| 查询目标 | 系统入口 | 查询路径 | 刷新频率 |
|---|---|---|---|
| Capex 项目 | [预算系统] | 投资管理 → Capex 项目 | 实时 |
| M&A 项目 | [项目系统 PJ] | M&A 项目 → 在执行项目 | 实时 |
| 历史投资回报 | [BI 系统] | 投资回报 → 历史 IRR/ROIC | 实时 |
| WACC 计算 | [BI 系统] | 估值工具 → WACC | 实时 |

### 查询维度二：资本结构

| 查询目标 | 系统入口 | 查询路径 | 刷新频率 |
|---|---|---|---|
| 资产负债结构 | [ERP/BI] | 财务结构 → 资产负债 | 每日 |
| 股息政策 | [ERP/司库] | 股东回报 → 股息记录 | 实时 |
| 市值/估值 | [行情系统] | 股票行情 → 市值/PE/PB | 实时 |

## 常用查询组合

**场景：Capex 投资审批**
```
1. [预算系统] → 投资管理 → 查询项目详情
2. [BI 系统] → 投资回报 → 查询 IRR/ROIC 测算
3. [BI 系统] → WACC → 确认当前 WACC
4. [ERP] → 现金流 → 评估资金来源
5. [BI 系统] → 现有投资组合 → 评估资源冲突
```

**场景：资本配置季度审视**
```
1. [BI 系统] → 投资回报 → 汇总所有项目 IRR
2. [BI 系统] → WACC → 确认当前 WACC
3. [ERP/BI] → 资产负债 → 确认当前资本结构
4. [司库系统 TRSY] → 资金计划 → 评估可用资本
5. [BI 系统] → 股东回报 → 确认股息政策执行
```
