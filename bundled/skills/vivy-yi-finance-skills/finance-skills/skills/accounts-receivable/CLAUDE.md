## 客户信用政策

### 客户分级

| 等级 | 评级标准 | 信用额度 | 账期 |
|---|---|---|---|
| A 级 | 战略客户/上市公司/高信用评分 | 最高 [XXX万] | 月结 60-90 天 |
| B 级 | 知名企业/长期合作/良好付款记录 | [XXX万] | 月结 30-45 天 |
| C 级 | 一般企业/新客户/付款记录一般 | [XXX万] | 月结 30 天或现结 |
| D 级 | 问题客户/信用评分低/历史逾期 | [XXX万] 或无信用额度 | 预收款或现结 |

### 信用额度审批

```
新客户首次信用额度审批：
  - ≤ 10 万：AR 主管审批
  - 10-50 万：AR 主管 + 信用管理经理
  - > 50 万：AR 主管 + 信用管理经理 + 财务总监

信用额度调整：
  - 额度内：正常交易
  - 超额度 ≤ 10%：AR 主管审批
  - 超额度 > 10%：须重新评估信用
```

## 收款条款配置

### 标准收款条款

```
标准月结条款：
  - 月结 30 天：发票日期后 30 天内付款
  - 月结 45 天：发票日期后 45 天内付款
  - 月结 60 天：发票日期后 60 天内付款
  - 月结 90 天：发票日期后 90 天内付款（战略客户）

特殊收款条款：
  - 预收款：发货/服务前收款
  - 分期收款：按里程碑分期付款（签约 30%/发货 50%/验收 20%）
  - 票据收款：银行承兑汇票/商业承兑汇票
  - 供应链金融：客户可提前收款（客户承担贴息）
```

### 到期日计算

```
到期日计算规则：
  - 月结 30 天：发票日期 + 30 天
  - 月结 45 天：发票日期 + 45 天
  - 月结 60 天：发票日期 + 60 天

节假日处理：
  - 到期日为周末：顺延至下一工作日
  - 到期日为节假日：提前至前一工作日

宽限期：
  - 标准宽限期：3 天（grace period）
  - 宽限期内不计入逾期
  - 超过宽限期开始计逾期
```

## 催款政策

### 逾期分级催款

| 逾期天数 | 催款方式 | 负责人 |
|---|---|---|---|
| 1-30 天 | 自动短信/邮件提醒 | AR 系统 |
| 31-60 天 | 人工催款函（邮件+电话）| AR 专员 |
| 61-90 天 | 正式催款函（书面）| AR 主管 |
| 91-180 天 | 停止接单 + 法律警告函 | AR 主管 + 法务 |
| > 180 天 | 法律诉讼/坏账核销 | 财务经理 + 法务 |

### 特殊催款规则

```
大额逾期（> [XXX万]）：
  → 逾期 30 天即升级至 AR 主管
  → 逾期 60 天即升级至财务经理

高风险客户识别：
  - 客户信用评级下降
  - 客户涉及法律诉讼/被执行
  - 客户经营异常（裁员/停产/资金链紧张）
  → 发现高风险信号立即停止发货/服务，启动收款程序
```

## 坏账准备政策

### 坏账准备计提规则

```
账龄分析法：
  - 1-30 天：0%（不计提）
  - 31-90 天：10%（一般准备）
  - 91-180 天：30%（较高准备）
  - 181-365 天：50%（高准备）
  - > 365 天：100%（全损）

客户信用等级调整：
  - A 级客户：实际损失率低，计提比例可降低
  - D 级客户：实际损失率高，计提比例可提高

个别计提法：
  - 已知客户财务困难/破产：按估计可收回金额计提
  - 诉讼中客户：按估计损失计提
```

## 审计与合规

### 收入确认规则

```
确认时点：
  - 商品销售：客户验收后确认收入
  - 服务提供：按服务期间或里程碑确认收入
  - 长期合同：按完工百分比法确认收入

确认金额：
  - 确认收入 = 开票金额（含税）
  - 税额不计入收入（增值税价外税）
```

## 地区/行业差异

```
客户规模差异：
  - 上市公司：付款规范，但合同审批周期长
  - 中小企业：付款灵活，但逾期风险较高
  - 政府/国企：账期长（60-90天），但风险低

行业差异：
  - 零售/电商：现结为主，无 AR 问题
  - 制造业：月结 30-60 天，有 AR 管理需求
  - 项目型业务：分期收款里程碑管理
  - 政府/教育：账期长，逾期成本低
```
