# 数据源清单 — [ERP]/[CRM]/[SO]/[INV]/[CASH]

## 数据源标签说明

| 标签 | 含义 | 典型来源 |
|---|---|---|
| [ERP] | 企业 ERP 系统（AR 模块）| SAP FI/AR / 用友 AR / 金蝶 AR |
| [CRM] | 客户关系管理系统 | Salesforce/用友 CRM/纷享销客 |
| [SO] | 销售订单/合同系统 | SAP SD / 用友 U8 / 金蝶 K3 |
| [INV] | 发票系统（销项）| 百望/航天信息/金税盘 |
| [CASH] | 资金/现金管理系统 | 司库系统/资金系统/网上银行 |

---

## [ERP] SAP FI/AR 模块（应收账款）

**覆盖范围：**
- 客户主数据（客户名称/信用额度/账期）
- 应收账款明细（各客户未清余额/账龄）
- 已开票应收（发票行项目）
- 已收款核销（收款凭证）
- 预收账款
- 客户争议标记
- 坏账准备

**查询路径：**

```
SAP 常用事务代码：
- FD10N：客户余额（AR 视图）
- FBL5N：客户行项目（未清项分析）
- FB03：显示凭证（查看收款记录）
- FD03：显示客户主数据

常用报表：
- 客户余额表（按客户）
- 账龄分析表（按账龄区间）
- 逾期分析表（已逾期项）
- TOP 客户应收排名
```

---

## [CRM] 客户关系管理系统

**覆盖范围：**
- 客户主数据（公司信息/联系人/地址）
- 信用评级（内部评级/外部评级）
- 信用额度（已用额度/可用额度）
- 销售合同（付款条款/账期）
- 客户沟通记录（催款历史/还款承诺）
- 客户风险信号（诉讼/经营异常）

**查询路径：**

```
CRM 系统：
- 客户管理 → 客户列表 → 查看信用信息
- 合同管理 → 销售合同 → 查看付款条款
- 跟进记录 → 催款沟通历史

外部数据查询：
- 天眼查/企查查：工商信息/股东/法律诉讼
- 中国执行信息公开网：http://zxgk.court.gov.cn
- 裁判文书网：https://wenshu.court.gov.cn
```

---

## [SO] SAP SD / 用友 / 金蝶（销售订单）

**覆盖范围：**
- 销售订单主数据（客户/产品/数量/价格）
- 订单状态（已确认/已发货/已开票/已完成）
- 收入确认记录（按收入确认时点）
- 交付进度（已发货数量/未发货数量）

**查询路径：**

```
SAP SD 常用事务代码：
- VA01：创建销售订单
- VA03：显示销售订单
- VL03N：显示交货单
- VF04： billing（开票）

用友/金蝶：
- 销售管理 → 销售订单查询
- 库存管理 → 销售出库单
- 应收管理 → 应收单据
```

---

## [INV] 开票系统（销项发票）

**覆盖范围：**
- 已开具发票（发票代码/号码/金额/日期/客户）
- 发票状态（正常/作废/红冲）
- 未开票应收（已确认收入但未开票）
- 电子发票

**查询路径：**

```
开票系统：
- 销项发票查询 → 按客户/按期间筛选
- 发票统计 → 月度/季度汇总
- 作废/红冲记录

常用报表：
- 销项发票汇总表（月度）
- 销项发票明细表（逐张）
- 未开票应收明细
```

---

## [CASH] 资金/现金管理系统

**覆盖范围：**
- 银行账户余额
- 银行流水（到账记录）
- 到账认领（AR 收款核销）
- 承兑汇票收付记录

**查询路径：**

```
司库系统（如有）：
- 到账查询 → 查看银行到账记录
- 付款指令 → 确认是否付款

网上银行：
- 企业网银 → 历史流水查询
- 到账通知 → 确认到账信息

ERP 资金模块：
- 收款认领 → 匹配客户和发票
- 预收账款 → 管理预收款
```

## 数据源组合使用

### 账龄分析
```
[ERP] 应收余额 → 按客户/按账龄汇总 → 生成账龄分析表
```

### 逾期催收
```
[ERP] 逾期清单 → [CRM] 客户信用信息 → [OA] 催款函发送记录 → 制定催款策略
```

### 客户信用评估
```
[CRM] 客户信息 → [ERP] 历史付款记录 → 天眼查/企查查 → 外部风险信号 → 评估信用额度
```

### 收款核销
```
[CASH] 到账流水 → [ERP] 收款认领 → 匹配 [INV] 发票 → 核销应收
```
