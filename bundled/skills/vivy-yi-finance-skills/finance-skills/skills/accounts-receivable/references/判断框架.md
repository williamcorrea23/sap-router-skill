# 判断框架 — 应收账款异常处理

## 核心原则

**应收账款管理的目标是"及时收回每一分钱，同时维护好客户关系"。**
遇到异常时，优先判断是"客户客观困难"还是"客户主观拖欠"，对高风险信号保持警惕。

---

## 异常类型与处理方式

### 类型一：客户逾期不付款

**典型场景：**
- 客户到期无理由不付款
- 客户以各种理由拖延付款（质量/服务问题）
- 客户提出不合理扣款要求

**判断流程：**

```
第一步：判断逾期原因

客观困难（客户问题）：
  - 客户资金周转困难
  - 客户内部付款流程长（国企/上市公司）
  - 客户对账单/发票有争议

主观拖欠（恶意拖欠）：
  - 客户有钱但不付
  - 客户故意拖延付款以获取现金流
  - 客户濒临破产想赖账

第二步：判断处理策略

客观困难：
  → 可适当延长账期（给予还款计划）
  → 评估客户未来还款能力
  → 必要时停止新订单

主观拖欠：
  → 态度坚定，限期付款
  → 停止新订单/发货
  → 必要时启动法律程序
```

---

### 类型二：客户信用评级下调

**典型信号：**
- 客户财务指标恶化（营收下降/亏损/现金流紧张）
- 客户涉及法律诉讼/被执行
- 客户高管频繁变动
- 客户被银行收紧授信

**判断标准：**

```
风险信号等级：

🔴 高风险信号：
  - 客户被申请破产/已进入破产程序
  - 客户实际控制人失联
  - 客户涉及大额诉讼（金额 > 我方应收款）
  - 客户银行账户被冻结

⚠️ 中风险信号：
  - 客户连续 2 年亏损
  - 客户裁员 > 30%
  - 客户被供应商追债（拖欠供应商货款）
  - 客户信用评级被外部机构下调

应对措施：

高风险信号：
  → 🔴 立即停止发货/服务
  → 成立专项收款小组
  → 准备法律诉讼材料
  → 评估坏账风险

中风险信号：
  → ⚠️ 收紧信用额度
  → 立即启动收款程序
  → 要求客户就存量应收提供担保
```

---

### 类型三：客户要求变更收款账户

**典型场景：**
- 客户要求将款项付至新账户
- 客户要求付至第三方账户
- 客户要求付至个人账户

**判断标准：**

```
🔴 高风险信号（立即暂停）：
  - 要求付至非客户名下账户（第三方账户）
  - 要求付至个人账户
  - 要求付至非客户注册地的异地账户
  - 变更理由不充分或无法核实

处理步骤：
  1. 🔴 立即暂停向该账户付款
  2. 联系客户原对接人核实
  3. 要求客户提供书面变更申请（加盖公章）
  4. 核实新账户是否为客户合法账户
  5. 如无法核实，升级至财务经理
```

---

### 类型四：客户争议（质量/数量/服务问题）

**典型场景：**
- 客户以质量问题拒付
- 客户以数量不足拒付
- 客户以服务不满意拒付
- 客户要求先开红字发票再付款

**处理流程：**

```
第一步：确认争议性质

质量问题：
  → 要求客户提供质量检验报告
  → 我方核查是否属实
  → 如属实：协商折让金额

数量问题：
  → 核查出库单/物流单据
  → 确认实际交付数量
  → 如确为我方问题：按实际数量结算

服务问题：
  → 核查服务合同/验收标准
  → 确认服务是否达标
  → 如未达标：协商解决方案

第二步：谈判策略

优先和解（部分折让后收款）：
  → 折让金额 ≤ 应收金额的 [X]%
  → 和解须有书面协议
  → 协议中明确"收到款项后双方无争议"

坚持原立场（证据充分时）：
  → 向客户出示合同/出库单/验收单
  → 态度坚定，限期付款
  → 如客户坚持，启动法律程序

第三步：收款决策

无争议部分：
  → 要求客户先付无争议部分
  → 有争议部分另案处理

争议部分：
  → 协商解决（折让/补货/服务补偿）
  → 或通过法律途径解决
```

### 类型五：坏账风险评估

**典型场景：**
- 客户已逾期 > 180 天
- 客户进入破产程序
- 客户确认无力偿还

**坏账判断标准：**

| 逾期天数 | 坏账概率 | 处理方式 |
|---|---|---|
| 1-30 天 | < 5% | 正常催款 |
| 31-90 天 | 5-15% | 加大催款力度 |
| 91-180 天 | 15-40% | 准备法律诉讼 |
| 181-365 天 | 40-70% | 法律诉讼/催款 |
| > 365 天 | > 70% | 核销坏账 |

**核销审批权限：**
```
坏账核销审批：
  - ≤ 5 万：AR 主管审批
  - 5-20 万：AR 主管 + 财务经理
  - > 20 万：AR 主管 + 财务经理 + 财务总监 + CEO
```

---

## 置信度评估

| 置信度 | 条件 | 操作 |
|---|---|---|
| 高 | 客户正常经营 + 无逾期 + 无风险信号 | ✅ 正常管理 |
| 中 | 有轻微逾期或低风险信号 | ⚠️ 关注催款 |
| 低 | 高风险信号或大额逾期 | 🔴 立即处理 |

---

## 升级触发条件

- 客户逾期 > 90 天且无还款计划
- 单一客户逾期 > [填空：如 100 万元]
- 客户要求变更收款账户（高风险）
- 客户涉及法律诉讼/破产
- 客户信用评级下调至 D 级
- 须核销坏账 > [填空：如 10 万元]
- 客户经营出现重大异常（停产/裁员/资金链断裂）
