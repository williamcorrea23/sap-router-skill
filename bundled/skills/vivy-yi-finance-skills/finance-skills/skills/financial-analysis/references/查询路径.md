# 查询路径 — 财务分析

## 数据源优先级

| 优先级 | 数据源 | 说明 |
|---|---|---|
| 1 | ERP 总账（[ERP]）| 科目余额/凭证/财务报表 | 最优先 |
| 2 | BI 系统（[BI]）| 可视化仪表盘/分析报表 | 分析展示 |
| 3 | 预算管理系统（[BK]）| 预算数据/预算执行 | 对比基准 |
| 4 | CRM 系统（[CRM]）| 销售数据/客户数据 | 收入分析 |
| 5 | HR 系统（[HR]）| 人力成本/员工数据 | 费用分析 |

---

## 常用查询场景

### 场景 1：利润表分析

**查询入口：**
```
ERP：
- 利润表（管理口径）：按部门/产品线细分
- 科目余额表：收入/成本/费用明细
- 辅助核算报表：按项目/客户/部门核算

BI 系统：
- 管理利润表仪表盘
- 部门损益仪表盘
- 产品线毛利仪表盘
```

**利润表分析维度：**
```
□ 收入分析：
  - 收入 vs 预算：[XXX万] vs [XXX万]（[X%]）
  - 收入 vs 同期：[XXX万] vs [XXX万]（[X%]）
  - 收入环比：[X%]

□ 成本分析：
  - 成本 vs 预算：[XXX万] vs [XXX万]（[X%]）
  - 成本 vs 同期：[XXX万] vs [XXX万]（[X%]）
  - 毛利率：[X%] vs 预算 [X%]

□ 费用分析：
  - 费用 vs 预算：[XXX万] vs [XXX万]（[X%]）
  - 费用率：[X%] vs 预算 [X%]
  - 主要超支科目：[...]

□ 利润分析：
  - 毛利 vs 预算：[XXX万] vs [XXX万]（[X%]）
  - 营业利润 vs 预算：[XXX万] vs [XXX万]（[X%]）
  - 净利润 vs 预算：[XXX万] vs [XXX万]（[X%])
```

## 数据源组合使用

### 月度财务分析
```
[ERP] 科目余额 → [BK] 预算数据 → [CRM] 销售数据 → 生成管理利润表
```

### 现金流分析
```
[ERP] 现金凭证 → [ERP] 应收应付 → [ERP] 存货 → 生成现金流量表
```

### KPI 监控
```
[ERP] 实时数据 → [BI] 仪表盘 → 异常预警 → 生成分析报告
```
