# 数据源清单 — [ERP]/[BI]/[BK]/[CRM]/[HR]

## 数据源标签说明

| 标签 | 含义 | 典型来源 |
|---|---|---|
| [ERP] | 企业 ERP 系统（总账）| SAP/用友/金蝶总账 |
| [BI] | BI 可视化系统 | Power BI/Tableau/帆软/用友 BI |
| [BK] | 预算管理系统 | SAP BPC/用友预算/金蝶预算 |
| [CRM] | 客户关系管理系统 | Salesforce/纷享销客/用友 CRM |
| [HR] | 人力资源系统 | 薪酬系统/考勤系统 |

---

## [ERP] SAP/用友/金蝶（总账）

**覆盖范围：**
- 科目余额表（收入/成本/费用/资产/负债）
- 辅助核算（部门/项目/客户/供应商）
- 会计凭证（逐笔明细）
- 财务报表（资产负债表/利润表/现金流量表）
- 管理利润表（按部门/产品线）

**查询路径：**

```
SAP 常用事务代码：
- FB03：显示凭证
- FAGLB03：总账科目余额报表
- S_ALR_00001234：管理利润表
- S_ALR_00001235：部门损益表
- FBL5N/FBL1N：客户/供应商行项目

用友/金蝶：
- 财务报表 → 利润表/资产负债表
- 辅助核算 → 部门/项目/客户明细
- 管理报表 → 部门利润表
```

---

## [BI] Power BI / Tableau / 帆软

**覆盖范围：**
- KPI 仪表盘（实时更新）
- 趋势分析图表（收入/毛利/费用趋势）
- 预算执行仪表盘
- 部门/产品/客户分析视图
- 自定义分析报表

**查询路径：**

```
Power BI：
- 工作区 → 财务分析文件夹
- 报表 → KPI 仪表盘
- 数据集 → 刷新状态

Tableau：
- 工作簿 → 财务分析工作簿
- 数据源 → 刷新状态

帆软：
- 决策管理系统 → 财务分析模块
```

---

## [BK] SAP BPC / 用友预算 / 金蝶预算

**覆盖范围：**
- 年度预算（收入/成本/费用预算）
- 预算版本（实际/预测/调整后）
- 预算执行进度
- 滚动预测
- 预算差异分析

**查询路径：**

```
SAP BPC：
- 预算编制 → 收入预算/成本预算
- 预算执行 → 执行进度报表
- 预算分析 → 差异分析

用友/金蝶：
- 预算管理 → 预算编制
- 预算执行 → 执行分析
- 滚动预测 → 预测模型
```

---

## [CRM] Salesforce / 纷享销客 / 用友 CRM

**覆盖范围：**
- 客户主数据（名称/分级/行业）
- 销售订单/合同
- 收入确认记录
- 客户贡献分析
- 销售管道（pipeline）

**查询路径：**

```
CRM 系统：
- 客户管理 → 客户列表
- 销售分析 → 客户贡献
- 销售管道 → 预计收入
- 合同管理 → 合同金额/账期
```

---

## [HR] 薪酬/考勤系统

**覆盖范围：**
- 员工人数/部门分布
- 薪酬总额（工资/社保/公积金）
- 人力成本分析
- 招聘计划/人员编制

**查询路径：**

```
HR 系统：
- 薪酬报表 → 人力成本汇总
- 人员编制 → 各部门人数
- 考勤数据 → 加班/出差统计

ERP：
- 工资凭证 → 人工成本
- 社保公积金 → 员工福利费
```

## 数据源组合使用

### 月度利润表分析
```
[ERP] 科目余额（收入/成本）→ [BK] 预算数据 → 对比分析
```

### 部门损益分析
```
[ERP] 辅助核算（按部门）→ [HR] 部门人数 → 人均效能分析
```

### 现金流分析
```
[ERP] 现金凭证 → [ERP] 应收应付变动 → [BI] 现金流仪表盘
```

### 产品/客户贡献分析
```
[CRM] 销售数据 → [ERP] 产品线核算 → [BI] 产品仪表盘
```
