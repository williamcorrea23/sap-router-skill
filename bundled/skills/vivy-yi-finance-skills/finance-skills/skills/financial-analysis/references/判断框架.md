# 判断框架 — 财务分析异常处理

## 核心原则

**财务分析的目标是"用数据说话，找到业务问题的根因"。**
遇到异常指标时，优先判断是"数据口径问题"、"短期波动"还是"业务实质变化"，再决定是否须深入分析。

---

## 异常类型与处理方式

### 类型一：毛利率异常（下降/低于目标）

**典型场景：**
- 毛利率环比下降 [X]%
- 毛利率低于目标值 [X%]
- 毛利率同比下降 [X]%

**判断流程：**

```
第一步：确认数据准确性

□ 数据来源是否正确（ERP 报表取数口径）？
□ 是否有会计政策变更影响？
□ 是否有一次性事项影响（滞销库存核销/资产减值）？
□ 成本分摊规则是否变更？

→ 如数据有问题：修正数据后再分析

第二步：判断是"量"还是"价"的问题

□ 销量变化：实际销量 vs 预算/同期
□ 售价变化：实际单价 vs 预算/同期
□ 单位成本变化：实际单位成本 vs 预算/同期

□ 收入下降 + 成本上升 = 量价双杀（最严重）
□ 收入下降 + 成本不变或下降 = 需求问题
□ 收入不变 + 成本上升 = 成本控制问题
□ 收入下降幅度 > 成本下降幅度 = 边际贡献减少

第三步：归因到具体产品/业务线

□ 哪个产品线毛利率下降最多？
□ 哪个客户贡献了主要变化？
□ 哪个部门/成本中心超支最多？

第四步：区分"战略性"还是"失控"

战略性毛利下降：
  → 产品结构优化（主动淘汰低毛利产品）
  → 为获取大客户接受短期毛利下降
  → 新市场开拓投入期
  → 须有明确的战略目标和回报计划

失控性毛利下降：
  → 成本上涨未及时传导至售价
  → 定价混乱/折扣失控
  → 原材料浪费/生产效率下降
  → 须立即制定改善计划
```

### 类型二：收入增长异常（下降/不及目标）

**典型场景：**
- 收入同比下降 > [X]%
- 收入环比下降 > [X]%
- 收入增速远低于目标/行业

**判断流程：**

```
第一步：确认数据准确性

□ 新增客户/新上线产品是否已计入？
□ 退费/返利/折扣是否已扣除？
□ 季节性因素是否考虑？（环比需谨慎）
□ 汇率变动是否影响？（如有海外收入）

→ 如数据有问题：修正数据后再分析

第二步：判断"新收入"还是"老收入"问题

新收入问题（获客/产品）：
  → 新客户获取减少
  → 新产品上市表现不及预期
  → 市场策略调整导致收入延迟

老收入下降（留存/复购）：
  → 客户流失率上升
  → 现有客户购买频次下降
  → 现有客户购买品类减少

□ 流失客户数：[X] 家
□ 流失收入贡献：[XXX万]
□ 留存客户收入变化：[X%]

第三步：按维度拆分

□ 哪个业务线/产品线收入下降最多？
□ 哪个区域/渠道收入下降最多？
□ 哪个客户群贡献下降最多？

第四步：结合市场/竞争分析

□ 行业整体增速是否放缓？
□ 主要竞争对手是否增长？
□ 市场份额是否下降？
□ 是否有替代品/新进入者？

□ 如行业整体下行：分析公司抗风险能力
□ 如竞争对手增长：公司竞争力问题
□ 如市场整体下降：考虑业务转型
```

### 类型三：费用率异常（费用增长快于收入）

**典型场景：**
- 费用增长率 > 收入增长率 [X]%
- 某项费用率大幅上升
- 费用预算执行率 > [X]%

**判断流程：**

```
第一步：判断费用类型

固定费用（相对稳定）：
  → 人员成本（工资/社保）
  → 房租/折旧
  → 管理费用

可变费用（随收入变动）：
  → 销售佣金/提成
  → 运输费/物流费
  → 包装费

半固定费用（有阶段跳跃）：
  → 市场推广费
  → 研发费用
  → IT 费用

→ 可变费用增长 > 收入增长：销售效率问题
→ 固定费用增长 > 收入增长：人员/规模失控
→ 半固定费用增长 > 收入增长：扩张是否合理？

第二步：定位到具体科目

□ 哪个费用科目增长最多？
□ 哪个部门费用超支最多？
□ 是否与战略投入相关（主动扩张）？

第三步：判断是否"战略性投入"

战略性投入（合理）：
  → 新市场开拓（市场费用）
  → 新产品研发（研发费用）
  → 人才储备（招聘/培训费用）
  →须有明确的投入产出预期

失控性增长（不合理）：
  → 人员扩张未带来相应产出
  → 费用审批流程缺失
  → 采购浪费/采购价上升
  → 须立即控制费用

第四步：制定改善计划

如为失控性增长：
  → 立即冻结非必要招聘
  → 加强费用审批
  → 评估供应商议价
  → 制定季度费用控制目标
```

### 类型四：现金流异常（流入/流出与利润不匹配）

**典型场景：**
- 净利润为正但经营现金流为负
- 经营现金流持续低于净利润
- 现金流大幅下降但利润未变

**判断流程：**

```
第一步：确认数据准确性

□ 是否完成银行调节？
□ 是否有大额在途资金？
□ 是否有未记账的收支？

第二步：分析利润 → 现金的差异

标准差异项：
□ 折旧/摊销（非现金费用，加回）
□ 营运资金变化：
  - 存货增加（现金占用，增加）
  - 应收账款增加（现金未收，增加）
  - 应付账款增加（现金未付，现金增加）
  - 预收账款增加（现金增加）

□ 大额非经营收支：
  - 投资收益（利润有但无现金）
  - 资产处置损益
  - 政府补贴

第三步：判断原因

净利润 > 经营现金流（差异大）：

□ 营运资金大幅增加：
  → 存货积压（制造业/贸易）
  → 客户回款变慢（DSO 上升）
  → 供应商账期缩短（DPO 下降）
  → 预收减少（客户议价能力下降）

□ 非现金费用过大：
  → 折旧摊销过高
  → 资产减值过大
  → 投资收益过大（主业盈利差）

□ 一次性事项：
  → 补缴税款
  → 预付大额费用
  → 预付供应商款项

第四步：风险评估

□ DSO 是否持续上升？（客户议价能力弱）
□ 存货周转天数是否上升？（产品滞销）
□ DPO 是否下降？（供应商议价能力弱）

□ 经营现金流持续为负 + 净利润为正：
  → 利润质量差
  → 须加速回款/控制存货
```

### 类型五：资产负债率持续上升

**典型场景：**
- 资产负债率 > 60%，且持续上升
- 负债增长快于资产增长
- 短期负债占比过高

**判断流程：**

```
第一步：区分负债类型

经营负债（合理）：
  → 应付账款（供应商融资，正常商业信用）
  → 预收账款（客户预付款，竞争力体现）
  → 应交税费（税务负债，正常）

金融负债（风险）：
  → 银行贷款（有息负债，有成本）
  → 债券（有息负债，有成本）
  → 租赁负债（有息负债）

→ 经营负债占比高：正常商业行为，风险低
→ 金融负债占比高：融资激进，风险高

第二步：判断负债用途

□ 负债用于：经营扩张（正常）
□ 负债用于：弥补亏损（危险）
□ 负债用于：股东分红（危险）
□ 负债用于：投资（需评估回报）

第三步：偿债能力评估

□ 流动比率是否 > 1.5？
□ 速动比率是否 > 1.0？
□ 经营现金流是否稳定？
□ 银行授信额度是否充足？

□ 偿债能力充足：风险可控
□ 偿债能力不足：🔴 高风险
```

## 置信度评估

| 置信度 | 条件 | 操作 |
|---|---|---|---|
| 高 | 数据完整 + 上月结账完成 + 口径一致 | ✅ 正常分析 |
| 中 | 数据完整，但部分口径调整 | ⚠️ 标注口径差异 |
| 低 | 数据不完整或上月未结账 | 🔴 暂停分析，核查 |

## 升级触发条件

- 毛利率低于预警值连续 2 个月
- 收入同比下降 > [X]%（[填空：如 10%]）
- 资产负债率 > 70%
- 流动比率 < 1.0
- 单月亏损 > [填空：如 50 万]
- 经营现金流持续为负 > 3 个月
