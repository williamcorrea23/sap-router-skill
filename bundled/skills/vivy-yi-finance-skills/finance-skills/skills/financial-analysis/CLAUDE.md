## KPI 体系配置

### 盈利能力指标

```
□ 毛利率 =（收入 - 成本）/ 收入
  - 目标值：[X%]
  - 预警值：[低于 X%] → ⚠️ 关注
  - 危险值：[低于 X%] → 🔴 预警

□ 营业利润率 = 营业利润 / 收入
  - 目标值：[X%]
  - 预警值：[低于 X%]
  - 危险值：[低于 X%]

□ 净利率 = 净利润 / 收入
  - 目标值：[X%]
  - 预警值：[低于 X%]
  - 危险值：[低于 X%]

□ 收入增长率 =（本期收入 - 上期收入）/ 上期收入
  - 目标值：[X%]
  - 预警值：[低于 X%]
  - 危险值：[低于 X% 或下降]
```

### 运营效率指标

```
□ 存货周转天数 = 365 / 存货周转率
  - 目标值：[X] 天
  - 预警值：[超过 X] 天 → 存货积压

□ 应收账款周转天数（DSO）= 365 / 应收账款周转率
  - 目标值：[X] 天
  - 预警值：[超过 X] 天 → 回款慢

□ 应付账款周转天数（DPO）= 365 / 应付账款周转率
  - 目标值：[X] 天
  - 预警值：[超过 X] 天 → 付款太快/供应商议价能力弱

□ 总资产周转率 = 收入 / 平均总资产
  - 目标值：[X] 次
  - 预警值：[低于 X] 次 → 资产利用效率低
```

### 财务风险指标

```
□ 资产负债率 = 总负债 / 总资产
  - 安全值：< 60%
  - 预警值：60-70%
  - 危险值：> 70%

□ 流动比率 = 流动资产 / 流动负债
  - 安全值：> 1.5
  - 预警值：1.0-1.5
  - 危险值：< 1.0

□ 速动比率 =（流动资产 - 存货）/ 流动负债
  - 安全值：> 1.0
  - 预警值：0.5-1.0
  - 危险值：< 0.5

□ 现金比率 = 现金 / 流动负债
  - 安全值：> 20%
  - 预警值：10-20%
  - 危险值：< 10%
```

### 成长能力指标

```
□ 收入增长率 =（本期 - 上期）/ 上期 × 100%
  - 目标值：[X%]
  - 预警值：[低于 X%]
  - 危险值：[负增长]

□ 利润增长率 =（本期 - 上期）/ 上期 × 100%
  - 目标值：[X%]
  - 预警值：[低于 X%]
  - 危险值：[下降]

□ 净资产增长率 =（期末 - 期初）/ 期初 × 100%
```
