# 判断框架 — 报销审核异常处理

## 核心原则

**报销审核没有"统一自动通过"规则，只有"通过/补正/驳回/升级"四选一的判断。**

遇到模糊情形时，优先原则是：宁可升级，不可漏放。

---

## 异常类型与处理方式

### 类型一：发票与实际业务不符

**典型场景：**
- 发票品名与报销内容不一致（开的是"办公用品"但报销的是"礼品"）
- 发票金额与付款记录不一致（报销 1000 元但付款记录 800 元）
- 发票日期早于出差申请日期（先有发票再有申请，逻辑矛盾）

**处理方式：**
```
第一步：要求员工提供补充证明（合同/付款截图/说明）
第二步：判断补充材料是否能合理解释差异
  - 能解释 → 降级为"中置信度"，标注"补充材料后通过"
  - 不能解释 → 驳回，标注"发票与实际业务不符"
第三步：同一员工出现 2 次及以上 → 自动升级至财务专员
```

---

### 类型二：超标准报销

**典型场景：**
- 住宿费超标（城市 A 类 500 元，实际报销 700 元）
- 餐饮超标（单次招待超过 500 元/人）
- 职级超标（基层员工按中层标准报销）

**处理方式：**

| 超标幅度 | 处理方式 |
|---|---|
| 5% 以内 | ⚠️ 标注超标，提示"可提交说明申请特批" |
| 5%-20% | 要求直属上级加签确认，说明业务必要性 |
| 20% 以上 | 驳回，要求重新报销或申请专项审批 |
| 虚假超标（事后修改票据）| 驳回 + 记录诚信档案 + 升级至财务经理 |

---

### 类型三：审批流程缺失

**典型场景：**
- 须事前申请的差旅，报销时无申请记录
- 事后补申请的出差（出发前未申请，事后补流程）
- 审批人被漏填（如须两级审批但只有一级）

**处理方式：**
```
第一步：判断缺失类型
  - 缺少事前申请 → 驳回，提示"须先完成出差申请再报销"
  - 审批人被漏填 → 退回员工，要求补充加签
  - 申请人顶替审批人（自己审自己）→ 驳回 + 记录异常

第二步：以下情形可特批
  - 紧急出差（出发前无法申请）→ 提供紧急情况说明 + 事后补申请
  - 外部不可抗力（天气/航班取消）→ 提供证明材料
```

---

### 类型四：跨期报销

**典型场景：**
- 4 月出差，5 月才提交报销（超 30 天）
- 上月费用本月才报销，但跨了季度

**处理方式：**

| 跨期时长 | 处理方式 |
|---|---|
| 30 天以内 | ✅ 正常通过（制度通常允许 30 天内报销）|
| 30-90 天 | ⚠️ 提示"超期"，要求说明延迟原因 |
| 90 天以上 | 驳回，提示"超过制度报销时限，联系财务申请特批" |
| 跨季度/年度 | 🔴 升级至财务经理，确认是否影响当期损益和预算 |

---

### 类型五：发票验真异常

**典型场景：**
- 发票在国家税务总局平台验真为"不存在"
- 发票状态显示"作废"或"红冲"
- 专票已被异地认证（失控发票）

**处理方式：**
```
验真平台：[https://inv-veri.chinatax.gov.cn](https://inv-veri.chinatax.gov.cn)

验真结果 = 不存在 / 作废 / 红冲 → ❌ 驳回，标注"发票无效"
验真结果 = 已认证（失控发票）→ ❌ 驳回 + 升级至财务经理
验真结果 = 正常，但金额/税号存疑 → 🔴 升级至税务专员
验真平台不可用（网络原因）→ 引导用户手动查验，提供截图
```

---

### 类型六：预算不足

**典型场景：**
- 报销时系统显示对应预算科目余额不足
- 当月预算已用完，但费用确实发生

**处理方式：**
```
第一步：确认预算类型
  - 刚性预算（不可超）→ 驳回报销，提示"须申请预算调整后再报销"
  - 柔性预算（可弹性）→ 提示超标额度，流转至财务经理审批

第二步：如有跨部门调拨可能
  - 引导申请人查询其他部门是否有闲置预算
  - 如有，填写预算调拨申请，再提交报销

第三步：特批路径
  - 财务经理可在权限内批准超预算报销
  - 超 20% 须 CFO 特批
```

## 置信度评估表

| 置信度 | 条件 | 操作 |
|---|---|---|
| 高 | 发票验真 ✅ + 标准明确 ✅ + 审批完整 ✅ + 预算充足 ✅ | ✅ 自动通过 |
| 中 | 验真 ✅ + 标准明确，但缺少一项辅助材料 | ⚠️ 标注提示，人工复核 |
| 低 | 标准模糊（无明确依据），但发票和审批均正常 | ⚠️ 降级人工复核 |
| 无 | 发票存疑 / 审批缺失 / 逻辑矛盾 | ❌ 不通过，要求补正 |

---

## 升级触发器

满足以下任一条件，自动升级至财务专员，不自行判断：

- 同一员工同一自然月内报销异常 ≥ 3 次
- 单笔金额超过 [填空：如 10 万元]
- 发票验真异常（假票/失控票）
- 疑似虚报（金额与业务明显不符）
- 涉及合同/采购的三单匹配争议
- 跨季度/跨年度的跨期报销
- 员工对驳回结论提出申诉
