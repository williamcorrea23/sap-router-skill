# 数据源清单 — [INT]/[ERP]/[OA]/[TAX]/[GOV]

## 数据源标签说明

| 标签 | 含义 | 典型来源 |
|---|---|---|
| [INT] | 企业内部制度/文档 | 报销制度/合同/SOP |
| [ERP] | 企业 ERP 系统 | SAP/用友/金蝶 |
| [OA] | OA 审批系统 | 钉钉/飞书/企微 |
| [FK] | 费控平台 | 分贝通/胜意/Coupa |
| [TAX] | 税务官方平台 | 国家税务总局 |
| [GOV] | 政府公开数据 | 天眼查/国家企业信用系统 |
| [MCP] | MCP 服务（已配置）| yuandian-law / pkulaw 等 |

---

## [INT] 企业内部制度/文档

**覆盖范围：**
- 费用报销制度（含标准表格）
- 差旅管理政策
- 采购管理制度
- 合同审批 SOP
- 公司职级体系文件

**获取方式：**
```
优先：从公司内网文档库/知识库读取
次优：要求用户提供最新版 PDF
```

**质量评估：**
```
时效性：⭐⭐⭐⭐  （以最新版本为准，注意制度版本号）
权威性：⭐⭐⭐⭐⭐ （公司内部最高标准）
可获取性：⭐⭐⭐  （制度通常为内部文档，须确保有权限）
```

## [ERP] SAP FI / CO 模块

**覆盖范围：**
- 供应商主数据（名称/银行账户/税号）
- 员工主数据（职级/部门/报销权限）
- 历史报销记录（近 3 年）
- 成本中心/项目编码映射
- 预算余额（CO 模块）

**查询路径：**

```
SAP FI（财务会计）:
- FK01/FK02/FK03：供应商主数据
- FB03：显示凭证
- FBL1N：供应商行项目（应付明细）

SAP CO（管理会计）:
- KOl1/KOb1：成本中心计划/实际对比
- CJ30：项目预算
- KB11N：手动预算过账

用友 U8:
- 应收/应付模块：供应商明细
- 总账模块：科目余额表/凭证查询
- UFO1：自定义报表

金蝶 K3:
- 应付款管理：供应商余额
- 总账：凭证/报表
- 预算管理：预算余额查询
```

**不覆盖：**
- OA 审批流程状态（须从 [OA] 获取）
- 发票影像（须从影像管理系统获取）
- 实时预算占用（须从 [FK] 确认）

---

## [OA] 钉钉/飞书/企业微信

**覆盖范围：**
- 出差申请记录（申请人/目的地/天数/预算）
- 审批历史（审批人/审批时间/审批意见）
- 合同审批记录
- 费用申请与报销关联关系

**查询路径：**

```
钉钉：
- 管理后台 → 审批记录 → 搜索申请人/单号
- 审批模板 ID：出差申请/费用报销/合同审批

飞书：
- 管理后台 → 审批记录 → 同上
- 支持通过 API 按日期/状态筛选

企业微信：
- 管理后台 → 审批 → 同上
- 支持按部门/申请人筛选
```

**不覆盖：**
- ERP 中的会计凭证信息（须从 [ERP] 获取）
- 发票影像（须上传附件或影像系统）
- 预算余额（须从 [FK] 确认）

---

## [FK] 分贝通/胜意费控/Coupa

**覆盖范围：**
- 实时预算余额（按科目/部门/项目）
- 预算占用（已申请未报销金额）
- 差旅标准校验（事前申请时自动校验）
- 供应商黑名单/白名单
- 费用标准引擎（自动判断是否超标）

**查询路径：**

```
分贝通：
- 企业后台 → 预算管理 → 部门/项目预算查询
- 支持 API：查询指定成本中心的预算余额

胜意费控：
- 预算中心 → 预算查询 → 按维度筛选
- 支持 ERP 对接：实时同步 SAP/用友数据

Coupa（外资企业常用）:
- 预算模块 → Budget Analysis
- 支持 What-if 场景模拟
```

**不覆盖：**
- 历史凭证信息（须从 [ERP] 获取）
- OA 审批流详情（须从 [OA] 获取）

---

## [TAX] 国家税务总局发票查验

**覆盖范围：**
- 增值税专用发票/普通发票真伪
- 发票状态（正常/作废/红冲/失控）
- 发票票面信息（金额/税号/品名）

**查询入口：**
```
https://inv-veri.chinatax.gov.cn

API（已配置 MCP 时可用）：
- 自动传入发票代码 + 号码 + 日期 + 金额
- 返回：一致/不一致/不存在
```

**不覆盖：**
- 发票影像（须要求员工上传扫描件）
- 发票与合同的匹配关系（须人工判断）
- 进项税额抵扣状态（须从税局系统查询）

---

## [GOV] 天眼查/启信宝/国家企业信用系统

**覆盖范围：**
- 供应商工商信息（注册资本/成立日期/经营状态）
- 是否在经营异常名录
- 严重违法失信名单
- 股东穿透（关联交易识别）
- 法定代表人限制高消费

**查询路径：**

```
天眼查（VIP）：
- https://www.tianyancha.com
- 支持按公司名/统一社会信用代码精确查询

启信宝：
- https://www.qixin.com
- 支持批量查询和导出

国家企业信用信息公示系统（免费）：
- http://www.gsxt.gov.cn
- 查询：经营异常/严重违法/行政处罚
```

## 数据源组合使用

### 常规报销审核
```
[INT]报销制度 → [ERP]员工职级 → [OA]出差申请 → [FK]预算余额 → [TAX]发票验真
```

### 采购类三单审核
```
[OA]合同审批 → [ERP]供应商主数据 → [GOV]供应商工商核查 → [OA]收货确认 → [ERP]付款记录
```

### 异常发票审核
```
[TAX]发票验真 → [ERP]历史记录核查 → [GOV]供应商背景核查 → [OA]审批历史
```

## Connector 映射

| 数据需求 | Connector | 工具 | 关键字段 |
|---------|-----------|------|---------|
| ERP 总账/供应商 | SAP | `sap_gl_query` | `LIFNR`, `NAME1`, `KOSTL`, `BELNR` |
| 差旅申请/审批 | SAP Concur | `concur_expense_list` | `reportId`, `status`, `amount`, `policyViolation` |
| 供应商工商核查 | 天眼查 | 手动/API | 注册资本/经营状态/股东穿透 |
| 发票验真 | 税务局 API | `tax_invoice_verify` | `invoice_code`, `invoice_no`, `amount` |
| 司库/现金 | Kyriba | `kyriba_cash_positions` | `account_id`, `balance` |
