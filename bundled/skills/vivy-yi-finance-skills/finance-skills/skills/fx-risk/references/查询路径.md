## 常用查询组合

**场景：外汇敞口核查**
```
1. [司库系统 TRSY] → 外汇管理 → 敞口报告 → 确认各币种敞口
2. [司库系统 TRSY] → 对冲工具 → 汇总已对冲金额
3. [司库系统 TRSY] → 市场数据 → 查询当前汇率
4. [司库系统 TRSY] → 计算 → 未对冲敞口 + 当前汇率 → 计算 VaR
```

**场景：对冲决策**
```
1. [司库系统 TRSY] → 敞口报告 → 查询须对冲敞口
2. [外部数据 BBG] → 外汇市场 → 查询当前远期汇率
3. [外部数据 BBG] → 外汇期权 → 查询期权波动率曲面
4. [司库系统 TRSY] → 对冲建议 → 系统推荐最优工具
```
