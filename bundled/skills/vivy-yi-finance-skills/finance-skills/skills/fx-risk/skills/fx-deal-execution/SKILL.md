# /fx-deal-execution — 外汇交易执行


## Examples

→ 示例：用户说"有一笔 500 万美元的远期合约要到期了，需要决定是否展期"，系统应调用本技能，分析远期合约展期决策。

→ 示例：用户说"今天 EUR/USD 波动很大，要不要趁机锁一个价格"，系统应调用本技能，评估即期锁汇机会。

→ 示例：用户说"帮我们执行一笔交叉货币互换，将浮动利率美元债务换成固定利率欧元"，系统应调用本技能，制定交叉货币互换执行方案。
## 第一步：交易授权确认

**交易基本信息：**
```
□ 交易方向：[买入 USD / 卖出 USD / 买入 EUR / 卖出 EUR]
□ 交易金额：[X] 万（币种）
□ 交易工具：[即期 / 远期外汇合约 / 外汇期权]
□ 参考汇率：[X.XXXX]（市场报价）
□ 敞口对冲目的：[对冲交易敞口 / 对冲会计敞口 / 对冲经济敞口]
```

**授权确认：**
```
□ 交易授权级别（根据金额）：
  → [X] 万以下：外汇交易员自主决定
  → [X]-[X] 万：须 Treasury Manager 审批
  → [X] 万以上：须 CFO 审批

□ 当前交易授权：[✅ 已授权 / ⚠️ 需审批 / 🔴 超出权限]
□ 审批人（如需）：[姓名]
□ 授权时间：[YYYY-MM-DD HH:MM]
```

## 升级触发条件

- 询价过程中发现市场异常波动（暂停交易并上报）
- 交易金额超过单笔限额（须更高层级审批）
- 结算未在 T+[X] 日完成（须立即联系交易对手和银行）
- 交易对手银行信用状况恶化（须重新评估对手名单）
- 实际执行汇率与参考汇率差异超过 [X]%（须分析原因）
- 须反向平仓或提前终止（产生平仓损益须专项审批）
