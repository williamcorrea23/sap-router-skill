# /payroll-calculation — 月度薪资核算


## Examples

→ 示例：用户说"这个月有员工离职，帮我们计算一下离职补偿金"，系统应调用本技能，计算离职补偿金。

→ 示例：用户说"帮我们审一下这个月的工资计算是否正确，有几个加班费异常"，系统应调用本技能，审核工资计算准确性。

→ 示例：用户说"销售团队这个月有提成调整，帮我重新算一下"，系统应调用本技能，执行销售提成重算。
## 第一步：获取薪资核算数据

**从 [PAYROLL] 获取薪资数据，从 [HR] 获取考勤数据：**
```
□ 薪资期间：[YYYY年MM月]
□ 在职员工：[X] 人
□ 薪资应发合计：[XXX元]
□ 扣款合计：[XXX元]
□ 实发合计：[XXX元]
```

---

## 第二步：考勤数据核对

**从 [HR] 获取考勤异常，核查薪资扣款：**
```
□ 加班时数：[X] 小时 — 应发加班费 [XXX元]
□ 事假天数：[X] 天 — 应扣 [XXX元]
□ 病假天数：[X] 天 — 应扣 [XXX元]
□ 迟到早退：[X] 次 — 应扣 [XXX元]
□ 旷工：[X] 天 — 应扣 [XXX元]

□ 考勤扣款合计：[XXX元]
□ 系统扣款合计：[XXX元]
□ 差异：[±XXX元]

□ 差异原因：[...]

□ 考勤异常人员：[X] 人
□ 须核查：[✅ 已核查 / ⚠️ 待核查]
```

---

## 第三步：应发工资核查

**核查应发项目：**
```
□ 应发工资合计：[XXX元]

□ 主要应发项目核查：
□ 基本工资：[XXX元] — [✅ 正常 / ⚠️ 需确认]
□ 岗位工资：[XXX元] — [✅ 正常 / ⚠️ 需确认]
□ 绩效工资：[XXX元] — [✅ 正常 / ⚠️ 需确认]
□ 加班费：[XXX元] — [✅ 正常 / ⚠️ 需确认]
□ 补贴：[XXX元] — [✅ 正常 / ⚠️ 需确认]

□ 异常应发项：[X] 项
□ 异常金额：[XXX元]
□ 异常人员：[X] 人
```

## 第四步：扣款核查

**核查各项扣款：**
```

□ 社保公积金个人扣款：
□ 养老保险：[XXX元] — [✅ 正确 / ⚠️ 错误]
□ 医疗保险：[XXX元] — [✅ 正确 / ⚠️ 错误]
□ 失业保险：[XXX元] — [✅ 正确 / ⚠️ 错误]
□ 住房公积金：[XXX元] — [✅ 正确 / ⚠️ 错误]

□ 个税扣款：[XXX元] — [✅ 正确 / ⚠️ 需重算]
□ 其他扣款：[XXX元] — [✅ 正常 / ⚠️ 需确认]

□ 扣款异常：[X] 项
```
