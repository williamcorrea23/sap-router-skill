# 数据源清单 — [PAYROLL]/[HR]/[ERP]

## 数据源标签说明

| 标签 | 含义 | 典型来源 |
|---|---|---|
| [PAYROLL] | 薪资核算系统 | SAP HCM/用友 NC/金蝶 HR/薪太软/薪酬雷达 |
| [HR] | 人力资源管理系统 | SAP SuccessFactors/北森/Workday/钉钉/飞书 |

---

## [PAYROLL] 薪资核算系统

**覆盖范围：**
- 员工薪资明细（应发/实发/扣款）
- 社保公积金计算与缴纳
- 个人所得税计算与申报
- 银行代发清单与结果
- 年终奖核算
- 离职薪资结算

**查询路径：**

```
薪资系统：
- 薪资核算 → 月度薪资明细
- 社保公积金 → 缴纳清单/增减员
- 个税申报 → 申报表/缴款凭证
- 银行代发 → 代发清单/失败清单
- 离职结算 → 离职薪资单

SAP HCM：
- PC00_M99_CWTR：薪资运行
- M03：薪资结果查询
- PU03：个税申报

用友 NC/金蝶 HR：
- 薪资核算 → 月度薪资表
- 社保管理 → 缴纳明细
- 个税申报 → 申报清单
```

---

## [HR] 人力资源管理系统

**覆盖范围：**
- 员工花名册（入职/转正/离职）
- 考勤记录（加班/请假/缺勤）
- 绩效考核结果
- 薪酬职级体系
- 社保公积金基数

**查询路径：**

```
HR 系统：
- 员工管理 → 花名册
- 考勤管理 → 月度考勤汇总
- 绩效管理 → 绩效评定结果
- 薪酬管理 → 薪资调整记录

SAP SuccessFactors/北森：
- 员工信息 → 入离职日期/岗位
- 考勤 → 加班申请/月度汇总
- 绩效 → 评定等级/系数
```

---

## [ERP] SAP / 用友 / 金蝶（总账）

**覆盖范围：**
- 应付职工薪酬科目（工资/奖金/社保）
- 应交税费-个人所得税
- 管理费用-工资/社保/公积金
- 制造费用-工资（生产人员）
- 销售费用-工资（销售人员）

**查询路径：**

```
SAP 常用事务代码：
- FAGLB03：科目余额表
- FB03：凭证明细
- FK03：供应商（涉及社保公积金）
- 薪资凭证过账：[ payroll posting ]

用友/金蝶：
- 总账 → 应付职工薪酬
- 工资凭证 → 薪资过账
- 社保公积金凭证 → 公司部分计提
- 个税凭证 → 代扣代缴
```

## 数据源组合使用

### 月度薪资核算闭环
```
[HR] 考勤数据 → [PAYROLL] 薪资计算 → [ERP] 薪资凭证过账
 → 银行代发 → [PAYROLL] 代发结果确认 → 失败重发
 → [ERP] 银行手续费凭证
```

### 社保公积金月度申报
```
[HR] 入离职变动 → [PAYROLL] 增减员确认 → [ERP] 社保公积金计提凭证
 → 银行扣款 → [PAYROLL] 缴纳结果确认 → 完税凭证归档
```

### 个税月度申报
```
[PAYROLL] 个税计算表 → 个税申报系统提交
 → 税款扣缴 → [ERP] 个税凭证 → 完税证明归档
 → 员工个税清单导出备查
```

### 年终奖发放
```
[HR] 绩效考核结果 → [PAYROLL] 年终奖计算
 → [ERP] 年终奖凭证 → 银行代发
 → [PAYROLL] 个税计算（单独计税或并入） → 申报系统录入
```
