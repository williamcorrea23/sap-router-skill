# 查询路径 — 薪资核算

## 数据源优先级

| 优先级 | 数据源 | 说明 |
|---|---|---|
| 1 | 薪资系统（[PAYROLL]）| 薪资计算/发放/个税 | 最优先 |
| 2 | HR 系统（[HR]）| 员工信息/考勤/绩效 | 考勤绩效数据 |
| 3 | ERP 总账（[ERP]）| 薪资凭证/社保凭证 | 账务核对 |

---

## 常用查询场景


## 数据源组合使用

### 月度薪资核算
```
[HR] 考勤 → [PAYROLL] 薪资计算 → [ERP] 凭证生成 → 银行代发 → 回单核对
```

### 社保公积金月度申报
```
[PAYROLL] 增减员 → [HR] 入职离职 → [ERP] 社保费用计提 → 银行扣款 → 回单核对
```

### 个税月度申报
```
[PAYROLL] 个税计算 → [ERP] 个税计提 → 个税申报系统提交 → 税款扣缴 → 申报回执
```

### 年终奖发放
```
[HR] 绩效考核 → [PAYROLL] 年终奖计算 → [ERP] 年终奖凭证 → [PAYROLL] 计税 → 银行代发
```
