## 薪资核算规则

### 月度薪资核算

```
□ 月度薪资组成：
  应发工资 = 基本工资 + 岗位工资 + 绩效工资 + 补贴 + 加班费 + 其他

□ 代扣代缴：
  实发工资 = 应发工资 - 社保个人 - 公积金个人 - 个税 - 其他扣款

□ 请假扣款：
  事假扣款 = 日工资 × 事假天数
  病假扣款 = 日工资 × 病假天数 × 扣款比例（[X]%）

□ 缺勤扣款：
  迟到/早退扣款 = [X] 元/次（按公司制度）
```

### 加班费核算

```
□ 工作日加班：时工资 × 1.5 × 加班小时数
□ 休息日加班：时工资 × 2.0 × 加班小时数（或补休）
□ 法定节假日加班：时工资 × 3.0 × 加班小时数

□ 加班调休规则：
□ 每月加班须在 [X] 个月内调休
□ 未调休按加班费标准计发
```
