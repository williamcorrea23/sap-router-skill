# 查询路径 — KPI 管理

## 查询入口

### 查询维度一：KPI 实际值

| 查询目标 | 系统入口 | 查询路径 | 刷新频率 |
|---|---|---|---|
| 财务 KPI（收入/利润） | [BI 系统] | 财务视图 → KPI 仪表板 | 每日 |
| 运营 KPI（DSO/库存周转） | [ERP] | 管理报告 → 运营指标 | 每日 |
| 业务 KPI（订单/转化率） | [CRM/OPS] | 业务分析 → KPI 看板 | 实时 |

### 查询维度二：KPI 历史与预测

| 查询目标 | 系统入口 | 查询路径 | 刷新频率 |
|---|---|---|---|
| KPI 历史趋势 | [BI 系统] | 历史数据 → 按指标/期间 | 实时 |
| KPI 预测值 | [BI 系统] | 预测分析 → KPI 预测 | 每周 |
| 行业对标数据 | [外部数据] | 行业数据 → KPI 对标 | 季度 |

## 常用查询组合

**场景：KPI 未达标分析**
```
1. [BI 系统] → KPI 仪表板 → 确认未达标 KPI
2. [ERP/CRM] → 查询相关业务明细 → 拆解未达标原因
3. [BI 系统] → 历史趋势 → 查询未达标 KPI 历史模式
4. [BI 系统] → 行业对标 → 查询行业均值与竞争对手
```

**场景：KPI 数据不一致排查**
```
1. [BI 系统] → 查询 KPI 定义与计算公式
2. [ERP] → 查询源系统数据
3. [CRM/OPS] → 查询业务系统数据
4. [BI 系统] → 数据血缘 → 追踪差异来源
```
