# /strategic-objective-setting — 战略目标设定


## Examples

→ 示例：用户说"3 年战略规划开始了，帮我把公司愿景翻译成具体的财务目标"，系统应调用本技能，执行战略目标设定和分解。

→ 示例：用户说"董事会要求明年的 EBITDA 增长 20%，帮我倒推各业务线的目标"，系统应调用本技能，执行目标倒推和资源规划。

→ 示例：用户说"帮我们设定一个合理的 ROIC 目标，作为 3 年规划的资本效率基准"，系统应调用本技能，基于行业基准设定 ROIC 目标。
## 第一步：愿景与战略主题确认

**公司愿景：**
```
□ 愿景声明：[一句话描述公司长期追求的目标]
□ 愿景实现时间：[YYYY 年]
□ 长期成功标志：[X] 项关键标志
```

**战略规划背景：**
```
□ 规划周期：[X] 年（[YYYY] - [YYYY]）
□ 基准年：[YYYY]
□ 上期战略评估：
  → 上期目标达成率：[X]%（整体）
  → 主要成就：[列表]
  → 主要偏差：[列表]

□ 本期战略主题选择：
  → [增长]：适用于市场扩张期
  → [效率]：适用于成本压力期
  → [转型]：适用于业务模式变革期
  → [国际化]：适用于出海阶段
  → 本期主题：[主 + 辅，如"增长为主，转型为辅"]
```

```

  → 收入目标：[X] 亿

  → 关键里程碑：[...]

**里程碑一致性检验：**
```

## 升级触发条件

- 长期目标与行业基准差距 > [X]%（过于激进或过于保守）
- 资源需求超过公司融资能力 [X]%
- 业务线目标之和无法覆盖公司目标
- 战略主题与股东/董事会期望存在重大分歧
- 目标 SMART 检验多项不通过
- 里程碑设定后发现与现有资源配置严重不匹配
