# 战略规划 Master Skill

## 能力描述

**这是什么：** 战略规划场景的主技能，整合战略假设管理、目标设定、执行追踪与战略调整，为 CFO 和战略团队提供完整的战略规划管理能力。

**解决什么问题：** 当用户需要编制 3-5 年战略规划、追踪战略执行、处理战略假设变化、或调整战略目标时，调用本技能获取完整的战略规划管理方案。

**边界：** 不涉及具体业务运营决策，专注于战略层面的规划与管理。

---

## 触发条件

满足以下任一场景时调用本技能：

- 用户提到"战略规划"、"3-5 年计划"、"战略目标"、"战略假设"
- 用户提到"战略执行"、"战略举措"、"战略调整"
- 用户要求"编制战略规划"、"追踪战略执行"、"处理战略偏差"

---


## Examples

→ 示例：用户说"我们开始做2026-2030年的战略规划，需要财务团队提供哪些输入"，系统应调用本技能，进入战略规划编制路径。

→ 示例：用户说"原材料价格上涨超预期，需要重新评估之前设定的战略假设"，系统应调用本技能，进入假设管理路径。

→ 示例：用户说"战略举措执行了一年半，需要评估一下对业绩的实际影响"，系统应调用本技能，进入执行追踪路径。
## 输入

```
□ 场景类型：[规划编制/假设管理/执行追踪/战略调整]
□ 规划周期：[3 年/5 年]
□ 涉及业务线：[列表]
□ 规划基准年：[YYYY]
□ 已获取信息：[描述]
```

## 原子能力调用顺序

```
路径一：战略规划编制
Step 1 → [战略假设分析]（strategic-assumption）
  → 输出：假设清单与验证

Step 2 → [战略目标设定]（strategic-target-setting）
  → 输出：目标体系

Step 3 → [战略路径设计]（strategic-path-design）
  → 输出：战略路径

Step 4 → [资源规划整合]（resource-planning）
  → 输出：资源配置方案

Step 5 → 汇总输出战略规划文档

路径二：战略假设变化
Step 1 → [战略假设分析]（strategic-assumption）
  → 输出：假设变化确认

Step 2 → [战略影响评估]（strategic-impact-assessment）
  → 输出：影响分析

Step 3 → [战略调整建议]（strategic-adjustment）
  → 输出：调整方案

Step 4 → 汇总输出战略调整报告
```
