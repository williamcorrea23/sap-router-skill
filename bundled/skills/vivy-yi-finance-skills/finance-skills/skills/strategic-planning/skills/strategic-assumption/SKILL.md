# /strategic-assumption — 战略假设分析


## Examples

→ 示例：用户说"我们 2026 年的战略假设是人民币汇率稳定，但现在看可能不对，帮我重新评估"，系统应调用本技能，识别假设偏差并更新战略影响分析。

→ 示例：用户说"帮我们梳理一下 3 年战略规划里的核心假设，哪些最容易被推翻"，系统应调用本技能，系统性地识别和评估战略假设。

→ 示例：用户说"主要原材料价格大幅波动，帮我评估对战略规划的影响"，系统应调用本技能，分析关键假设变化下的战略敏感性。
## 第一步：战略假设识别

**战略规划概览：**
```
□ 战略周期：[X] 年（[YYYY] 至 [YYYY]）
□ 战略主题：[增长/效率/转型/国际化]
□ 战略目标：[列表]
□ 当前所处阶段：[规划/执行/评估]
```

**假设来源识别：**
```
□ 外部假设（公司无法控制）：
  → 市场假设：市场规模/增速/份额
  → 竞争假设：竞争对手行为/进入者
  → 政策假设：监管环境/贸易政策
  → 技术假设：技术路线/颠覆性创新

□ 内部假设（公司可以影响）：
  → 资源假设：资金/人才/产能
  → 能力假设：技术/运营/组织能力
  → 执行假设：执行速度/效率/优先级

□ 财务假设：
  → 收入假设：增长率/市场份额
  → 成本假设：原材料/人工/规模效应
  → 融资假设：融资成本/可获得性
```

## 第二步：假设结构化记录

**假设清单：**
```
| # | 假设类别 | 假设描述 | 关联战略目标 | 假设来源 | 置信度 |
|---|---------|---------|------------|---------|--------|
| 1 | [市场] | [描述] | [目标] | [分析师/内部] | [高/中/低] |
| 2 | [竞争] | [描述] | [目标] | [内部] | [高/中/低] |
| 3 | [财务] | [描述] | [目标] | [内部] | [高/中/低] |

□ 假设置信度评估：
  → 高置信度：有充分数据支持，近 3 年未发生重大变化
  → 中置信度：有一定依据，但存在不确定性
  → 低置信度：依赖外部因素，缺乏历史验证
```

**关键假设识别（Critical Assumptions）：**
```
□ 高影响力假设（对战略目标影响 > [X]%）：
  → 假设 1：[描述] — 关联目标 [名称] — 置信度 [低]
  → 假设 2：[描述] — 关联目标 [名称] — 置信度 [中]

□ 假设失效风险评估：
  → 假设 1：失效概率 [X]%，影响程度 [高/中/低]
  → 假设 2：失效概率 [X]%，影响程度 [高/中/低]
```

```
□ 触发事件追踪：[是/否] — 触发事件清单：
  → 市场假设：主要竞争对手重大行动
```

```
  → 变化信号：[有/无] — [描述]

□ 假设 2：[描述]
  → ...
```

## 第四步：假设变化影响评估

**假设失效影响分析：**
```
□ 假设 1 已失效/即将失效：
  → 假设内容：[原假设] → [新情况]
  → 变化程度：[轻微/显著/根本性]
  → 对战略目标影响：
    · [目标1]：[原预测值] → [新预测值]（[±X]%）
    · [目标2]：[原预测值] → [新预测值]（[±X]%）

□ 连锁影响评估：
  → 假设 1 变化是否影响其他假设：[是/否]
  → 受影响假设：[假设2] — 影响方式 [描述]
```

**战略脆弱性评估：**
```
□ 战略对假设的敏感度：
  → 高度敏感（假设变化 → 目标偏差 > [X]%）：[战略/目标列表]
  → 中度敏感（假设变化 → 目标偏差 [X-X]%）：[战略/目标列表]
  → 低度敏感：[战略/目标列表]

□ 战略韧性评估：
  → 韧性强的方面：[描述]
  → 韧性弱的方面：[描述]
```

## 第五步：战略调整建议

**应对策略选择：**
```
□ 针对失效假设的应对策略：
  → 规避：改变战略方向，不再依赖该假设
  → 缓解：采取措施降低假设失效的影响
  → 接受：承认风险，维持现状，准备应急计划
  → 利用：利用假设变化带来的机会

□ 建议策略（针对高风险假设）：
  → 假设 1：[策略] — 理由 [描述]
  → 预期效果：[描述]
```

**战略调整建议：**
```

  → 针对假设：[假设名称]

  → 预期效果：[量化描述]
  → 实施难度：[高/中/低]
  → 资源需求：[描述]

□ 调整方案 2：[描述]
  → ...
```

## 升级触发条件

- 关键假设被验证为已失效
- 假设变化导致战略目标偏差 > [X]%
- 须调整战略方向或资源配置（> [X] 万）
- 假设变化引发连锁反应（多个假设同时受影响）
- 竞争对手或市场出现根本性变化信号
- 须召开战略会议重新评估
