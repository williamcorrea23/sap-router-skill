## 常用查询组合

**场景：3-5 年战略规划编制**
```
1. [BI 系统] → 历史数据 → 提取过去 3-5 年核心指标
2. [BI 系统] → 战略仪表板 → 评估当前战略执行
3. [外部数据] → 行业研究 → 获取行业增速/趋势预测
4. [外部数据] → 宏观数据 → 获取 GDP/利率预测
5. [BI 系统] → 业务线规划 → 汇总各业务线意见
```

**场景：战略假设验证**
```
1. [外部数据] → 行业数据 → 验证行业增速假设
2. [BI 系统] → 历史趋势 → 验证历史增长率
3. [外部数据] → 竞品分析 → 验证竞争格局假设
```
