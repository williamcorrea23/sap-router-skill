当专业 IFRS 9 系统不可用时，按以下步骤手动完成 ECL 计量。

---

```
□ 敞口清单（按客户/产品分）：

□ 到期日：[YYYY-MM-DD]
□ 抵押品：[有/无 — 类型/价值]

```

## 步骤二：Stage 分类

```
□ 对每个客户/产品执行 Stage 判断：

□ Stage 3（如有）：
□ 逾期 90 天以上？→ 是 → Stage 3
□ 其他客观减值证据？→ 是 → Stage 3

□ Stage 2（如无 Stage 3）：
□ 逾期 30-89 天？→ 是 → Stage 2
□ PD 评级下降 [X] 级？→ 是 → Stage 2
□ Watch List？→ 是 → Stage 2
□ 宏观高风险行业？→ 是 → Stage 2（辅助判断）

□ Stage 1（其余正常客户）：
□ 无上述条件 → Stage 1
```

```

□ Stage 2 ECL 计算：
□ EAD：[X] 万元
□ PD（存续期，当前评级）：[X]%
□ LGD：[X]%

□ EAD：[X] 万元
```
