# /investment-decision-report — 投资决策报告生成

## 第一步：汇总所有分析结果

```
□ DCF 结果：
□ NPV：[X] 万元
□ IRR：[X]%（门槛 [X]%）
□ 回收期：[X] 年 [X] 个月

□ 敏感性分析：
□ 悲观情景 NPV：[X] 万元 — [✅ >0 / ❌ <0]
□ 最敏感变量：[变量名]

□ 战略价值（如已评估）：
□ 实物期权价值：[X] 万元
□ 调整后项目价值：[X] 万元

□ 可比参照（如已评估）：
□ IRR vs 可比均值：[X]% vs [X]% — 偏差 [±X]%

---

## 第二步：生成决策建议

```
□ 建议结论：
□ ✅ 强烈推荐：NPV > 0 + IRR 达标 + 悲观 NPV > 0
□ ✅ 推荐：NPV > 0 + IRR 达标（或有战略理由）
□ ⚠️ 慎重考虑：NPV > 0 但 IRR 临界，或悲观 < 0 但有战略理由
□ ❌ 不建议：NPV < 0 且无充分战略理由

□ 建议审批层级（对照 CLAUDE.md 门槛）：
□ 单笔 [X]-[X] 万元 → CFO 审批
□ [X]-[X] 万元 → IC 审批
□ > [X] 万元 → 董事会审批

□ 建议条件（如通过）：
□ [条件 1]：建议 [描述]
□ [条件 2]：建议 [描述]
```
