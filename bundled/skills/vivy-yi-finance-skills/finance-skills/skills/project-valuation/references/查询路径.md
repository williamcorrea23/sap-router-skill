# 查询路径 — 项目估值（无 Connector 时）

## 手动流程说明

当专业估值工具/BI 系统不可用时，按以下步骤手动完成项目估值。

---

## 步骤一：收集项目数据

```
□ 项目基本信息：
□ 项目名称：[...]
□ 投资金额：[X] 万元
□ 建设期/投资期：[X] 个月
□ 运营期：[X] 年
□ 预计投产日期：[YYYY-MM]

□ 财务假设（从业务部门获取）：
□ 预计销量：[X] 单位/年（按年拆分）
□ 产品单价：[X] 元/单位
□ 单位可变成本：[X] 元/单位
□ 年固定成本（不含折旧）：[X] 万元
□ 折现率：[X]%

□ 资本性支出：
□ 初始投资：[X] 万元
□ 预计残值：[X] 万元
□ 折旧方式：[直线法/工作量法]
```

---

## 步骤二：建立 Excel 估值模型

```
□ Sheet 1：假设（Assumptions）
□ 投资金额、收入、成本、折现率等全部假设

□ Sheet 2：现金流预测（CF Forecast）
□ 年度：[Year 1] - [Year N]
□ 收入：[每年计算]
□ 可变成本：[每年计算]
□ 固定成本：[每年计算]
□ EBITDA：[每年计算]
□ 折旧：[每年计算]
□ EBIT：[每年计算]
□ 所得税：[每年计算]
□ 税后净利：[每年计算]
□ 加回折旧：[每年计算]
□ 资本支出：[仅 Year 1 / 逐年]
□ 营运资本变动：[如适用]
□ 自由现金流（FCF）：[每年计算]

□ Sheet 3：DCF 分析
□ NPV = Σ FCFt / (1+WACC)^t
□ IRR（用 Excel IRR 函数）
□ 投资回收期（简单）

□ Sheet 4：敏感性分析
□ 单变量：销量/价格/成本/折现率变化对 NPV 影响
□ 双变量：销量 + 价格对 NPV 的联合影响
```
