# 判断框架 — 项目估值与投资决策

## 核心原则

项目估值的核心是"用合理方法量化项目价值，支持投资决策"。遇到判断时，优先区分"可量化价值"和"战略价值"，确保决策有数据支撑。

---

## 估值方法选择判断

```
□ 项目特征判断：

□ 稳定现金流项目（运营改善/效率提升）
  → 首选：DCF（NPV + IRR）
  → 补充：敏感性分析

□ 高增长/高不确定性项目（新产品线/扩张）
  → 首选：DCF + 实物期权
  → 理由：传统 DCF 低估战略灵活性价值

□ 短期项目（设备更新/技术改造）
  → 首选：回收期 + IRR
  → 理由：现金流稳定，时间跨度短

□ 纯财务比较（多个相似规模项目）
  → 首选：NPV 排序 + IRR 对比
  → 理由：直观、可比
```

---

## IRR 判断标准

```
□ IRR 解读：
□ IRR > WACC + 风险溢价 → ✅ 项目可行
□ IRR 在 WACC 附近（±[X]%）→ ⚠️ 临界，需其他指标支持
□ IRR < WACC → ❌ 项目不可行（除非有战略价值）

□ 风险调整：
□ 高风险项目须提高折现率要求
□ IRR 门槛 = WACC + 项目特定风险溢价
```

---

## NPV 判断标准

```
□ NPV 解读：
□ NPV > 0 → ✅ 项目创造价值
□ NPV ≈ 0 → ⚠️ 边际项目，结合 IRR 和战略判断
□ NPV < 0 → ❌ 项目摧毁价值（除非战略必需）

□ 敏感性分析 NPV：
□ 基准情景 NPV > 0，悲观情景 NPV < 0 → ⚠️ 风险较高
□ 所有情景 NPV > 0 → ✅ 项目稳健
□ 所有情景 NPV < 0 → ❌ 项目不可行
```

---

## 投资决策矩阵

```
                    NPV > 0          NPV ≈ 0          NPV < 0
IRR > 门槛          ✅ 强烈推荐      ⚠️ 慎重考虑       ❌ 拒绝
IRR ≈ 门槛          ⚠️ 慎重考虑      ❌ 拒绝           ❌ 拒绝
IRR < 门槛          ⚠️ 慎重考虑      ❌ 拒绝           ❌ 拒绝
```

---

## 实物期权判断

```
□ 何时考虑实物期权：
□ 高不确定性项目（研发/新产品线）
□ 分阶段投资（可选择退出/追加）
□ 市场条件变化时可调整策略
□ 独占性/专利保护（战略价值高）

□ 常用实物期权类型：
□ 延迟期权（Option to Defer）：是否推迟投资
□ 扩张期权（Option to Expand）：是否扩大投资规模
□ 收缩期权（Option to Contract）：是否缩减规模
□ 放弃期权（Option to Abandon）：是否放弃项目

□ 何时可忽略实物期权：
□ 项目现金流稳定可预测
□ 项目不可延迟（竞争压力）
□ 市场条件变化对项目影响小
```

## 升级触发条件

- IRR < 最低回报率要求
- NPV < [X] 万元（即使 IRR 达标）
- 悲观情景下 NPV < 0
- 须追加资本支出 > [X] 万元
- 涉及新业务线/重大战略决策
