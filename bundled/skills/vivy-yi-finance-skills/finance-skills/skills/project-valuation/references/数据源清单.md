# 数据源清单 — 项目估值

## 估值数据需求与来源

### 财务与市场数据

| 数据项 | 用途 | 首选数据源 | 备选数据源 |
|--------|------|-----------|-----------|
| 历史财务数据 | 基础假设参照 | ERP 管理账 | 财务分析 |
| 销量预测 | 收入假设 | 销售部/市场部 | 市场研究报告 |
| 价格预测 | 收入假设 | 销售部/市场部 | 行业分析报告 |
| 成本结构 | 成本假设 | 财务部/生产部 | 标准成本系统 |
| 折现率（WACC）| NPV/IRR 计算 | 财务部（定期测算）| 外部顾问 |

---

### 行业与市场数据

| 数据项 | 用途 | 首选数据源 | 备选数据源 |
|--------|------|-----------|-----------|
| 行业增长率 | 宏观假设 | 国家统计局 | 行业协会 |
| 竞争对手数据 | 可比项目参照 | 上市公司年报 | Bloomberg |
| 市场研究报告 | 行业趋势 | 投行研报 | Strategy&/McKinsey |
| 利率数据 | 折现率基础 | 央行基准利率 | 彭博 |

---

### 可比交易数据

| 数据项 | 用途 | 首选数据源 | 备选数据源 |
|--------|------|-----------|-----------|
| 可比公司 EV/EBITDA | 可比估值参照 | Bloomberg | Capital IQ |
| 可比交易倍数 | 交易法估值 | M&A 数据库 | 投行交易库 |
| 行业平均回报率 | IRR 门槛参照 | 内部历史数据 | 行业报告 |

---

## 估值模型工具

```
Excel 财务模型：
□ 标准结构：假设 → 预测 → DCF / IRR / NPV → 敏感性分析
□ 敏感性分析表：单变量/双变量敏感性
□ 情景分析：基准/乐观/悲观

Python / Julia（高级分析）：
□ 蒙特卡洛模拟（不确定性量化）
□ 实物期权定价（Black-Scholes / 二叉树）
□ 自动化敏感性分析

专业估值软件：
□ FactSet / Capital IQ：可比公司/交易数据
□ Bloomberg：实时市场数据
```
