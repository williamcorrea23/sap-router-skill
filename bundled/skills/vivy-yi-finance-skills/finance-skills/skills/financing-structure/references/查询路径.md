## 常用查询组合

**场景：融资成本分析**
```
1. [司库系统 TRSY] → 融资管理 → 汇总所有融资成本
2. [外部数据 BBG] → 利率数据 → 获取市场基准利率
3. [司库系统 TRSY] → 对冲工具 → 分析对冲后实际成本
4. [BI 系统] → 财务费用 → 分析利息支出趋势
```

**场景：新融资方案评估**
```
1. [司库系统 TRSY] → 授信管理 → 查询可用授信
2. [外部数据 BBG] → 利率数据 → 获取当前市场利率
3. [评级机构] → 查询评级 → 评估信用评级变化
4. [BI 系统] → 现金流预测 → 评估还款能力
```
