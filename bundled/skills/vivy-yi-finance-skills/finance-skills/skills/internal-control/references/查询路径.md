## 查询入口

| 查询目标 | 系统入口 | 查询路径 | 刷新频率 |

## 常用查询组合

**场景：发现内控缺陷**
```
1. [内控系统 IC] → 缺陷管理 → 新增缺陷 → 录入信息
2. [审计系统 AUD] → 查询历史类似缺陷 → 参考整改方案
3. [ERP] → 查询相关流程操作日志 → 确认影响范围
```

**场景：设计新内控流程**
```
1. [内控系统 IC] → 流程库 → 参考同类流程
2. [ERP] → 查询相关业务数据量 → 评估控制频率
3. [审计系统 AUD] → 查询相关审计发现 → 识别控制重点
```
