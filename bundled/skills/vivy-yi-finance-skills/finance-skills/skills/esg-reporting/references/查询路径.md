## 常用查询组合

**场景：年度 ESG 报告编制**
```
1. [ESG 平台] → 环境数据 → 提取 E 指标
2. [HR 系统] → 社会数据 → 提取 S 指标
3. [合规系统 CMP] → 治理数据 → 提取 G 指标
4. [ESG 平台] → GRI 对照 → 核对披露完整性
5. [外部数据] → 评级数据 → 查询最新评级及反馈
```

**场景：碳排放数据核查**
```
1. [ESG 平台] → 碳排放 → 查询范围 1/2/3 数据
2. [ERP] → 能源数据 → 交叉核对用电量
3. [ESG 平台] → 数据质量 → 标注估算数据
```
