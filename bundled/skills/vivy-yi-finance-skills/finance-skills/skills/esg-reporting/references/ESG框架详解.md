# ESG 报告 — 框架与标准详解

本文件为 ESG 报告场景提供详尽的国际国内框架、披露标准、量化指标、评级方法、报告流程。配合 5 个 atomic skill 使用。

---

## 一、国际披露框架全景

### 1.1 主流框架对比

| 框架 | 制定方 | 发布时间 | 适用范围 | 中国采用 |
|------|-------|---------|---------|---------|
| **GRI** | GRI（Global Reporting Initiative） | 1997 | 通用标准，全球通用 | 已采纳 |
| **SASB** | 永续会计准则委员会 | 2018 | 行业特定（77 行业） | 部分采纳 |
| **TCFD** | 气候相关财务披露工作组 | 2017 | 气候相关 | 已采纳 |
| **ISSB IFRS S1/S2** | 国际可持续准则理事会 | 2023 | 通用（ISSB） | 对接中 |
| **CDP** | Carbon Disclosure Project | 2000 | 气候/水/森林 | 已采纳 |
| **MSCI ESG Rating** | MSCI | 2010 | 评级（非披露标准） | 评级 |
| **Sustainalytics** | Sustainalytics | 1992 | 评级 | 评级 |
| **EcoVadis** | EcoVadis | 2007 | 评级（供应链） | 评级 |
| **CSDDD**（欧盟） | 欧盟 | 2024 | 强制（人权和环境） | 不适用 |

### 1.2 ISSB 与 GRI 的关系

```
GRI Standards (2021):
├── 通用标准（GRI 1, 2, 3）
├── 行业标准（GRI 11-40+，2023+）
└── 议题标准（GRI 200/300/400）

ISSB Standards:
├── IFRS S1: 一般可持续相关披露
│   - 治理、战略、风险管理、指标与目标
└── IFRS S2: 气候相关披露
    - 范围 1/2/3 排放
    - 气候相关风险与机遇
    - 情景分析
```

**实务选择**：
- **GRI 优先**：欧盟/全球通用、强调利益相关方导向
- **ISSB 优先**：投资者导向、强调财务重要性
- **混合模式**：多数大公司用 GRI + TCFD/ISSB S2

**中国趋势**：
- 财政部 2024 年发布《企业可持续披露准则》征求意见稿
- 2027 年 A+H 股强制执行（与 ISSB 对接）
- 上交所/深交所 2026 年起 ESG 强制披露范围扩大

## 二、GRI 标准详解

### 2.1 GRI 三层结构

```
GRI 1: 基础（Foundation）
   - 报告原则
   - 通用要求

GRI 2: 一般披露（General Disclosures）
   - 组织概况、战略、治理、道德、利益相关方参与

GRI 3: 实质性议题方法（Material Topics）
   - 议题识别、评估、决策流程

GRI 4: 行业披露（行业特定）
   - GRI 11 油气、12 煤炭、... 40+ 行业

议题标准（GRI 200/300/400）：
GRI 200: 经济议题（经济绩效、市场表现、反腐败等）
GRI 300: 环境议题（能源、水、排放、废弃物等）
GRI 400: 社会议题（雇佣、健康安全、培训、客户等）
```

### 2.2 双重重要性（Double Materiality）

**GRI 2021 新增**：重大性议题应同时考虑：

```
                    利益相关方重要性
                          ↑
                          │
                    ●     │     ●
                  高      │      高
                双重     │     单一
                重要性   │     重要
                  ●      │      ●
                ────────┼─────────→
                低       │        高
                          │      投资者重要性
                          ↓
```

```
1. 财务重要性（Financial Materiality）
   - 议题对公司财务的潜在影响
   - 投资者导向
   - ISSB/CSDS 强调

2. 影响重要性（Impact Materiality）
   - 公司对环境/社会的影响
   - 利益相关方导向
   - GRI 强调

3. 双重重要性（Double Materiality）
   - 两者交集或并集
   - CSRD/ESRS 强制要求
   - 中国 CSDS 2024 草案采纳
```

**评估方法**：

```
财务重要性评估：
- 影响公司现金流、负债、资本成本
- 短期（< 1 年）/ 中期（1-5 年）/ 长期（> 5 年）
- 量化方法：DCF / 情景分析 / 敏感性

影响重要性评估：
- 影响规模（scale）
- 影响范围（scope）
- 不可补救性（irremediability）
- 发生概率（likelihood）
```

### 2.3 GRI 议题标准示例

**GRI 305: 排放**
```
305-1: 直接（范围 1）温室气体排放
305-2: 间接（范围 2）能源间接排放
305-3: 其他间接（范围 3）排放
305-4: 温室气体排放强度
305-5: 减排
305-6: 臭氧消耗物质（ODS）排放
305-7: 氮氧化物（NOx）、硫氧化物（SOx）及其他主要气体排放
```

**GRI 403: 职业健康与安全**
```
403-1: 职业健康安全管理体系
403-2: 危害识别、风险评估和事件调查
403-3: 职业健康服务
403-4: 工人参与、协商和职业健康安全沟通
403-5: 工人职业健康安全培训
403-6: 工人健康促进
403-7: 业务关系中职业健康安全影响的预防和缓解
403-8: 职业健康安全管理体系覆盖的工人
403-9: 工伤
403-10: 工作相关的健康问题
```

## 三、SASB 标准详解

### 3.1 SASB 行业分类

SASB 按 **11 个行业、77 个子行业** 分类：

| 行业 | 子行业数 | 关键指标示例 |
|------|---------|------------|
| **消费品** | 16 | 产品安全、广告道德 |
| **Extractives & Minerals Processing** | 8 | 环境影响、社区关系 |
| **金融** | 14 | 普惠金融、数据安全 |
| **食品和饮料** | 12 | 食品安全、水资源 |
| **医疗保健** | 8 | 产品质量、获取性 |
| **基础设施** | 8 | 环境影响、社区影响 |
| **可再生资源** | 5 | 可持续采购、碳汇 |
| **资源转化** | 7 | 能源效率、废弃物 |
| **服务业** | 6 | 数据隐私、员工发展 |
| **技术与通讯** | 8 | 数据安全、隐私 |
| **交通运输** | 6 | 排放、安全 |

### 3.2 SASB 五大维度

```
1. Environment（环境）
2. Social Capital（社会资本）
3. Human Capital（人力资本）
4. Business Model & Innovation（商业模式与创新）
5. Leadership & Governance（领导力与治理）
```

### 3.3 SASB 行业标准示例：半导体

```
Environment:
- 温室气体排放（Scope 1+2）
- 能源管理
- 水资源管理
- 危险废弃物管理

Social Capital:
- 客户隐私
- 数据安全

Human Capital:
- 员工健康与安全
- 员工多元化

Business Model & Innovation:
- 产品生命周期管理
- 包装材料管理

Leadership & Governance:
- 商业道德
- 竞争行为
```

## 四、TCFD/ISSB S2 框架详解

### 4.1 TCFD 四大支柱

```
1. 治理（Governance）
   - 董事会监督气候风险
   - 管理层的角色和责任

2. 战略（Strategy）
   - 气候相关风险与机遇
   - 对业务/战略/财务规划的影响
   - 情景分析（2°C / 1.5°C 情景）

3. 风险管理（Risk Management）
   - 气候风险识别与评估流程
   - 整合到整体风险管理
   - 优先级排序

4. 指标与目标（Metrics & Targets）
   - Scope 1/2/3 排放
   - 行业特定指标
   - 气候相关目标
```

### 4.2 气候情景分析

**常用情景**（IPCC / IEA / NGFS）：

| 情景 | 来源 | 关键假设 |
|------|------|---------|
| 1.5°C 情景（Net Zero 2050） | IEA NZE | 2050 净零碳排放，温升 1.5°C |
| 2°C 情景 | IEA SDS / RCP 2.6 | 2100 温升 2°C |
| 现行政策情景 | IEA STEPS | 不新增政策 |
| 延迟转型情景 | NGFS Delayed | 政策延迟到 2030 |
| 物理风险高情景 | IPCC RCP 8.5 | 物理风险高（洪水、热浪） |

**财务影响评估**：

```
物理风险：
├── 急性（极端天气）
│   - 工厂停产损失
│   - 供应链中断
│   - 资产减值
└── 慢性（海平面、温度）
    - 制冷成本上升
    - 海平面上升影响沿海资产

转型风险：
├── 政策与法律
│   - 碳价上升
│   - 排放法规
│   - 诉讼风险
├── 技术
│   - 低碳技术替代
│   - 研发投入需求
├── 市场
│   - 客户偏好变化
│   - 原材料成本
└── 声誉
    - 利益相关方关注
    - 评级机构观点
```

### 4.3 Scope 1/2/3 排放

**范围 1：直接排放**
```
- 化石燃料燃烧（自有锅炉、车辆）
- 工业过程排放（水泥、化工）
- 逸散排放（制冷剂、SF6）
```

**范围 2：外购能源间接排放**
```
- 外购电力
- 外购热力
- 外购蒸汽

计算方法：
- Location-based：基于电网平均排放因子
- Market-based：基于供应商实际排放因子
- 选择：通常同时披露两个
```

**范围 3：价值链间接排放（15 类）**
```
上游：
1. 采购的商品和服务
2. 资本货物
3. 燃料和能源相关（不含 Scope 1/2）
4. 上游运输和配送
5. 运营产生的废弃物
6. 商务出行
7. 员工通勤
8. 上游租赁资产

下游：
9. 下游运输和配送
10. 销售产品的加工
11. 销售产品的使用
12. 销售产品的报废处理
13. 下游租赁资产
14. 特许经营
15. 投资
```

**范围 3 数据挑战**：
```
- 类别 11（产品使用）通常最大，估算困难
- 类别 1（采购）和 12（报废）需要供应商数据
- 常见使用：行业平均 + 经济投入产出模型（EEIO）
- 中国企业范围 3 披露比例：< 30%（2024）
```

## 五、量化指标参考值

### 5.1 气候相关指标

**绝对排放**：
```
Scope 1（tCO₂e）：公司直接排放
Scope 2（tCO₂e）：外购电力/热力
Scope 3（tCO₂e）：价值链排放
```

**强度指标**：
```
碳强度（tCO₂e/单位收入）
碳强度（tCO₂e/单位产量）
能源强度（GJ/单位产量）
```

**行业基准（参考）**：

| 行业 | Scope 1+2 强度（tCO₂e/百万美元收入） | Scope 3 占比 |
|------|-----------------------------------|------------|
| 油气 | 500 - 1500 | 80% - 90% |
| 公用事业 | 1500 - 4000 | 50% - 70% |
| 钢铁 | 1500 - 2500 | 20% - 30% |
| 水泥 | 1200 - 2000 | 10% - 20% |
| 化工 | 500 - 1000 | 70% - 80% |
| 汽车 | 50 - 100 | 85% - 95% |
| 房地产 | 30 - 80 | 50% - 70% |
| 科技 | 10 - 30 | 90% - 95% |
| 金融服务 | 5 - 20 | 60% - 80% |

### 5.2 社会指标

**员工**：
```
- 员工总数（按性别/年龄/地区/合同类型）
- 流失率（自愿/非自愿）
- 培训时长（人均小时）
- 工伤率（每 200,000 工时）
- 死亡率（员工 + 承包商）
- 男女同工同酬比
- 多元化（管理层/董事会）
```

**社区**：
```
- 社会投资金额
- 志愿者时长
- 受益人数
- 当地雇佣比例
```

### 5.3 治理指标

```
- 董事会规模
- 独立董事比例
- 多元化（性别/专业）
- 董事会会议次数
- 出席率
- 高管薪酬与 ESG 挂钩比例
- 反腐败培训覆盖率
- 数据安全事件数
- 重大处罚事件
```

## 六、ESG 评级方法对比

### 6.1 主流评级机构

| 评级 | 评分体系 | 重点 | 中国 A 股覆盖 |
|------|---------|------|------------|
| **MSCI ESG** | AAA - CCC（7 档） | 财务实质性 | 600+ |
| **Sustainalytics** | 风险分数（0-40+） | 风险导向 | 1000+ |
| **FTSE Russell** | 0-5 分（5 档） | 行业实质性 | 500+ |
| **S&P Global CSA** | 0-100 分 | 投资者导向 | 200+ |
| **恒生 ESG** | 0-100 分（A+ 6 档） | 中国本地 | 2000+ |
| **Wind ESG** | AAA - D（7 档） | 中国本地 | 8000+ |
| **商道融绿** | A+ - D（10 档） | 中国本地 | 8000+ |

### 6.2 MSCI 评级方法

```
行业权重（Industry Materiality）：
- 选择行业实质性议题
- 不同行业权重不同

关键议题评估（Key Issue Score）：
- 治理（G）：相对通用
- 环境和社会的权重因行业而异
  - 油气：环境 > 社会
  - 金融：治理 > 社会
  - 消费品：社会 > 环境

评分方法：
- 风险敞口（Exposure）
- 风险管理（Management）
- 最终评级 = f(风险敞口, 风险管理)
```

### 6.3 提升 ESG 评级的关键

**MSCI 视角**：
```
- 治理结构（董事会独立性、薪酬政策）
- 商业道德（反腐败机制）
- 数据隐私（金融/科技高权重）
- 碳排放（能源/工业高权重）
- 员工发展
- 供应链管理
```

**行业针对性**：
| 行业 | 关键议题 | 重点指标 |
|------|---------|---------|
| 银行 | 普惠金融、数据安全 | 数字鸿沟、客户隐私 |
| 钢铁 | 碳排放、水资源 | Scope 1+2、用水强度 |
| 房地产 | 绿色建筑、社区影响 | 绿色认证、社区投入 |
| 科技 | 数据隐私、人才发展 | 安全事件、员工留存 |

## 七、报告流程与周期

### 7.1 年度报告时间线（中国 A+H 股）

```
1 月 - 2 月：内部准备启动
├── 重大性议题确认（每年评估）
├── 数据收集准备
└── ESG 治理会议启动

3 月 - 4 月：数据收集与核实
├── 内部数据（HR/环境/安全/采购）
├── 外部数据（供应商、客户调研）
├── 第三方审验/鉴证

5 月 - 6 月：报告撰写与设计
├── 章节内容撰写
├── 数据可视化
├── 案例收集

7 月：报告审核
├── ESG 委员会审议
├── 董事会审议
├── 外部审计/鉴证（如适用）

8 月：报告发布
├── 港交所同步披露
├── ESG 评级机构送评
├── 投资者沟通
```

### 7.2 数据治理

**数据收集机制**：
```
集团总部 ESG 部门：
├── 制定 ESG 数据收集模板
├── 设计 ESG 数据治理流程
├── 监督各子公司/部门填报
└── 汇总并交叉验证

各子公司/部门：
├── 环保部门：能源/排放/废弃物
├── HR 部门：员工/培训/安全
├── 采购部门：供应链 ESG
├── 合规部门：反腐败/合规事件
├── IT 部门：数据安全
└── 投资者关系：ESG 沟通
```

**ESG 信息系统**：
```
推荐功能：
- 指标模板管理
- 数据自动采集（IoT 集成）
- 报告自动生成
- 历史数据归档
- 评级跟踪

可选供应商：
- Workiva（北美主流）
- Diligent
- 妙盈科技（Miotech，中国）
- 商道纵横
```

## 八、独立鉴证（Assurance）

### 8.1 鉴证类型

```
有限保证（Limited Assurance）
- 适用于大多数情况
- 鉴证范围较小
- 工作量较少
- 鉴证结论："未发现重大错报"

合理保证（Reasonable Assurance）
- 适用于关键指标
- 鉴证范围大
- 工作量大（接近财务报表审计）
- 鉴证结论："在所有重大方面公允反映"
```

### 8.2 鉴证范围

**常见可鉴证指标**：
```
- Scope 1+2 GHG 排放
- 能源消耗
- 员工总数与流失率
- 工伤率与死亡率
- 反腐败培训覆盖率
- 供应链审查比例
```

**难鉴证指标**：
```
- Scope 3 排放
- 社区影响（投入金额）
- 多元化（管理层）
- 间接影响数据
```

### 8.3 鉴证准则

| 标准 | 适用范围 | 鉴证类型 |
|------|---------|---------|
| ISAE 3000 | 通用 | 有限/合理 |
| ISAE 3410 | GHG | 有限/合理 |
| AA1000AS | 通用 | 有限/合理 |
| HKEX ESG Guide | 港股 | 有限（自愿） |
| CSAE 3001 | 中国内地 | 有限/合理 |

## 九、合规要求（中国）

### 9.1 A 股强制披露

**上交所/深交所 ESG 披露要求（2024 年修订）**：

```
强制披露范围：
- 报告期内持续督导的上市公司
- 重点关注：行业重大风险、监管关注

披露要求：
- 《可持续发展报告》与年报同步
- 至少包含：
  - 双重重要性评估
  - 治理结构
  - 战略与政策
  - 风险与机遇
  - 指标与目标
- 至少 1 项关键指标做第三方鉴证
```

### 9.2 H 股强制披露

**港交所 ESG Guide**：

```
不披露就解释（Comply or Explain）：
- 治理（Governance）
- 环境（Environmental）：排放、资源使用、气候
- 社会（Social）：雇佣、健康安全、发展、培训、供应链、产品责任、反腐败、社区

2025 年起升级为 Climate-related Disclosure（强制）：
- 治理
- 战略
- 风险管理
- 指标与目标
```

### 9.3 国有控股企业

**国资委要求**（2024）：

```
央企控股上市公司：
- 2025 年起 100% 披露 ESG 报告
- 控股上市公司建立 ESG 治理体系
- ESG 报告与年报同步

地方国企：
- 2026 年起分阶段强制
- 鼓励设立 ESG 委员会
```

## 十、报告设计最佳实践

### 10.1 报告结构（推荐）

```
封面：标题、年份、Logo

开篇：
- 主席致辞（CEO/董事会主席）
- 关于本报告（覆盖范围、报告期、标准）
- 高管寄语

主体：
- 集团概况
- ESG 治理（董事会声明、ESG 委员会）
- 重大性议题（双重重要性矩阵）
- 战略与目标
- 绩效与进展
  - E：气候变化、能源、水、废弃物、生物多样性
  - S：员工、产品、社区、供应链
  - G：治理结构、商业道德、风险管理

附录：
- GRI 索引
- SASB 索引
- TCFD 索引
- 数据表（多年对比）
- 鉴证报告
- 联系方式
```

### 10.2 数据展示

**关键绩效指标（KPI）**：
```
- 用具体数字 + 单位
- 提供 3-5 年趋势
- 设定目标并跟踪进展
- 行业基准对比（可选）
```

**可视化**：
```
- 重大性矩阵图
- Scope 1/2/3 饼图
- 多元化堆叠柱状图
- 碳排放强度趋势线
- 案例照片（员工/项目）
```

### 10.3 常见错误

```
- 大量"宣传式"内容缺乏数据
- 数据无对比（只有今年，没有去年）
- 重大性议题"和稀泥"（全部"高"重要性）
- 范围 3 几乎不披露
- 没有第三方鉴证
- 治理描述空洞
- 利益相关方沟通流于形式
```

## 十一、ESG 与财务报告的衔接

### 11.1 ISSB 强调的"连接性"（Connectivity）

```
ESG 信息
  ↓ 影响
财务信息（同一公司的财务报表）
  ↓ 决定
投资者决策
```

**具体表现**：
- Scope 1/2/3 排放影响未来成本
- 气候风险影响资产估值
- 员工多元化影响生产力
- 治理结构影响投资回报

### 11.2 EU CSRD 强制要求

```
ESRS（European Sustainability Reporting Standards）：
├── ESRS E1: 气候变化
├── ESRS E2: 污染
├── ESRS E3: 水和海洋资源
├── ESRS E4: 生物多样性和生态系统
├── ESRS E5: 资源利用和循环经济
├── ESRS S1: 员工
├── ESRS S2: 价值链工人
├── ESRS S3: 影响社区
├── ESRS S4: 消费者和最终用户
├── ESRS G1: 商业行为
└── ESRS 1/2: 通用披露
```

**中国对应**：
- 财政部 2024 年发布《企业可持续披露准则》征求意见稿
- 基本结构与 ESRS/ISSB 对接
- 2027 年 A+H 股先行
- 2028-2030 年逐步扩大
