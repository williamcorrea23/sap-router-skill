# /esg-reporting — ESG 报告编制


## Examples

→ 示例：用户说"年度 ESG 报告要开始准备了，帮我规划一下时间表和工作内容"，系统应调用本技能，制定 ESG 报告编制计划。

→ 示例：用户说"MSCI ESG 评级访谈下个月，财务需要准备哪些数据"，系统应调用本技能，准备 MSCI 访谈财务数据清单。

→ 示例：用户说"帮我们审一下 ESG 报告里的碳排放数据，数字之间对不上"，系统应调用本技能，核查 ESG 报告数据一致性。
## 第一步：报告范围与框架确认

**报告基本信息：**
```
□ 报告年度：[YYYY 年]
□ 报告周期：[年度/季度/半年]
□ 报告边界：[全集团/母公司+主要子公司/特定业务]
□ 覆盖实体：[X] 家

□ 披露框架选择：
  → GRI（全球报告倡议组织）：[✅ 采用 / 不采用]
  → SASB（可持续会计准则委员会）：[✅ 采用 / 不采用]
  → TCFD（气候相关财务披露工作组）：[✅ 采用 / 不采用]
  → 中国 ESG 披露标准（中证/深交所）：[✅ 采用 / 不采用]

□ 数据收集截止：[YYYY-MM-DD]
□ 报告发布目标：[YYYY-MM-DD]
```

---

## 第二步：ESG 数据收集

**E（环境）数据：**
```
□ 温室气体排放（范围 1, 2, 3）：
  → 范围 1（直接排放）：[X] 吨 CO2e
  → 范围 2（外购能源）：[X] 吨 CO2e
  → 范围 3（价值链）：[X] 吨 CO2e（如披露）

□ 能源消耗：
  → 总能耗：[X] MWh
  → 可再生能源比例：[X]%

□ 水资源：
  → 取水量：[X] 立方米
  → 废水排放量：[X] 立方米

□ 废弃物：
  → 一般废弃物：[X] 吨
  → 危险废弃物：[X] 吨
  → 回收率：[X]%

□ 数据来源：
  → 各工厂/子公司填报率：[X]%
  → 数据质量：[✅ 可靠 / ⚠️ 部分估算 / 🔴 质量存疑]
```

**S（社会）数据：**
```
□ 员工数据：
  → 总员工数：[X] 人
  → 女性员工占比：[X]%
  → 管理层女性占比：[X]%
  → 员工流失率：[X]%

□ 健康与安全：
  → 工伤率：[X]（LTIR）
  → 死亡事故：[X] 起

□ 社区投资：
  → CSR 投入：[X] 万元
  → 志愿服务时长：[X] 小时
```

**G（治理）数据：**
```
□ 董事会构成：
  → 董事会成员：[X] 人
  → 独立董事占比：[X]%
  → 女性董事占比：[X]%

□ 治理实践：
  → 独立董事比例：[X]%（符合 [X]% 要求）
  → 审计委员会由独立董事构成：[是/否]
  → 反腐培训覆盖率：[X]%
```

## 第三步：披露框架核对

**GRI 对照检查：**
```
□ 核心指标披露率：[X]/[X]（[X]%）
  → 已披露：GRI [302-1] 能源消耗，[305-1] 范围1排放...
  → 未披露/不适用：GRI [X] — 原因 [不适用/数据不可得]

□ 重大性议题识别：
  → 重大 E 议题：[气候变化、水资源...]
  → 重大 S 议题：[员工安全、多元化...]
  → 重大 G 议题：[商业道德、供应链...]
```

**TCFD 对照检查：**
```
□ 治理：气候治理架构 [✅ 已披露 / ⚠️ 部分披露 / 🔴 未披露]
□ 战略：气候相关风险与机遇 [✅ 已披露 / ⚠️ 部分披露 / 🔴 未披露]
□ 风险管理：气候风险评估流程 [✅ 已披露 / ⚠️ 部分披露 / 🔴 未披露]
□ 指标与目标：气候指标与目标 [✅ 已披露 / ⚠️ 部分披露 / 🔴 未披露]
```

## 第四步：报告撰写

**报告结构（参考 GRI 标准模板）：**
```
□ 执行摘要：[X] 页
□ 关于我们：[X] 页
□ ESG 治理：[X] 页
□ 环境绩效：[X] 页
□ 社会绩效：[X] 页
□ 治理绩效：[X] 页
□ 关键指标与目标：[X] 页
□ 附录/数据表：[X] 页
```

**报告撰写要点：**
```
□ 叙事连贯性：
  → ESG 战略与业务战略的关联：[描述]
  → ESG 绩效与业务成果的链接：[描述]

□ 关键信息：
  → 年度亮点 [3-5 条]
  → 与上期对比（同比变化）
  → 与行业对标

□ 数据可视化：
  → 趋势图：3-5 年历史数据
  → 分布图：各业务单元/地区分布
  → 仪表盘：关键指标达成
```

```

  → 法务/合规审阅：[✅ 已审阅 / ⏳ 待审阅]
  → 高管审阅（CFO/CEO）：[✅ 已审阅 / ⏳ 待审阅]
  → 董事会审阅：[✅ 已审阅 / ⏳ 待审阅]

□ 审阅意见：[X] 条
  → 已采纳：[X] 条
  → 未采纳：[X] 条 — 理由
```

```
□ 鉴证机构（如聘请）：[机构名称]
□ 鉴证范围：[有限鉴证/合理鉴证]
□ 鉴证标准：[ISAE 3000/AA1000]
□ 鉴证费用：[X] 万
□ 鉴证结论：[无保留意见/保留意见]
□ 鉴证声明页将包含于报告中：[是/否]
```

## 升级触发条件

- 关键 ESG 数据缺失 > [X]% 且无法在截止日前补充
- 数据质量存疑（内部数据与外部信息不一致）
- 披露框架存在重大缺失（GRI 核心指标缺失 > [X]%）
- 审阅中发现重大事实错误或误导性陈述
- 外部鉴证发现数据问题须重新核实
- 发布前遭遇 ESG 争议事件（数据造假指控、ESG 评级下调）
