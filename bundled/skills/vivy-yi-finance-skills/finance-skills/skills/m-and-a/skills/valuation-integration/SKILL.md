# /valuation-integration — 估值综合分析


## Examples

→ 示例：用户说"三种估值方法结果不一样，最终交易价格怎么定，帮我综合分析一下"，系统应调用本技能，整合多种估值方法并给出定价建议。

→ 示例：用户说"管理层对估值区间有争议，需要一个中立角度的分析"，系统应调用本技能，生成中立估值综合报告。

→ 示例：用户说"交易估值确定了，帮我评估一下协同效应能不能支撑这个价格"，系统应调用本技能，评估协同效应支撑的交易价格合理性。

## 升级触发条件

- 各方法估值差异 > [X]%（方法论存在争议）
- 报价显著高于估值上限（须重新谈判）
- 须对估值假设进行重大调整
- 涉及重大协同效应（须单独量化）
- 涉及或有对价/ earn-out 结构设计
- 涉及上市公司披露要求（股价敏感性）
