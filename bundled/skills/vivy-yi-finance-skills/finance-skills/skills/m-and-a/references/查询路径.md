## 常用查询组合

**场景：尽调财务分析**
```
1. [外部数据] → 尽调数据库 → 下载目标公司历史财务
2. [BI 系统] → 可比公司 → 查询同类公司关键指标
3. [外部数据 BBG] → 可比交易 → 查询同类交易估值倍数
4. [ERP] → 查询本公司数据 → 用于协同效应分析
```

**场景：估值谈判**
```
1. [BI 系统] → DCF 模型 → 计算估值区间
2. [外部数据 BBG] → 可比公司 → 查询行业平均倍数
3. [外部数据] → 可比交易 → 查询近期交易价格
4. [BI 系统] → 协同效应 → 分析整合后价值增量
```
