# 查询路径 — 审计配合

## 数据源优先级

| 优先级 | 数据源 | 说明 |
|---|---|---|
| 1 | ERP 总账（[ERP]）| 账套数据/凭证/余额表 | 最优先 |
| 2 | 档案管理系统（[DOC]）| 合同/发票/审批单 | 原始凭证 |
| 3 | 银行系统（[BANK]）| 银行对账单/回单 | 银行函证 |

---

## 数据源组合使用

### 审计进场准备
```
[ERP] 科目余额 → 编制余额表 → 提供审计资料清单
[ERP] 序时账 → 抽查凭证 → 准备合同/发票 [DOC]
```

### 函证程序
```
[BANK] 银行对账单 → 发函 → 回函核对 → 差异处理
[ERP] 往来明细 → 发函 → 回函核对 → 替代程序
```

### 审计调整确认
```
审计调整台账 → 管理层评估 → [ERP] 过账 → 调整后余额核对
```
