# 判断框架 — 审计配合异常处理

## 核心原则

**审计配合的目标是"资料完整、答疑清晰、调整合规、报告及时"。**
遇到异常时，优先判断是"资料缺失"还是"业务实质问题"，确保审计高效推进、报表公允。

---

## 异常类型与处理方式

### 类型一：资料缺失或无法提供

**典型场景：**
- 审计师要求的资料系统内无记录
- 历史档案已销毁或无法找到
- 资料存在但格式不符合审计要求

**判断流程：**

```
第一步：确认缺失资料

□ 缺失资料名称：[描述]
□ 审计要求提供日期：[YYYY-MM-DD]
□ 资料用途：[描述]

□ 缺失原因分析：
□ 系统无记录（从未录入/数据丢失）
□ 档案未归档（档案室无此文件）
□ 档案已销毁（超过保管期限）
□ 第三方持有（银行/税务局等）

□ 是否有替代资料：[是/否]
□ 替代资料：[描述]
```

```
第二步：评估缺失影响

□ 对报表影响：
□ 影响金额：[XXX万]
□ 影响科目：[科目名称]
□ 是否重大：[是/否 — 通常为报表重要性水平 X%]

□ 对审计意见影响：
□ 无影响 → 可出具无保留意见
□ 可能保留/无法表示意见 → 须评估

□ 对具体科目影响：
□ [科目A]：[影响描述]
□ [科目B]：[影响描述]
```

```
第三步：制定应对方案

□ 替代方案：
□ 替代资料确认：
□ [替代资料A] — 审计可接受程度 [高/中/低]
□ [替代资料B] — 审计可接受程度 [高/中/低]

□ 管理层说明：
□ 须提供管理层声明书说明
□ 须说明缺失原因及替代措施

□ 审计范围受限应对：
□ 范围受限程度：[轻微/重大]
□ 是否签署审计范围受限声明

□ 补充归档（如可能）：
□ 能否补充归档：[是/否]
□ 补充时间：[YYYY-MM-DD]
□ 补充后审计师是否接受
```

### 类型二：审计调整分歧

**典型场景：**
- 审计师提出调整，管理层不认同
- 调整影响利润，管理层希望回调
- 税务调整与财务处理存在差异

**判断流程：**

```
第一步：确认调整内容

□ 调整类型：
□ 审计调整（调整分录）
□ 重分类调整（不影响利润）
□ 税务调整（与财务处理差异）

□ 调整分录：
借：[科目A] [XXX万]
  贷：[科目B] [XXX万]

□ 对报表影响：
□ 对资产负债表影响：[XXX万]
□ 对利润表影响：[XXX万]
□ 对所得税影响：[XXX万]

□ 提出方：[审计师/管理层]

第二步：分析分歧原因

□ 分歧类型：
□ 会计判断分歧（收入确认时点/资产减值等）
□ 估计分歧（坏账准备/折旧年限等）
□ 分类分歧（费用资本化/非经常性损益等）
□ 税务处理分歧

□ 分歧合理性分析：
□ 审计师立场：[描述]
□ 管理层立场：[描述]
□ 会计准则依据：[条款]
□ 行业惯例：[描述]

第三步：评估调整影响

□ 如接受调整：
□ 影响利润：[±XXX万]
□ 影响税后利润：[±XXX万]
□ 是否影响业绩考核：[是/否]

□ 如不接受调整：
□ 审计师可能出具：[无保留/保留/无法表示意见]
□ 风险程度：[低/中/高]

第四步：确定最终处理

□ 处理方案：

□ 接受审计调整：
→ 管理费用 [XXX万]
→ 利润减少 [XXX万]
→ 须说明对业绩的影响

□ 不接受调整但妥协：
→ 部分接受：[调整项] — 调整金额 [XXX万]
→ 坚持立场：[调整项] — 坚持理由 [描述]

□ 坚持立场：
→ 准备说明材料
→ 沟通审计合伙人
→ 评估出具非标意见风险

□ 最终决定：
□ 调整金额：[XXX万]
□ 不调整金额：[XXX万]
□ 管理层声明：[描述]
```

### 类型三：发现内控缺陷

**典型场景：**
- 审计师发现账实不符
- 发现未经审批的重大交易
- 发现舞弊迹象

**判断流程：**

```
第一步：确认缺陷详情

□ 缺陷类型：
□ 设计缺陷（制度本身有问题）
□ 运行缺陷（制度执行不到位）

□ 缺陷级别：
□ 重大缺陷（影响报表可靠性）
□ 重要缺陷（影响效率但不重大）
□ 一般缺陷（轻微问题）

□ 缺陷描述：
□ 发现环节：[描述]
□ 涉及金额：[XXX万]
□ 涉及人员：[姓名/部门]
□ 发现时间：[YYYY-MM-DD]

□ 是否涉及舞弊：[✅ 否 / ⚠️ 疑似 / 🔴 是]

第二步：评估影响

□ 对报表影响：
□ 高风险 — 可能导致报表重大错报
□ 中风险 — 可能导致报表重要错报
□ 低风险 — 不会导致报表错报

□ 对内控报告影响：
□ 须在内部控制审计报告中披露
□ 可能影响内控有效性评价

□ 舞弊风险（如涉及）：
□ 舞弊类型：[挪用/侵占/虚假记录]
□ 涉及金额：[XXX万]
□ 涉及人员：[内部/外部]

第三步：制定整改方案

□ 即时整改：
□ 纠正已发现的错报
□ 暂停相关人员职责（如涉及舞弊）
□ 保护证据

□ 内控整改：
□ 整改措施：[描述]
□ 整改责任人：[X]
□ 整改完成日期：[YYYY-MM-DD]

□ 舞弊处理（如涉及）：
□ 启动内部调查
□ 评估是否报警
□ 通知保险理赔（如有）
□ 完善反舞弊机制

第四步：报告与沟通

□ 内部报告：
□ 报告对象：[CFO/CEO/董事会/审计委员会]
□ 报告时间：[YYYY-MM-DD]

□ 审计沟通：
□ 告知审计师：[是/否]
□ 审计师是否在报告中披露

□ 外部报告（如舞弊）：
□ 报告监管机构：[是/否]
□ 报告公安机关：[是/否]
```

### 类型四：银行函证异常

**典型场景：**
- 银行不回函
- 回函金额与账面不符
- 回函显示冻结/抵押

**判断流程：**

```
第一步：确认函证状态

□ 函证项目：
□ 银行名称：[名称]
□ 函证内容：[存款/借款/担保/其他]
□ 函证金额：[XXX万]
□ 发函日期：[YYYY-MM-DD]

□ 回函状态：
□ 已回函：[回函日期]
□ 未回函：[已发函 X 天]
□ 二次发函：[是/否]

□ 回函内容：
□ 金额一致：[✅ 是 / ⚠️ 否]
□ 信息相符：[✅ 是 / ⚠️ 否]
□ 存在冻结/抵押：[✅ 否 / 🔴 是]

第二步：分析未回函原因

□ 未回函原因：
□ 银行处理周期长（正常 X 周内）
□ 地址/联系人错误
□ 银行不予配合（须高层沟通）
□ 账户已销户

□ 其他考虑：
□ 该账户重要性：[高/中/低]
□ 替代程序可行性：[描述]
```

```
第三步：处理金额不符

□ 回函金额：[XXX万]
□ 账面金额：[XXX万]
□ 差异：[±XXX万]

□ 差异原因分析：
□ 记账时间差（在途款项）
□ 费用/利息未入账
□ 错误记账
□ 未达账项

□ 核对步骤：
□ 取得银行对账单逐笔核对
□ 识别未达账项
□ 确认是否须调账

□ 调账建议：
□ 须调账：[✅ 是 / ⚠️ 否]
□ 调整金额：[XXX万]
□ 调整分录：[描述]
```

```
第四步：处理冻结/抵押

□ 冻结/抵押信息：
□ 冻结金额：[XXX万]
□ 抵押物：[描述]
□ 抵押权人：[名称]
□ 冻结/抵押原因：[描述]

□ 合规性检查：
□ 是否已披露：[✅ 是 / 🔴 否]
□ 是否在报表中反映：[✅ 是 / 🔴 否]

□ 影响评估：
□ 对报表影响：[描述]
□ 对现金流影响：[描述]
□ 对持续经营假设影响：[✅ 无 / ⚠️ 可能]

□ 须报告：[CFO/CEO/董事会]
```

### 类型五：往来函证异常

**典型场景：**
- 往来单位不回函
- 回函金额与账面不符
- 回函显示争议或拒付

**判断流程：**

```
第一步：确认函证状态

□ 往来单位：[名称]
□ 函证内容：[应收账款/应付账款/其他往来]
□ 函证金额：[XXX万]
□ 发函日期：[YYYY-MM-DD]

□ 回函状态：
□ 已回函确认：[✅ 金额一致]
□ 已回函不符：[⚠️ 金额不符]
□ 未回函：[🔴 未回]
□ 二次发函：[是/否]

□ 回函金额：[XXX万]
□ 账面金额：[XXX万]
□ 差异：[±XXX万]

第二步：分析差异原因

□ 差异原因：
□ 记账时间差（发票未到/在途货物）
□ 销售退回/折让未入账
□ 争议项（质量/数量问题）
□ 重复记账
□ 错误记账

□ 争议性质：
□ 金额争议（数量/单价）
□ 品质争议（质量问题）
□ 付款争议（条件未满足）

□ 是否存在舞弊风险：[✅ 否 / ⚠️ 疑似]

第三步：制定催收/处理方案

□ 如为应付账款差异：
□ 多计应付 → 调账
□ 争议项 → 与供应商协商
□ 长期挂账 → 清理

□ 如为应收账款差异：
□ 少计应收账款 → 催收
□ 争议项 → 协商解决
□ 无法收回 → 计提坏账

□ 处理时间：[YYYY-MM-DD]

第四步：替代程序

□ 如长期不回函：
□ 替代程序：
□ 检查期后收款记录
□ 检查发货单/验收单
□ 检查发票/合同
□ 管理层说明

□ 替代程序充分性：[✅ 充分 / ⚠️ 不够充分]
```

## 置信度评估

| 置信度 | 条件 | 操作 |
|---|---|---|---|
| 高 | 函证全部回函 + 账账相符 + 档案齐全 | ✅ 正常分析 |
| 中 | 部分函证未回，但替代程序充分 | ⚠️ 补充替代程序后再分析 |
| 低 | 主要科目函证失败 + 存在重大差异 | 🔴 优先解决函证问题 |

## 升级触发条件

- 审计调整影响利润 > [X] 万
- 存在重大审计调整分歧
- 发现重大内控缺陷或舞弊迹象
- 银行/往来函证差异 > [X] 万
- 审计报告出具前仍有未决事项
