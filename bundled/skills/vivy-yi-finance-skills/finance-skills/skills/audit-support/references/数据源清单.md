# 数据源清单 — [ERP]/[DOC]/[BANK]

## 数据源标签说明

| 标签 | 含义 | 典型来源 |
|---|---|---|
| [ERP] | 企业 ERP 系统 | SAP/用友/金蝶总账 |
| [DOC] | 档案管理系统/合同管理系统 | 文档管理系统/飞书/钉钉审批 |
| [BANK] | 银行系统/银企直联 | 网上银行/ERP 银行模块 |

---

## [ERP] SAP / 用友 / 金蝶（总账）

**覆盖范围：**
- 科目余额表（月度/年度）
- 序时账（凭证明细）
- 财务报表（BS/PL/CF）
- 应收应付明细
- 固定资产卡片
- 银行日记账
- 税务申报记录

**查询路径：**

```
SAP 常用事务代码：
- FAGLB03：科目余额表
- FBL5N：客户行项目（应收账款）
- FBL1N：供应商行项目（应付账款）
- FBL3N：科目行项目
- S_ALR_00001236：财务报表执行
- AFAR：在建工程
- AW01N：资产主数据

用友/金蝶：
- 总账 → 科目余额表
- 总账 → 序时账
- UFO report：财务报表
- 应收管理 → 客户明细
- 应付管理 → 供应商明细
```

## [DOC] 档案管理系统 / 合同管理系统

**覆盖范围：**
- 采购/销售合同
- 发票/收据
- 付款审批单
- 固定资产采购合同
- 租赁协议
- 股东会/董事会决议
- 担保合同
- 内部制度文件

**查询路径：**

```
档案管理系统：
- 合同管理 → 合同台账
- 档案借阅 → 审批记录

飞书/钉钉：
- 审批记录 → 付款审批
- 合同审批

物理档案室：
- 凭证档案（按年度）
- 合同档案（按类别）
- 工商档案
```

## [BANK] 银行系统 / 银企直联

**覆盖范围：**
- 银行对账单
- 银行回单
- 银行承兑汇票
- 信用证记录
- 贷款合同
- 冻结/抵押记录

**查询路径：**

```
网上银行：
- 账户管理 → 对账单下载
- 交易记录 → 明细查询
- 回单查询

银企直联（ERP）：
- 银行日记账
- 电子回单

柜台查询：
- 存款证明
- 冻结/抵押证明
- 贷款余额证明
```

## 数据源组合使用

### 货币资金审计
```
[ERP] 银行日记账 → [BANK] 银行对账单 → 发函 → 回函核对 → 差异调节
```

### 应收账款审计
```
[ERP] 客户明细 → 账龄分析 → 发函 → 回函核对 → 替代程序（发货单/发票）
```

### 固定资产审计
```
[ERP] 固定资产卡片 → [DOC] 采购合同/验收单 → 盘点 → 折旧测算 → 减值评估
```

### 审计调整
```
审计调整分录 → 管理层评估 → [ERP] 过账 → 调整后余额核对 → 审计确认
```

## Connector 映射

|  | 数据需求 | Connector | 工具 | 关键字段 |
|------|---------|-----------|------|---------|
| 财务数据/凭证 | SAP | `sap_gl_query` | `BELNR`, `RACCT`, `BUDAT`, `DMBTR` |
