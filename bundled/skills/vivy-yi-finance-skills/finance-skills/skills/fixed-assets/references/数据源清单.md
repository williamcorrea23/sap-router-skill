# 数据源清单 — [FA]/[ERP]/[PO]

## 数据源标签说明

| 标签 | 含义 | 典型来源 |
|---|---|---|
| [FA] | 固定资产系统 | SAP 资产模块/用友资产/金蝶固定资产 |
| [ERP] | 企业 ERP 系统（总账）| SAP/用友/金蝶总账 |
| [PO] | 采购管理系统 | SAP MM/用友采购/金蝶采购 |

---

## [FA] SAP 资产模块 / 用友资产 / 金蝶固定资产

**覆盖范围：**
- 固定资产台账（资产卡片）
- 折旧计算与计提
- 资产增加/减少/变动
- 资产减值准备
- 资产盘点管理
- 在建工程转固
- 使用权资产（IFRS 16）
- 租赁管理

**查询路径：**

```
SAP 资产模块常用事务代码：
- AS03：显示资产卡片
- AFAB：折旧运行
- AIAB：资产待报废
- ABAVN：资产报废
- AZMN：资产减少
- S_ALR_00001233：资产账簿报表

用友/金蝶固定资产：
- 资产台账 → 资产清单
- 折旧清单 → 月折旧计提表
- 资产变动 → 增加/减少/减值记录
- 盘点管理 → 盘点差异处理
```

---

## [ERP] SAP / 用友 / 金蝶（总账）

**覆盖范围：**
- 固定资产原值科目
- 累计折旧科目
- 固定资产减值准备科目
- 固定资产清理科目
- 在建工程科目
- 使用权资产科目
- 租赁负债科目
- 资产减值损失科目
- 营业外收支（资产处置损益）

**查询路径：**

```
SAP 常用事务代码：
- FAGLB03：科目余额表
- FB03：凭证明细
- S_ALR_00001234：固定资产报表

用友/金蝶：
- 总账 → 固定资产（原值/累计折旧/减值）
- 应收应付 → 相关（不直接涉及）
- 固定资产系统数据 ← → 总账数据须一致
```

---

## [PO] SAP MM / 用友采购 / 金蝶采购

**覆盖范围：**
- 资产采购订单/合同
- 采购申请/审批
- 验收单/入库单
- 供应商发票
- 资产请购流程

**查询路径：**

```
SAP MM：
- ME21N：创建采购订单
- ME2L：按供应商查看采购订单
- MIGO：货物移动/验收
- MIRO：发票校验

用友/金蝶采购：
- 采购订单 → 采购合同管理
- 验收单 → 资产验收记录
- 到货查询 → 在建工程/固定资产入账依据
```

## 数据源组合使用

### 月度折旧核对
```
[FA] 折旧运行 → [FA] 折旧清单 → [ERP] 折旧凭证 → 金额一致性核对
```

### 资产增加审核
```
[PO] 采购订单 → [PO] 验收单 → [FA] 资产增加台账 → [ERP] 入账凭证
```

### 盘点差异处理
```
[FA] 资产台账 → [FA] 盘点差异表 → [FA] 盘盈/盘亏确认 → [ERP] 入账
```

### 减值测试
```
[FA] 资产账面价值 → [FA] 可收回金额评估 → [ERP] 减值损失入账 → [FA] 减值准备更新
```

### 在建工程转固
```
[PO] 验收单 → [ERP] 在建工程转出 → [FA] 资产增加登记 → [FA] 折旧开始计算
```
