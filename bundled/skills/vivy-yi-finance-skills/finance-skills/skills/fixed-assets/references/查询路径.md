# 查询路径 — 固定资产管理

## 数据源优先级

| 优先级 | 数据源 | 说明 |
|---|---|---|
| 1 | 固定资产系统（[FA]）| 资产台账/折旧/变动 | 最优先 |
| 2 | ERP 总账（[ERP]）| 固定资产科目余额 | 账务核对 |
| 3 | 采购系统（[PO]）| 资产采购申请/验收 | 入账依据 |

## 数据源组合使用

### 月度折旧核对
```
[FA] 折旧计算表 → [ERP] 折旧入账凭证 → 对比金额一致

### 资产增加审核
```
[PO] 采购订单 → [FA] 验收单 → [ERP] 发票 → 入账凭证

### 盘点差异处理
```
[FA] 资产台账 → [FA] 盘点差异表 → [ERP] 盘盈/盘亏入账

### 减值测试
```
[FA] 资产账面价值 → [FA] 可收回金额 → [ERP] 减值入账
```
