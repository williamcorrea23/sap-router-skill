# /fixed-assets-master — 固定资产管理主流程

## 完整固定资产管理流程

```
Step 1：折旧分析（月度）
  → 执行 depreciation-analysis
  → 确保折旧计提准确

Step 2：资产变动核查
  → 核查本月新增/减少资产
  → 新增资产：执行 asset-acquisition-review
  → 减少资产：执行 asset-disposal-review

Step 3：盘点核查（定期/异常时）
  → 核查账实一致性
  → 盘盈：按 asset-disposal-review（反向）
  → 盘亏：执行 asset-disposal-review

Step 4：减值评估（年度/异常时）
  → 评估减值迹象
  → 计算可收回金额
  → 执行减值处理

Step 5：综合管理报告
  → 汇总 Step 1-4 发现的问题
  → 输出固定资产健康度报告
```
