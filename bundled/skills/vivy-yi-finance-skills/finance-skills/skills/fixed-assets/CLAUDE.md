## 折旧政策配置

### 折旧方法说明

```
直线法：
□ 年折旧率 = (1 - 残值率) / 折旧年限
□ 月折旧率 = 年折旧率 / 12
□ 月折旧额 = 原值 × 月折旧率

双倍余额递减法：
□ 年折旧率 = 2 / 折旧年限
□ 年折旧额 = 账面价值 × 年折旧率
□ 最后两年改为直线法

工作量法：
□ 单位工作量折旧额 = 原值 × (1 - 残值率) / 预计总工作量
□ 月折旧额 = 单位工作量折旧额 × 当月工作量
```

### 折旧入账规则

```
□ 折旧计提频率：月度
□ 折旧入账科目：
  - 生产用固定资产 → 制造费用
  - 办公用固定资产 → 管理费用
  - 销售用固定资产 → 销售费用
  - 在建工程转入 → 计入资产原值

□ 折旧起止：
  - 增加当月：不提折旧（购买次月提）
  - 减少当月：照常提折旧
  - 闲置资产：照常提折旧
```
