# GTS 일상 운영 (Tier 1) Best Practice

- **GTS On-Premise**: ✓

- **한국 특화**: ✓ (수출입통관·FTA)

## 일간 운영

### Compliance Management
- [ ] **Sanction Party List Screening** — 일간 자동 실행
- [ ] **Embargo Check** — 금수국 거래 차단
- [ ] **License Determination** — 라이선스 적용 거래

### Customs Management
- [ ] **Customs Declaration** — 한국 관세청 연동
- [ ] **Tariff Classification** — HS 코드 정확성
- [ ] **Customs Document** — 신고서 작성·전송

### Trade Preference (FTA)
- [ ] **Preference Determination** — 원산지 결정
- [ ] **Long-term Vendor Declaration** — 공급자 신고서
- [ ] **Certificate of Origin** — 원산지 증명서
