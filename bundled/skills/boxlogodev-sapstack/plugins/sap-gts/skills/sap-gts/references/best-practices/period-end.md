# GTS 분기·연 마감 (Tier 2) Best Practice

## 분기 마감

### 무역 통계
- [ ] **Export Statistics** — 수출 통계 (관세청 의무)
- [ ] **Import Statistics** — 수입 통계
- [ ] **Country-of-origin Report** — 원산지별 집계

### Compliance 평가
- [ ] 제재 위반 사례 0건
- [ ] License 활용율
- [ ] FTA 활용률 (Origin Determination 비율)

## 연 마감

### 연간 보고
- [ ] **Annual Export Compliance Report** (DRDA 등)
- [ ] **FTA Annual Statement** — 활용 실적
- [ ] **Customs Audit** — 관세청 사후 심사 대응

### Master Data 정비
- [ ] HS 코드 마스터 갱신 (신년 분류 변경)
- [ ] 라이선스 마스터 갱신
- [ ] 공급자 declaration 재취득

### Tariff Updates
- 신년 관세율 적용 (1월 1일)
- FTA 단계별 인하 적용
- 부가가치세·소비세 변경 반영

## 한국 특화

- 관세청 사후 심사 대비
- FTA Annual Compliance (한·미 FTA 등)
- AEO (Authorized Economic Operator) 재인증
