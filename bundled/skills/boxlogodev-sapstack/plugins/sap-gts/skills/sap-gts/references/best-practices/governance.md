# GTS 거버넌스·무역 컴플라이언스 (Tier 3) Best Practice

## 1. 수출 통제 (Export Control)

### 1.1 규제 프레임워크
- **한국 전략물자 수출 통제** (산업통상자원부)
- **미국 EAR** (Export Administration Regulations)
- **EU 듀얼유즈** (Council Regulation 2021/821)
- **UN 제재** (UN Security Council Resolutions)

### 1.2 제재 스크리닝
- 매 거래 자동 스크리닝
- 매치 시 거래 차단 + 검토 워크플로우
- 결과 보존: 5년 (감사 대응)

## 2. 통관 거버넌스

### 2.1 한국 통관 절차
- **수출 통관**: 신고 → 검사 → 적재 → 출항
- **수입 통관**: 도착 → 신고 → 검사 → 통관 → 반출
- **TSC** (Tariff Service Center): HS 코드 자문
- **UNI-PASS**: 전자 통관 시스템

### 2.2 AEO (Authorized Economic Operator)
- 한국 AEO 인증 (KCS)
- 인증 → 통관 간소화·우선 검사
- 3년 재인증 필요

## 3. FTA 거버넌스

### 3.1 원산지 관리
- **WO** (Wholly Obtained): 완전 생산
- **CTC** (Change in Tariff Classification): 세번 변경
- **VA** (Value Added): 부가가치 기준
- **Specific Process**: 특정 공정

### 3.2 원산지 증명
- **KCS-FTA**: 한국 자율 증명
- **EUR.1**: 한·EU FTA
- **US FTA**: 자율 증명 + 사후 검증
- **CPTPP**: 자율 증명

### 3.3 사후 검증 대응
- 협정 상대국 검증 요청 대응
- 증빙 문서 보존: 5년 (한국 기준)
- 부정 발급 시 벌금·관세 추징

## 4. 위험 관리

### 4.1 무역 위험 분류
- **High**: 제재국·금수품·전략물자
- **Medium**: 신규 시장·신규 품목
- **Low**: 기존 거래·표준 품목

### 4.2 위험 대응
- High → 별도 승인 워크플로우
- Medium → 분기 검토
- Low → 표준 절차

## 5. 컴플라이언스 지표

| 지표 | 임계 |
|---|---|
| 제재 스크리닝 매치 처리율 | 100% (24시간 내) |
| 통관 신고 정확율 | > 98% |
| FTA 활용률 | > 80% (협정 대상 거래 중) |
| 사후 검증 통과율 | 100% |
| AEO 인증 유지 | 활성 |

## 6. 외부 감사·검증

- 관세청 사후 심사: 분기 대응 가능 상태 유지
- 외부 무역 감사: 연 1회 권장
- 내부 통제 평가: 분기 1회
