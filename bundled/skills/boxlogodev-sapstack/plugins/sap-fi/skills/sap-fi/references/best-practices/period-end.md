## 기간마감 절차 (Period-End Close Process)

### 📋 Pre-Close 체크 (T-5 ~ T-1일, 기간말 전 5~1일)

#### 기간 잠금 상태 확인
- [ ] **T-code OB52 — 회계기간 현황 조회**
  - 현재 개방 기간 확인
  - 폐기할 기간 확인 (이월기간 설정)
  - 담당: 마스터 데이터 관리자
  - 액션: 목표 마감 기간이 정확히 설정되었는지 재확인

#### 미처리 IDoc 및 배치작업 확인
- [ ] **T-code BD87 — IDoc 모니터링 (EDI)**
  - 폐기된 IDoc 상태 확인 (Status 29-39)
  - 미처리 구매/판매 오더 IDoc 재전송 필요 여부 판단
  - 담당: 인터페이스 담당자
  - 액션: 정기 배치 완료 여부 확인 (ZEDIFM101, ZMMBATCH 등)

#### 환율 테이블 최신화
- [ ] **T-code OB08 — 외화 환율 마지막 확인**
  - 기간말 환율 공식(한은, 기업은행) 반영 확인
  - 차입금/장기채무 통화별 환율 동기화
  - 담당: 재정담당자
  - 액션: 환율 입력 후 T-code ABASTEST로 신뢰성 검증

#### 미결 항목 정리
- [ ] **T-code F.13 — 미결 항목 정리**
  - 60일 이상 미결 항목 처리 여부 재확인
  - 선급금/미청구 입고 정상화
  - 담당: 재무회계 담당자

#### 파킹 전표 최종 확인
- [ ] **T-code FV50 — 파킹 전표 조회**
  - 기간말에 파킹 상태인 전표 즉시 최종화
  - 기한경과 파킹 전표는 역분개 또는 재작성
  - 담당: 재무회계 담당자

---

### 🔄 Close 실행 단계 (T일, 기간마감일)

#### Step 1: GR/IR 정리 (Material Management 연계)
- [ ] **T-code MR11 — GR/IR 우발채무 시뮬레이션**
  - 60일 이상 미청구 입고 → 예상 비용 계산
  - 선택 항목: 실제 청구 또는 근사가격으로 청구 시뮬
  - 담당: 구매/원가 담당자
  - 절차:
    1. 시뮬레이션(S) 모드 실행
    2. 생성될 전표 검증
    3. 실행(A) 모드로 최종 생성

#### Step 2: 감가상각 계산
- [ ] **T-code AFAB — 감가상각 배치 실행**
  - 유형자산 감가상각 자동 계산
  - 당월 신규 자산 감가상각 시작 여부 확인
  - 담당: 재산 담당자
  - 주의사항:
    - 선취소각(Accelerated) 설정 확인 (K-GAAP vs IFRS)
    - 소수점 넷째자리 반올림 정책 일관성
    - 폐기/처분 자산 선행 제거 완료

#### Step 3: 외화 평가
- [ ] **T-code FAGL_FC_VAL — 외화 자산/부채 평가**
  - 외화 채권/채무 기간말 환율로 평가
  - 환차익/환차손 자동 생성
  - 담당: 재정담당자
  - 설정:
    - 평가 기준 환율(T-code OB08)
    - 유예 계정(예: 4199 장기차입금 환차이연)
    - 당월 신규 거래 포함 여부

#### Step 4: 미결 정리 최종
- [ ] **T-code F.13 — 미결 정리 재실행**
  - Step 1~3의 전표 생성으로 인한 신규 미결 정리
  - 환차손익 선급금 상각
  - 담당: 재무회계 담당자

#### Step 5: 기간 마감 (잠금)
- [ ] **T-code OB52 — 회계기간 폐기**
  - 기간 상태를 "열림" → "마감"으로 변경
  - 폐기 범위: 재무회계(FI), 자산회계(AA), 원가회계(CO)
  - 담당: 마스터 데이터 관리자
  - 최종 확인:
    - 이월 기간(Next Period) 자동 개방 여부 확인
    - 폐기 후 역거래 불가상태 재확인

### ✅ Post-Close 검증 (T+1 ~ T+3일)

#### 재무제표 검증
- [ ] **T-code S_ALR_87012284 — 재무상태 보고서**
  - 자산/부채/자본 합계 검증
  - 기간 대비 변화액 타당성 검토
  - 담당: 재정담당자
  - 비교:
    - 전월 같은 항목 대비 증감률
    - 연간 누적 대비 월간 기여도

#### 손익 대사
- [ ] **T-code S_ALR_87012979 — 손익계산서**
  - 매출, 비용, 영업이익, 순이익 검증
  - 예산 대비 실적 대비율(Variance %)
  - 담당: 예산/경영분석 담당자

#### Trial Balance (TB) 정합성
- [ ] **T-code S_ALR_87012269 — 시산표**
  - 차변 = 우변 균형 확인
  - 부정 잔액 계정 조사
  - 담당: 재무회계 담당자
  - 확인항목:
    - 임시계정(99xx) 전액 정리 확인
    - 특별항목(예: 자산재평가) 별도 라인 표시

#### GR/IR 정리 효과 검증
- [ ] **T-code MB5S — 기간말 GR/IR 최종 확인**
  - 미정리 GR/IR 금액 < 전월의 5% 확인
  - aged된 항목 원인 분석
  - 담당: 구매/원가 담당자

## 주요 위험 시나리오 & 대응

| 시나리오 | 원인 | 대응 액션 | 심각도 |
|---------|------|---------|--------|
| GR/IR 미정리 후 마감 | 구매 부서 미완료 | MR11 재실행 + 이월 보고 | 🟠 High |
| 음수 GR/IR 발생 | 중복 입고 또는 선인수정 | MB51 및 GR이력 재조사 | 🔴 Critical |
| 환율 입력 누락 | 마감일 환율 공시 지연 | 전일 환율 사용 + 차월 조정 사항 기록 | 🟠 High |
| 파킹 전표 기간 사이 혼재 | 년도 변경 또는 기간 오기입 | FV50 조회 → 역분개 → 재작성 | 🔴 Critical |
| TB 불균형 발생 | IDoc 누락 또는 자동배치 실패 | SM37 배치 이력 조회 → 재실행 | 🔴 Critical |

## K-SOX & IFRS 준수 사항

- **감가상각 정책일관성**: 자산 인수 시점별 상각 시작일 명시 (IFRS 16 리스 계약 포함)
- **환차손익 회계처리**: 선금액/장기차입금 환차이연 기간 명시
- **재무제표 이중 검토**: 재정담당 > 경영진 의사결정 구조 문서화
- **마감 일정 통제**: 기간 폐기 권한은 부사장급 이상 승인 구조
