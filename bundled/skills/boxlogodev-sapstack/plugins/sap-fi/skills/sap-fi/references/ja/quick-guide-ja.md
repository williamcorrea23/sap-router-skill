# sap-fi クイックガイド (日本語)

> SAP FI (財務会計) クイックリファレンス. 詳細は `SKILL.md` および `references/closing-checklist.md` 参照.

## 🔑 環境インテーク

1. SAP リリース (ECC 6.0 EhPx / S/4HANA 19xx-23xx)
2. 展開モデル (オンプレミス / RISE / Cloud PE)
3. 会計年度バリアント (K1 4月始まり日本標準 / K4 暦年 / カスタム)
4. 会社コード (ユーザー提供 — 推測禁止)
5. エラーメッセージ番号 + 発生した T-code

## 📚 コアモジュール

### AP — 買掛金
- **FB60 / MIRO** 仕訳エラー:
  - 税コード未割当 → **FTXP** 確認
  - 許容差超過 → **OMR6** 会社コードごとの限度
  - GR ベース IV 不一致 → PO 明細 Invoice タブ vs 入庫数量
- **F110** 支払処理:
  - 支払方法欠落 (LFB1-ZWELS)
  - 銀行未決定 → **FBZP**
  - DME ファイル未生成 → **DMEE** ツリー (支払方法ごと)
- 源泉徴収税 (拡張) — **WTAD** + SPRO 割当

### AR — 売掛金
- **FD32** (ECC) / **UKM_BP** (S/4 FSCM): 与信管理
- **F150** 督促 → **FBMP** 督促手続
- **VKM1 / VKM3**: 与信ブロック解除 (受注 / 出荷)
- 前受金: **F-37** (請求), **F-29** (仕訳), **F-39** (消込)

### GL — 総勘定元帳
- フィールドステータス競合 — 3 つのソース: **OBC4** (伝票タイプ) + **OB14** (記帳キー) + **FS00** (勘定科目)
- 外貨評価 — **FAGL_FC_VAL** (必ずテストランから)
- 自動消込 — **F.13** (テストラン優先)
- 残高繰越 — **F.16** (ECC) / **FAGLGVTR** (新 G/L)

### AA — 固定資産会計
- 減価償却処理 — **AFAB**
- 資産振替 — **ABUMN**
- 年次クローズ — **AJAB**

## 🚨 よくある問題

### "FB01 で仕入先統制勘定に直接仕訳できない"
- 根本原因: LFB1-AKONT は統制勘定
- 解決: **FB60** (仕入先請求書) または特別 G/L (F-47/F-48) を使用. FB01 は統制勘定への直接仕訳不可.

### "F110 で何も選択されない"
- 仕入先に支払方法なし (XK03 → LFB1.ZWELS 空)
- 明細がまだ期日前 — 期日確認
- 会社コードが支払処理で非活性

### "MIRO で税コード不一致"
- 税コードが仕訳禁止状態 (FTXP)
- 会社コード固有税務手続変更
- 反対仕訳 + 正しいコードで再入力

### "期間クローズ済 — 仕訳不可"
- OB52 → 権限グループ調整 (前期間を安易に開かない)
- 年次繰越状況確認 (F.16 / FAGLGVTR)

## ECC vs S/4HANA 要点

- **GL**: BSEG/BKPF → ACDOCA (S/4 ユニバーサルジャーナル)
- **AR 与信**: FD32 → UKM_BP (FSCM)
- **AA**: クラシック AA → 新固定資産会計 (S/4 必須)
- **銀行**: FI12 → BAM (Bank Account Management)

## 🇯🇵 日本ローカル考慮

- **消費税**: 日本独自の消費税率管理 (税抜・税込・軽減税率)
- **インボイス制度** (2023~): 適格請求書発行事業者番号 (T番号)
- **会計基準**: J-GAAP (日本基準) + IFRS 並行
- **印紙税 (Stamp Duty)**: 高額契約書の電子印紙対応
- **会計年度 (4月-3月)**: K1 fiscal year variant
- **税務調査**: CDPOS/CDHDR 7年以上保存必須

## ⚠️ スコープ外

- 販売請求書 (SD 使用)
- コストセンター会計 (CO 使用)
- 製造原価追跡 (CO + PP 使用)
- 資金管理 (TR プラグイン)
