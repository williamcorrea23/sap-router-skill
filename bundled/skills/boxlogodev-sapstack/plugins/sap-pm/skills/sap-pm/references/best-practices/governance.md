## 체크리스트

### 산업안전보건법 준수
- [ ] **보전 기록 5년 보관** — 한국 산업안전보건법 §44
  - 정책: 모든 보전 이력을 SAP에서 추적 가능하게 유지
  - 범위:
    ```
    - 예방보전 계획 및 실적
    - 정기 점검 결과
    - 고장 발생 및 조치 기록
    - 부품 교체 이력
    - 기술자 자격증 및 교육 이력
    ```
  - 도구: IW29(설비 이력), IW38(오더 기록), Archiving(장기 보관)
  - 감사: 연 1회 검증(감시기관 불시 점검 대비)

- [ ] **안전 관련 점검 기록** — 한국 산업안전보건법 §49
  - 정책: 특별 안전 점검 결과 5년 보관
  - 대상: 
    ```
    - 승강기, 보일러, 압력용기 (정기 점검)
    - 전기 안전(월 1회)
    - 기계 안전(분기 1회)
    ```
  - 도구: PM 모듈 커스텀 체크리스트 (IP30 예방보전 연계)
  - 기록: PDF 또는 SAP DMS(Document Management) 저장

### 예방보전 이행률 KPI
- [ ] **연간 예방보전 이행률 목표** 
  - 정책: 예방보전 비율 60% 이상 유지
  - 계산:
    ```
    예방보전 이행률 = 실제 예방보전 건수 / 계획 예방보전 건수 × 100%
    
    예시:
    - 월 계획 예방보전: 30건
    - 월 실제 예방보전: 28건
    - 이행률: 93.3% (목표: 90%)
    ```
  - 효과:
    ```
    예방보전 비율 ↑ → 긴급 보전 감소 → 가동률 ↑, 비용 ↓
    예방보전 비율 ↓ → 고장 증가 → 생산 차질, 비용 ↑
    ```
  - 빈도: 월 1회 추적, 분기별 보고

### 장비 폐기/신규 절차
- [ ] **IM01 자산 폐기 승인 프로세스** — T-code: IM01 (FI-AA)
  - 정책: PM에서 제안 → FI(자산담당)에서 승인 → 폐기 처리
  - 절차:
    ```
    1. IW29로 설비 이력 분석
       - 고장율: MTBF 목표의 50% 이하?
       - 복구비용: 신규 설비 가격의 70% 이상?
       - 기술 낙후: 산업 표준에 미달?
    
    2. PM에서 제안 문서 작성
       - 폐기 대상 설비
       - 폐기 사유
       - 신규 설비 제안 (필요시)
       - 예상 비용 절감
    
    3. 경영진 승인 (투자심의위원회)
       - 폐기 자산 감가상각 완료 확인
       - 보험 청구 여부 검토
    
    4. IM01 폐기 처리
       - 상태: 'ABST'(폐기)로 변경
       - 시스템 기록: 폐기 일자, 폐기 방법
    
    5. 물리적 폐기 실행
       - 설비 제거 및 폐기
       - 폐기증명서 수취
    ```
  - 주의: 폐기 자산은 감가상각 완료 후 처리

### 신규 설비 도입 프로세스
- [ ] **신규 설비 PM 파라미터 설정** — T-code: IM01 → PM
  - 절차:
    ```
    1. 자산 등록 (IM01, FI-AA)
       - 자산 번호 부여
       - 감가상각 방법 설정
    
    2. PM 마스터 등록 (IP01)
       - Equipment ID 부여
       - 위치 설정 (플랜트, 구역)
       - 담당 기술자 배정
    
    3. BOM 및 부품 등록 (MM01)
       - 예비부품 마스터 생성
       - 재고 위치 설정 (보통 Central Store)
    
    4. 예방보전 계획 수립 (IP01/IP10)
       - 점검 주기 설정 (월/분기/년)
       - 표준 작업 시간 설정
       - 필요 부품 목록 정의
    
    5. 담당 기술자 교육
       - 매뉴얼 제공
       - 공급업체 기술 교육
       - 초기 100시간: 고강도 모니터링
    ```
  - 빈도: 신규 설비 도입 시마다

### 보전전략(Maintenance Strategy) 연간 검토
- [ ] **보전전략 재평가** — 연간 1회 (11월)
  - 정책: 산업 동향, 기술 발전, 설비 노후화 반영
  - 검토 항목:
    ```
    1. 설비별 보전 방식 재평가
       - 예방보전 vs 상태기반(CBM) 비율
       - 예: 컨베이어 → 예방보전 100% 유지
         압축기 → 상태기반 50% 도입 검토
    
    2. 외주 vs 자체 보전 비율
       - 자체: 내부 역량 활용 (숙련도, 비용 효율)
       - 외주: 전문 기술 (복잡한 설비, 보증 유지)
       - 혼합: 대부분의 경우
    
    3. 신기술 도입 평가
       - IoT 센서: 상태 모니터링 자동화
       - AI 예측: 고장 사전 예측
       - 드론 점검: 고공 설비 점검 비용 절감
    
    4. 부품 전략
       - 재고 vs JIT(Just In Time)
       - 신뢰성 부품 vs 저가 부품 트레이드오프
    
    5. 인력 계획
       - 기술자 나이: 은퇴 인력 > 신입 배치?
       - 교육: 신기술 교육 필요?
       - 급여 경쟁력: 이직 위험?
    ```
  - 산출물: 연간 보전 전략서 (경영진 승인)

### 보전 예산 관리
- [ ] **연간 보전 예산 책정** — 매년 9월
  - 기준:
    ```
    보전 예산 = 설비 자산가액 × 1~3% (산업 평균)
    
    예시:
    - 설비 자산가액: ₩50억
    - 보전 예산: ₩7,500만 ~ ₩15억
    - 배분:
      * 인력비(급여+복리): 60~65%
      * 자재비(부품+소모품): 20~25%
      * 외주비(전문가, 도급): 10~15%
    ```
  - 조정: 분기별 실적 대비 ±10% 편차 분석

### 보전 기술자 자격 및 교육
- [ ] **기술자 자격증 관리** — HR 연계
  - 정책: 주요 설비별 자격증 보유자 최소 2명
  - 대상:
    ```
    - 승강기 기술사
    - 보일러 기술자
    - 전기 안전 관리자
    - 대형 기계 기술자
    ```
  - 관리: HR 정보 시스템 연계 (자격증 만료일 추적)
  - 비용: 연간 교육 예산 편성

### 외주 보전업체 관리
- [ ] **외주업체 성과평가** — 분기별
  - 평가지표:
    ```
    - 납기 준수율: 계약 기한 내 완료 %
    - 품질: 재작업 비율
    - 원가: 견적 대비 실제비용
    - 안전: 안전사고 발생 여부
    - 고객만족도: 내부 만족도 조사
    ```
  - 결과:
    - 평가점수 ≥ 80점: 계약 갱신
    - 60~80점: 개선 계획 수립
    - < 60점: 계약 해지 검토

## 문제해결 시나리오

### Scenario 1: 고장율 급증
```
증상: MTBF 목표(720h) 대비 실제(400h) 45% 악화
원인:
- 신입 기술자 부족 → 예방보전 누락
- 부품 품질 저하 → 조기 마모
해결:
1. IW29로 고장 패턴 분석
2. 부품 공급업체 변경 검토
3. 기술자 교육 강화
4. 임시 외주 보전 확대
조치: 보전전략 개정 (예방보전 100% 강화)
```

### Scenario 2: 보전비 예산초과 30%
```
상황: 예산 ₩10억 → 실제 ₩13억
원인: 긴급 고장 6건 (통상 2~3건) → ₩3억 추가 소요
대응:
1. 긴급 고장 근본원인 분석
   - 예비부품 부족? → MMBE 재고 재평가
   - 기술자 역량 부족? → 외주 강화
   - 설비 노후화? → 신규 도입 검토
2. 위험관리: 차기년 예산 15% 증액 검토
3. 보전 효율성 추진과제: IoT 센서 도입 타당성 분석
```

## 감사 체크리스트 (Annual Audit)

```
[ ] 보전 기록 보관 확인
    - 최근 5년 이력: 완전성 100%
    - 디지털 보관(DMS): SAP 연계 확인

[ ] 안전 관련 점검 기록
    - 정기 점검(승강기, 보일러): 누락 0건
    - 월간 전기 안전 점검: 12개월 완료

[ ] 예방보전 계획 vs 실적
    - 5개 설비 표본: 이행률 ≥ 60%

[ ] 기술자 자격증 현황
    - 자격증 보유자: 주요 설비별 2명 이상
    - 자격증 만료 예정: 갱신 계획 수립

[ ] 외주업체 평가
    - 분기 평가 완료: 4회 기록
    - 저성과 업체: 개선 또는 해지

[ ] 설비 폐기/신규 절차
    - 폐기 설비 3건: 감가상각 완료 확인
    - 신규 설비 IP01 등록: 완료

[ ] 예산 관리
    - 연간 예산 vs 실제: ±15% 범위

[ ] 보전전략 문서
    - 최신 버전 (11월): 경영진 승인
```
