# HCM 일간 운영 Best Practice

## 적용 범위
- ECC: ✓ | S/4HANA: 부분 지원(SuccessFactors 권장)

## 개요
인사 관리의 일일 점검 체크리스트. 인사 변경, 근태 관리, 급여 시뮬레이션을 포함합니다.

---

## 체크리스트

### 일간 인사변경 처리
- [ ] **PA30 인사변경 관리** — T-code: PA30
  - 왜: 조직 최신 상태 유지, 급여 정확성 확보
  - 빈도: 매일 09:00 (야간 승인 건 확인)
  - 세부:
    ```
    처리 대상:
    - 신입 입사: 채용 공고 → 최종 합격 → 입사
      필수 정보: 직급, 기본급, 근무지, 직책
    
    - 인사이동(전배, 승진):
      변경 사항: 부서, 직급, 급여(변경 시)
      승인 절차: 매니저 → HRM → CFO
    
    - 퇴직 처리:
      퇴직일 설정, 미지급금 정산, 퇴직금 계산
      마지막 급여: 다음 급여일 처리 또는 별도 지급
    
    - 급여 변경:
      인상/인하 사유(성과평가, 직급 변경, 시장 조정)
      기본급, 수당, 공제 항목 별도 관리
    ```
  
  - 승인 프로세스:
    ```
    PA30 입력 → 임시 저장(TEMP)
    → 매니저 검토 → HRM 승인 → 시스템 반영(MAIN)
    반영 시점: 신청월 말 또는 명시된 발효일
    ```

### 근태 이상 확인
- [ ] **PT_BAL00 / PT60 근태 이상** — T-code: PT_BAL00 또는 PT60
  - 왜: 근무 규칙 위반 조기 포착, 급여 정정 예방
  - 빈도: 매일 10:00 (출근 후)
  - 세부:
    ```
    이상 유형 및 대응:
    1. 지각(Late Arrival) > 30분
       조치: 지각 기록 → 차감 시간 급여 감액
       예: 지각 1시간 → 시급 × 1시간 감액
    
    2. 조퇴(Early Departure) > 1시간
       조치: 조퇴 기록 → 차감 시간 급여 감액
    
    3. 무단결근(Absent)
       조치: 결근 1일 → 일급 × 1일 감액 + 규정상 제약(경고, 감봉)
    
    4. 초과근무(Overtime) > 계획
       조치: 초과근무 기록 → 할증급 계산(보통 1.5배)
    
    5. 휴가 미신청 사용
       조치: 적법한 휴가인지 확인 → 병가인 경우 진단서 요청
    ```
  
  - 기록 위치: PT_BAL00 또는 임금대장(PT60)

### 급여 시뮬레이션
- [ ] **PC00_M99_CALC 급여 시뮬레이션** — T-code: PC00_M99_CALC (SIM)
  - 왜: 월 급여 전 검증, 오류 사전 방지
  - 빈도: 매달 급여 실행 전(보통 T-3일)
  - 세부:
    ```
    시뮬레이션 프로세스:
    1. 범위 선택: 월(예: 2026-04), 급여 형태(정규, 초과근무 등)
    2. 직원 필터: 전체 또는 특정 부서
    3. 실행(SIM=시뮬레이션 모드): 실제 반영 안 함
    4. 결과 검토:
       - 기본급: 명단 대비 일치 여부
       - 수당: 초과근무, 보너스 등 정확한가?
       - 공제: 세금, 보험료 올바른가?
       - 순급여: 예상치와 비교
    5. 이상 건 확인:
       - PA30로 인사변경 정정
       - PT_BAL00로 근태 정정
       - 수당 및 공제 항목 수정
    ```
  
  - 체크리스트:
    ```
    [ ] 신입/퇴직자: 부분 월 급여 계산 확인
    [ ] 인상/인하: 변경액 정확성 확인
    [ ] 초과근무: 시간 × 시급 × 1.5배 정확성
    [ ] 휴가 미지급(Carry-over): 차년도 휴가 설정 확인
    [ ] 적립금(퇴직연금): 적립률 정확성
    [ ] 세금: 누진세 계산 정확성
    ```

### 초과근무 모니터링
- [ ] **초과근무 데이터 검증** — T-code: PT60 또는 CATS
  - 왜: 근로기준법 준수(한국 52시간 제한), 건강/안전 관리
  - 빈도: 주 1회(목요일)
  - 세부:
    ```
    한국 법규:
    - 주 최대 초과근무: 12시간(휴일 제외)
    - 일 최대 초과근무: 4시간(휴일 제외)
    - 연 평균 주당 근무: 52시간 이하
    
    모니터링 항목:
    1. 개인별 주간 초과근무: 12시간 초과 여부
    2. 개인별 월간 초과근무: 누적 추이
    3. 부서별 평균 초과근무: 이상 부서 파악
    4. 위험군(고(高) 초과근무): 상담 및 휴무 권고
    
    조치:
    - 12시간 초과: 경고 + 휴무 강제
    - 누적 초과시간 > 월 200시간: 경보 + 원인 분석
    - 건강 이상 신호: 건강검진 권고
    ```

### 급여 관련 협의
- [ ] **HR-재무 일일 협의**
  - 시간: 매일 14:00 (15분 회의)
  - 참석: HRM 담당자, 급여 담당자
  - 의제:
    ```
    - PA30 미승인 건 확인
    - 초과근무 이상 건 검토
    - 당월 급여 일정 최종 확인
    - 특별 지급(인센티브, 상여금) 신청
    ```
