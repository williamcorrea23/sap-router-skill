# HCM 기간말 마감 Best Practice

## 적용 범위
- **ECC**: ✓
- **S/4HANA**: 부분 지원(SuccessFactors 권장)
- **한국 특화**: ✓ (4대보험, 원천세, 연말정산)

## 마감 프로세스

### Phase 1: 급여 실행 전 준비 (월 20일)

#### 인사변경 최종 확인
- [ ] **T-code PA30 — 월간 인사변경 마감**
  - 월 20일까지 완료
  - 확인사항:
    - 신입/퇴직 건: 급여 시뮬레이션 반영 여부 확인
    - 인상/인하: 금액 정확성 재검토
    - 휴가 사용: 반영 완료 여부
  - 승인: HRM 담당자 서명
  - 결과: PA30 변경사항 최종 리스트

#### 근태 데이터 검증
- [ ] **T-code PT60 — 근태 데이터 확정**
  - 월 22일까지 완료
  - 확인사항:
    - 지각/조퇴/결근: 모두 기록 완료
    - 초과근무: 시간 정확성 확인
    - 휴가: 신청 및 승인 완료
    - 미기록 건: 현업 담당자 협의하여 정정
  - 산출물: 근태 최종 확정서

#### 급여 항목 사전 검증
- [ ] **T-code PC00_M99_CALC (SIM 모드) — 급여 시뮬레이션**
  - 월 23일 실행
  - 프로세스:
    1. 시뮬레이션 범위: 해당월 + 전체 직원
    2. 급여 계산 실행 (SIM mode = 실제 반영 안 함)
    3. 결과 검토:
       ```
       - 직원 수: 예상치와 일치?
       - 총 기본급: 전월과 변동 사유 파악
       - 초과근무: 건수 × 시간 × 할증율 정확?
       - 공제: 세금, 보험 정상?
       ```
    4. 이상 건 조사:
       - 기본급 이상: PA30 재확인
       - 초과근무 이상: PT60 재확인
       - 공제 이상: 세율 설정(PKPLA) 확인
    5. 결과 승인: CFO 또는 HR팀장 사인
  - 산출물: 시뮬레이션 리포트 (이상 건 포함)

### Phase 2: 급여 실행 (월 25~26일)

#### PA03 릴리즈
- [ ] **T-code PA03 — 급여 실행 릴리즈**
  - 월 25일 오전 실행
  - 프로세스:
    1. 릴리즈 대상: 모든 변경된 PA30 오더
    2. PA03로 메뉴: 릴리즈
    3. 확인: 모든 PA30 오더가 released 상태로 변경
  - 검증: PA30 상태 변경 이력(T-code PA30, 변경 탭) 확인

#### PC00_M99_CALC 실행
- [ ] **T-code PC00_M99_CALC — 급여 계산 실행**
  - 월 25~26일 실행
  - 프로세스:
    1. 급여 유형: 정규급여 선택
    2. 범위: 해당월 + 전체 직원
    3. 실행 모드: CALC (실제 계산)
    4. 결과:
       ```
       - 총 지급액: 예상치와 비교 (±3% 허용)
       - 개별 직원 순급여: 샘플 5-10건 정확성 확인
       - 오류 메시지: 있으면 즉시 수정 후 재실행
       ```
  - 산출물: 급여 계산 결과 리포트

#### PC00_M99_CEDT 재무 전기
- [ ] **T-code PC00_M99_CEDT — FI 전표 생성**
  - PC00_M99_CALC 완료 후 즉시 실행
  - 프로세스:
    1. 급여 계산 결과 확인
    2. PC00_M99_CEDT 메뉴로 이동
    3. 코스트 센터별 급여비 배분 자동 계산
    4. FI 전표 자동 생성 (GL 계정별)
  - 검증:
    - 생성된 전표 확인: T-code SA38로 리포트 조회
    - FI 계정(예: GL 5101 급여비)에 월간 금액 적재 여부
  - 산출물: 급여 전표(FI 문서번호)

### Phase 3: 급여 지급 (월 마지막 영업일)

#### 급여 지급 실행
- [ ] **T-code PA03 / HR-PAY 급여 지급**
  - 월 마지막 영업일 실행
  - 방법:
    - 자동이체: HR시스템 → 은행 인터페이스 자동 송금
    - 또는 수동: FI 지급 시스템(F110 전자이체)
  - 확인:
    - 지급액 = PC00_M99_CALC 순급여 합계
    - 지급 계좌 정확성 확인 (직원별)
    - 지급 완료 로그 보관

#### 급여 명세서 배포
- [ ] **T-code HR-PAY / HRESS0008 — 급여명세서 생성 및 배포**
  - 월 마지막 영업일 또는 익월 초
  - 프로세스:
    1. 급여 명세서 자동 생성
    2. PDF 또는 종이 인쇄
    3. 직원 배포 또는 포탈 공개
  - 확인사항:
    - 각 직원 명세서 정확성 spot check (10건)
    - 개인정보(주민번호) 마스킹 여부 (한국 법규)

### Phase 4: 한국 월간 특화 마감 (익월 초)

#### 4대보험 신고 준비
- [ ] **T-code HRPA_REPORTING_KOR — 4대보험 신고 자료 생성**
  - 익월 초(1~5일)
  - 대상:
    - 국민연금: 9% 적립 + 기업 부담금
    - 건강보험: 3.4% 적립 + 기업 부담금
    - 고용보험: 0.9% 적립 + 기업 부담금
    - 산재보험: 산업별 요율 적용
  - 신고:
    - 각 공단별 온라인 신고(사이버 신청)
    - 또는 세무사 위임
  - 문서: 월간 신고 확인서 보관

#### 원천세 신고
- [ ] **T-code HRPA_TAX_KOR — 급여 원천세 신고**
  - 익월 10일까지 (필수)
  - 신고 대상: 근로소득세
  - 금액:
    - 과세 대상: 기본급 + 수당 - 비과세 항목
    - 원천세 = 과세 대상 × 세율(누진세)
  - 신고 방법:
    - 국세청 홈택스: 법인 담당자 로그인 → 급여지급 내역 신고
    - 또는 세무사 위임
  - 문서: 월간 신고 확인서 보관
  - 미신고 시 가산세 부과 (필수)

### Phase 5: 분기별/반기별 마감

#### 한국 분기말 특화
- [ ] **T-code HRPA_QUARTERLY_KOR — 고용보험 정산**
  - 4월, 7월, 10월, 1월 초
  - 실행:
    - 분기별 고용보험료 정산
    - 초과 납부분 환급 신청 또는 차기 납부액 조정
  - 문서: 고용보험 정산 내역 보관

### Phase 6: 연간 마감 (12월 ~ 2월)

#### 연말정산 준비
- [ ] **T-code HRPA_YEAR_END_KOR — 연말정산 자료 생성 (1월)**
  - 1월 초 ~ 중순 실행
  - 자료 생성:
    ```
    - 근로소득 원천징수영수증: 직원별
    - 장기근속 퇴직금(별도): 퇴직자
    - 소득공제: 교육비, 의료비, 기부금 등
    - 세액공제: 자녀세액공제, 근로소득세액공제
    ```
  - 직원 제출:
    - 추가 서류: 의료비 영수증, 교육비 영수증, 기부금 영수증
    - 제출 기한: 2월 말까지

#### 연말정산 계산
- [ ] **T-code PC00_M99_JAHR — 연말정산 계산 (2월)**
  - 2월 중순 ~ 말 실행
  - 프로세스:
    1. 연간 급여 합계 재계산
    2. 총 원천세 합계
    3. 최종 납부세액 계산
    4. 개별 직원별 환급 또는 추가 납부 산정
  - 결과:
    - 세무서 제출용 연말정산 신고서
    - 직원 고지서(환급/납부액)
  - 지급: 3월 월급에 연말정산 결과 적용

#### 퇴직연금 적립
- [ ] **T-code HRPA_PENSION_KOR — 연간 퇴직연금 적립**
  - 연 1회(12월 말 또는 계약 조항에 따라)
  - 금액:
    - 확정급여형(DB): 계리 평가액
    - 확정기여형(DC): 월 급여 × 연 12개월 × 적립율(보통 8.33%)
  - 신고:
    - 퇴직연금 기금 운영사에 납입
    - 연금사업자 확인서 보관
  - 세무 처리: FI 비용 배부

## 주요 위험요소

| 위험 | 증상 | 대응 |
|-----|------|------|
| 월급 지연 | PA03 릴리즈 미완료 또는 PC00_M99_CALC 실패 | 즉시 원인 파악, 임시 지급 후 정정 |
| 세금 신고 누락 | 익월 10일 경과 후 미신고 | 가산세 부과 → 세무사 긴급 신고 |
| 퇴직자 연말정산 오류 | 퇴직금 + 급여 별도 계산 미실시 | 별도 연말정산 계산서 재발행 |
| 4대보험료 부과 오류 | 보험료 과다 납부 또는 과소 납부 | 다음 분기 정산 시 조정 또는 환급 신청 |
