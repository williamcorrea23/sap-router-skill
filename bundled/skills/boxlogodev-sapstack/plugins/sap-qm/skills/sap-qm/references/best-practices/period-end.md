# QM 기간마감 Best Practice

## 적용 범위
- ECC: ✓ | S/4HANA: ✓

## 개요
월 마감 프로세스에서 품질 부문 담당자가 수행해야 할 정산, 검증 작업 목록.

---

## 기간 마감 프로세스

### Phase 1: Pre-flight 검사 (T-5일)

#### 1. 미결 검사 로트 확인
- [ ] **QA33 미결 로트 현황** — T-code: QA33
  - 필터: Status ∈ ('계획', '진행중') AND Creation Date ≤ 당월
  - 조치:
    ```
    - 당월 생산 로트: 마감 전 완료 추진
    - 차기월 예정 검사: 차기월 이월 확인
    - 고령 로트(>7일): 완료 또는 취소 결정
    ```

#### 2. 미처리 사용결정 확인
- [ ] **QA32 미처리 사용결정** — T-code: QA32
  - 필터: Status = '검사완료' AND Creation Date ≤ 당월
  - 조치:
    ```
    각 로트별 처리:
    - ACCEPT: 재고 해제 → MM 재고 증가 확정
    - REJECT: 공급업체 반품 처리 → ME22로 반품오더 생성
    - SCRAP: 폐기 처리 → 원가 반영
    - REWORK: 재공정 처리 → WIP 재계산 필요
    ```

#### 3. 미해결 품질통보 확인
- [ ] **QM10 미해결 품질통보** — T-code: QM10
  - 필터: Status = 'OPEN' AND Created ≤ 당월
  - 조치:
    ```
    각 통보별:
    - 종료 가능: QM01로 상태 '완료'(COMPLETED)
    - 미해결: 다음월 이월 또는 취소 결정
    ```

#### 4. 차단 재고 현황
- [ ] **MMBE 차단(Blocked) 재고** — T-code: MMBE
  - 필터: Restricted-use stock ≠ 0
  - 목표: 월말 차단재고 = 0 또는 최소화
  - 조치:
    ```
    각 부품별:
    - QA32로 재검사 → 사용결정
    - 또는 MB1C로 폐기 처리
    ```

### Phase 2: 월 마감 실행 (T-day)

#### 5. 검사 결과 통계 산출
- [ ] **검사 요약 보고서** — T-code: QSU / 커스텀 리포트
  - 산출:
    ```
    월별 검사 통계:
    - 검사 로트: 총 ___ 건
      * 수입 검사: ___ 건 (합격 ___, 부적합 _%)
      * 공정 검사: ___ 건 (합격 _%, 부적합 _%)
      * 완성품 검사: ___ 건 (합격 _%, 부적합 _%)
    
    - 공급업체별 통계:
      * A 공급업체: 합격률 _%, 반품률 _%
      * B 공급업체: 합격률 _%, 반품률 _%
    
    - 불량원인별 집계:
      * 형상 공차 초과: __%
      * 표면 흠: __%
      * 기능 부적합: __%
    ```

#### 6. 차단 재고 원가 반영
- [ ] **MMBE 차단재고 분류** — T-code: MMBE
  - 분류:
    ```
    차단 재고 = 회복 불가능(완전 폐기) + 회복 가능(재검사)
    
    회복 불가능:
    - MB1C로 폐기 처리 → 재고 감소 + 원가 손실
    
    회복 가능:
    - 차기월 이월 → 별도 재고 계정으로 관리
    - 또는 재공정 → WIP 증가로 다시 계산
    ```

#### 7. 공급업체 품질 평가
- [ ] **공급업체 품질 월별 평가** — T-code: QE3 / 커스텀
  - 평가:
    ```
    각 공급업체별:
    - 합격률 = ACCEPT / (ACCEPT+REJECT+SCRAP) × 100%
    - 반품률 = REJECT / (총입고량) × 100%
    - 부적합 원인 TOP 3 분석
    
    평가 결과:
    - A등급(합격률 ≥ 98%): 계약 갱신
    - B등급(95~98%): 개선 계획 수립
    - C등급(< 95%): 대체 공급업체 검토
    
    기록: 평가 결과 저장 → 차기월 단가 협상 근거
    ```

#### 8. 품질 KPI 계산
- [ ] **월별 품질 KPI** — 커스텀 대시보드
  - 계산:
    ```
    1. 수입 검사 합격률
       = 수입 합격 로트 / 총 수입 로트 × 100% (목표: 98%)
    
    2. 공정 검사 합격률
       = 공정 합격 로트 / 총 공정 로트 × 100% (목표: 99%)
    
    3. 완성품 합격률
       = 완성품 합격 로트 / 총 완성품 로트 × 100% (목표: 99.5%)
    
    4. 품질통보 처리율
       = (신규+미해결) 통보 / 월말 해결 통보 × 100%
    
    5. 공급업체 평균 합격률
       = 전체 공급업체 합격률 평균 (목표: 98%)
    ```

### Phase 3: 검증 (T+3일)

#### 9. 품질 손실 비용 분석
- [ ] **차단재고/부적합 원가** — T-code: MMBE + 원가분석
  - 분석:
    ```
    월간 품질 손실:
    - 부적합 부품 원가: ₩ ___
    - 폐기 부품 원가: ₩ ___
    - 재검사 비용: ₩ ___
    - 재공정 비용: ₩ ___
    - 고객 품질통보 대응 비용: ₩ ___
    
    합계: ₩ ___
    
    비율: 품질 손실 / 총 생산비 × 100% (목표: < 1%)
    ```

#### 10. FI와 대사
- [ ] **차단재고 vs FI 계정** — T-code: FBL5N + MMBE
  - 대사:
    ```
    QM 측: MMBE 차단재고 금액 합계
    FI 측: T-Account "차단재고"(예: 127000) 잔액
    
    차이 분석:
    - 재고평가 단가 변동
    - 반품 처리 오류
    - 폐기 미처리
    ```

### Phase 4: 마감 확인 (T+5일)

#### 11. 검사계획 유효성 재평가
- [ ] **QP01 검사계획 검토** — T-code: QP01
  - 평가:
    ```
    최근 3개월 검사 결과 기반:
    - 합격률 100%인 부품: 샘플링 축소 가능?
    - 불량률 높은 부품: 검사 강화 필요?
    - 새 부품 공정능력: 정규 검사로 전환?
    ```
  - 변경: 검사계획 수정 (QP01) → 효과적 시점 설정

#### 12. 월 마감 Lock
- [ ] **QM 기간 마감**
  - 효과: QA33, QA32 데이터 수정 불가
  - 예외: CFO 승인 후만 선금(후차월) 수정 가능

## 특수 상황 처리

### 대량 부적합 발생
```
상황: 공급업체 X 부품 500개 입고 → 검사 결과 25% 부적합 (125개)
처리:
1. QA32에서 REJECT 처리 → 125개 차단재고 생성
2. ME22로 반품오더 생성 → 공급업체 협의
3. QM02로 품질통보 생성 → 근본원인 분석
4. 영향 범위 파악: 이미 투입된 부품은?
   → 공정에 투입 완료: 생산 중단 또는 재작업
   → 재고에 있음: 차단 후 반품

월말 처리:
- 반품 승인까지 차단재고 유지
- 원가 영향: 미수금 계상 또는 반품오더 미정산
```

### 고객 품질 반품
```
상황: 완성품 출고 후 고객으로부터 부적합 반품 신고
처리:
1. QM02로 품질통보 생성 (높은 우선순위)
2. 원인 분석: 제조 오류? 운송 손상? 고객 오사용?
3. 책임 귀속:
   - 제조 오류: 리콜 비용 + 고객 보상
   - 운송 손상: 물류비 청구
   - 고객 오사용: 교육 (AS 대응)
4. 재발 방지: 검사 기준 강화 또는 패킹 개선

월말 처리:
- 환급액: 매출 역분(Return) 처리 (SD)
- 원가: 품질 손실 비용 증가
```

## 품질 통계 대시보드 예시

```
품질 현황 (2026년 4월)
=========================================

검사 현황
- 수입 검사: 120로트 / 합격 118 (합격률 98.3%) ✓
- 공정 검사: 85로트 / 합격 84 (합격률 98.8%) ✓
- 완성품 검사: 60로트 / 합격 60 (합격률 100%) ✓

공급업체 평가
- A공급업체: 합격률 99.2% (A등급)
- B공급업체: 합격률 96.5% (B등급) → 개선 계획
- C공급업체: 합격률 94.2% (C등급) → 대체 검토 중

불량 원인 분석 (파레토)
1. 형상 공차 초과: 35건 (60%)
   → 공급업체 B에서 발생
   → 공정능력 개선 요청 완료
2. 표면 흠: 15건 (25%)
   → 내부 공정에서 발생
   → 라인 A 조정 완료
3. 기능 부적합: 8건 (15%)
   → 부품 조립 오류
   → 작업 교육 실시

품질 손실
- 부적합 부품 원가: ₩3,400,000
- 폐기 부품 원가: ₩1,200,000
- 재검사 비용: ₩800,000
합계: ₩5,400,000 (생산비 대비 0.85%) ✓

품질통보 현황
- 신규 통보: 5건 (모두 공급업체 품질)
- 해결 통보: 3건
- 미해결: 2건 (조사 중)

목표 달성률
- 수입 검사 합격률 98%: 98.3% ✓
- 공정 검사 합격률 99%: 98.8% (△ 0.2%)
- 완성품 검사 합격률 99.5%: 100% ✓
- 품질 손실율 < 1%: 0.85% ✓
- 공급업체 평균 합격률 98%: 96.6% (△ 1.4% → 개선 중)
```
