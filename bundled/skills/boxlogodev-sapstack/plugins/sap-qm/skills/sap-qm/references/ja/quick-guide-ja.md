# SAP QM クイックガイド (日本語)

## 環境ヒアリング
1. SAP リリース (ECC EhP / S/4HANA 年度)
2. デプロイモデル (On-Premise / RISE / Cloud PE)
3. QM 有効化状態 (資材タイプ別検査設定)
4. 業種 (製造、製薬、食品、自動車部品)

## 日本ローカル

### ISO / IATF 認証
- ISO 9001: 品質マネジメントシステム (ほぼ全製造業)
- IATF 16949: 自動車部品 (OEM サプライヤ必須)
- 検査計画 (QP01) が認証審査の中核証憑

### 製薬 GMP
- 規制当局 (PMDA/厚労省) 基準
- バリデーション 3 段階: IQ → OQ → PQ
- 逸脱管理: 品質通知 (QM01) で追跡

### 食品 HACCP
- CCP (重要管理点) モニタリング
- 検査特性 (MIC) を CCP 項目にマッピング

### サプライヤ品質管理
- 受入検査強化 (Inspection Type 01)
- サプライヤ評価 (ME61): 品質スコア自動反映
- 不適合時: 品質通知 (Q3) → 是正処置要求
