# QM 거버넌스 Best Practice

## 적용 범위
- ECC: ✓ | S/4HANA: ✓

## 개요
품질 부문의 정책, 승인 체계, 법규 준수 항목. ISO/IATF 인증, GMP/HACCP 준수를 포함합니다.

## 체크리스트

### ISO/IATF 심사 대비
- [ ] **검사계획 승인 절차** — T-code: QP01
  - 정책: 검사계획 변경 → 기술팀+품질팀+경영진 3단계 승인
  - 절차:
    ```
    1. 기술팀: 검사특성 정의 (측정 방법, 기준)
    2. 품질팀: 샘플링 계획 검토 (AQL 적정성)
    3. 경영진: 최종 승인 (비용 vs 품질 트레이드오프)
    
    문서화: QP01 이력 추적 → 감사 대비
    ```
  - 감시: 분기 1회 검사계획 정적 감사(Static Audit)
    ```
    샘플: 무작위 선택 부품 5개
    검증: QP01 존재 여부, 승인자 서명, 기한 유효성
    ```

- [ ] **IATF 16949 특수 요구사항** — Automotive
  - 정책: 자동차 부품 공급업체는 IATF 16949 인증 필수
  - 관리:
    ```
    - 공급업체 리스트: IATF 인증 여부 확인
    - 인증서 만료: 재인증 추적 (만료 3개월 전 알림)
    - 감사: 2년마다 정기 감사 + 분기 현장 확인
    
    비인증 공급업체:
    - 인증 유예 기간: 6개월
    - 그 이후: 대체 공급업체 전환
    ```

- [ ] **ISO 9001 내부 감사** — 연 2회
  - 범위:
    ```
    1. 문서 관리: QP*, QA*, QM* 문서 버전 관리
    2. 검사 기록: QA33, QA32 데이터 완전성
    3. 샘플링 실행: AQL 기준 준수 확인
    4. SPC 관리도: 통제 상태 유지 확인
    5. 교정 기록: 측정 장비 보정 이력(Calibration)
    ```
  - 산출: 부적합 사항 → 시정 계획 수립

### 샘플링 절차 적정성 연간 검토
- [ ] **샘플링 계획 및 AQL 검증** — 연 1회(1월)
  - 정책: 국제 표준(ANSI Z1.4 또는 ISO 2859-1) 준수
  - 검토:
    ```
    AQL(Acceptable Quality Level) 재평가:
    - 중요 특성(Critical): AQL 0.25% ~ 0.65%
    - 주요 특성(Major): AQL 1.0% ~ 2.5%
    - 경미 특성(Minor): AQL 4.0% ~ 10%
    
    산업 특성 반영:
    - 자동차: 엄격함(0.65%)
    - 일반산업: 표준(1.0%)
    - 비경영 부품: 관대함(2.5%)
    
    예시:
    부품 "피스톤": 중요(Critical) → AQL 0.65%
    입고 로트: 1,000개
    샘플 크기: n=200 (ANSI Z1.4)
    수용 기준: 부적합 ≤ 3개 → 합격
    ```
  - 기록: AQL 검토 문서 저장 (감시 대비)

### 한국 GMP/HACCP 특수 요구사항
- [ ] **의료기기 GMP 준수** (해당 시 필수) — 식약청
  - 정책: 의료기기 제조 시설은 GMP 인증 필수
  - 관리 항목:
    ```
    1. 검사실 환경 관리
       - 온습도: 기준값 ±범위 내 유지 (기록)
       - 청결도: 월 1회 공기질 검사
       - 측정 장비: 월 1회 보정(Calibration)
    
    2. 제품 추적성
       - Batch/Lot 기반 추적: 입고~출고까지 기록
       - 부적합 발생 시: 동일 Batch 제품 모두 추적
       - 보관 기간: 제품 유효기간 + 1년
    
    3. 비상 조치
       - 부적합 발견: 즉시 차단(Quarantine)
       - 고객 통보: 48시간 이내
       - 수거 계획 수립: 식약청 신고
    ```
  - 감시: 연 2회 자체 감사 + 식약청 정기 감시

- [ ] **식품 HACCP 준수** (해당 시 필수) — 농림축산식품부
  - 관리 항목:
    ```
    1. CCP(Critical Control Point) 모니터링
       - 온도: 냉동 -18°C 이하 유지
       - 시간: 보관 기간 기준 충족
       - pH: 산도 기준 범위
    
    2. 검사 기록: 월 1회 제출 (축산물품질평가원)
    
    3. 부적합 대응: 즉시 격리 + 통보
    ```

### 검사계획 변경관리
- [ ] **QP01 변경관리 프로세스** — T-code: QP01
  - 변경 트리거:
    ```
    1. 불량률 변화
       - 합격률 100% 지속(6개월) → 샘플링 축소
       - 부적합 발생 → 샘플링 강화
    
    2. 신 제품/신 공급업체
       - 초기 검사: AQL 강화(0.25%)
       - 안정화(3개월 합격률 ≥ 98%) → 정규 검사로 전환
    
    3. 공정능력 변화
       - Cpk ≥ 1.33 → 검사 축소 검토
       - Cpk < 1.0 → 검사 강화 필수
    ```
  - 승인: QP01 변경 → 승인자 서명(전자서명 가능)

### 측정 시스템 적절성 검증
- [ ] **MSA(Measurement System Analysis)** — 연 1회
  - 검증 항목:
    ```
    1. 정확도(Accuracy)
       - 표준 부품으로 측정 → 표준값 대비 편차
       - 허용 기준: ±2%
    
    2. 재현성(Repeatability)
       - 동일 부품을 같은 측정자가 10회 측정
       - 범위 ÷ 평균 < 10% (GRR: Gauge R&R)
    
    3. 재현성(Reproducibility)
       - 동일 부품을 다른 측정자가 측정
       - 측정자 간 편차 확인 → 교육 필요 여부
    ```
  - 기록: MSA 결과 → 측정 장비 파일 보관

### SPC 관리도 운영 기준
- [ ] **통제한계(Control Limits) 설정** — 통계학적 기준
  - 설정 방법:
    ```
    X-bar chart (평균):
    - 상한 = X_mean + 3σ
    - 하한 = X_mean - 3σ
    - σ = 범위(R) ÷ d2
    
    예시:
    - 평균: 50.0mm
    - 표준편차: 0.2mm
    - 상한: 50.6mm
    - 하한: 49.4mm
    
    관리도 판독:
    - 점이 한계 벗어남 → 공정이상 신호
    - 8개 연속 점이 중앙선 한쪽 → 공정 편향 신호
    ```
  - 빈도: 월 1회 관리도 검토

### 공급업체 평가 및 관리
- [ ] **공급업체 품질 협력 계약** — 계약서
  - 계약 조건:
    ```
    1. 품질 기준
       - 합격률: 99% 이상
       - 반품률: 0.5% 이하
       - AQL: 1.0% (또는 협의)
    
    2. 불이행 시 페널티
       - 합격률 95~98%: 경고 + 개선 계획
       - 합격률 < 95%: 단가 2% 인하
       - 반복 불이행(3회): 계약 해지
    
    3. 인센티브
       - 합격률 99.5% 이상 6개월 유지: 단가 1% 할인
    ```
  - 계약서 기재: SAP 공급업체 마스터(MM01)에 기록

### 부적합 처리 및 시정 조치
- [ ] **8D(Eight Disciplines) Report 작성** — 중대 부적합 시
  - 절차:
    ```
    D1. 팀 구성: 관련 부서 대표(PP, QM, 공급업체)
    D2. 문제 정의: 부적합 내용, 영향 범위
    D3. 임시 대응: 즉시 조치(격리, 반품)
    D4. 근본원인: 왜? 왜? 5Why 분석
    D5. 시정 조치: 근본원인 제거 계획
    D6. 실행 및 검증: 조치 실행 → 효과 확인
    D7. 재발 방지: 프로세스 개선
    D8. 인정: 팀 성취 인정 (문화 조성)
    ```
  - 기록: 8D Report 저장 (QM02 연계)

### 품질 교육 및 역량 관리
- [ ] **품질 담당자 자격 및 교육** 
  - 정책:
    ```
    1. 최소 자격
       - 검사 담당: 고등학교 이상 + 1년 경력
       - 검사 감독: 고등학교 이상 + 3년 경력
       - 검사 매니저: 대학 이상 + 5년 경력
    
    2. 필수 교육
       - 신입: 기초 검사 교육 (40시간)
       - 연간: 직무 교육 (20시간)
       - 특수: ISO 9001, SPC, MSA 교육
    
    3. 자격증
       - 검사 기술자 자격증
       - 통계 관리사 자격증(권장)
       - 품질경영 기사(권장)
    ```
  - 관리: HR 정보 시스템에 교육 이력 기록

## 감사 체크리스트 (Annual Audit)

```
[ ] ISO 9001 기준 충족
    - 문서 관리: 버전 관리 100%
    - 검사 기록: 완전성 100%
    - 샘플링: 기준 준수 100%

[ ] IATF 16949 (자동차 부품)
    - 공급업체 인증: 유효 ≥ 95%
    - 부품 추적성: 100%
    - 반품 처리: 즉시성 확인

[ ] GMP/HACCP (의료·식품)
    - 환경 모니터링: 월간 기록 완전
    - 제품 추적: Batch 추적 가능
    - 부적합 대응: 48시간 이내

[ ] 검사계획 검증
    - QP01 유효성: 샘플 5건 확인
    - 승인 기록: 3단계 승인 100%
    - 변경 관리: 추적 완전

[ ] 측정 시스템
    - MSA 보정(Calibration): 유효 ≥ 95%
    - 측정 장비: 예방보전 기록 완전
    - 표준품: 검증 이력 보유

[ ] SPC 운영
    - 관리도: 월간 작성 12회
    - 통제 상태: 이상 신호 대응 기록

[ ] 공급업체 협력
    - 품질 계약: 서명 완전
    - 평가 기록: 월간 12회 완전
    - 부적합: 8D Report 작성률 ≥ 90%

[ ] 품질 교육
    - 신입 교육: 40시간 기록
    - 연간 교육: 20시간 기록
    - 자격증: 갱신 완료
```
