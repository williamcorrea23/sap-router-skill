## 中国本地化

### ISO / IATF 认证
- ISO 9001: 质量管理体系 (几乎所有制造业)
- IATF 16949: 汽车零部件 (主机厂供应商必备)
- 检验计划 (QP01) 是认证审核核心证据

### 制药 GMP
- 药监部门 (NMPA) 标准
- 验证三阶段: IQ → OQ → PQ
- 偏差管理: 经质量通知 (QM01) 追踪

### 食品 HACCP
- CCP (关键控制点) 监控
- 检验特性 (MIC) 映射到 CCP 项

### 供应商质量管理
- 强化来料检验 (Inspection Type 01)
- 供应商评估 (ME61): 自动反映质量评分
- 不合格 → 质量通知 (Q3) → 纠正措施
