# SuccessFactors 페이롤·연 마감 (Tier 2) Best Practice

## 월 마감 (Payroll)

### Pre-Payroll (D-3 ~ D-1)
- [ ] **Time Off** 마감 — 미승인 휴가 모두 처리
- [ ] **Time Tracking** 마감 — 미승인 시간 모두 처리
- [ ] **Compensation Changes** — 이번 달 적용 변경 확정

### Payroll Run (D-Day)
- [ ] **Test Run** → 결과 검토
- [ ] **Production Run** → 페이슬립 생성
- [ ] **ERP Replication** → FI/CO 전기
- [ ] **Bank File** → 은행 전송

### Post-Payroll (D+1 ~ D+3)
- [ ] 페이슬립 직원 배포
- [ ] 4대보험 EDI 신고
- [ ] 원천세 신고 준비

## 분기 마감

- [ ] **Performance Cycle** 마감 (분기 또는 반기)
- [ ] **Goal Plan** 갱신
- [ ] **Talent Review** 데이터 정합성

## 연 마감

### 연말정산 (한국)
- [ ] 직원 연말정산 자료 수집 (1월~2월)
- [ ] EC Payroll → 연말정산 처리
- [ ] 환급/추가 징수 정확성
- [ ] 1월 ~ 2월 신고

### Compensation Cycle
- [ ] 연봉 협상 사이클 (보통 1월~3월)
- [ ] 인상률 결정·적용
- [ ] Total Rewards Statement 배포

### Performance Cycle
- [ ] 연간 평가 마감
- [ ] Calibration (등급 분포 조정)
- [ ] 인사 결정 (승진·이동)

### 연간 보고
- [ ] HR 분석 보고서 (CHRO 대상)
- [ ] 다양성·인구 통계
- [ ] 인건비 분석 (CFO 협업)

## 한국 특화

- 연말정산 의무 (1~2월)
- 4대보험 연 정산 (4월)
- 임금 인상 가이드라인 (정부·고용노동부)
