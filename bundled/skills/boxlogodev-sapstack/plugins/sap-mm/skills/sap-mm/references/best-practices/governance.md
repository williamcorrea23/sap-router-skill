## 체크리스트

### 🔐 Release Strategy 통제

#### Release Strategy 정기 검토
- [ ] **T-code SPRO → MM → Release Processing**
  - 시기: 연 2회 (상반기/하반기)
  - 목표: 구매오더 승인 기준(금액, 체계) 최신화
  - 정책 검토:
    - 팀장 승인: 예) 5천만원 이하
    - 부장 승인: 예) 5천만원~1억원
    - CFO 승인: 예) 1억원 초과
  - 담당: 구매담당자 / CFO
  - 액션: 기준 변경 시 시스템 설정 즉시 반영

#### Release 권한 정기 점검
- [ ] **T-code SU01Z — Purchasing Authority Review**
  - 분기별 구매 권한 보유자 목록 재검증
  - 확인:
    - 권한 보유자가 실제 업무(구매 결정)에 관여하는가?
    - 이직/전보된 인원 권한 회수 여부
    - 신규 구매담당자 권한 신청 여부
  - 담당: 보안팀 / 인사팀
  - 빈도: 분기(3개월)
  - 문서: 권한 검증 결과

### 📦 재고 관리 통제

#### 재고 ABC 정책 정기 재검토
- [ ] **T-code MC.1 또는 수동 — Inventory ABC Policy**
  - 시기: 연 2회 (상반기/하반기)
  - 검토 항목:
    - A급(고가) 자재: 재고 한도, 발주 주기 설정
    - B급(중가) 자재: 표준 재고 관리
    - C급(저가) 자재: 최소 관리(일괄 발주)
  - 담당: 자재담당자 / 원가담당자
  - 액션: 정책 변경 시 각 자재별 매개변수 업데이트

#### 보안재고(Safety Stock) 설정
- [ ] **T-code MM02 또는 MRP 설정 — Safety Stock Level**
  - 월간 검토:
    - 수급 불안정 자재 → 보안재고 증가
    - 안정적 자재 → 보안재고 감소
    - 계절성 자재(예: 에어컨 실외기) → 시기별 조정
  - 담당: 자재담당자
  - 빈도: 월 1회 또는 수요 변화 시
  - 목표: 재고 부족율 < 5%, 과잉재고 < 10%

#### 단종/폐기 자재 관리
- [ ] **T-code MM02 또는 MB51 — Obsolete Material Process**
  - 월간 단종 자재 정기 검토:
    - 12개월 미거래 자재 목록화
    - 제품 생산 종료 자재 확인
    - 손상/불량 자재 선별
  - 담당: 자재담당자
  - 폐기 절차:
    1. 폐기 승인(부서장 또는 CFO)
    2. 조정 전표(MB1A) 생성 → 재고 제거 + 비용 처리
    3. 폐기 증거(사진, 인수증) 보관
  - 빈도: 월 1회 (월말)

### 🤝 공급업체 관리

#### 공급업체 성과 정기 평가
- [ ] **T-code ME61 — Vendor Rating**
  - 월간 공급업체 평가:
    - **납기율**: On-time Delivery % (목표 ≥95%)
    - **품질**: Defect Rate % (목표 ≤2%)
    - **가격**: Price Competitiveness (시장가 대비)
    - **신용**: Payment History (연체 없음)
  - 담당: 구매담당자
  - 빈도: 월 1회
  - 평가 등급:
    - **A급**: 종합점수 ≥80점 (선호)
    - **B급**: 60~80점 (개선 요청)
    - **C급**: <60점 (변경 검토)

#### 공급업체 기술 및 신용 조사
- [ ] **T-code ME01 — Vendor Master Review**
  - 반기별 공급업체 주요 정보 재검증:
    - 회사 재무 상태 (연체 기록, 신용 평점)
    - 생산 능력 변화 (확장, 축소)
    - 산업 뉴스 (인수/합병, 경영진 변경)
  - 담당: 구매담당자 / 신용관리
  - 액션: 신용도 하락 공급업체는 대체 계획 수립

#### 공급업체 다원화
- [ ] **T-code ME11 — Supplier Diversification**
  - 분기별 검토:
    - 주요 자재별 공급업체 수 (최소 2개 권장)
    - 단일 공급업체 의존율 (80% 이상 시 위험)
    - 신규 공급업체 발굴 및 자격 심사
  - 담당: 구매담당자
  - 액션: 단일 의존도 높은 자재는 대체 공급업체 개발

### 💰 가격 관리

#### 자재 가격 협상 및 계약 관리
- [ ] **T-code ME11 + 수동 — Price Contract Management**
  - 분기별 협상:
    - 구매액 상위 20% 자재 가격 재협상
    - 장기 계약 갱신 (1년 단위)
    - 환율/유가 변동에 따른 가격 조정 조항 검토
  - 담당: 구매담당자 / CFO
  - 문서: 가격 계약서 서명 및 보관
  - 빈도: 분기 1회

#### 표준원가 정기 검토
- [ ] **T-code CK11N — Standard Cost Maintenance**
  - 월간 표준원가 업데이트:
    - 실제 구매 가격 vs 표준원가 차이 분석
    - 차이가 ±10% 초과 시 표준원가 조정
  - 담당: 원가담당자
  - 빈도: 월 1회 (월초)
  - 문서: 표준원가 변경 내역 기록

### 📊 GR/IR 관리

#### GR/IR 미정산 모니터링
- [ ] **T-code MB5S — GR/IR Aging Analysis**
  - 월간 모니터링:
    - 30일, 60일, 90일+ aged GR/IR 현황
    - 미정산 원인 분류(청구 미도착, 품질 이슈 등)
    - 누적 GR/IR 잔액 vs 예비금(Buffer) 상태
  - 담당: 구매/원가담당자
  - 목표: 60일 이상 미정산 항목 월 5% 이하
  - 액션: Aged 항목 정산 강행 또는 배부 대기금 설정

#### 발주-입고-청구 주기 분석
- [ ] **T-code ME2N + MB51 — PO-GR-Invoice Cycle**
  - 분기별 분석:
    - 평균 발주→입고 소요일수
    - 평균 입고→청구 소요일수
    - 기간별 추이 (개선 확인)
  - 담당: 구매담당자
  - 목표: 발주→입고 5일 이내, 입고→청구 10일 이내
  - 액션: 기간 단축을 위한 프로세스 개선

### 🔍 정기 감시

#### 월간 MM 거버넌스 체크
- [ ] **T-code MB52, MB5S, ME28 — Monthly MM Review**
  - 점검 항목:
    - 재고 잔액 대비 회전율(Inventory Turnover)
    - 미처리 구매오더 현황 (기한도과 건수)
    - GR/IR 미정산 잔액
    - 폐기 자재 처리 현황
  - 담당: 자재담당자
  - 빈도: 월 1회
  - 문서: 월간 거버넌스 점검표

#### 분기 감사(Quarterly Audit)
- [ ] **분기별 MM 내부통제 평가**
  - 항목:
    - 공급업체 성과 평가 결과 (개선 건수)
    - 구매 Release 권한 준수율
    - 재고 불일치 현황 (조정 건수)
    - GR/IR 정산 진행도
    - 폐기 자재 승인 절차 준수
  - 담당: 내부감사 / 구매담당자
  - 문서: 분기 감사 보고서

#### 연간 K-SOX 평가
- [ ] **연간 MM 내부통제 평가**
  - 평가 항목:
    - 구매 권한 분리 운영 현황
    - Release Strategy 설정과 실제 운영 일치도
    - 공급업체 선정 프로세스 준수
    - 재고 조정 승인 프로세스
    - GR/IR 정산 정책 준수
  - 담당: CFO / 감사위원회
  - 결과: 내부통제 평가 보고서 (최고 경영진 서명)

## 위험 시나리오 & 대응

| 시나리오 | 원인 | 예방 액션 | 심각도 |
|---------|------|---------|--------|
| 구매 Release 권한 오용 | 승인 기준 미준수 | 분기 권한 검증 | 🔴 Critical |
| 공급업체 부도로 공급 중단 | 신용도 미평가 | 반기 신용 조사 + 다원화 | 🔴 Critical |
| 재고 부족으로 생산 중단 | 보안재고 미설정 | 월간 보안재고 검토 | 🔴 Critical |
| GR/IR 미정산 누적 과다 | 정산 프로세스 미흡 | 월간 MB5S 모니터링 | 🟠 High |
| 불량 자재 입고 미적발 | 품질 검사 미흡 | 분기 공급업체 품질평가 | 🟠 High |
| 폐기 자재 미승인 처리 | 절차 우회 | 월간 거버넌스 체크 | 🟡 Medium |

## K-SOX 준수 체크사항

- **구매 내부통제**: Release Strategy 정의, 권한 분리, 정기 검증
- **공급업체 관리**: 성과 평가, 신용도 모니터링, 계약 관리
- **재고 통제**: ABC 정책, 보안재고 설정, 조정 승인
- **기록 보관**: 구매 계약서, 공급업체 평가, 재고 조정 전표 최소 7년

- **구매 거버넌스**: 발주 기준, Release Strategy 정책 문서화

- **재고 정책**: ABC 분류, 보안재고 기준, 폐기 절차
