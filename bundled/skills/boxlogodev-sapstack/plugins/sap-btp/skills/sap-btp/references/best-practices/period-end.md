# BTP 비용·라이선스 마감 (Tier 2) Best Practice

## 월말

### 비용 정산
- [ ] **BTP Usage Report** — 서비스별 사용량
- [ ] **Cost Allocation** — 부서·프로젝트별 분배
- [ ] **Budget vs Actual** — 차이 > 10% 분석

### 라이선스 관리
- [ ] **Entitlement Usage** — quota 대비 사용
- [ ] **CPEA 잔액** — Cloud Platform Enterprise Agreement
- [ ] **Service expiration** — 만료 임박 서비스 갱신

## 분기

### 라이선스 재검토
- 사용량 추세 → 다음 분기 entitlement 조정
- 불필요 서비스 → 분기 단위 정리
- 신규 서비스 → 비용·가치 평가 후 결정

### Capacity Planning
- 사용량 90%+ 메트릭 → 증설 결정
- 사용량 30% 이하 → downsizing 검토

## 연말

### 연간 계약 갱신
- BTP Enterprise 계약 갱신 협상 (영업과 협업)
- CPEA Renewal 또는 신규 계약 전환
- 전사 라이선스 통합 (계열사 협의)

### Year-End 보고
- 연간 BTP 비용 보고서 (CFO 대상)
- 비용 절감 사례 정리
- 다음 연도 예산 제안

## 한국 특화

- 연결산 시 BTP 비용 → FI 분개 정확성 검증
- 부가세 처리 (해외 서비스 → 부가세 별도)
- 외환 환산 (USD/EUR 비용 → KRW)
