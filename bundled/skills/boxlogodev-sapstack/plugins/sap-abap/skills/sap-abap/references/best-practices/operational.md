## 일간 (Daily)

### 개발 환경 정상성
- [ ] **ATC (ABAP Test Cockpit)** — 변경 객체 사전 점검
  - 신규 코드 푸시 전 ATC 클린 보장
  - 위반 0건 정책 (예외는 별도 승인)
- [ ] **ST22** — 본인 작성 코드의 덤프 우선 분석
- [ ] **SLG1** — Application Log 신규 ERROR 점검

### 코드 리뷰
- [ ] PR/트포 1건 이상 리뷰 (분담·로테이션)
- [ ] 리뷰 체크: 명명 규칙, 예외 처리, SQL 효율, 한국어 메시지

### 디버깅 안전 수칙

- [ ] SE16N edit mode 운영 환경에서 사용 금지
