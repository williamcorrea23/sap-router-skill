# Contributors

sapstack은 SAP 운영자·컨설턴트·개발자 커뮤니티의 기여로 성장합니다. 이 파일에는 의미 있는 기여를 한 분들을 기록합니다.

## 메인테이너

- **[@BoxLogoDev](https://github.com/BoxLogoDev)** — 프로젝트 창시자·메인테이너. SAP 운영 자문 파이프라인 설계 및 sapstack 마켓플레이스 구축.

## 기여자

> 아직 외부 기여자가 없습니다. 첫 PR을 환영합니다 — [CONTRIBUTING.md](CONTRIBUTING.md) 참조.

## 기여 분야

기여하실 수 있는 분야:

| 영역 | 예시 |
|---|---|
| **신규 모듈** | SAP GTS/IBP/SAC/Ariba/Concur/Datasphere 등 |
| **기존 모듈 깊이** | T-code/Note/IMG/Best Practice 보강 |
| **다국어** | en/zh/ja/de/vi SKILL 본문 검수 |
| **산업 가이드** | 화학/제약/자동차/의료/공공 등 |
| **국가 CVI** | `country/` 신규 국가 (템플릿: `country/templates/cvi-template.md`) |
| **AI 도구 통합** | MCP / VS Code Extension / Cursor / Continue 호환 레이어 |
| **테스트·검증** | scripts/check-*.sh quality gate 강화 |

## 기여 절차

1. [CONTRIBUTING.md](CONTRIBUTING.md) 확인
2. Issue 등록 (큰 변경의 경우 선행 논의)
3. Fork → 브랜치 생성 → 변경 → 로컬 quality gate 통과 확인
4. PR 제출 (Summary + Test Plan 포함)
5. CI 통과 + 리뷰 후 머지

## 라이선스

기여하는 모든 코드와 문서는 [MIT License](LICENSE)로 배포되는 데 동의하는 것으로 간주됩니다.

## 감사 인사

이 프로젝트의 가치는 사용해주시는 분들로부터 옵니다. 버그 리포트, 사용 사례 공유, 문서 제안 모두 큰 도움이 됩니다.

기여 등록은 GitHub username 또는 이메일 + 변경 요약 형태로 받으며, 의미 있는 기여를 한 분은 PR 머지 시 또는 별도 PR로 위 "기여자" 섹션에 추가됩니다.
