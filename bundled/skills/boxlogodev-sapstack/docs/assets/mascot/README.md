# 표준씨 (Standard) — 마스코트 자산

이 디렉토리는 sapstack 마스코트 **표준씨(Standard)**의 이미지 자산을 담습니다.
브랜드 가이드는 [MASCOT.md](../../../MASCOT.md)를 참고하세요.

## 필요한 파일 (직접 투입)

아래 파일명 그대로 이 디렉토리에 넣어주세요. README가 이 경로를 참조합니다.

| 파일명 | 말풍선 | 사용처 |
|--------|--------|--------|
| `standard-ko.png` | 「SAP에서는 스탠다드라서 안됩니다.」 | `README.md` (한국어) |
| `standard-en.png` | 「In SAP, it's standard, so it can't be changed.」 | `README.en.md` 및 ja/zh/de/vi 폴백 |

> 다른 언어(ja/zh/de/vi)용 현지화 말풍선 버전은 커뮤니티 기여 환영입니다.
> 추가 시 `standard-ja.png` 처럼 ISO 언어 코드 suffix를 붙이고 해당 README를 교체하세요.

## 권장 사양

- 형식: PNG (투명 배경 권장) 또는 SVG
- 비율: 정사각형(1:1) 권장 — 도안 기준 약 1240×1240
- README 표시 폭: 280px (코드에서 `width="280"`로 제어)
- 스타일: 흑백 라인아트 (도안 원본 유지)
