# 모듈 횡단 기간마감 오케스트레이션

> Tier 2 — Period-End BP | 적용: FI/CO/MM/SD/PP/PM/QM | ECC: ✓ | S/4: ✓

## 1. 기간마감 전체 흐름

SAP 기간마감은 **모듈 간 의존성**이 있어 순서가 중요합니다.
잘못된 순서로 실행하면 데이터 불일치가 발생합니다.

```
PP/QM → MM → SD → PM → CO → FI → 연결결산
(생산) (자재) (판매) (보전) (관리회계) (재무회계) (그룹)
```

## 3. 분기/연말 추가 작업

### 분기
- [ ] 세금 신고 (부가세 확정 신고)
- [ ] 법인세 중간예납 (한국: 3/6/9월)
- [ ] K-SOX 분기 통제 평가

### 연말 (추가)
- [ ] 연말 감가상각 (연 1회 실행인 경우)
- [ ] 자산 재평가 (선택)
- [ ] 재고 실사 (MI01/MI04/MI07)
- [ ] 충당금 설정 (대손충당금, 퇴직급여 등)
- [ ] 이월 (F.16 — 잔액 이월)
- [ ] 신 회계연도 개설 (OB52)

## 4. 기간마감 사전검증 체크리스트 (Pre-Flight)

마감 시작 전 아래 항목을 반드시 확인:

- [ ] **기간 열림 확인** — OB52에서 해당 기간 Open 상태
- [ ] **미처리 IDoc 확인** — BD87에서 Error/Waiting 상태 IDoc
- [ ] **미처리 배치 잡 확인** — SM37에서 실패한 주기적 잡
- [ ] **환율 테이블 업데이트** — OB08에서 당월 환율 입력
- [ ] **마스터데이터 변경 동결** — 마감 중 마스터 변경 자제 공지
