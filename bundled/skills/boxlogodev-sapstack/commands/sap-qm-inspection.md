# SAP QM 품질검사 분석

입력: `$ARGUMENTS`

## 목표
품질검사 관련 이슈를 체계적으로 분석합니다. 검사로트 상태 확인 → 검사계획 검증 → 결과 기록 안내 → 사용결정 → 후속조치 순서로 진행합니다.

## 안전 규칙
- 사용결정(Usage Decision)은 사용자 확인 후에만 안내
- 차단 재고 해제는 품질 담당자 승인 필요

## 실행 절차

### Step 1 — 환경 확인
- 검사 유형 (01 수입검사 / 03 공정검사 / 04 출하검사 / 기타)
- 자재번호 또는 검사로트번호
- 플랜트 + SAP 릴리스
- 적용 품질 표준 (ISO 9001 / IATF 16949 / GMP / HACCP)

### Step 2 — 검사로트 상태 확인
1. **QA03** — 검사로트 조회
   - 로트 상태: REL (릴리즈) / CALC (결과 기록) / UD (사용결정 완료)
   - 검사계획 할당 여부
   - 샘플 크기 확인
2. **QA33** — 검사로트 목록 (미결 로트 일괄 확인)

### Step 3 — 검사계획 검증
1. **QP03** — 검사계획 확인
   - MIC (검사특성) 적정성
   - 샘플링 절차 (AQL 기준)
   - 유효기간 확인
2. 검사계획 미할당 시: QM 구성 점검 (자재유형별 검사 설정)

### Step 4 — 결과 기록 안내
1. **QE01/QE51N** — 결과 기록
   - 정량적 (수치) vs 정성적 (합/부) 구분
   - 규격 상한/하한 vs 실측값 비교
   - SPC 관리도 참고 (해당 시)

### Step 5 — 사용결정 (Usage Decision)
1. **QA11** — 사용결정 입력
   - A (합격) → 재고 자유 사용 전환
   - R (불합격) → 차단 재고 유지 → 품질통보 (QM01) 생성
   - 조건부 합격 → 부분 입고 처리
2. 후속 자동 조치:
   - 재고 전기 (합격 시)
   - 공급업체 평가 점수 반영 (수입검사 시)
   - 품질통보 자동 생성 (불합격 시)

### Step 6 — 품질통보 연계 (불합격 시)
1. **QM01** — 품질통보 생성
   - Q1 (고객 클레임) / Q2 (내부) / Q3 (공급업체)
   - 결함 코드 + 원인 코드 입력
   - 시정조치 작업 할당
2. **8D 프로세스** 매핑 (해당 시)

## 업종별 특화

### 자동차 (IATF 16949)
- PPAP (생산부품승인) 검증
- 공정 능력 지수 (Cp, Cpk) 확인
- 고객 특별 요구사항 반영

### 제약 (GMP)
- 밸리데이션 상태 확인
- 일탈(Deviation) 처리 절차
- 안정성 시험 이력

### 식품 (HACCP)
- CCP 모니터링 결과
- 이력추적 (Batch traceability)
