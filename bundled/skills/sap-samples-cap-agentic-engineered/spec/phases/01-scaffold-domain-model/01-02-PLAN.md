```cds
using { risk } from '../db/schema';

service RiskService {
  @readonly
  entity GLTransactions as projection on risk.GLTransactions;

  // Unbound action: triggers AI Core inference on all transactions
  // Handler implementation in Phase 2 (risk-service.js)
  action analyzeRisks() returns String;
}
```

Key decisions:
- `@readonly` on entity because transactions come from BDC -- app does not create/update/delete GL data
