# Roadmap: Financial Risk Analyzer (MCP-First)

## Overview

This roadmap rebuilds the Financial Risk Analyzer in this directory using **MCP-first development** — every SAP code artifact is generated by querying the relevant SAP MCP server (CAP, Fiori, UI5) before writing code. Phase 1 scaffolds the domain model, Phase 2 builds the backend service, Phase 3 builds the Fiori frontend, Phase 4 integrates end-to-end. Domain logic (feature columns, risk labels, AI Core client, mock predictor) is reused from `../prototype/` since it has no SAP framework dependency.

**MCP Server Mapping:**
- **CAP MCP** (`@cap-js/mcp-server`): CDS entities, services, handlers, actions
- **Fiori MCP** (`@sap-ux/fiori-mcp-server`): UI annotations, manifest.json, floorplan config
- **UI5 MCP** (`@ui5/mcp-server`): Controllers, XML views, bootstrap, control APIs
