## Symptom

You are doing a system conversion to SAP S/4HANA. The following SAP S/4HANA Transition Worklist item is applicable in this case.

## Solution

## Description

Software Maintenance was part of the IS-SW industry solution.

It supports the process of distributing new levels of software code to licensed customers, specifically: maintenance level management (MLM), to distribute maintenance code/corrections/fixes; and Version Change Protection (VCP), to automatically distribute a new software product or version change to licensed customers (providing a new license).

IS-SW and Software Maintenance Processing functionality are not available in SAP S/4HANA.

There is no specific, targeted successor solution in SAP S/4HANA for these capabilities.
