You are doing a system conversion to SAP S/4HANA. The following SAP S/4HANA Transition Worklist item is applicable in this case.

## Solution

## Description

With SAP S/4HANA, transaction PEG01( Pegging: Collective Processing ) and its programs are not available. The old transaction PEG01 is replaced by transaction PEG01N ( A&D GPD Pegging: Collective Processing ).

## Business Process related information

Transaction PEG01 ( Pegging: Collective Processing ) is not available with SAP S/4HANA. Customers need to use transaction PEG01N as the successor transaction.

| Transaction not available in SAP S/4HANA   | Transaction PEG01 (replaced by PEG01N)   |
|--------------------------------------------|------------------------------------------|

## Required and Recommended Action(s)

Start using transaction PEG01N in SAP S/4HANA and educate the end user.
