## Symptom

You are doing a system conversion to SAP S/4HANA, on-premise edition. The following SAP S/4HANA Transition Worklist item is applicable in this case.

## Solution

## Description

WebDynpro Applications of SAP ERP Materials Management (MM) are not available with SAP S/4HANA, on-premise edition 1511any more.

The corresponding functionality is replaced with existing SAP GUI transactions, Fiori applications or Smart Business within SAP S/4HANA, on-premise edition 1511.

The coverage of Fiori applications will grow over time.
