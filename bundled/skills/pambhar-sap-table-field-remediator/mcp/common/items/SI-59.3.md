## Symptom

You are doing a system conversion from SAP ERP to SAP S/4HANA or an upgrade from a lower to a higher SAP S/4HANA release and are using the functionality described in this Note.

The following SAP S/4HANA Simplification Item SI12 Public Sector_FI-LOC-PSHU is applicable in this case.

## Solution

## Description

Packages GLO_REP_EAPS_HU, GLO_REP_EAPS_SFWS_01 and it contents become obsolete because of no customer usage. Objects stored in the package and corresponding subpackages are removed including their references from other objects.

## Business Process Related Information

Public Sector Localization for Hungary stored in packages

GLO_REP_EAPS_HU, GLO_REP_EAPS_SFWS_01 and it's subpackages are not being used and have been deprecated in S/4HANA OP2023 and became obsolete in all the supported lower versions.

Regarding S/4HANA OP2023 release, for transition period of 2 years, the packages and their subpackages have been moved to SW component 'S4DEPREC' if the restoration of the functionality is needed.

## IMPORTANT:

To complete the installation procedure for S4TWL - Public Sector Localization for Hungary Deprecation you have to implement the automatic correction instructions from this Note.

Remember, that you cannot use your own coding/functionality anymore which is based on the removed coding/functionality, explained in this Note.
