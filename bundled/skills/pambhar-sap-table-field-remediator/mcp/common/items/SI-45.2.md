## Symptom

You execute a system conversion from ERP to S/4HANA or an upgrade from a lower S/4HANA release to a higher one (2021 or higher).

From S/4HANA 2021 onwards the Financial Services Business Partner table Additional Partner Numbers (table name BPID001 ) is obsolete. The data of this table have to be migrated to table BUT0ID. This note describes activities that have to be executed in order to perform the migration in the source system before the conversion.

## Reason and Prerequisites

Financial Services Business Partner table Additional Partner Numbers (BPID001) must not be used any more from S/4HANA 2021 onwards. The table is very similar to the central business partner table Identification Numbers (BUT0ID), therefore the table the migration into the Identification Numbers table BUT0ID which was optional up to now will be enforced now.

## Solution

## Procedure for determining the relevance

The migration needs to be executed as soon as there is at least one entry in table BPID001 in any of the clients of your system apart from 000 and 066.

There will also be a consistency check class that is triggered by the SUM process. SUM process will be stopped if there are data in table BPID001 in one of the clients.

## Required and recommended actions

In case there are entries in table BPID001 these need to be migrated to table BUT0ID and deleted afterwards.

In SAP note 1556353 you can find a detailed description what needs to be done to execute this migration.

The main steps are the following:

Creation of a mapping for customizing (BPID001 -> BUT0ID)

BADI implementation in case of customer-specific enhancement in table BPID001

Activation of business function CA_FSBP_MIGRATION_ID (creates new customizing entries in table BUT0ID)

Migration of BPID001 data to BUT0ID table (including migration of change documents)

The migration is implemented via the ASU toolbox. This means that you will be guided through the whole process supported by the system after starting ASU toolbox. For details how to start this guided process please check note 1556353 and the referenced note 1000009.
