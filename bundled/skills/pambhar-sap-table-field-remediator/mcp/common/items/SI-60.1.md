## Symptom

You are doing a system conversion to SAP S/4HANA. The following SAP S/4HANA Transition Worklist item is applicable in this case.

## Solution

## Description

In SAP S/4HANA Retail for merchandise management, retail power lists are available, but defined as "not strategic any more".  See SAP note 2269324 - Compatibility Scope Matrix for SAP S/4HANA, on-premise edition , refer to ID 104 in the xls.

## Business Process related information

Move to content basded on strategic technologies, for example CDS (Core Data Services), or OData services (e.g. API_PURGPRICINGCONDITIONRECORD_SRV, API_SLSPRICINGCONDITIONRECORD_SRV).

## Required and Recommended Action

Move to strategic technology before expiration of use rights as explained in note 2269324.

## How to Determine Relevancy

This Simplification Item is relevant if Retail Power Lists are used. This is the case if any of transactions WR_MEK2, WR_MEK3, WR_VK12, WR_VK13 or programs R_WR_WRAP_MEK2, R_WR_WRAP_MEK3, R_WR_WRAP_VK12, R_WR_WRAP_VK13 are used.
