## Symptom

You are doing a system conversion to SAP S/4HANA. The following SAP S/4HANA Transition Worklist item is applicable in this case.

## Solution

## Description

The Construction Progress Report and Valuation function is not available in SAP S/4HANA 1511.

## Business Process related information

The Construction Progress Report and Valuations functions and related transactions are not available in SAP S/4HANA 1511.
