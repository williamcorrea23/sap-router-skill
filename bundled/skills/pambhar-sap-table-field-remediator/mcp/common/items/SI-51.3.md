## Symptom

You are doing a system conversion to SAP S/4HANA. The following SAP S/4HANA Transition Worklist item is applicable in this case.

## Solution

## Description

Software License Management was part of the IS-HT-SW industry solution.

It supports business processes for licensing software, such as: Software License creation, Version Change processing, Product Bundle creation, Software License pricing.

IS-SW and thus Software License Management capabilities are not available in SAP S/4HANA.

There is currently no functional equivalent available in SAP S/4HANA.
