## Symptom

You are doing a system conversion from SAP ERP to SAP S/4HANA or an upgrade from a lower to a higher SAP S/4HANA release and are using the function described in this note. The following SAP S/4HANA Simplification Item is applicable in this case.

## Solution

## Description

The group asset function in Asset Accounting is part of the SAP S/4HANA compatibility scope, which comes with limited usage rights. For more details on the compatibility scope and its expiry date and links to further information please refer to SAP Note 2269324. In the compatibility matrix attached to SAP note 2269324, group assets can be found under ID 428.

SAP does not plan to provide a group asset function for Asset Accounting in the future.

Note that postings to aggregated assets are not possible with SAP S/4HANA.

## Business Process-Related Information

For a description of the functions in the compatibility scope, see the SAP S/4HANA Feature Scope Description in section FI-AA: Group Assets.

## Required and Recommended Action(s)

Until the expiry date of the compatibility pack license, you can continue to use the group asset function. However, we recommend that you define your future strategy early in before this date because the group asset function must no longer be used after this date.

Note that you need to adjust roles, user menus and authorizations to remove group asset transactions, since transactions AS21, AS22, AS23, AS24, AS25 and AS26 are no longer supported after the compatibility scope expiry date.

Instead of the group asset function, you can use one of the succession solutions described in SAP Note 3532580.
