## Symptom

You are doing a system conversion to SAP S/4HANA, on-premise edition. The following SAP S/4HANA Transition Worklist item is applicable in this case.

## Solution

With SAP S/4HANA, on-premise edition 1709 FPS1, the access to DCS based Market Data, Commodity Curves and Financial Transactions has been simplified. Details about the simplification can be found in the attached document 'Function in Details - Simplified DCS Access'.
