## Symptom

You are doing a system conversion from SAP ERP to SAP S/4HANA or an upgrade from a lower to a higher SAP S/4HANA release and are using the functionality described in this note. The following SAP S/4HANA Simplification Item is applicable in this case.

## Solution

## Description

Profit Center Planning in classical Profit Center Accounting (PCA) is part of the SAP S/4HANA compatibility scope, which comes with limited usage rights. For more details on the compatibility scope and it's expiry date and links to further information please refer to SAP note 2269324. In the compatibility matrix attached to SAP note 2269324, you can find the topic under the ID 427.

This means that you should migrate from Classical Profit Center Planning to designated alternative functionalities before expiry of the compatibility pack license. This note shall provide information on what to do to move to the alternative functionality.

## Business Process related information

Going forward with SAP S/4HANA, SAP follows a separate product strategy for actual postings and for planning in EC-PCA.

Profit Center Planning based on table GLPCP is in compatibility scope and must not be used beyond they expiry date of the compatibility package usage rights.

Actual postings for classical Profit Center Accounting, using GLPCA and GLPCT, will be adopted to S/4HANA product standards and, thus, can still be used after expiry of usage rights. Please refer to SAP note 3572033 for more details.

Planning in classic PCA in SAP S/4HANA is active if the active indicator is set in transaction 0KE5 (field PCRCH for the corresponding controlling area in the table TKA00 is set to '2') and the online transfer of data records in PCA tables has been activated using transaction 1KEF. Please note that you cannot deactivate Profit Center Planning independently of actual values in PCA. This means, you need to make sure legal compliance by organizational measures after expiration of usage rights.

## Required and Recommended Action(s)

As alternatives to planning in classical PCA you have the following options:

SAP Analytics Cloud for planning with SAP S/4HANA (SAC)

SAP S/4HANA G/L planning transactions

Refer to SAP note 2425255 - Profit Center Accounting in the universal journal in SAP S/4HANA, on-premise edition , section 8, for more information.
