## Symptom

You are doing a system conversion to SAP S/4HANA 2023. The following SAP S/4HANA Transition Worklist item is applicable in this case.

## Reason and Prerequisites

Renovation

## Solution

## Description

App and transaction in the SAP S/4HANA solution for EoD memo record reconciliation are deprecated as of SAP S/4HANA 2023.

Currently the functionality is deprecated (alternative exists with roadmap).

## Business Process related information

The following objects have been deprecated:

| Fiori App ID/Transaction Code   | Description                         |
|---------------------------------|-------------------------------------|
| F4867                           | Reconcile Cash Flows - Memo Records |
| FCLM_CR_MMRD                    | Reconcile End-of-Day Memo Records   |

## Required and Recommended Action(s)

Inform your business users.

## How to Determine Relevancy

This Transition Worklist item is relevant for you if you are using one or more of the deprecated objects that are listed above.

## Other Terms

SAP S/4HANA, Upgrade, F4867, FCLM_CR_MMRD
