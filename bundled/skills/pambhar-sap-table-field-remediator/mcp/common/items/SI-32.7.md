## Symptom

You are doing a system conversion to SAP S/4HANA. The following SAP S/4HANA Transition Worklist item is applicable in this case.

## Solution

## Description

Hierarchy Graphic in Project System is part of the SAP S/4HANA compatibility scope, which comes with limited usage rights. For more details on the compatibility scope and it's expiry date and links to further information please refer to SAP note 2269324. In the compatibility matrix attached to SAP note 2269324, hierarchy graphic in Project System can be found under the ID 465.

Hierarchy graphic is used to visualize the WBS element hierarchy. Is is categorized as not part of the target architecture (Functionality available in SAP S/4HANA 1511 delivery but not considered as future technology). In order to display the project hierarchy (in a non-graphical way) you can use the Project Builder (CJ20N) or any other transaction listing project elements hierarchically.

## Business Process related information

Hierarchy Graphic can be used as part of the SAP S/4HANA compatibility scope until end of expiry date. There is no alternative planned for a graphical visualization.

In order to display the project hierarchy (in a non-graphical way) you can use the Project Builder (CJ20N) or any other transaction listing project elements hierarchically.
