## Symptom

You are doing a system conversion to SAP S/4HANA, on-premise edition. The following SAP S/4HANA Transition Worklist item is applicable in this case.

## Solution

## Description

With the installation of the SAP S/4HANA on-premise edition 1511, certain countryspecific transaction codes and programs in the application area of Financials have been replaced with newer transactions or programs (compared to EhP7 for SAP ERP 6.0). For more details about the affected transactions and programs and their replacements, refer to SAP Notes 2029012 and 2227963.
