## Symptom

You are doing a system conversion to SAP Commercial Project Management for SAP S/4HANA. The following SAP S/4HANA Transition Worklist item is applicable in this case.

## Solution

## Description

There is no interface of SAP Commercial Project Management with SAP Cloud for Customer in SAP Commercial Project Management for SAP S/4HANA .

## Required and Recommended Actions

Inform key users and end users.
