## Symptom

You are doing a system conversion to SAP S/4HANA, on-premise edition. The following SAP S/4HANA Transition Worklist item is applicable in this case.

## Reason and Prerequisites

Reason: Conversion from SAP ERP to SAP S/4HANA or upgrade to a higher SAP S/4HANA release.

Prerequisite: Business Function Set DIMP for Discrete Industries and Mill Products or Business Function Set DIMP_PS for Discrete Industries / Mill Products and Public Services are active in SAP ERP.

## Solution

## Description

The functionality of JIT Outbound as part of the automotive industry solution is not the target solution in SAP S/4HANA and therefore deprecated. Even if it is not yet defined when JIT Outbound will become obsolete and be replaced, which means it's still in maintenance until communicated otherwise, it will receive no new features and innovations. SAP recommends implementing JIT Supply to Production as the strategic next-generation solution.

## Business Process related information

JIT Supply to Production supports the following key features of JIT Outbound:

Internal replenishment to a production supply area from a source storage location using summarized JIT calls

Replenishment to a production supply area from an external supplier using summarized JIT calls

Manage control cycles for JIT supply to production

Create JIT calls as replenishment requests manually

Plan replenishment to production supply areas based on demands from production

Send JIT calls through EDI messages to external suppliers and other communication partners

Action framework, defining

allowed actions per internal processing status

change of internal processing status by action execution

interlinked actions, combining multiple actions to be processed together subsequently

Monitor JIT calls JIT Supply to Production differs from JIT Outbound in the following key features:

Internal replenishment to a production supply area from a warehouse managed source storage location is supported with Extended Warehouse Management as the strategic warehouse management solution in SAP S/4HANA. An integration to the classic Warehouse Management, as supported by JIT Outbound, is not available and not planned in future

Goods receipt processing in external replenishment to a production supply area using summarized JIT calls is supported using Inbound Cargo Registration (ICR) as part of SAP S/4HANA for Manufacturing Logistics.

Processing forwarded sequenced JIT calls are not yet supported with JIT Supply to Production

Replenishment of supermarkets close to production lines through demand-driven replenishment planning is supported with JIT Supply to Production, but not with JIT Outbound

## How to Determine Relevancy

The Simplification Item is relevant, if:

The Business Function DIMP_SDUD is active (check with transaction SFW2 or SFW_BROWSER), and

Outbound JIT calls exist (check database table JITOHD for entries)

## Required and recommended action(s)

When the check item is relevant, there are two alternative options to proceed with the conversion to SAP S/4HANA:

Evaluate and implement the next-generation Just-in-Time (JIT) Supply to Production solution in SAP S/4HANA instead of continuing with JIT Outbound.

Apply for an exemption for this item in the SAP S/4HANA Simplification Item Check and continue with the current JIT Outbound implementation and consider a migration to the JIT Supply to Production solution at a later point in time.
