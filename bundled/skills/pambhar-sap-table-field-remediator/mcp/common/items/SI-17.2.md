## Symptom

You are doing a system conversion to SAP S/4HANA. The following SAP S/4HANA transition worklist item is applicable in this case.

## Solution

## Description

Master agreements are not available in SAP S4/HANA.

Read the following information in case the pre-upgrade check in your system issues a message for the check ID  SI11_MASTER_AGREEMENTS "Master Agreements": This check issues a warning if it finds entries in the database which indicate that master agreements were created in some client of the system. The warning message is for your information, no futher mandatory action is required.

Read the following information in case the custom code analysis has detected customer coding related to this transition worklist item: SAP objects which are used by the detected customer code shall not be used any more. Adapt the customer code accordingly.
