## Symptom

You are upgrading from a lower to a higher SAP S/4HANA release. If you have defined an implementation of classic BAdI DMEE_BADI_01 in the past and you are using transaction DMEEX, then this SAP S/4HANA Simplification Item is applicable.

Please note: If SAP S/4HANA  Simplification Item S4TWL - DMEE (3370503) is relevant for you, this SAP S/4HANA Simplification Item might also become relevant for you after you migrate from DMEE to DMEEX.

## Solution

## Business Value

SAP has decided to replace legacy classic BAdI DMEE_BADI_01 (when using DMEEX) with Enhancement Spot ES_DMEEX and BAdI DMEE_OUTPUT_POSTPROCESSING as calling the new BAdI is more performance optimized.

## Business Process related information

Classic BAdI DMEE_BADI_01 is deprecated when using DMEEX from release SAP S/4HANA 2023 FPS2 and obsolete from release SAP S/4HANA 2025.

There will no longer be any official support for the classic BAdI DMEE_BADI_01 in DMEEX and it will be removed from the system without further notice.

## Required and Recommended Action(s)

Migrate all your classic BAdI implementations of DMEE_BADI_01 to new BAdIs with Enhancement Spot.

## How to Determine Relevancy

You have defined an implementation of classic BAdI DMEE_BADI_01 and the format mapping tree in the filter of the BAdI is DMEEX.

Version 1: Initial release.
