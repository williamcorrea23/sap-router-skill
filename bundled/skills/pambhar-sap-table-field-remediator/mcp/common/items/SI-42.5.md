## Symptom

This note is only relevant for customers who use China GTI (Golden Tax Interface) solution and convert their system to SAP S/4HANA On-Premise Edition 1610 Feature Pack 1, SAP S/4HANA On-Premise Edition 1709 or SAP S/4HANA On-Premise Edition 1809.

Starting from release SAP S/4HANA On-Premise Edition 1610 Feature Pack 1 the GTI solution is using a new data base design and old generated GTI invoices will not be avalible in the GTI applications.

## Reason and Prerequisites

This note is relevant for you when your start release for the system conversion is an ERP release or is an S/4HANA Finance release, where you use the localization of China GTI solution in the system.

## Solution

In the new data base design for the GTI solution invoices will be stored in 6 new tables and the old invoices can be migarated into these new tables as decribed in note 2727947.
