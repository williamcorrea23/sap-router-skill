| Business Impact |       3209838 | S4TWL - Deprecation of Average Daily Balance Key figure Report App ( F3987 ) |

## Symptom

You want to use Average Daily Balance Key figure Report app F3987.

## Reason and Prerequisites

The Average Daily Balance Key figure Report App will be deprecated in on premise S/4 HANA 2022 and will become obsolete.

We recommend you to switch to the successor app W0166,  as soon as possible.

## Solution

## Description

The Average Daily Balance Key figure Report App will be deprecated in on premise S/4 HANA 2022 and will become obsolete.

We recommend you to switch to the successor app W0166,  as soon as possible.

## Business process related information

The Average Daily Balance (ADB) is a figure that represents the average balance of an account for a specific period. The app F3987 allows you to calculate ADBs for intramonth dates based on the posting date of FI documents.You can also calculate sevaral other key figures, such as the year-to-date average balance. This app therefore allows you to calculate efficiently and accurately the ADB at any point during a year.

With the new app W0166, you can display average daily balances and spot balances. You can do this for each ledger and all company codes. Using this app, you can add the measures and dimensions by using drag and drop functions to provide a user-friendly interface. Also, this allows you to display data in a hierarchical format.

## Required and Recommended Actions

Anyone using F3987 app should switch to the successor app W0166.

## How to Determine Relevancy

If you are using F3987 app, this deprecation note is applicable for you.

## Other Terms

F3987, ADB, ADB Key figure report, ADBKFC
