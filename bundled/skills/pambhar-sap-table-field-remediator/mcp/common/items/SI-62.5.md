## Symptom

You want to use SAP S/4HANA 2021 FPS02 or higher. This SAP Note provides information about the restrictions of this release with regard to the SAP Waste and Recycling functions.

## Reason and Prerequisites

Update to SAP S/4HANA, 2021 FPS02 or higher

## Solution

## Description

As of SAP S/4HANA, 2021 FPS02, it is no longer possible to create an additional service order in EWACONTS when creating a waste disposal order.

## Business process information

There is currently no alternative way to create service orders in the SAP Waste and Recycling core.

## Required and recommended actions

The existing underlying CS functions were dispensed with due to the activities in the context of CS compatibility. Therefore, the processes in W&R have been adjusted accordingly.

If the function is required by customers in the future, a request for this should be created using the Customer Influence Portal . SAP will then review a corresponding future implementation.

If this is needed quickly, an implementation via a customer-specific enhancement is recommended.

## Relevancy

This change is relevant if you use transaction EWACONTS in connection with the creation of CS service orders in the context of waste.

See the corresponding simplification element and the list of simplifications in SAP S/4HANA.
