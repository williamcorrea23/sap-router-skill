## Symptom

You are doing a system conversion to SAP S/4HANA. The following SAP S/4HANA Transition Worklist item is applicable in this case.

## Solution

## Description

In SAP S/4HANA Retail for merchandise management, functionality of retail markdown plan is available, but defined as "not strategic any more".  See SAP note 2269324 Compatibility Scope Matrix for SAP S/4HANA, on-premise edition , refer to ID 104 in the xls.

## Business Process related information

Move to available functionality of markdown profile integrating in the price planning workbench.

Alternatively integrate with solutions offered by third-party providers.

## Required and Recommended Action

Move to price planning workbench as an alternative, or integrate with a third-party solution before expiration of use rights as explained in note 2269324.

## How to Determine Relevancy

This Simplification Item is relevant if markdown planning is used. This is the case if any of transactions WMF1, WMF2, WMF3, WMF4, WMF5, WMF6, WMF7,WMF_ARCHR, WMF_SARA, WMR1, WMR2, WMR3, WMR4 or programs RWMFA200, RWMFA210, RWMFA300, SAPMWMFA are used.

This also can be checked via transaction SE16N. Enter table WMFH, or WMFP and check whether there are any entries.
