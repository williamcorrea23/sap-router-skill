## Symptom

You are doing a system conversion from SAP ERP to SAP S/4HANA or an upgrade from a lower to a higher SAP S/4HANA release and are using the functionality described in this note.

## Solution

## Description

The add-ons listed below are part of the SAP S/4HANA compatibility scope, which comes with limited usage rights. For more details on the compatibility scope and links to further information please refer to SAP note 2269324.

| Add-on technical software component   | Add-on product                               | Vendor      |   Compatibility Package ID |
|---------------------------------------|----------------------------------------------|-------------|----------------------------|
| RECA                                  | SAP Common Area Maintenance Expense Recovery | SAP SE      |                        109 |
| FATCA                                 | SAP Tax Classification and Reporting         | SAP Fioneer |                        112 |
| FB4U                                  | budget-based benefits selection              | SAP SE      |                        125 |

## Required and Recommended Action(s)

Use rights of the compatibility package licenses for the add-ons above will expire as described in note 2269324. SAP does not plan to provide alternatives for these applications and will also not release these add-ons for SAP S/4HANA 2025 and higher. Therefore, we recommend you save any business relevant data of these add-ons before an upgrade to SAP S/4HANA 2025 or higher and uninstall the add-ons.

Information about the uninstallation process you find in the following SAP notes.

1604155 - Release strategy and Maintenance Information for the ABAP add-on RECA 1497354 - Release strategy for the ABAP add-on FB4U
