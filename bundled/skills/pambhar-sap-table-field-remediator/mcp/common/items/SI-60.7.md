| Business Impact |       2371539 | S4TWL - Retail Rapid Replenishment, and Customer Replenishment |

Symptom You are doing a system conversion to SAP S/4HANA. The following SAP S/4HANA Transition Worklist item is applicable in this case.

## Solution

## Description

In SAP S/4HANA Retail for merchandise management, rapid replenishment, and customer replenishment are available, but defined as "not strategic any more".  See SAP note 2269324 - Compatibility Scope Matrix for SAP S/4HANA, on-premise edition , refer to ID 104 in the xls.

In a previous version of this note, the Retail Puirchase Order Planning Workbench was covered as well. This is now described in a separate note, see 3472264.

## Business Process related information

It is planned to modernize Rapid Replenishment and Customer Replenishment with release SAP S/4HANA 2023 FPS03. With this modernization the functionality will become part of the perpetual scope of S/4HANA.

## Required and Recommended Action

No special actions for is needed when converting from SAP ERP to SAP S/4HANA.

## How to Determine Relevancy

This Simplification Item is relevant if rapid replenishment, or customer replenishment is used.

This is the case if transactions WRP1R, WVM1, WVM3, WVM4, WVM6, WVM7 are used.
