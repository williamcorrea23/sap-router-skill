## Symptom

You are doing a system conversion to SAP S/4HANA, on-premise edition. The following SAP S/4HANA Transition Worklist item is applicable in this case.

## Solution

## Description

With the integration of SAP Contract Accounts Receivable and Payable (FI-CA) with FSCM Biller Direct it is possible to offer end-customers the convenience of receiving invoices, viewing account statuses, and making payments online as a selfservice. Integration of FSCM Biller Direct with BRIM/FI-CA is part of the SAP S/4HANA compatibility scope, which comes with limited usage rights. For more details on the compatibility scope and it's expiry date and links to further information please refer to SAP note 2269324. In the compatibility matrix attached to SAP note 2269324, 'BRIM - Integration with FSCM Biller Direct' can be found under the ID 437.

## Business Process related information

The functionality Biller Direct with FI-CA is not the target architecture, but still available and can be used within SAP S/4HANA, on-premise edition.

## Required and Recommended Action(s)

To check if you use FSCM Biller Direct, call the transaction SE16 to check for entries in the table DEBPPEVENTLOG.

Customers are using normally own branded web based portals. SAP therefore alternatively offers APIs which are available with SAP S/4HANA on-premise 2021:
