| Business Impact |       2267288 | S4TWL - Selected Project System Interfaces |

## Symptom

You are doing a system conversion to SAP S/4HANA. The following SAP S/4HANA Transition Worklist item is applicable in this case.

## Solution

## Description

Selected Project System Interface are not available in SAP S/4HANA 1511 and higher releases:

OpenPS for Microsoft Project for a bidirectional data exchange with Microsoft Project and transaction OPENPS for the download of activities to Palm Pilot

MPX up- and download functions including transactions CN06 and CN07 and related functions in PS reporting transactions such as CN41 etc.

Export to Graneda and Microsoft Access (see also 1670678 - New features in SAP GUI for Windows 7.30 )

## Business Process related information

The selected interfaces listed above will not be available anymore in SAP S/4HANA 1511 and higher releases. Alternative solutions such leveraging PS BAPIs or OData APIs should be considered.

| Transaction Codes not available in SAP S/4HANA   | CN06   | MPX Download: Standard Network    |
|--------------------------------------------------|--------|-----------------------------------|
|                                                  | CN07   | MPX Upload: Standard Network      |
|                                                  | OPENPS | Download Activities to Palm Pilot |

## Required and Recommended Action(s)

Knowledge Transfer to key- and end user.
