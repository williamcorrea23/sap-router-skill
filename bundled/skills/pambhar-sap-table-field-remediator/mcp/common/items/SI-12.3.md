## Symptom

You are doing a system upgrade to SAP S/4HANA 2022 or higher and use the legacy MDI cost center replication. The following SAP S/4HANA Transition Worklist item is applicable in this case.

## Reason and Prerequisites

The legacy MDI cost center replication is only available in SAP S/4HANA releases from 2020 until 2023. Fore more information regarding the legacy MDI Cost Center Replication please check the SAP Help documentation. Afterwards it is not possible to use this old functionality anymore.

## Solution

Before upgrading to SAP S/4HANA 2022 or higher, check whether the functionality is used or not. If not, there are no further actions. If yes, a migration to the new model is recommended and mandatory with SAP S/4HANA release 2023 at the latest. See SAP Note 3232417.

## How to Determine Relevancy

This Simplification Item is relevant if you have entries in table DRFC_APPL_SYS, field OUTB_IMPL = FI_CO_CCTR. You can check if the replication model is active in table DRFC_APPL
