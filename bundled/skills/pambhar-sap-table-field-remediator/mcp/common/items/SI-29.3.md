## Symptom

Following apps have been deprecated in SAP S/4HANA 2022 and are obsolete since 2023 FPS00. It will be no longer available on the SAP Fiori launchpad after this. You can find the details of the successor apps in the below table.

| Deprecated App Id   | Technical Name          | Deprecated App Name                                 | Successor App(s)                        |
|---------------------|-------------------------|-----------------------------------------------------|-----------------------------------------|
| W0004               | EAMS_WDA_ONL_OIF        | Manage Notification List                            | Find Maintenance Notification ( F2071 ) |
| W0011               | EAMS_WDA_MD_IC_OVP      | Display Master Data Information Center (Planner)    | Find Technical Object ( F2072 )         |
| W0012               | EAMS_WDA_MD_IC_OVP      | Display Master Data Information Center (Technician) | Find Technical Object ( F2072 )         |
| W0026               | EAMS_WDA_MPOS_MPLAN_OIF | Manage Maintenance                                  | Manage Maintenance Plans (F5325) &      |

|       |                      | Plan and Item List                                    | Manage Maintenance Items (F5356)                                        |
|-------|----------------------|-------------------------------------------------------|-------------------------------------------------------------------------|
| W0016 | EAMS_WDA_JOBLIST_OIF | Display Job List                                      | Perform Maintenance Jobs (F5104A),Report and Repair Malfunction (F2023) |
| W0018 | EAMS_WDA_ONL_OIF     | Manage Order List                                     | Find Maintenance Orders - F2175                                         |
| W0019 | EAMS_WDA_ONL_IC_OVP  | Manage Orders and Notifications in Information Center | Find Maintenance Orders - F2175                                         |
| W0020 | EAMS_WDA_CONF_OIF    | Confirm Jobs                                          | Perform Maintenance Jobs (F5104A),Report and Repair Malfunction (F2023) |

## Reason and Prerequisites

Functionality is obsolete.

## Solution

You can use the successor apps which are available on the SAP Fiori launchpad.

## Other Terms

POWL applications
