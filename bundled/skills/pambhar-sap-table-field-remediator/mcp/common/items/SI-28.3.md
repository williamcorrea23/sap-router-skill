## Symptom

You are doing a system conversion to SAP S/4HANA, on-premise edition. The following SAP S/4HANA Transition Worklist item is applicable in this case.

## Solution

## Description

With SAP S/4HANA, on-premise edition 1511 Documents@web is not available.

Documents @ Web transactions like CVW* are not available in SAP S/4HANA, onpremise edition 1511. This is obsolete now and PLM WUI can be used for the same

## Business Process related information

Use PLM WUI.

| Transaction not available in SAP S/4HANA on-premise edition 1511   | CVW*   |
|--------------------------------------------------------------------|--------|

## Required and Recommended Action(s)

Check if Web Document is used anywhere. Use PLM WUI or FIORI tiles for CV transactions.
