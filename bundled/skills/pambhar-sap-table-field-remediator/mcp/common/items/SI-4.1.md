## Symptom

You are doing a system conversion to SAP Commercial Project Management for SAP S/4HANA. The following SAP S/4HANA Transition Worklist item is applicable in this case.

## Solution

## Description

With SAP Commercial Project Management for SAP S/4HANA , the Rate Card application is not available. Note that a functional equivalent is available.

## Required and Recommended Actions

Use alternative transactions like VK11 for appropriate business requirements.

Inform key users and end users.

## Other Terms

Rate Card, SAP Commercial Project Management, rate, CA-CPD, S/4HANA
