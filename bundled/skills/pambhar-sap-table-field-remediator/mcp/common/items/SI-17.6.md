## Symptom

You are performing a system conversion from ERP to SAP S/4HANA or an upgrade from a lower SAP S/4HANA release to a higher release or feature package (2023 Feature Pack 2 or higher).

In S/4HANA, the Financial Services (Treasury) business partner table Standing Instructions Correspondence (table name VTBSTA2 ) is obsolete. The data in this table has not been required since Release ERP 604 (Enhancement Package 4 for ERP 600). As of this release, the correspondence framework must be used in Treasury Management. However, due to an error in the technical delivery, the table VTBSTA2 can still be maintained in transaction BP. Release S/4HANA 2025 deactivates the maintainability of the table.

The upgrade to the on-premise release SAP S/4HANA 2027 will force a deletion of the table content in the SUM process.

## Reason and Prerequisites

The Financial Services business partner table "Standing Instructions: Correspondence" is no longer supported as of SAP S/4HANA 2025. As of Release ERP 604, a separate correspondence framework exists in Treasury Management that is used to process correspondence for Treasury contracts. There is no other SAP application for which data must be retained in the standing instructions correspondence (table VTBSTA2). The data does not yet have to be deleted during the upgrade to SAP S/4HANA 2025. The deletion will be forced as part of the SUM process in the successor release (expected to be SAP S/4HANA 2027).

## Solution

## Procedure for determining relevance

This simplification item is relevant if the database table VTBSTA2 contains at least one entry.

## Required and recommended actions

A deletion of the data is not yet enforced in release S/4HANA 2023 and release S/4HANA 2025, but will become available as of the next subsequent release S/4HANA 2027. If you no longer need the data, SAP recommends that you delete the table content already during the upgrade to S/4HANA 2023, Feature Pack 2, or S/4HANA 2025.
