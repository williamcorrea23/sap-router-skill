| Business Impact |       2323655 | Social Intelligence Non-availability |

## Symptom

You are doing a system conversion to SAP S/4HANA, on-premise edition. The following SAP S/4HANA Transition Worklist item is applicable in this case.

## Reason and Prerequisites

Social Intelligence data harvesting features were provided in ERP EhP7 and EhP8. Customer could harvest data from social sources like Twitter, Facebook, etc. After text analysis using HANA, the sentiment data used to be stored in underlying BS_FND tables. Consent Management and Archiving features were also provided.

These features will not be available in S/4 from the initial release

## Solution

As of now, there is no alternative for this solution in S/4 HANA
