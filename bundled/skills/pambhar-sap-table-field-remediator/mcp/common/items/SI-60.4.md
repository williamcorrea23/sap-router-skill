## Symptom

You are doing a system conversion to SAP S/4HANA. The following SAP S/4HANA Transition Worklist item is applicable in this case.

## Solution

## Description

In SAP S/4HANA Retail for merchandise management, bonus buy functionality via transactions VBK1, VBK2, VBK3, VBK6 is available, but defined as "not strategic any more".  See SAP note 2269324 - Compatibility Scope Matrix for SAP S/4HANA, onpremise edition , refer to ID 104 in the xls.

## Business Process related information

Similar functionality is avaialble through transactions RDMBB*.

In addition SAP Omnichannel Promotion Pricing service (OPPS) provides superior functionality part of SAP's Industry Cloud offerings.

## Required and Recommended Action

Move to bonus buy functionality available through transactions RDMBB* as an alternative, or integrate with SAP Omnichannel Promotion Pricing service (OPPS) before expiration of use rights as explained in note 2269324.

## How to Determine Relevancy

This Simplification Item is relevant if bonus buy funcationality via IDocs of type WPDBBY is used.

This is the case if any of the transactions VBK1, VBK2, VBK3, VBK6 is used. This also can be checked via transaction SE16N. Enter table EDIDC and check whether there are any entries for EDIDC-MESTYP = WPDBBY.

## Other Terms

VBK1, VBK2, VBK3, VBK6, WPDBBY
